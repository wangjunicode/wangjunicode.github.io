# 游戏布料与软体物理模拟完全指南

## 一、布料物理的核心挑战

布料是游戏中最具挑战性的物理模拟对象：
- **高维自由度**：一件衣服有数千个顶点，每个顶点有3个位置自由度
- **强约束性**：布料既不可拉伸又不可压缩，约束矛盾难以精确求解
- **实时性要求**：游戏需要在 1ms~3ms 内完成一帧的布料模拟
- **稳定性要求**：数值积分不稳定时会出现爆炸（顶点飞出）或折叠（穿模）

### 1.1 主流布料算法对比

| 算法 | 精度 | 速度 | 稳定性 | 适用场景 |
|------|------|------|--------|---------|
| 弹簧-质点（Spring-Mass） | 中 | 快 | 差（显式） | 简单旗帜 |
| PBD（Position Based Dynamics） | 中 | 快 | **极好** | 角色服装 |
| XPBD（扩展PBD） | 高 | 中 | 极好 | 高品质衣物 |
| 有限元法（FEM） | 极高 | 慢 | 好 | 离线CG |
| GPU并行PBD | 中 | **极快** | 极好 | 大规模仿真 |

## 二、Position Based Dynamics 算法详解

### 2.1 PBD核心思想

PBD 是 Matthias Müller 等人在 2007 年提出的游戏物理模拟方法，核心思路：

> **不直接模拟力，而是直接修正位置以满足约束**

PBD每帧流程：
```
1. 外力积分：v += dt * f_ext / m；x_pred = x + dt * v
2. 约束求解：迭代修正 x_pred 使所有约束满足
3. 速度更新：v = (x_pred - x) / dt；x = x_pred
```

### 2.2 距离约束推导

距离约束要求两顶点间距离保持为 d₀：

```
C(p1, p2) = |p1 - p2| - d0 = 0
```

PBD 位置修正量：

```
Δp1 = -w1/(w1+w2) * (|p1-p2| - d0) * (p1-p2)/|p1-p2|
Δp2 = +w2/(w1+w2) * (|p1-p2| - d0) * (p1-p2)/|p1-p2|
```

其中 `w = 1/mass`（质量的倒数），固定顶点的 `w = 0`。

### 2.3 完整 C# PBD 布料实现

```csharp
using UnityEngine;
using System.Collections.Generic;
using Unity.Collections;
using Unity.Jobs;
using Unity.Mathematics;
using Unity.Burst;

/// <summary>
/// 基于Position Based Dynamics的布料模拟器
/// 支持距离约束、弯曲约束、碰撞处理
/// </summary>
public class PBDClothSimulator : MonoBehaviour
{
    [Header("网格配置")]
    public int gridWidth  = 20;      // 布料网格宽度（顶点数）
    public int gridHeight = 20;      // 布料网格高度（顶点数）
    public float gridSpacing = 0.1f; // 顶点间距（米）

    [Header("物理参数")]
    public Vector3 gravity = new Vector3(0, -9.81f, 0);
    public float damping   = 0.99f;  // 速度衰减（0~1，越小越阻尼）
    public float mass      = 1.0f;   // 每个顶点质量
    public int   solverIterations = 5; // 约束求解迭代次数

    [Header("约束强度")]
    [Range(0, 1)] public float stretchStiffness = 0.9f;  // 拉伸约束刚度
    [Range(0, 1)] public float shearStiffness   = 0.8f;  // 剪切约束刚度
    [Range(0, 1)] public float bendStiffness    = 0.3f;  // 弯曲约束刚度

    [Header("碰撞")]
    public Transform[] collisionSpheres;  // 用于碰撞的球体列表

    // 顶点数据
    private Vector3[] positions;
    private Vector3[] prevPositions;
    private Vector3[] velocities;
    private float[]   inverseMasses;  // 固定顶点=0，自由顶点=1/mass
    private bool[]    pinned;         // 是否固定

    // 约束数据
    [System.Serializable]
    public struct DistanceConstraint
    {
        public int   idx1, idx2;
        public float restLength;     // 静止长度
        public float stiffness;      // 刚度（0~1）
    }

    private List<DistanceConstraint> constraints = new List<DistanceConstraint>();

    // 渲染相关
    private Mesh clothMesh;
    private MeshFilter meshFilter;
    private int[] triangles;

    void Start()
    {
        InitializeMesh();
        InitializePhysics();
        BuildConstraints();
    }

    void InitializeMesh()
    {
        int vertCount = gridWidth * gridHeight;
        clothMesh = new Mesh { name = "ClothMesh" };

        // 生成顶点
        var verts = new Vector3[vertCount];
        var uvs   = new Vector2[vertCount];
        for (int y = 0; y < gridHeight; y++)
        for (int x = 0; x < gridWidth;  x++)
        {
            int i = y * gridWidth + x;
            verts[i] = new Vector3(x * gridSpacing, 0, y * gridSpacing);
            uvs[i]   = new Vector2((float)x / (gridWidth - 1), (float)y / (gridHeight - 1));
        }

        // 生成三角形
        int triCount = (gridWidth - 1) * (gridHeight - 1) * 2;
        triangles = new int[triCount * 3];
        int t = 0;
        for (int y = 0; y < gridHeight - 1; y++)
        for (int x = 0; x < gridWidth  - 1; x++)
        {
            int i = y * gridWidth + x;
            triangles[t++] = i;
            triangles[t++] = i + gridWidth;
            triangles[t++] = i + 1;
            triangles[t++] = i + 1;
            triangles[t++] = i + gridWidth;
            triangles[t++] = i + gridWidth + 1;
        }

        clothMesh.vertices  = verts;
        clothMesh.uv        = uvs;
        clothMesh.triangles = triangles;
        clothMesh.RecalculateNormals();

        meshFilter = gameObject.AddComponent<MeshFilter>();
        var renderer = gameObject.AddComponent<MeshRenderer>();
        meshFilter.mesh = clothMesh;
    }

    void InitializePhysics()
    {
        int n = gridWidth * gridHeight;
        positions     = new Vector3[n];
        prevPositions = new Vector3[n];
        velocities    = new Vector3[n];
        inverseMasses = new float[n];
        pinned        = new bool[n];

        // 初始化位置（从网格顶点复制）
        clothMesh.vertices.CopyTo(positions, 0);
        positions.CopyTo(prevPositions, 0);

        float invM = 1.0f / mass;
        for (int i = 0; i < n; i++)
        {
            inverseMasses[i] = invM;
        }

        // 固定顶部两行顶点
        for (int x = 0; x < gridWidth; x++)
        {
            int topRow = (gridHeight - 1) * gridWidth + x;
            inverseMasses[topRow] = 0f;
            pinned[topRow] = true;
        }
    }

    void BuildConstraints()
    {
        // 结构约束（水平/垂直相邻顶点）
        for (int y = 0; y < gridHeight; y++)
        for (int x = 0; x < gridWidth;  x++)
        {
            int i = y * gridWidth + x;

            // 水平约束
            if (x < gridWidth - 1)
                AddDistanceConstraint(i, i + 1, stretchStiffness);

            // 垂直约束
            if (y < gridHeight - 1)
                AddDistanceConstraint(i, i + gridWidth, stretchStiffness);

            // 剪切约束（对角线）
            if (x < gridWidth - 1 && y < gridHeight - 1)
            {
                AddDistanceConstraint(i, i + gridWidth + 1, shearStiffness);
                AddDistanceConstraint(i + 1, i + gridWidth, shearStiffness);
            }

            // 弯曲约束（隔一顶点，抵抗折叠）
            if (x < gridWidth - 2)
                AddDistanceConstraint(i, i + 2, bendStiffness);
            if (y < gridHeight - 2)
                AddDistanceConstraint(i, i + gridWidth * 2, bendStiffness);
        }

        Debug.Log($"[PBD Cloth] 约束数量: {constraints.Count}");
    }

    void AddDistanceConstraint(int idx1, int idx2, float stiffness)
    {
        float restLength = Vector3.Distance(positions[idx1], positions[idx2]);
        constraints.Add(new DistanceConstraint
        {
            idx1 = idx1, idx2 = idx2,
            restLength = restLength,
            stiffness = stiffness
        });
    }

    void FixedUpdate()
    {
        float dt = Time.fixedDeltaTime;
        int n = positions.Length;

        // ---- Step 1: 外力积分 ----
        for (int i = 0; i < n; i++)
        {
            if (inverseMasses[i] == 0) continue;  // 固定顶点跳过
            
            velocities[i] += gravity * dt;
            velocities[i] *= damping;
            
            prevPositions[i] = positions[i];
            positions[i] += velocities[i] * dt;
        }

        // ---- Step 2: 迭代约束求解 ----
        for (int iter = 0; iter < solverIterations; iter++)
        {
            SolveDistanceConstraints();
            SolveCollisionConstraints();
        }

        // ---- Step 3: 速度更新 ----
        for (int i = 0; i < n; i++)
        {
            if (inverseMasses[i] == 0) continue;
            velocities[i] = (positions[i] - prevPositions[i]) / dt;
        }

        // ---- Step 4: 更新Mesh ----
        clothMesh.vertices = positions;
        clothMesh.RecalculateNormals();
        clothMesh.RecalculateBounds();
    }

    void SolveDistanceConstraints()
    {
        foreach (var c in constraints)
        {
            int   i1 = c.idx1, i2 = c.idx2;
            float w1 = inverseMasses[i1], w2 = inverseMasses[i2];
            float wSum = w1 + w2;
            if (wSum < 1e-6f) continue;

            Vector3 delta = positions[i1] - positions[i2];
            float   dist  = delta.magnitude;
            if (dist < 1e-6f) continue;

            // PBD位置修正
            float correction = (dist - c.restLength) / dist;
            Vector3 dp = correction * delta * c.stiffness;

            positions[i1] -= dp * (w1 / wSum);
            positions[i2] += dp * (w2 / wSum);
        }
    }

    void SolveCollisionConstraints()
    {
        if (collisionSpheres == null) return;

        foreach (var sphere in collisionSpheres)
        {
            if (sphere == null) continue;
            
            float   radius = sphere.localScale.x * 0.5f;
            Vector3 center = sphere.position;

            for (int i = 0; i < positions.Length; i++)
            {
                if (inverseMasses[i] == 0) continue;

                Vector3 diff = positions[i] - center;
                float   dist = diff.magnitude;

                if (dist < radius)
                {
                    // 将顶点推出球体表面
                    positions[i] = center + diff.normalized * (radius + 0.001f);
                }
            }
        }
    }

    /// <summary>
    /// 外部接口：在运行时固定或释放顶点
    /// </summary>
    public void PinVertex(int index, bool pinIt)
    {
        if (index < 0 || index >= positions.Length) return;
        inverseMasses[index] = pinIt ? 0f : 1.0f / mass;
        pinned[index] = pinIt;
    }

    /// <summary>
    /// 外部接口：施加局部冲击力（如爆炸、风）
    /// </summary>
    public void ApplyImpulse(Vector3 center, float radius, Vector3 force)
    {
        for (int i = 0; i < positions.Length; i++)
        {
            if (inverseMasses[i] == 0) continue;
            
            float dist = Vector3.Distance(positions[i], center);
            if (dist < radius)
            {
                float falloff = 1.0f - dist / radius;
                velocities[i] += force * falloff * inverseMasses[i];
            }
        }
    }
}
```

### 2.4 XPBD：加入刚度物理意义

标准PBD的刚度参数与仿真步长耦合，XPBD（Extended PBD）通过引入**顺应性系数α**解决了这个问题：

```csharp
/// <summary>
/// XPBD距离约束求解器
/// α参数具有真实物理意义，与仿真步长无关
/// </summary>
public static class XPBDSolver
{
    /// <summary>
    /// 求解一个距离约束
    /// </summary>
    /// <param name="p1">顶点1位置</param>
    /// <param name="p2">顶点2位置</param>
    /// <param name="w1">顶点1逆质量</param>
    /// <param name="w2">顶点2逆质量</param>
    /// <param name="restLength">静止长度</param>
    /// <param name="compliance">顺应性 α（0=刚性, 大=柔软）单位: m²/N</param>
    /// <param name="lambda">拉格朗日乘子（每步开始前重置为0）</param>
    /// <param name="dt">仿真步长</param>
    public static void SolveDistanceXPBD(
        ref Vector3 p1, ref Vector3 p2,
        float w1, float w2,
        float restLength, float compliance,
        ref float lambda, float dt)
    {
        float wSum = w1 + w2;
        if (wSum < 1e-6f) return;

        Vector3 delta = p1 - p2;
        float   dist  = delta.magnitude;
        if (dist < 1e-6f) return;

        float C = dist - restLength;  // 约束值（应为0）
        
        // XPBD拉格朗日乘子更新
        float alphaHat = compliance / (dt * dt);  // α̃ = α/dt²
        float deltaLambda = (-C - alphaHat * lambda) / (wSum + alphaHat);
        lambda += deltaLambda;

        Vector3 grad = delta / dist;  // 约束梯度
        
        p1 += deltaLambda * w1 * grad;
        p2 -= deltaLambda * w2 * grad;
    }
    
    /*
     * 典型compliance值参考：
     * 刚性布料（帆布）: α = 1e-8  m²/N
     * 普通T恤：        α = 1e-6  m²/N
     * 丝绸：           α = 1e-4  m²/N
     * 橡皮筋：         α = 1e-2  m²/N
     */
}
```

```csharp
            {
                maxDistance       = isFixed ? 0f : 0.5f,   // 0=完全固定
                collisionSphereDistance = isFixed ? 0f : 0.2f
            };
        }

        cloth.coefficients = coefficients;
        Debug.Log($"[Cloth] 约束配置完成，顶部固定阈值Y={fixedThreshold:F3}");
    }

    /// 运行时模拟风力效果
    /// </summary>

        {
            // 使用正弦波模拟阵风

            cloth.externalAcceleration = new Vector3(windStrength, 0, Mathf.Sin(Time.time) * 1f);
        }
    }
```

## 四、GPU Compute Shader 布料模拟

对于需要同时运行大量布料（如草地飘动、群体角色）的场景，GPU布料是最佳方案：

### 4.1 GPU布料 Compute Shader

```hlsl
// GPUCloth.compute
#pragma kernel IntegratePositions
#pragma kernel SolveConstraints  
#pragma kernel UpdateVelocities
#pragma kernel UpdateMesh

#define GROUP_SIZE 64

// 顶点数据Buffer
struct ClothVertex
{
    float3 position;
    float3 prevPosition;
    float3 velocity;
    float  inverseMass;   // 0 = 固定
};

// 距离约束Buffer
struct DistConstraint
{
    int   idx1, idx2;
    float restLength;
    float stiffness;
};

RWStructuredBuffer<ClothVertex> _Vertices;
StructuredBuffer<DistConstraint> _Constraints;
RWBuffer<float3> _MeshPositions;    // 输出到Mesh的位置

float3 _Gravity;
float  _DeltaTime;
float  _Damping;
int    _VertexCount;
int    _ConstraintCount;

// ---- 积分阶段：施加外力，预测新位置 ----
[numthreads(GROUP_SIZE, 1, 1)]
void IntegratePositions(uint3 id : SV_DispatchThreadID)
{
    uint i = id.x;
    if (i >= (uint)_VertexCount) return;

    ClothVertex v = _Vertices[i];
    if (v.inverseMass < 1e-6) return;  // 固定顶点不移动

    // 施加重力
    v.velocity += _Gravity * _DeltaTime;
    v.velocity *= _Damping;

    v.prevPosition = v.position;
    v.position += v.velocity * _DeltaTime;

    _Vertices[i] = v;
}

// ---- 约束求解阶段（Gauss-Seidel迭代）----
// 注意：GPU并行时约束求解可能产生竞争，使用Jacobi（取平均）更稳定
[numthreads(GROUP_SIZE, 1, 1)]
void SolveConstraints(uint3 id : SV_DispatchThreadID)
{
    uint ci = id.x;
    if (ci >= (uint)_ConstraintCount) return;

    DistConstraint c = _Constraints[ci];
    
    ClothVertex v1 = _Vertices[c.idx1];
    ClothVertex v2 = _Vertices[c.idx2];
    
    float wSum = v1.inverseMass + v2.inverseMass;
    if (wSum < 1e-6) return;

    float3 delta = v1.position - v2.position;
    float  dist  = length(delta);
    if (dist < 1e-6) return;

    float correction = (dist - c.restLength) / dist;
    float3 dp = correction * delta * c.stiffness;

    // Jacobi方式：写回需要原子操作，这里简化为直接写（实际项目需要双Buffer）
    if (v1.inverseMass > 1e-6)
        _Vertices[c.idx1].position -= dp * (v1.inverseMass / wSum);
    if (v2.inverseMass > 1e-6)
        _Vertices[c.idx2].position += dp * (v2.inverseMass / wSum);
}

// ---- 速度更新阶段 ----
[numthreads(GROUP_SIZE, 1, 1)]
void UpdateVelocities(uint3 id : SV_DispatchThreadID)
{
    uint i = id.x;
    if (i >= (uint)_VertexCount) return;

    ClothVertex v = _Vertices[i];
    if (v.inverseMass > 1e-6)
        v.velocity = (v.position - v.prevPosition) / _DeltaTime;
    
    _Vertices[i] = v;
}

// ---- 输出到Mesh ----
[numthreads(GROUP_SIZE, 1, 1)]
void UpdateMesh(uint3 id : SV_DispatchThreadID)
{
    uint i = id.x;
    if (i >= (uint)_VertexCount) return;
    _MeshPositions[i] = _Vertices[i].position;
}
```

### 4.2 GPU布料 C# 控制器

```csharp
using UnityEngine;

/// <summary>
/// GPU Compute Shader 布料模拟控制器
/// 支持大量布料实例同时运行（如旗帜阵列、群体角色）
/// </summary>
public class GPUClothController : MonoBehaviour
{
    [Header("Compute Shader")]
    public ComputeShader clothShader;
    
    [Header("仿真参数")]
    public Vector3 gravity = new Vector3(0, -9.81f, 0);
    public float damping  = 0.99f;
    public int solverIterations = 8;
    public int substeps = 2;    // 子步数（提高稳定性）
    
    [Header("风力")]
    public Vector3 windForce = new Vector3(5, 0, 0);
    public float windTurbulence = 1.0f;

    // GPU Buffer
    private ComputeBuffer vertexBuffer;
    private ComputeBuffer constraintBuffer;
    private GraphicsBuffer meshPositionBuffer; // 直接写入Mesh顶点

    // Compute Shader Kernel索引
    private int integrateKernel;
    private int solveKernel;
    private int velocityKernel;
    private int meshKernel;

    private Mesh clothMesh;
    private int vertexCount;
    private int constraintCount;

    // Shader参数ID
    private static readonly int GravityID          = Shader.PropertyToID("_Gravity");
    private static readonly int DeltaTimeID         = Shader.PropertyToID("_DeltaTime");
    private static readonly int DampingID           = Shader.PropertyToID("_Damping");
    private static readonly int VertexCountID       = Shader.PropertyToID("_VertexCount");
    private static readonly int ConstraintCountID   = Shader.PropertyToID("_ConstraintCount");

    void Start()
    {
        GetKernels();
        InitializeFromMesh();
    }

    void GetKernels()
    {
        integrateKernel = clothShader.FindKernel("IntegratePositions");
        solveKernel     = clothShader.FindKernel("SolveConstraints");
        velocityKernel  = clothShader.FindKernel("UpdateVelocities");
        meshKernel      = clothShader.FindKernel("UpdateMesh");
    }

    void InitializeFromMesh()
    {
        var meshFilter = GetComponent<MeshFilter>();
        clothMesh = meshFilter.mesh;
        clothMesh.MarkDynamic(); // 允许每帧修改

        vertexCount = clothMesh.vertexCount;
        var verts = clothMesh.vertices;

        // 初始化顶点Buffer数据
        // ClothVertex: position(3) + prevPosition(3) + velocity(3) + inverseMass(1) = 10 floats
        int stride = 10 * sizeof(float);
        float[] vertData = new float[vertexCount * 10];
        
        float invMass = 1.0f;
        float yMax = float.MinValue;
        foreach (var v in verts) yMax = Mathf.Max(yMax, v.y);

        for (int i = 0; i < vertexCount; i++)
        {
            int base_ = i * 10;
            vertData[base_ + 0] = verts[i].x;
            vertData[base_ + 1] = verts[i].y;
            vertData[base_ + 2] = verts[i].z;
            vertData[base_ + 3] = verts[i].x;  // prevPosition
            vertData[base_ + 4] = verts[i].y;
            vertData[base_ + 5] = verts[i].z;
            // velocity = 0
            vertData[base_ + 9] = (verts[i].y >= yMax - 0.01f) ? 0f : invMass;
        }

        vertexBuffer = new ComputeBuffer(vertexCount, stride);
        // vertexBuffer.SetData(vertData);  // 实际项目中填充数据

        Debug.Log($"[GPU Cloth] 初始化 {vertexCount} 顶点");
    }

    void FixedUpdate()
    {
        float subDt = Time.fixedDeltaTime / substeps;
        
        for (int step = 0; step < substeps; step++)
        {
            // 加入风力扰动
            Vector3 totalGravity = gravity + windForce 
                + Random.insideUnitSphere * windTurbulence;

            SetCommonParams(subDt, totalGravity);
            
            // 积分
            DispatchKernel(integrateKernel, vertexCount);
            
            // 约束迭代
            for (int iter = 0; iter < solverIterations; iter++)
                DispatchKernel(solveKernel, constraintCount);
            
            // 更新速度
            DispatchKernel(velocityKernel, vertexCount);
        }

        // 输出到Mesh
        DispatchKernel(meshKernel, vertexCount);
        
        // 重新计算法线（可以用Compute Shader优化）
        clothMesh.RecalculateNormals();
    }

    void SetCommonParams(float dt, Vector3 g)
    {
        foreach (var k in new[] { integrateKernel, solveKernel, velocityKernel, meshKernel })
        {
            clothShader.SetBuffer(k, "_Vertices", vertexBuffer);
            if (k == solveKernel)
                clothShader.SetBuffer(k, "_Constraints", constraintBuffer);
        }
        
        clothShader.SetVector(GravityID, g);
        clothShader.SetFloat(DeltaTimeID, dt);
        clothShader.SetFloat(DampingID, damping);
        clothShader.SetInt(VertexCountID, vertexCount);
        clothShader.SetInt(ConstraintCountID, constraintCount);
    }

    void DispatchKernel(int kernel, int count)
    {
        int groups = Mathf.CeilToInt(count / 64f);
        clothShader.Dispatch(kernel, groups, 1, 1);
    }

    void OnDestroy()
    {
        vertexBuffer?.Release();
        constraintBuffer?.Release();
        meshPositionBuffer?.Release();
    }
}
```

### 5.2 角色服装动态碰撞配置

```csharp
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 角色服装自动碰撞体配置
/// 遍历骨骼层级，为指定骨骼自动添加ClothSphereCollider对
/// </summary>
public class CharacterClothColliderAuto : MonoBehaviour
{
    [Header("骨骼路径")]
    public string[] boneNames = {
        "Hips", "Spine", "Spine1", "Spine2", "Chest",
        "LeftUpperLeg", "LeftLowerLeg",
        "RightUpperLeg", "RightLowerLeg"
    };

    [Header("碰撞球半径")]
    public float defaultRadius = 0.1f;
    public float legRadius     = 0.08f;

    private Cloth cloth;

    void Start()
    {
        cloth = GetComponent<Cloth>();
        if (cloth == null)
        {
            Debug.LogError("[ClothCollider] 未找到Cloth组件");
            return;
        }

        AutoConfigureColliders();
    }

    void AutoConfigureColliders()
    {
        var animator = GetComponentInParent<Animator>();
        if (animator == null) return;

        var sphereList = new List<SphereCollider>();

        foreach (string boneName in boneNames)
        {
            // 查找骨骼Transform
            Transform bone = FindBoneRecursive(transform.root, boneName);
            if (bone == null)
            {
                Debug.LogWarning($"[ClothCollider] 未找到骨骼: {boneName}");
                continue;
            }

            // 添加或复用SphereCollider
            var sphere = bone.gameObject.GetComponent<SphereCollider>()
                      ?? bone.gameObject.AddComponent<SphereCollider>();
            
            bool isLeg = boneName.Contains("Leg");
            sphere.radius = isLeg ? legRadius : defaultRadius;
            sphere.isTrigger = false;
            
            sphereList.Add(sphere);
        }

        cloth.sphereColliders = sphereList.ToArray();
        Debug.Log($"[ClothCollider] 自动配置了 {sphereList.Count} 个碰撞球");
    }

    Transform FindBoneRecursive(Transform parent, string name)
    {
        if (parent.name == name) return parent;
        foreach (Transform child in parent)
        {
            var found = FindBoneRecursive(child, name);
            if (found != null) return found;
        }
        return null;
    }
}
```

## 七、最佳实践总结

### 物理稳定性

1. **使用子步（Substeps）**：将大步长拆成多个小步，显著提升稳定性
2. **XPBD替代PBD**：刚度参数有物理意义，调参更直觉
3. **避免超薄布料**：弯曲刚度不能为0，否则产生数值震荡
4. **固定点至少两个**：单点固定时布料会无限旋转

### Unity Cloth 使用

1. **Cloth.ClearTransformMotion()** 在角色传送（位移突变）后调用，防止抖动
2. **cloth.enabled = false** 关闭模拟，比销毁组件性能更好
3. **碰撞体数量 ≤ 16**：Unity Cloth的碰撞体数量有上限
4. **MeshCollider 禁用**：Unity Cloth不支持MeshCollider，只支持SphereCollider和CapsuleCollider

### GPU布料

1. **双Buffer方式**：解决并行约束求解的竞争写入问题
2. **颜色图分组**：将约束按颜色图（Graph Coloring）分组，同组内无依赖可并行
3. **预热阶段**：GPU布料启动时跑50帧预热，避免初始形状突变

布料模拟是物理系统中最需要"调参艺术"的领域，理解算法本质远比熟悉API更重要。从最简单的PBD出发，根据项目实际需求逐步引入XPBD、GPU加速和精确碰撞，是务实的工程路径。
