# 游戏个性化推荐与内容分发系统完全指南

## 一、为什么游戏需要个性化推荐

### 1.1 核心业务价值

```
传统游戏运营模式（全量推送）：
  运营 → 制作活动 → 全量推送给所有玩家 → 低相关性 → 用户忽略 → 留存下降

个性化推荐模式：
  玩家行为数据 → 构建用户画像 → 精准匹配内容 → 高相关性 → 用户参与 → 留存↑付费↑

典型效果（行业数据参考）：
  ├── 活动参与率：提升 25%~40%
  ├── 皮肤购买转化率：提升 15%~30%
  ├── 新关卡首次挑战率：提升 20%~35%
  └── 用户流失率：降低 10%~20%
```

### 1.2 游戏推荐的特殊挑战

| 挑战 | 说明 | 解决思路 |
|------|------|---------|
| **新玩家冷启动** | 新注册玩家无行为数据 | 基于人口属性的初始化画像 |
| **游戏进度约束** | 推荐内容必须符合玩家当前进度 | 游戏进度过滤层 |
| **探索-利用平衡** | 只推荐熟悉内容导致无新鲜感 | UCB/Thompson采样 |
| **时序性强** | 游戏阶段不同，偏好差异极大 | 时间衰减特征权重 |
| **隐式反馈为主** | 玩家很少显式评分 | 行为信号的隐式反馈建模 |

### 3.2 行为信号采集与隐式评分计算

```csharp
namespace GameRecommend
{
    /// <summary>
    /// 行为信号采集器
    /// 负责将玩家的各种游戏行为转化为可量化的偏好信号
    /// </summary>
    public class BehaviorCollector
    {
        // 各行为信号的权重配置
        private static readonly Dictionary<string, float> SignalWeights = new Dictionary<string, float>
        {
            ["view"]            = 0.1f,  // 查看（弱信号）
            ["click"]           = 0.3f,  // 点击
            ["start"]           = 0.5f,  // 开始（关卡/活动）
            ["complete"]        = 1.0f,  // 完成（强正向信号）
            ["retry"]           = 0.4f,  // 重试（参与意愿强）
            ["share"]           = 0.8f,  // 分享（高认可）
            ["purchase"]        = 1.5f,  // 购买（最强正向信号）
            ["skip"]            = -0.2f, // 跳过（弱负向信号）
            ["early_quit"]      = -0.5f, // 中途退出（负向信号）
            ["negative_review"] = -1.0f  // 投诉/负面反馈
        };

        // 时间衰减参数（近期行为权重更高）
        private const float TIME_DECAY_HALFLIFE_DAYS = 7f; // 7天半衰期

        /// <summary>
        /// 记录行为事件并更新画像
        /// </summary>
        public void RecordBehavior(
            PlayerProfile profile,
            string contentId,
            ContentType contentType,
            string action,
            float value = 1f)
        {
            // 获取或创建交互记录
            if (!profile.ContentHistory.TryGetValue(contentId, out var interaction))
            {
                interaction = new ContentInteraction
                {
                    ContentId = contentId,
                    Type = contentType
                };
                profile.ContentHistory[contentId] = interaction;
            }

            // 更新行为计数
            switch (action)
            {
                case "view":     interaction.ViewCount++;  break;
                case "click":    interaction.ClickCount++; break;
                case "complete": interaction.IsCompleted = true; break;
            }

            interaction.TotalDwellTime  += value;
            interaction.LastInteractTime = DateTimeOffset.UtcNow.ToUnixTimeSeconds();

            // 重新计算隐式评分
            interaction.Rating = CalculateImplicitRating(interaction, action, value);

            // 更新偏好特征
            UpdatePreferenceFeatures(profile, contentType, action, value);
        }

        /// <summary>
        /// 计算隐式评分（0~5分制）
        /// </summary>
        private float CalculateImplicitRating(
            ContentInteraction interaction,
            string latestAction,
            float latestValue)
        {
            float score = 0f;

            // 基础行为得分
            if (SignalWeights.TryGetValue(latestAction, out float weight))
                score += weight * latestValue;

            // 完成度奖励
            if (interaction.IsCompleted) score += 1.5f;

            // 点击率（CTR）信号：点击/查看比率
            if (interaction.ViewCount > 0)
                score += Mathf.Clamp(interaction.ClickCount / (float)interaction.ViewCount * 2f, 0f, 1f);

            // 停留时间信号（对于视频/剧情内容）
            float dwellScore = Mathf.Log10(1f + interaction.TotalDwellTime / 60f) * 0.5f; // 以分钟为单位
            score += dwellScore;

            // 时间衰减：越近期的交互，评分权重越高
            float daysSinceInteract = (DateTimeOffset.UtcNow.ToUnixTimeSeconds()
                - interaction.LastInteractTime) / 86400f;
            float decayFactor = Mathf.Pow(0.5f, daysSinceInteract / TIME_DECAY_HALFLIFE_DAYS);
            score *= decayFactor;

            return Mathf.Clamp(score, 0f, 5f);
        }

        /// <summary>
        /// 根据内容类型和行为更新玩家偏好维度
        /// </summary>
        private void UpdatePreferenceFeatures(
            PlayerProfile profile, ContentType contentType, string action, float value)
        {
            float delta = SignalWeights.TryGetValue(action, out float w) ? w * 0.05f : 0f;

            switch (contentType)
            {
                case ContentType.PVPMode:
                    profile.CompetitionPreference = Mathf.Clamp01(
                        profile.CompetitionPreference + delta);
                    break;
                case ContentType.Story:
                    profile.NarrativePreference = Mathf.Clamp01(
                        profile.NarrativePreference + delta);
                    break;
                case ContentType.Skin:
                case ContentType.Character:
                    profile.CollectionPreference = Mathf.Clamp01(
                        profile.CollectionPreference + delta);
                    break;
            }
        }
    }
}
```

## 四、推荐算法实现

### 4.1 基于物品的协同过滤（Item-CF）

```csharp
namespace GameRecommend
{
    /// <summary>
    /// 基于物品的协同过滤推荐算法
    ///
    /// 原理：
    ///   如果玩家A喜欢物品X，而X与Y高度相似（被类似玩家群体喜欢），
    ///   则推荐Y给玩家A
    ///
    /// 优点：
    ///   - 不需要其他用户的实时数据（物品相似度可离线计算）
    ///   - 对新用户（冷启动）比User-CF更友好
    ///   - 可解释性强（"因为你喜欢X，所以推荐Y"）
    /// </summary>
    public class ItemBasedCF
    {
        // 物品相似度矩阵（从服务器预计算并缓存到本地）
        // key: itemId, value: {similarItemId → similarityScore}
        private Dictionary<string, Dictionary<string, float>> _itemSimilarityMatrix =
            new Dictionary<string, Dictionary<string, float>>();

        // 物品元信息
        private Dictionary<string, ContentItem> _itemCatalog =
            new Dictionary<string, ContentItem>();

        public void LoadSimilarityMatrix(ItemSimilarityData data)
        {
            _itemSimilarityMatrix.Clear();
            foreach (var entry in data.Entries)
            {
                if (!_itemSimilarityMatrix.ContainsKey(entry.ItemId))
                    _itemSimilarityMatrix[entry.ItemId] = new Dictionary<string, float>();
                _itemSimilarityMatrix[entry.ItemId][entry.SimilarItemId] = entry.Score;
            }
        }

        /// <summary>
        /// 为指定玩家生成推荐列表
        /// </summary>
        /// <param name="profile">玩家画像</param>
        /// <param name="topK">推荐数量</param>
        /// <param name="filters">过滤条件（如进度约束、已拥有物品排除）</param>
        public List<RecommendResult> Recommend(
            PlayerProfile profile,
            int topK = 10,
            RecommendFilter filters = null)
        {
            // 1. 找出玩家有强交互的物品集合（"种子物品"）
            var seedItems = GetSeedItems(profile, maxSeeds: 20);
            if (seedItems.Count == 0) return new List<RecommendResult>();

            // 2. 计算候选物品的预测得分
            var candidateScores = new Dictionary<string, float>();

            foreach (var (seedId, seedRating) in seedItems)
            {
                if (!_itemSimilarityMatrix.TryGetValue(seedId, out var similars)) continue;

                foreach (var (candidateId, simScore) in similars)
                {
                    // 过滤已交互物品（不重复推荐）
                    if (profile.ContentHistory.ContainsKey(candidateId)) continue;

                    // 预测得分 = Σ (种子物品评分 × 与种子的相似度)
                    if (!candidateScores.ContainsKey(candidateId))
                        candidateScores[candidateId] = 0f;
                    candidateScores[candidateId] += seedRating * simScore;
                }
            }

            // 3. 应用过滤条件（进度约束、内容类型、已解锁等）
            if (filters != null)
            {
                var filtered = new Dictionary<string, float>();
                foreach (var (id, score) in candidateScores)
                {
                    if (_itemCatalog.TryGetValue(id, out var item) &&
                        PassesFilter(item, profile, filters))
                    {
                        filtered[id] = score;
                    }
                }
                candidateScores = filtered;
            }

            // 4. 探索-利用平衡（加入少量随机推荐，避免信息茧房）
            InjectExploration(candidateScores, profile, explorationRatio: 0.15f);

            // 5. 按得分排序，取TopK
            var sorted = new List<KeyValuePair<string, float>>(candidateScores);
            sorted.Sort((a, b) => b.Value.CompareTo(a.Value));

            var results = new List<RecommendResult>();
            for (int i = 0; i < Mathf.Min(topK, sorted.Count); i++)
            {
                var (itemId, score) = (sorted[i].Key, sorted[i].Value);
                results.Add(new RecommendResult
                {
                    ContentId = itemId,
                    Score = score,
                    Strategy = RecommendStrategy.ItemCF,
                    Reason = BuildReasonText(itemId, seedItems, profile)
                });
            }

            return results;
        }

        /// <summary>
        /// 获取玩家的种子物品（有明确偏好的物品）
        /// </summary>
        private List<(string id, float rating)> GetSeedItems(PlayerProfile profile, int maxSeeds)
        {
            var seeds = new List<(string, float)>();

            foreach (var (id, interaction) in profile.ContentHistory)
            {
                if (interaction.Rating >= 2.0f) // 只选择正向交互作为种子
                    seeds.Add((id, interaction.Rating));
            }

            // 按评分降序排列，取最多maxSeeds个
            seeds.Sort((a, b) => b.Item2.CompareTo(a.Item2));
            if (seeds.Count > maxSeeds)
                seeds = seeds.GetRange(0, maxSeeds);

            return seeds;
        }

        /// <summary>
        /// 探索注入：以一定比例随机插入新内容，防止推荐陷入局部最优
        /// </summary>
        private void InjectExploration(
            Dictionary<string, float> scores,
            PlayerProfile profile,
            float explorationRatio)
        {
            // 获取候选池中玩家未见过的随机内容
            var unseenItems = new List<string>();
            foreach (var (id, item) in _itemCatalog)
            {
                if (!profile.ContentHistory.ContainsKey(id) &&
                    !scores.ContainsKey(id))
                {
                    unseenItems.Add(id);
                }
            }

            int explorationCount = Mathf.RoundToInt(scores.Count * explorationRatio);
            for (int i = 0; i < Mathf.Min(explorationCount, unseenItems.Count); i++)
            {
                int randomIdx = UnityEngine.Random.Range(0, unseenItems.Count);
                string randomItem = unseenItems[randomIdx];
                unseenItems.RemoveAt(randomIdx);

                // 探索物品给予较低基础分，使其不会压过强推荐
                float maxScore = scores.Count > 0 ? new System.Linq.Lookup<float,string>(null).Count : 1f;
                scores[randomItem] = 0.1f; // 低分但确保能展示
            }
        }

        private bool PassesFilter(ContentItem item, PlayerProfile profile, RecommendFilter filter)
        {
            // 进度约束
            if (filter.RequireProgressCheck && item.RequiredLevel > profile.MaxLevelCompleted)
                return false;

            // 内容类型过滤
            if (filter.AllowedTypes != null && filter.AllowedTypes.Count > 0 &&
                !filter.AllowedTypes.Contains(item.Type))
                return false;

            // 付费内容对非付费用户的屏蔽
            if (filter.OnlyFreeContent && item.IsPaid && !profile.IsPayingUser)
                return false;

            return true;
        }

        private string BuildReasonText(string itemId, List<(string, float)> seeds, PlayerProfile profile)
        {
            if (seeds.Count == 0) return "猜你喜欢";
            var topSeed = seeds[0];
            if (_itemCatalog.TryGetValue(topSeed.Item1, out var seedItem))
                return $"因为你喜欢 {seedItem.Name}";
            return "为你精选";
        }
    }

    [Serializable]
    public class ContentItem
    {
        public string ItemId;
        public string Name;
        public ContentType Type;
        public int RequiredLevel;   // 解锁所需等级
        public bool IsPaid;
        public string[] Tags;
        public float Popularity;    // 全局热度（0~1）
    }

    public class RecommendFilter
    {
        public bool RequireProgressCheck = true;
        public List<ContentType> AllowedTypes;
        public bool OnlyFreeContent = false;
    }

    public enum RecommendStrategy
    {
        ItemCF,        // 协同过滤
        ContentBased,  // 内容相似
        Popular,       // 热门推荐
        Contextual,    // 上下文推荐（场景感知）
        Rule,          // 规则推荐
        Exploration    // 探索推荐
    }

    [Serializable]
    public class RecommendResult
    {
        public string ContentId;
        public float Score;
        public RecommendStrategy Strategy;
        public string Reason;  // 推荐理由文案
        public int Position;   // 推荐位置（排名）
    }

    [Serializable]
    public class ItemSimilarityData
    {
        public List<ItemSimilarityEntry> Entries;
    }

    [Serializable]
    public class ItemSimilarityEntry
    {
        public string ItemId;
        public string SimilarItemId;
        public float Score; // 余弦相似度
    }
}
```

### 4.2 基于规则的降级推荐（冷启动 & 离线容灾）

```csharp
namespace GameRecommend
{
    /// <summary>
    /// 规则推荐引擎
    /// 用途：
    ///   1. 新用户冷启动（无行为数据时）
    ///   2. 网络故障时的本地降级
    ///   3. 配合协同过滤的保底兜底
    /// </summary>
    public class RuleBasedRecommender
    {
        private List<ContentItem> _allItems;

        public RuleBasedRecommender(List<ContentItem> allItems)
        {
            _allItems = allItems;
        }

        /// <summary>
        /// 冷启动推荐：基于用户基本属性（无行为数据）
        /// </summary>
        public List<RecommendResult> ColdStartRecommend(PlayerProfile profile, int topK = 10)
        {
            var candidates = new List<(ContentItem item, float score)>();

            foreach (var item in _allItems)
            {
                float score = 0f;

                // 规则1：全局热度（所有用户都推热门内容）
                score += item.Popularity * 2f;

                // 规则2：进度匹配（推荐与当前进度匹配的内容）
                int progressGap = item.RequiredLevel - profile.MaxLevelCompleted;
                if (progressGap <= 0)
                    score += 1f;  // 已解锁内容加分
                else if (progressGap <= 3)
                    score += 0.5f; // 即将解锁的内容（引导挑战感）
                else
                    score -= 1f;  // 太超前的内容降分

                // 规则3：新内容加权（近7天新上线）
                // 此处简化，实际用item的上线时间判断
                score += 0.3f;

                // 规则4：付费用户推荐付费内容
                if (profile.IsPayingUser && item.IsPaid)
                    score += 0.5f;

                candidates.Add((item, score));
            }

            // 排序取Top
            candidates.Sort((a, b) => b.score.CompareTo(a.score));

            var results = new List<RecommendResult>();
            for (int i = 0; i < Mathf.Min(topK, candidates.Count); i++)
            {
                results.Add(new RecommendResult
                {
                    ContentId = candidates[i].item.ItemId,
                    Score = candidates[i].score,
                    Strategy = RecommendStrategy.Rule,
                    Reason = i == 0 ? "最受玩家欢迎" : "猜你喜欢"
                });
            }
            return results;
        }

        /// <summary>
        /// 上下文感知推荐：基于当前游戏场景
        /// </summary>
        public List<RecommendResult> ContextualRecommend(
            PlayerProfile profile,
            GameContext context,
            int topK = 5)
        {
            var results = new List<RecommendResult>();

            switch (context.Scene)
            {
                case GameScene.AfterLevelComplete:
                    // 通关后推荐：下一关 + 相关皮肤
                    results.AddRange(RecommendNextLevel(profile, context.CurrentLevelId));
                    results.AddRange(RecommendRelatedSkins(context.CurrentLevelId));
                    break;

                case GameScene.AfterLevelFailed:
                    // 失败后推荐：推荐提升战力的道具/角色
                    results.AddRange(RecommendPowerUpItems(profile));
                    break;

                case GameScene.Lobby:
                    // 大厅推荐：活动 + 限时内容
                    results.AddRange(RecommendTimeLimitedContent(profile));
                    break;

                case GameScene.Shop:
                    // 商店推荐：个性化商品
                    results.AddRange(RecommendShopItems(profile, topK));
                    break;
            }

            return results.GetRange(0, Mathf.Min(topK, results.Count));
        }

        private List<RecommendResult> RecommendNextLevel(PlayerProfile profile, string currentLevelId)
        {
            // 查找下一关卡逻辑（简化示例）
            return new List<RecommendResult>
            {
                new RecommendResult
                {
                    ContentId = $"level_{profile.MaxLevelCompleted + 1}",
                    Strategy = RecommendStrategy.Contextual,
                    Reason = "继续挑战下一关"
                }
            };
        }

        private List<RecommendResult> RecommendRelatedSkins(string levelId) => new List<RecommendResult>();
        private List<RecommendResult> RecommendPowerUpItems(PlayerProfile profile) => new List<RecommendResult>();
        private List<RecommendResult> RecommendTimeLimitedContent(PlayerProfile profile) => new List<RecommendResult>();
        private List<RecommendResult> RecommendShopItems(PlayerProfile profile, int topK) => new List<RecommendResult>();
    }

    public class GameContext
    {
        public GameScene Scene;
        public string CurrentLevelId;
        public long Timestamp;
    }

    public enum GameScene { Lobby, AfterLevelComplete, AfterLevelFailed, Shop, CharacterSelect }
}
```

```csharp
using UnityEngine;
using System.Collections;
using GameRecommend;

/// <summary>
/// 推荐系统客户端总入口
/// 职责：协调在线推荐（服务端）与离线推荐（本地），管理缓存与曝光追踪
/// </summary>
public class RecommendManager : MonoBehaviour
{
    public static RecommendManager Instance { get; private set; }

    [Header("服务端配置")]
    [SerializeField] private string _apiEndpoint = "https://recommend.game.com/api/v1";
    [SerializeField] private int _cacheExpireMinutes = 30;

    private PlayerProfile _currentProfile;
    private BehaviorCollector _behaviorCollector;
    private RuleBasedRecommender _ruleRecommender;
    private ItemBasedCF _itemCF;

    // 推荐结果缓存（按展示位置分组）
    private Dictionary<string, List<RecommendResult>> _resultCache =
        new Dictionary<string, List<RecommendResult>>();
    private float _lastFetchTime;

    // 曝光记录（用于去重，防止同一物品反复展示）
    private HashSet<string> _exposedItemsThisSession = new HashSet<string>();

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); return; }

        _behaviorCollector = new BehaviorCollector();
        _itemCF = new ItemBasedCF();
    }

    public void Initialize(PlayerProfile profile, List<ContentItem> catalog)
    {
        _currentProfile = profile;
        _ruleRecommender = new RuleBasedRecommender(catalog);
    }

    /// <summary>
    /// 获取指定展示位的推荐内容（同步接口，优先返回缓存）
    /// </summary>
    public List<RecommendResult> GetRecommendations(
        string placement,      // 展示位标识，如 "home_banner", "lobby_card", "shop_hot"
        int count = 5,
        GameContext context = null)
    {
        // 1. 有效缓存直接返回
        if (_resultCache.TryGetValue(placement, out var cached) &&
            Time.realtimeSinceStartup - _lastFetchTime < _cacheExpireMinutes * 60f)
        {
            TrackImpressions(cached); // 记录曝光
            return cached;
        }

        // 2. 无缓存 → 本地降级推荐（同时异步拉取服务端结果）
        List<RecommendResult> fallback;
        if (context != null)
            fallback = _ruleRecommender.ContextualRecommend(_currentProfile, context, count);
        else
            fallback = _ruleRecommender.ColdStartRecommend(_currentProfile, count);

        // 异步拉取服务端个性化推荐（下次展示时使用）
        StartCoroutine(FetchServerRecommendations(placement, count));

        TrackImpressions(fallback);
        return fallback;
    }

    private IEnumerator FetchServerRecommendations(string placement, int count)
    {
        string url = $"{_apiEndpoint}/recommend?placement={placement}&count={count}" +
                     $"&user_id={_currentProfile.UserId}";

        using var request = UnityEngine.Networking.UnityWebRequest.Get(url);
        request.timeout = 5;
        yield return request.SendWebRequest();

        if (request.result == UnityEngine.Networking.UnityWebRequest.Result.Success)
        {
            var response = JsonUtility.FromJson<RecommendResponse>(request.downloadHandler.text);
            if (response?.Results != null)
            {
                _resultCache[placement] = response.Results;
                _lastFetchTime = Time.realtimeSinceStartup;
                Debug.Log($"[Recommend] 拉取 {placement} 推荐成功: {response.Results.Count} 条");
            }
        }
    }

    /// <summary>
    /// 记录玩家行为（供算法学习）
    /// </summary>
    public void TrackBehavior(string contentId, ContentType type, string action, float value = 1f)
    {
        _behaviorCollector.RecordBehavior(_currentProfile, contentId, type, action, value);

        // 异步上报到服务端（用于更新模型）
        StartCoroutine(ReportBehaviorToServer(contentId, action, value));
    }

    private void TrackImpressions(List<RecommendResult> results)
    {
        foreach (var result in results)
        {
            if (!_exposedItemsThisSession.Contains(result.ContentId))
            {
                _exposedItemsThisSession.Add(result.ContentId);
                TrackBehavior(result.ContentId, ContentType.Level, "view");
            }
        }
    }

    private IEnumerator ReportBehaviorToServer(string contentId, string action, float value)
    {
        var payload = JsonUtility.ToJson(new BehaviorEvent
        {
            UserId = _currentProfile.UserId,
            ContentId = contentId,
            Action = action,
            Value = value,
            Timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds()
        });

        byte[] body = System.Text.Encoding.UTF8.GetBytes(payload);
        using var req = new UnityEngine.Networking.UnityWebRequest($"{_apiEndpoint}/behavior", "POST");
        req.uploadHandler   = new UnityEngine.Networking.UploadHandlerRaw(body);
        req.downloadHandler = new UnityEngine.Networking.DownloadHandlerBuffer();
        req.SetRequestHeader("Content-Type", "application/json");
        req.timeout = 5;
        yield return req.SendWebRequest();
    }

    [Serializable]
    private class RecommendResponse { public List<RecommendResult> Results; }

    [Serializable]
    private class BehaviorEvent
    {
        public string UserId;
        public string ContentId;
        public string Action;
        public float Value;
        public long Timestamp;
    }
}
```

## 七、指标体系与效果评估

```csharp
/// <summary>
/// 推荐系统评估指标计算工具
/// </summary>
public static class RecommendMetrics
{
    /// <summary>
    /// 点击率（CTR）：点击次数 / 展示次数
    /// </summary>
    public static float CalcCTR(int clicks, int impressions) =>
        impressions > 0 ? (float)clicks / impressions : 0f;

    /// <summary>
    /// Precision@K：TopK推荐中，被用户实际交互的比例
    /// </summary>
    public static float PrecisionAtK(List<string> recommended, HashSet<string> interacted, int k)
    {
        int hits = 0;
        for (int i = 0; i < Mathf.Min(k, recommended.Count); i++)
            if (interacted.Contains(recommended[i])) hits++;
        return k > 0 ? (float)hits / k : 0f;
    }

    /// <summary>
    /// NDCG@K（归一化折损累计增益）：考虑排名顺序的推荐质量指标
    /// 值域0~1，越高越好
    /// </summary>
    public static float NDCG_K(List<string> recommended, Dictionary<string, float> relevanceScores, int k)
    {
        float dcg = 0f, idcg = 0f;

        // DCG
        for (int i = 0; i < Mathf.Min(k, recommended.Count); i++)
        {
            float rel = relevanceScores.TryGetValue(recommended[i], out float r) ? r : 0f;
            dcg += rel / Mathf.Log(i + 2, 2); // log₂(i+2) 为折损因子
        }

        // 理想DCG（按相关度最优排序）
        var sortedScores = new List<float>(relevanceScores.Values);
        sortedScores.Sort((a, b) => b.CompareTo(a));
        for (int i = 0; i < Mathf.Min(k, sortedScores.Count); i++)
            idcg += sortedScores[i] / Mathf.Log(i + 2, 2);

        return idcg > 0 ? dcg / idcg : 0f;
    }

    /// <summary>
    /// 多样性分数：推荐列表中不同类型内容的比例
    /// 多样性过低 → 信息茧房；多样性过高 → 相关性下降
    /// </summary>
    public static float DiversityScore(List<ContentItem> recommended)
    {
        if (recommended.Count == 0) return 0f;
        var types = new HashSet<ContentType>();
        foreach (var item in recommended) types.Add(item.Type);
        return (float)types.Count / recommended.Count;
    }
}
```

## 八、最佳实践总结

### 8.1 推荐系统成熟度路线图

```
阶段1（起步）：规则推荐
  全局热度榜 + 进度约束过滤 + 运营人工干预
  → 快速上线，无需大量数据

阶段2（成长）：协同过滤
  Item-CF / User-CF + 行为数据采集
  → 开始个性化，需要≥10万DAU的数据量

阶段3（成熟）：深度学习
  双塔模型 / Wide&Deep / DeepFM
  → 特征交叉能力强，需要专门ML团队维护

阶段4（领先）：实时强化学习
  Bandit算法 + 实时特征 + 在线学习
  → 毫秒级响应，动态最优化
```

### 8.2 核心设计原则

| 原则 | 说明 |
|------|------|
| **质量 > 数量** | 展示5个高相关推荐，胜过展示20个低质量推荐 |
| **可解释性** | 始终提供推荐理由文案，提升用户信任感 |
| **尊重游戏进度** | 推荐被锁定的内容是最大的用户体验杀手 |
| **多元化** | 每个推荐列表保证至少2种内容类型，避免单一重复 |
| **及时降级** | 服务端不可用时，本地规则推荐必须无缝接管 |
| **数据飞轮** | 推荐→行为→数据→优化模型→更好推荐，形成正向循环 |

### 8.3 隐私合规注意事项

- **最小化数据收集**：只采集与推荐直接相关的游戏内行为，不采集位置等敏感信息
- **用户控制权**：提供"重置推荐偏好"功能，允许用户清除画像数据
- **透明告知**：在隐私政策中明确说明个性化推荐的数据使用方式
- **GDPR/PIPL合规**：欧盟和中国玩家的数据处理需符合当地法规，支持数据导出与删除

> **工程建议**：首次集成推荐系统时，建议从"全局热度 + 进度过滤"的规则方案起步，同时建立完整的行为数据采集管道。当积累足够数据后，再逐步迭代到协同过滤乃至深度学习方案，避免一步到位导致工程复杂度失控。
