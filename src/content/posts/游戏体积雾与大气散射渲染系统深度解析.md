# 游戏体积雾与大气散射渲染系统深度解析

## 一、物理基础：光在介质中的传播

### 1.1 大气散射的物理模型

自然界中光线在传播过程中会与大气中的粒子发生相互作用，产生散射现象。游戏中逼真的天空与雾效正是基于这套物理模型构建的。

**两种核心散射类型：**

| 散射类型 | 适用场景 | 粒子尺寸 | 视觉特征 |
|---------|---------|---------|---------|
| Rayleigh散射 | 空气分子、小粒子 | << 光波长 | 蓝天、日出红霞 |
| Mie散射 | 雾、云、尘埃 | ≈ 光波长 | 前向散射、光晕 |

**Rayleigh散射相函数：**

$$F_{Rayleigh}(\theta) = \frac{3}{16\pi}(1 + \cos^2\theta)$$

**Mie散射的Henyey-Greenstein相函数：**

$$F_{HG}(\theta, g) = \frac{1}{4\pi} \cdot \frac{1 - g^2}{(1 + g^2 - 2g\cos\theta)^{3/2}}$$

其中 `g` 是各向异性因子，`g=0` 为各向同性，`g>0` 表示前向散射（雾效），`g<0` 表示后向散射。

### 1.2 体积渲染方程（VRE）

光在参与介质中传播时，需要综合考虑吸收、外散射、内散射等因素：

$$L(x, \omega) = \int_0^d T(x, x_t) \cdot \sigma_s(x_t) \cdot L_{scat}(x_t, \omega) \, dt$$

其中透射率 $T$ 的计算：

$$T(a, b) = e^{-\int_a^b \sigma_e(x)dx}$$

## 二、Ray Marching 体积雾实现

### 2.1 基础 Ray Marching 算法

Ray Marching 是实现体积雾的核心技术——沿视线方向步进采样，累积散射贡献。

```hlsl
// VolumetricFog.hlsl - 核心 Ray Marching 算法

#ifndef VOLUMETRIC_FOG_INCLUDED
#define VOLUMETRIC_FOG_INCLUDED

#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"
#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/DeclareDepthTexture.hlsl"

// 雾效参数
CBUFFER_START(FogParams)
    float4 _FogColor;
    float  _FogDensity;       // 散射系数
    float  _FogAbsorption;    // 吸收系数
    float  _FogHeight;        // 雾层高度
    float  _FogHeightFalloff; // 高度衰减系数
    float  _MieG;             // Mie 各向异性因子
    int    _MarchSteps;       // 步进次数
    float  _MaxDistance;      // 最大步进距离
CBUFFER_END

// Henyey-Greenstein 相函数
float HenyeyGreenstein(float cosTheta, float g)
{
    float g2 = g * g;
    float denom = 1.0 + g2 - 2.0 * g * cosTheta;
    return (1.0 / (4.0 * PI)) * (1.0 - g2) / (denom * sqrt(denom));
}

// 高度相关的雾密度
float GetFogDensity(float3 worldPos)
{
    float heightFog = exp(-max(0.0, worldPos.y - _FogHeight) * _FogHeightFalloff);
    return _FogDensity * heightFog;
}

// 透射率计算
float GetTransmittance(float density, float stepSize)
{
    return exp(-(density + _FogAbsorption) * stepSize);
}

// 主 Ray Marching 函数
float4 RayMarchFog(float3 rayOrigin, float3 rayDir, float sceneDepth)
{
    float stepSize = min(sceneDepth, _MaxDistance) / (float)_MarchSteps;
    
    float3 accumulatedLight = 0.0;
    float transmittance = 1.0;
    
    // 获取主光源方向
    Light mainLight = GetMainLight();
    float3 lightDir = mainLight.direction;
    float3 lightColor = mainLight.color;
    
    // cosTheta：视线与光源方向夹角
    float cosTheta = dot(rayDir, lightDir);
    float phaseHG = HenyeyGreenstein(cosTheta, _MieG);
    
    [loop]
    for (int i = 0; i < _MarchSteps; i++)
    {
        float t = (i + 0.5) * stepSize;
        float3 samplePos = rayOrigin + rayDir * t;
        
        float density = GetFogDensity(samplePos);
        if (density < 0.0001) continue;
        
        // 计算步进透射率
        float stepTransmittance = GetTransmittance(density, stepSize);
        
        // 阴影采样（简化版本，使用主光源阴影）
        float shadowAtten = mainLight.shadowAttenuation;
        
        // 散射贡献 = 密度 * 相函数 * 光照 * 当前透射率
        float3 scattering = density * phaseHG * lightColor * shadowAtten;
        scattering *= _FogColor.rgb;
        
        // 积分累加（使用精确积分公式）
        float3 scatteringIntegral = scattering * (1.0 - stepTransmittance) / (density + _FogAbsorption);
        accumulatedLight += transmittance * scatteringIntegral;
        
        transmittance *= stepTransmittance;
        
        // 早退优化：透射率过低时停止
        if (transmittance < 0.01) break;
    }
    
    return float4(accumulatedLight, 1.0 - transmittance);
}

#endif
```

```csharp
// VolumetricFogRenderFeature.cs
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

public class VolumetricFogRenderFeature : ScriptableRendererFeature
{
    [System.Serializable]
    public class Settings
    {
        public RenderPassEvent renderPassEvent = RenderPassEvent.BeforeRenderingPostProcessing;
        public Shader fogShader;
        [Range(8, 64)] public int marchSteps = 32;
        [Range(0f, 1f)] public float fogDensity = 0.05f;
        public Color fogColor = Color.white;
        [Range(-1f, 1f)] public float mieG = 0.76f;
        public float maxDistance = 100f;
        public float fogHeight = 0f;
        public float fogHeightFalloff = 0.5f;
    }

    public Settings settings = new Settings();
    private VolumetricFogPass _fogPass;

    public override void Create()
    {
        if (settings.fogShader == null) return;
        _fogPass = new VolumetricFogPass(settings);
        _fogPass.renderPassEvent = settings.renderPassEvent;
    }

    public override void AddRenderPasses(ScriptableRenderer renderer, ref RenderingData renderingData)
    {
        if (_fogPass == null) return;
        renderer.EnqueuePass(_fogPass);
    }
}

public class VolumetricFogPass : ScriptableRenderPass
{
    private readonly VolumetricFogRenderFeature.Settings _settings;
    private Material _fogMaterial;
    private RTHandle _fogTexture;
    
    private static readonly int FogColorId = Shader.PropertyToID("_FogColor");
    private static readonly int FogDensityId = Shader.PropertyToID("_FogDensity");
    private static readonly int MieGId = Shader.PropertyToID("_MieG");
    private static readonly int MarchStepsId = Shader.PropertyToID("_MarchSteps");
    private static readonly int MaxDistanceId = Shader.PropertyToID("_MaxDistance");
    private static readonly int FogHeightId = Shader.PropertyToID("_FogHeight");
    private static readonly int FogHeightFalloffId = Shader.PropertyToID("_FogHeightFalloff");

    public VolumetricFogPass(VolumetricFogRenderFeature.Settings settings)
    {
        _settings = settings;
        _fogMaterial = CoreUtils.CreateEngineMaterial(settings.fogShader);
    }

    public override void OnCameraSetup(CommandBuffer cmd, ref RenderingData renderingData)
    {
        var descriptor = renderingData.cameraData.cameraTargetDescriptor;
        // 半分辨率渲染，提升性能
        descriptor.width /= 2;
        descriptor.height /= 2;
        descriptor.depthBufferBits = 0;
        descriptor.colorFormat = RenderTextureFormat.ARGBHalf;
        
        RenderingUtils.ReAllocateIfNeeded(ref _fogTexture, descriptor, 
            FilterMode.Bilinear, name: "_VolumetricFogTexture");
        
        ConfigureTarget(_fogTexture);
        ConfigureClear(ClearFlag.Color, Color.clear);
    }

    public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
    {
        if (_fogMaterial == null) return;
        
        CommandBuffer cmd = CommandBufferPool.Get("VolumetricFog");
        
        // 更新 Shader 参数
        _fogMaterial.SetColor(FogColorId, _settings.fogColor);
        _fogMaterial.SetFloat(FogDensityId, _settings.fogDensity);
        _fogMaterial.SetFloat(MieGId, _settings.mieG);
        _fogMaterial.SetInt(MarchStepsId, _settings.marchSteps);
        _fogMaterial.SetFloat(MaxDistanceId, _settings.maxDistance);
        _fogMaterial.SetFloat(FogHeightId, _settings.fogHeight);
        _fogMaterial.SetFloat(FogHeightFalloffId, _settings.fogHeightFalloff);
        
        using (new ProfilingScope(cmd, new ProfilingSampler("Volumetric Fog")))
        {
            // 绘制全屏 Quad 执行体积雾计算
            Blitter.BlitCameraTexture(cmd, renderingData.cameraData.renderer.cameraColorTargetHandle, 
                _fogTexture, _fogMaterial, 0);
            
            // 将雾效叠加到场景（加法混合）
            cmd.SetGlobalTexture("_VolumetricFogTexture", _fogTexture);
        }
        
        context.ExecuteCommandBuffer(cmd);
        CommandBufferPool.Release(cmd);
    }

    public override void OnCameraCleanup(CommandBuffer cmd)
    {
        // 清理 RTHandle
    }
    
    public void Dispose()
    {
        _fogTexture?.Release();
        CoreUtils.Destroy(_fogMaterial);
    }
}
```

## 三、大气散射天空盒实现

### 3.1 Precomputed Atmospheric Scattering

实时计算完整的大气散射开销极大，工业实践中通常使用**预计算查找表（LUT）**存储散射结果。

```hlsl
// AtmosphericScattering.hlsl

#define PLANET_RADIUS 6371000.0   // 地球半径（米）
#define ATM_RADIUS    6471000.0   // 大气顶部半径
#define RAYLEIGH_SCALE_HEIGHT 8500.0
#define MIE_SCALE_HEIGHT      1200.0

// Rayleigh 散射系数（RGB，对应波长 650nm/510nm/475nm）
static const float3 RAYLEIGH_BETA = float3(5.5e-6, 13.0e-6, 22.4e-6);
// Mie 散射系数
static const float3 MIE_BETA = float3(21e-6, 21e-6, 21e-6);

struct AtmosphereParams
{
    float3 sunDirection;
    float  sunIntensity;
    float3 viewPos;        // 相对于星球中心的位置
};

// 射线与球体相交检测
bool RaySphereIntersect(float3 ro, float3 rd, float radius, out float t0, out float t1)
{
    float3 oc = ro;
    float a = dot(rd, rd);
    float b = 2.0 * dot(oc, rd);
    float c = dot(oc, oc) - radius * radius;
    float disc = b * b - 4.0 * a * c;
    
    if (disc < 0.0) return false;
    
    float sqrtDisc = sqrt(disc);
    t0 = (-b - sqrtDisc) / (2.0 * a);
    t1 = (-b + sqrtDisc) / (2.0 * a);
    return true;
}

// 大气密度计算
float2 GetDensity(float height)
{
    float2 density;
    density.x = exp(-height / RAYLEIGH_SCALE_HEIGHT); // Rayleigh
    density.y = exp(-height / MIE_SCALE_HEIGHT);       // Mie
    return density;
}

// 单次散射积分
float3 ComputeSkyColor(AtmosphereParams params, float3 rayDir, int numSteps, int numLightSteps)
{
    float t0, t1;
    float3 planetCenter = float3(0, -PLANET_RADIUS, 0);
    float3 viewPos = params.viewPos - planetCenter;
    
    if (!RaySphereIntersect(viewPos, rayDir, ATM_RADIUS, t0, t1))
        return float3(0, 0, 0);
    
    t0 = max(t0, 0.0);
    float stepSize = (t1 - t0) / (float)numSteps;
    
    float3 totalRayleigh = float3(0, 0, 0);
    float3 totalMie = float3(0, 0, 0);
    float2 opticalDepth = float2(0, 0);
    
    float cosTheta = dot(rayDir, params.sunDirection);
    float phaseR = (3.0 / (16.0 * PI)) * (1.0 + cosTheta * cosTheta);
    float phaseM = HenyeyGreenstein(cosTheta, 0.76);
    
    for (int i = 0; i < numSteps; i++)
    {
        float t = t0 + (i + 0.5) * stepSize;
        float3 samplePos = viewPos + rayDir * t;
        float height = length(samplePos) - PLANET_RADIUS;
        
        float2 density = GetDensity(height) * stepSize;
        opticalDepth += density;
        
        // 计算到太阳的光学深度（次级 Ray March）
        float2 lightOpticalDepth = float2(0, 0);
        float lt0_light, lt1_light;
        if (RaySphereIntersect(samplePos, params.sunDirection, ATM_RADIUS, lt0_light, lt1_light))
        {
            float lightStep = lt1_light / (float)numLightSteps;
            for (int j = 0; j < numLightSteps; j++)
            {
                float3 lightPos = samplePos + params.sunDirection * ((j + 0.5) * lightStep);
                float lightHeight = length(lightPos) - PLANET_RADIUS;
                lightOpticalDepth += GetDensity(lightHeight) * lightStep;
            }
        }
        
        // 透射率
        float3 attenuation = exp(
            -(RAYLEIGH_BETA * (opticalDepth.x + lightOpticalDepth.x) + 
              MIE_BETA * 1.1 * (opticalDepth.y + lightOpticalDepth.y))
        );
        
        totalRayleigh += density.x * attenuation;
        totalMie += density.y * attenuation;
    }
    
    return params.sunIntensity * (
        totalRayleigh * RAYLEIGH_BETA * phaseR + 
        totalMie * MIE_BETA * phaseM
    );
}
```

```hlsl
Shader "Custom/AtmosphericSkybox"
{

            {

            };

            float  _SunIntensity;
            float  _SunDiscSize;
            float4 _GroundColor;
            
            Varyings vert(Attributes IN)
            {
                Varyings OUT;
                OUT.positionCS = TransformObjectToHClip(IN.positionOS.xyz);
                OUT.worldDir = IN.positionOS.xyz;
                return OUT;
            }
            
            float4 frag(Varyings IN) : SV_Target
            {
                float3 rayDir = normalize(IN.worldDir);
                
                // 地面处理
                if (rayDir.y < 0)
                    return float4(_GroundColor.rgb, 1.0);
                
                AtmosphereParams params;
                params.sunDirection = normalize(_SunDirection.xyz);
                params.sunIntensity = _SunIntensity;
                // 相机位于海平面上方 200m
                params.viewPos = float3(0, PLANET_RADIUS + 200.0, 0);
                
                float3 color = ComputeSkyColor(params, rayDir, 16, 8);
                
                // 添加太阳光盘
                float sunDot = dot(rayDir, params.sunDirection);
                float sunDisc = smoothstep(1.0 - _SunDiscSize, 1.0 - _SunDiscSize * 0.5, sunDot);
                color += sunDisc * _SunIntensity * float3(1.0, 0.9, 0.7);
                
                // Reinhard 色调映射
                color = color / (color + float3(1, 1, 1));
                
                return float4(color, 1.0);
            }
            ENDHLSL
        }
    }
}
```

## 四、移动端体积雾性能优化

### 4.1 时间抖动 + 重投影（Temporal Reprojection）

```csharp
// TemporalFogManager.cs - 时域累积优化
public class TemporalFogManager : MonoBehaviour
{
    private RTHandle[] _fogHistory = new RTHandle[2];
    private int _currentFrame = 0;
    private Matrix4x4 _prevViewProj;
    
    // Halton 序列抖动采样点
    private static readonly Vector2[] HaltonSequence = GenerateHaltonSequence(8);
    
    private static Vector2[] GenerateHaltonSequence(int count)
    {
        var seq = new Vector2[count];
        for (int i = 0; i < count; i++)
        {
            seq[i] = new Vector2(Halton(i + 1, 2), Halton(i + 1, 3));
        }
        return seq;
    }
    
    private static float Halton(int index, int baseN)
    {
        float result = 0f;
        float denom = 1f;
        while (index > 0)
        {
            denom *= baseN;
            result += (index % baseN) / denom;
            index /= baseN;
        }
        return result;
    }
    
    public void Update()
    {
        // 每帧使用不同的采样偏移（抖动）
        Vector2 jitter = (HaltonSequence[_currentFrame % 8] * 2f - Vector2.one);
        Shader.SetGlobalVector("_FogJitter", new Vector4(jitter.x, jitter.y, 0, 0));
        
        // 传递历史帧数据用于重投影
        Shader.SetGlobalTexture("_FogHistoryTexture", _fogHistory[1 - (_currentFrame & 1)]);
        Shader.SetGlobalMatrix("_PrevViewProjMatrix", _prevViewProj);
        
        _prevViewProj = Camera.main.projectionMatrix * Camera.main.worldToCameraMatrix;
        _currentFrame++;
    }
}
```

## 五、高级效果：体积光柱（God Rays）

### 5.1 屏幕空间体积光

```hlsl
// GodRays.hlsl - 径向模糊实现屏幕空间体积光

Texture2D _OcclusionMask;   // 光源遮挡蒙版
float2    _LightPosScreen;  // 光源屏幕坐标 (0-1)
float     _Density;         // 光柱密度
float     _Weight;          // 光柱权重
float     _Decay;           // 衰减系数
float     _Exposure;        // 曝光强度
int       _NumSamples;      // 采样数量

float4 GodRaysPS(float2 uv : TEXCOORD0) : SV_Target
{
    // 径向采样方向（从当前像素指向光源）
    float2 deltaUV = (uv - _LightPosScreen) * (1.0 / _NumSamples) * _Density;
    
    float3 color = float3(0, 0, 0);
    float illuminationDecay = 1.0;
    float2 sampleUV = uv;
    
    for (int i = 0; i < _NumSamples; i++)
    {
        sampleUV -= deltaUV;
        // 采样遮挡蒙版（天空区域为1，物体为0）
        float3 sample = SAMPLE_TEXTURE2D(_OcclusionMask, sampler_LinearClamp, sampleUV).rgb;
        sample *= illuminationDecay * _Weight;
        color += sample;
        illuminationDecay *= _Decay;
    }
    
    return float4(color * _Exposure, 1.0);
}
```

## 六、最佳实践总结

### 6.1 性能指标参考

| 配置 | 步进次数 | 分辨率 | 时域 | 典型耗时（移动端） |
|------|---------|--------|------|-----------------|
| 低质量 | 8 | 半分辨率 | 否 | ~0.8ms |
| 中质量 | 16 | 半分辨率 | 是 | ~1.2ms |
| 高质量 | 32 | 全分辨率 | 是 | ~3.5ms |

### 6.2 工程关键点

1. **半分辨率渲染** 是最重要的优化手段，配合双边滤波上采样，视觉损失极小
2. **时域积累**（Temporal Reprojection）可将有效采样数提升 4~8 倍，是 TAA 思路的延伸
3. **早退优化**：透射率 < 1% 时停止步进，可节省约 15-30% 的计算量
4. **噪声抖动**：使用 Halton/Blue Noise 序列替代均匀采样，消除层状噪点
5. **体积阴影缓存**：对静态场景预计算 3D 阴影纹理（Voxel Shadow Map），避免每帧重算
6. **距离裁剪**：超过 200m 的区域使用解析雾（Exponential Fog）替代体积雾
7. **高度雾分离**：将低频高度雾单独处理，可大幅减少体积雾的绘制范围

## 七、总结

体积雾与大气散射是提升游戏视觉品质的重要手段，核心要点：

- **物理基础**：理解 Rayleigh/Mie 散射，才能调出正确的视觉感受
- **Ray Marching**：体积雾的核心算法，步进次数与质量/性能直接挂钩
- **预计算优化**：大气散射 LUT 预计算是移动端可用的工业标准方案
- **时域积累**：充分利用帧间信息，以极低代价大幅提升质量
- **自适应质量**：根据帧率动态调节参数，在各档次设备上都能流畅运行

结合 URP 的 ScriptableRenderPass 架构，可以将体积雾系统灵活接入任何项目，并通过 Volume Component 实现逐区域的精细控制。
