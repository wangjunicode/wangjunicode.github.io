# 游戏触觉反馈与手柄振动系统设计完全指南

## 一、触觉反馈的价值与分类

### 1.1 为什么触觉反馈很重要

触觉反馈（Haptic Feedback）是游戏沉浸感的"第五维度"——当玩家击中敌人感受到震手的冲击、开枪时感受到枪口的连续震动、攀爬时感触到粗糙的岩石质感，这些触觉信号能极大提升沉浸感和操作精准度。研究表明，合理的触觉反馈能提升玩家留存率约 15%~20%。

### 1.2 触觉反馈技术分类

| 类型 | 代表硬件 | 技术原理 | 精度 |
|------|---------|---------|------|
| 偏心旋转马达（ERM） | Android 低端机、Xbox 360 | 不平衡旋转质量块 | 低（仅开/关） |
| 线性谐振执行器（LRA） | iPhone Taptic Engine、Switch HD Rumble | 音圈驱动线性振动 | 高（频率+幅度控制） |
| 压电振动器 | 部分专业设备 | 压电晶体变形 | 极高 |
| 自适应扳机（Adaptive Trigger） | PS5 DualSense | 电机改变扳机阻力 | 特殊（力反馈） |
| 超声波触觉 | 未来 AR/VR 设备 | 超声波空气压力 | 非接触式 |

### 2.1 iOS Core Haptics（苹果触觉引擎）

iOS 13+ 提供了 Core Haptics API，支持精细的 AHAP（Apple Haptic and Audio Pattern）格式：

```csharp
using UnityEngine;
using System.Runtime.InteropServices;

public class iOSHapticEngine
{
    // 导入原生 iOS 插件函数
    #if UNITY_IOS && !UNITY_EDITOR
    [DllImport("__Internal")]
    private static extern void _PlayTransientHaptic(float intensity, float sharpness);
    
    [DllImport("__Internal")]
    private static extern void _PlayContinuousHaptic(float intensity, float sharpness, float duration);
    
    [DllImport("__Internal")]
    private static extern void _PlayAHAPPattern(string ahapJson);
    
    [DllImport("__Internal")]
    private static extern bool _IsHapticsSupported();
    #endif
    
    // 瞬态触觉（单次敲击感）
    public static void PlayTransient(float intensity = 1f, float sharpness = 0.5f)
    {
        intensity = Mathf.Clamp01(intensity);
        sharpness = Mathf.Clamp01(sharpness);
        
        #if UNITY_IOS && !UNITY_EDITOR
        if (_IsHapticsSupported())
            _PlayTransientHaptic(intensity, sharpness);
        #elif UNITY_EDITOR
        Debug.Log($"[Haptic-iOS] Transient: intensity={intensity}, sharpness={sharpness}");
        #endif
    }
    
    // 连续触觉（持续震动）
    public static void PlayContinuous(float intensity, float sharpness, float duration)
    {
        #if UNITY_IOS && !UNITY_EDITOR
        if (_IsHapticsSupported())
            _PlayContinuousHaptic(intensity, sharpness, duration);
        #elif UNITY_EDITOR
        Debug.Log($"[Haptic-iOS] Continuous: intensity={intensity}, sharpness={sharpness}, duration={duration}s");
        #endif
    }
    
    // AHAP 模式播放（最强大的自定义触觉）
    public static void PlayAHAPPattern(TextAsset ahapFile)
    {
        if (ahapFile == null) return;
        
        #if UNITY_IOS && !UNITY_EDITOR
        if (_IsHapticsSupported())
            _PlayAHAPPattern(ahapFile.text);
        #endif
    }
```

### 2.2 Android 振动器（Vibrator / VibrationEffect）

```csharp
// Android 触觉封装
using UnityEngine;

public class AndroidHapticEngine
{
    private static AndroidJavaObject vibrator;
    private static bool isInitialized = false;
    private static int androidApiLevel = 0;
    
    private static void Initialize()
    {
        if (isInitialized) return;
        
        #if UNITY_ANDROID && !UNITY_EDITOR
        using var unityPlayer = new AndroidJavaClass("com.unity3d.player.UnityPlayer");
        using var activity = unityPlayer.GetStatic<AndroidJavaObject>("currentActivity");
        vibrator = activity.Call<AndroidJavaObject>("getSystemService", "vibrator");
        
        using var buildVersion = new AndroidJavaClass("android.os.Build$VERSION");
        androidApiLevel = buildVersion.GetStatic<int>("SDK_INT");
        #endif
        
        isInitialized = true;
    }
    
    // 简单振动（毫秒）
    public static void Vibrate(long milliseconds)
    {
        Initialize();
        
        #if UNITY_ANDROID && !UNITY_EDITOR
        if (vibrator == null) return;
        
        if (androidApiLevel >= 26)  // Android 8.0+
        {
            // 使用 VibrationEffect
            using var vibrationEffect = new AndroidJavaClass("android.os.VibrationEffect");
            var effect = vibrationEffect.CallStatic<AndroidJavaObject>(
                "createOneShot", milliseconds, -1);  // -1 = DEFAULT_AMPLITUDE
            vibrator.Call("vibrate", effect);
        }
        else
        {
            vibrator.Call("vibrate", milliseconds);
        }
        #elif UNITY_EDITOR
        Debug.Log($"[Haptic-Android] Vibrate {milliseconds}ms");
        #endif
    }
    
    // 振动模式（节拍器式振动）
    public static void VibratePattern(long[] pattern, int repeat = -1)
    {
        Initialize();
        
        #if UNITY_ANDROID && !UNITY_EDITOR
        if (vibrator == null) return;
        
        if (androidApiLevel >= 26)
        {
            using var vibrationEffect = new AndroidJavaClass("android.os.VibrationEffect");
            var effect = vibrationEffect.CallStatic<AndroidJavaObject>(
                "createWaveform", pattern, repeat);
            vibrator.Call("vibrate", effect);
        }
        else
        {
            vibrator.Call("vibrate", pattern, repeat);
        }
        #endif
    }
    
    // 自定义振幅序列（Android 8.0+ 支持幅度控制）
    public static void VibrateWaveform(long[] timings, int[] amplitudes, int repeat = -1)
    {
        Initialize();
        
        #if UNITY_ANDROID && !UNITY_EDITOR
        if (vibrator == null || androidApiLevel < 26) 
        {
            // 降级为简单振动
            Vibrate(50);
            return;
        }
        
        // 检查幅度控制能力
        bool hasAmplitudeControl = vibrator.Call<bool>("hasAmplitudeControl");
        if (!hasAmplitudeControl)
        {
            VibratePattern(timings, repeat);
            return;
        }
        
        using var vibrationEffect = new AndroidJavaClass("android.os.VibrationEffect");
        var effect = vibrationEffect.CallStatic<AndroidJavaObject>(
            "createWaveform", timings, amplitudes, repeat);
        vibrator.Call("vibrate", effect);
        #endif
    }
    
    // 预定义触觉效果（Android 11+）
    public static void PlayPredefinedEffect(PredefinedEffect effect)
    {
        Initialize();
        
        #if UNITY_ANDROID && !UNITY_EDITOR
        if (vibrator == null || androidApiLevel < 30) 
        {
            Vibrate(20);
            return;
        }
        
        using var vibrationEffect = new AndroidJavaClass("android.os.VibrationEffect");
        int effectId = (int)effect;
        var vEffect = vibrationEffect.CallStatic<AndroidJavaObject>("createPredefined", effectId);
        vibrator.Call("vibrate", vEffect);
        #endif
    }
    
    public enum PredefinedEffect
    {
        Click = 0,
        DoubleClick = 1,
        Tick = 2,
        HeavyClick = 5
    }
    
    public static void Cancel()
    {
        #if UNITY_ANDROID && !UNITY_EDITOR
        vibrator?.Call("cancel");
        #endif
    }
}
```

## 三、主机手柄 HD 振动

### 3.1 PS5 DualSense 触觉系统

DualSense 提供了三大核心触觉功能：
- **HD 振动**：高保真双马达振动，区分频率和振幅
- **自适应扳机**：L2/R2 扳机可编程阻力
- **麦克风阵列**：（非振动，但影响触觉设计）

```csharp
// Unity PS5 DualSense 振动封装
// 需要 Unity PS5 SDK 包（com.unity.platforms.ps5）
using UnityEngine;

#if UNITY_PS5
using Unity.PSVita;  // 实际使用 PS5 SDK 的命名空间（具体按官方文档）
#endif

public class DualSenseHapticController : MonoBehaviour
{
    [Header("振动参数")]
    [SerializeField, Range(0f, 1f)] private float leftMotorIntensity = 0f;
    [SerializeField, Range(0f, 1f)] private float rightMotorIntensity = 0f;
    
    [Header("自适应扳机")]
    [SerializeField] private TriggerMode leftTriggerMode = TriggerMode.Off;
    [SerializeField] private TriggerMode rightTriggerMode = TriggerMode.Off;
    
    public enum TriggerMode
    {
        Off,
        Feedback,       // 连续阻力（如拉弓）
        Weapon,         // 阶段性阻力（扣扳机感）
        Vibration,      // 高频振动（引擎运转感）
        SlopeFeedback   // 斜坡阻力
    }
    
    private Gamepad gamepad;
    
    private void Update()
    {
        gamepad = Gamepad.current;
        if (gamepad == null) return;
        
        // 通过 Input System 设置振动
        SetMotorSpeeds(leftMotorIntensity, rightMotorIntensity);
    }
    
    public void SetMotorSpeeds(float left, float right)
    {
        if (gamepad == null) return;
        
        gamepad.SetMotorSpeeds(
            Mathf.Clamp01(left), 
            Mathf.Clamp01(right)
        );
    }
    
    // 枪械射击触觉反馈
    public void SimulateGunShot(WeaponType weapon)
    {
        StopAllCoroutines();
        
        switch (weapon)
        {
            case WeaponType.Pistol:
                StartCoroutine(PistolRecoilPattern());
                break;
            case WeaponType.Shotgun:
                StartCoroutine(ShotgunRecoilPattern());
                break;
            case WeaponType.Rifle:
                StartCoroutine(RifleAutoFirePattern());
                break;
            case WeaponType.Sniper:
                StartCoroutine(SniperRecoilPattern());
                break;
        }
    }
    
    // 手枪：单次强烈短震
    private System.Collections.IEnumerator PistolRecoilPattern()
    {
        SetMotorSpeeds(0.8f, 0.6f);  // 左强右弱（枪托后坐力 vs 手指触感）
        yield return new WaitForSeconds(0.06f);
        SetMotorSpeeds(0.3f, 0.2f);
        yield return new WaitForSeconds(0.04f);
        SetMotorSpeeds(0f, 0f);
    }
    
    // 霰弹枪：极强后坐力
    private System.Collections.IEnumerator ShotgunRecoilPattern()
    {
        SetMotorSpeeds(1f, 1f);
        yield return new WaitForSeconds(0.1f);
        SetMotorSpeeds(0.5f, 0.3f);
        yield return new WaitForSeconds(0.08f);
        SetMotorSpeeds(0.1f, 0f);
        yield return new WaitForSeconds(0.05f);
        SetMotorSpeeds(0f, 0f);
    }
    
    // 步枪：全自动连射节奏
    private System.Collections.IEnumerator RifleAutoFirePattern()
    {
        for (int i = 0; i < 5; i++)
        {
            SetMotorSpeeds(0.6f, 0.5f);
            yield return new WaitForSeconds(0.04f);
            SetMotorSpeeds(0.2f, 0.1f);
            yield return new WaitForSeconds(0.04f);
        }
        SetMotorSpeeds(0f, 0f);
    }
    
    // 狙击枪：压力蓄积 + 爆发
    private System.Collections.IEnumerator SniperRecoilPattern()
    {
        // 扣扳机过程（渐增阻力感）
        SetMotorSpeeds(0.05f, 0.05f);
        yield return new WaitForSeconds(0.2f);
        // 击发瞬间
        SetMotorSpeeds(1f, 0.9f);
        yield return new WaitForSeconds(0.08f);
        SetMotorSpeeds(0.4f, 0.2f);
        yield return new WaitForSeconds(0.1f);
        SetMotorSpeeds(0f, 0f);
    }
    
    // 环境触觉：走路在不同地面上的振动
    public void SimulateFootstep(SurfaceType surface)
    {
        StopAllCoroutines();
        StartCoroutine(FootstepPattern(surface));
    }
    
    private System.Collections.IEnumerator FootstepPattern(SurfaceType surface)
    {
        float left = 0f, right = 0f;
        float duration = 0f;
        
        switch (surface)
        {
            case SurfaceType.Grass:
                left = 0.15f; right = 0.1f; duration = 0.12f;
                break;
            case SurfaceType.Concrete:
                left = 0.35f; right = 0.25f; duration = 0.08f;
                break;
            case SurfaceType.Metal:
                left = 0.5f; right = 0.45f; duration = 0.05f;
                break;
            case SurfaceType.Mud:
                left = 0.2f; right = 0.25f; duration = 0.2f;
                break;
            case SurfaceType.Water:
                left = 0.1f; right = 0.15f; duration = 0.3f;
                break;
        }
        
        SetMotorSpeeds(left, right);
        yield return new WaitForSeconds(duration);
        // 逐渐消退
        SetMotorSpeeds(left * 0.3f, right * 0.3f);
        yield return new WaitForSeconds(duration * 0.5f);
        SetMotorSpeeds(0f, 0f);
    }
    
    public enum WeaponType { Pistol, Shotgun, Rifle, Sniper }
    public enum SurfaceType { Grass, Concrete, Metal, Mud, Water }
    
    private void OnDestroy()
    {
        gamepad?.SetMotorSpeeds(0f, 0f);
    }
}
```

```csharp
// DualSense 自适应扳机效果封装
// 基于 Unity Input System + PS5 SDK 扩展

using UnityEngine;
using UnityEngine.InputSystem;
using UnityEngine.InputSystem.DualShock;  // Unity 提供的 DualShock 扩展

public class AdaptiveTriggerController : MonoBehaviour
{
    [Header("弓箭蓄力设置")]
    [SerializeField] private float maxDrawForce = 1f;   // 最大拉弓阻力
    [SerializeField] private float drawSpeed = 1f;
    
    private float currentDrawAmount = 0f;
    private bool isDrawing = false;
    
    // 模拟弓箭蓄力（L2 阻力随蓄力增加）
    public void StartBowDraw()
    {
        isDrawing = true;
        StartCoroutine(BowDrawRoutine());
    }
    
    public void ReleaseBow()
    {
        isDrawing = false;
        currentDrawAmount = 0f;
        // 释放扳机阻力
        SetTriggerEffect(TriggerEffect.Off, 0f);
        
        // 释放时右马达短暂强振（弓弦弹回感）
        var gamepad = Gamepad.current;
        if (gamepad != null)
        {
            gamepad.SetMotorSpeeds(0f, 0.7f);
            Invoke(nameof(StopVibration), 0.08f);
        }
    }
    
    private System.Collections.IEnumerator BowDrawRoutine()
    {
        while (isDrawing)
        {
            currentDrawAmount = Mathf.Clamp01(currentDrawAmount + Time.deltaTime * drawSpeed);
            
            // 随蓄力增加扳机阻力（从 0.2 到 maxDrawForce）
            float triggerForce = Mathf.Lerp(0.2f, maxDrawForce, currentDrawAmount);
            SetTriggerEffect(TriggerEffect.SliderFeedback, triggerForce);
            
            // 满弓时微振动（手抖感）
            if (currentDrawAmount >= 0.95f)
            {
                var g = Gamepad.current;
                float tremble = Mathf.Sin(Time.time * 60f) * 0.15f * currentDrawAmount;
                g?.SetMotorSpeeds(Mathf.Abs(tremble), 0f);
            }
            
            yield return null;
        }
    }
    
    // 车辆驾驶：油门踏板感
    public void SetAccelerationFeedback(float throttle, float wheelSlip)
    {
        if (wheelSlip > 0.5f)
        {
            // 轮胎打滑 → 扳机振动
            SetTriggerEffect(TriggerEffect.Vibration, 0.7f);
        }
        else if (throttle > 0.8f)
        {
            // 大油门 → 强阻力
            SetTriggerEffect(TriggerEffect.ContinuousFeedback, throttle);
        }
        else
        {
            SetTriggerEffect(TriggerEffect.ContinuousFeedback, throttle * 0.3f);
        }
    }
    
    // 游泳/水中移动：水的阻力感
    public void SetWaterResistanceFeedback(float velocity)
    {
        float resistance = Mathf.Clamp01(velocity / 5f) * 0.6f;
        SetTriggerEffect(TriggerEffect.ContinuousFeedback, resistance);
    }

    {

        // DualSenseTrigger.SetEffect(TriggerSide.Right, effect, intensity);
        
        #if UNITY_EDITOR
        // Debug.Log($"[AdaptiveTrigger] Effect={effect}, Intensity={intensity:F2}");
        #endif

    {
        Gamepad.current?.SetMotorSpeeds(0f, 0f);
    }

    {
        Off,
        ContinuousFeedback,  // 持续阻力
        SectionFeedback,     // 区段阻力（扣到某点才有感觉）
```

### 4.1 统一触觉接口设计

```csharp
// 统一触觉反馈接口
using UnityEngine;

public interface IHapticProvider
{
    bool IsSupported { get; }
    void PlayImpact(HapticImpactStyle style);
    void PlayNotification(HapticNotificationType type);
    void PlaySelection();
    void PlayCustomPattern(HapticPattern pattern);

// 触觉强度枚举
public enum HapticImpactStyle
{
    Light,      // 轻触（UI 点击）
    Medium,     // 中等（一般操作反馈）
    Heavy,      // 强烈（命中/爆炸）
    Rigid,      // 刚性（碰撞硬物）
    Soft        // 柔软（踩到软物）
}

public enum HapticNotificationType
{
    Success,
    Warning,
    Error
}

// 自定义触觉模式定义
[System.Serializable]
public class HapticPattern
{
    public string name;
    public HapticKeyframe[] keyframes;
}

[System.Serializable]
public struct HapticKeyframe
{
    public float time;          // 时间点（秒）
    public float leftIntensity;  // 左马达强度 [0,1]
    public float rightIntensity; // 右马达强度 [0,1]
    public float frequency;      // 频率（Hz，高精度设备用）
```

### 4.2 平台适配层实现

```csharp
// iOS 触觉适配器
public class iOSHapticProvider : IHapticProvider
{
    public bool IsSupported => Application.platform == RuntimePlatform.IPhonePlayer;
    
    public void PlayImpact(HapticImpactStyle style)
    {
        float intensity = style switch
        {
            HapticImpactStyle.Light => 0.3f,
            HapticImpactStyle.Medium => 0.6f,
            HapticImpactStyle.Heavy => 1.0f,
            HapticImpactStyle.Rigid => 0.9f,
            HapticImpactStyle.Soft => 0.2f,
            _ => 0.5f
        };
        
        float sharpness = style switch
        {
            HapticImpactStyle.Light => 0.3f,
            HapticImpactStyle.Medium => 0.5f,
            HapticImpactStyle.Heavy => 0.8f,
            HapticImpactStyle.Rigid => 1.0f,
            HapticImpactStyle.Soft => 0.1f,
            _ => 0.5f
        };
        
        iOSHapticEngine.PlayTransient(intensity, sharpness);
    }
    
    public void PlayNotification(HapticNotificationType type)
    {
        switch (type)
        {
            case HapticNotificationType.Success:
                iOSHapticEngine.PlayTransient(0.7f, 0.4f);
                break;
            case HapticNotificationType.Warning:
                iOSHapticEngine.PlayTransient(0.6f, 0.6f);
                break;
            case HapticNotificationType.Error:
                // 三连击模式
                PlayErrorPattern();
                break;
        }
    }
    
    private async void PlayErrorPattern()
    {
        for (int i = 0; i < 3; i++)
        {
            iOSHapticEngine.PlayTransient(0.8f, 0.8f);
            await System.Threading.Tasks.Task.Delay(100);
        }
    }
    
    public void PlaySelection()
    {
        iOSHapticEngine.PlayTransient(0.15f, 0.5f);
    }
    
    public void PlayCustomPattern(HapticPattern pattern)
    {
        // 将 HapticPattern 转换为 AHAP JSON 并播放
        string ahapJson = ConvertToAHAP(pattern);
        // iOSHapticEngine.PlayAHAPPattern(ahapJson);
    }
    
    private string ConvertToAHAP(HapticPattern pattern)
    {
        // AHAP 格式转换（Apple Haptic and Audio Pattern）
        var sb = new System.Text.StringBuilder();
        sb.Append("{\"Version\":1,\"Pattern\":[");
        
        bool first = true;
        foreach (var kf in pattern.keyframes)
        {
            if (!first) sb.Append(",");
            first = false;
            
            sb.Append("{\"Event\":{");
            sb.Append($"\"Time\":{kf.time:F3},");
            sb.Append("\"Type\":\"HapticTransient\",");
            sb.Append("\"Parameters\":[");
            sb.Append($"{{\"ParameterID\":\"HapticIntensity\",\"ParameterValue\":{kf.leftIntensity:F2}}},");
            sb.Append($"{{\"ParameterID\":\"HapticSharpness\",\"ParameterValue\":{kf.sharpness:F2}}}");
            sb.Append("]}}");
        }
        
        sb.Append("]}");
        return sb.ToString();
    }
    
    public void SetContinuousIntensity(float intensity)

// Android 触觉适配器
public class AndroidHapticProvider : IHapticProvider
{
    public bool IsSupported => Application.platform == RuntimePlatform.Android;
    
    public void PlayImpact(HapticImpactStyle style)
    {
        long duration = style switch
        {
            HapticImpactStyle.Light => 10,
            HapticImpactStyle.Medium => 30,
            HapticImpactStyle.Heavy => 60,
            HapticImpactStyle.Rigid => 50,
            HapticImpactStyle.Soft => 20,
            _ => 25
        };
        AndroidHapticEngine.Vibrate(duration);
    }
    
    public void PlayNotification(HapticNotificationType type)
    {
        switch (type)
        {
            case HapticNotificationType.Success:
                AndroidHapticEngine.VibratePattern(new long[] { 0, 20, 60, 30 });
                break;
            case HapticNotificationType.Warning:
                AndroidHapticEngine.VibratePattern(new long[] { 0, 50, 50, 50 });
                break;
            case HapticNotificationType.Error:
                AndroidHapticEngine.VibratePattern(new long[] { 0, 80, 40, 80, 40, 80 });
                break;
        }
    }
    
    public void PlaySelection()
    {
        AndroidHapticEngine.Vibrate(8);
    }
    
    public void PlayCustomPattern(HapticPattern pattern)
    {
        var timings = new long[pattern.keyframes.Length * 2];
        var amplitudes = new int[pattern.keyframes.Length * 2];
        
        float lastTime = 0f;
        for (int i = 0; i < pattern.keyframes.Length; i++)
        {
            var kf = pattern.keyframes[i];
            float gap = kf.time - lastTime;
            
            timings[i * 2] = (long)(gap * 1000f);
            amplitudes[i * 2] = 0;  // 静音间隔
            
            timings[i * 2 + 1] = 30;  // 振动持续时间
            amplitudes[i * 2 + 1] = (int)(kf.leftIntensity * 255);
            
            lastTime = kf.time + 0.03f;
        }
        
        AndroidHapticEngine.VibrateWaveform(timings, amplitudes);
    }
    
    public void SetContinuousIntensity(float intensity)
    {
        long[] timings = { 0, 50 };
        int[] amplitudes = { 0, (int)(intensity * 255) };
        AndroidHapticEngine.VibrateWaveform(timings, amplitudes, 0);  // 循环
    }
    
    public void Stop()
    {
        AndroidHapticEngine.Cancel();
    }
}

// 手柄触觉适配器（Xbox / DualShock 4 / DualSense 通用）
public class GamepadHapticProvider : IHapticProvider
{
    private Coroutine currentPattern;
    private MonoBehaviour host;
    
    public bool IsSupported => UnityEngine.InputSystem.Gamepad.current != null;
    
    public GamepadHapticProvider(MonoBehaviour host)
    {
        this.host = host;
    }
    
    public void PlayImpact(HapticImpactStyle style)
    {
        if (currentPattern != null) host.StopCoroutine(currentPattern);
        currentPattern = host.StartCoroutine(ImpactRoutine(style));
    }
    
    private System.Collections.IEnumerator ImpactRoutine(HapticImpactStyle style)
    {
        var gamepad = UnityEngine.InputSystem.Gamepad.current;
        if (gamepad == null) yield break;
        
        (float left, float right, float duration) = style switch
        {
            HapticImpactStyle.Light  => (0.2f, 0.15f, 0.05f),
            HapticImpactStyle.Medium => (0.5f, 0.4f,  0.08f),
            HapticImpactStyle.Heavy  => (1.0f, 0.9f,  0.12f),
            HapticImpactStyle.Rigid  => (0.8f, 0.8f,  0.06f),
            HapticImpactStyle.Soft   => (0.2f, 0.3f,  0.15f),
            _ => (0.5f, 0.4f, 0.08f)
        };
        
        gamepad.SetMotorSpeeds(left, right);
        yield return new WaitForSeconds(duration);
        
        // 淡出
        gamepad.SetMotorSpeeds(left * 0.3f, right * 0.3f);
        yield return new WaitForSeconds(duration * 0.5f);
        gamepad.SetMotorSpeeds(0f, 0f);
    }
    
    public void PlayNotification(HapticNotificationType type)
    {
        if (currentPattern != null) host.StopCoroutine(currentPattern);
        currentPattern = host.StartCoroutine(NotificationRoutine(type));
    }
    
    private System.Collections.IEnumerator NotificationRoutine(HapticNotificationType type)
    {
        var gamepad = UnityEngine.InputSystem.Gamepad.current;
        if (gamepad == null) yield break;
        
        if (type == HapticNotificationType.Success)
        {
            gamepad.SetMotorSpeeds(0.3f, 0.6f);
            yield return new WaitForSeconds(0.06f);
            gamepad.SetMotorSpeeds(0.1f, 0.8f);
            yield return new WaitForSeconds(0.1f);
        }
        else if (type == HapticNotificationType.Error)
        {
            for (int i = 0; i < 3; i++)
            {
                gamepad.SetMotorSpeeds(0.8f, 0.8f);
                yield return new WaitForSeconds(0.06f);
                gamepad.SetMotorSpeeds(0f, 0f);
                yield return new WaitForSeconds(0.05f);
            }
        }
        
        gamepad.SetMotorSpeeds(0f, 0f);
    }
    
    public void PlaySelection() => PlayImpact(HapticImpactStyle.Light);
    
    public void PlayCustomPattern(HapticPattern pattern)
    {
        if (currentPattern != null) host.StopCoroutine(currentPattern);
        currentPattern = host.StartCoroutine(CustomPatternRoutine(pattern));
    }
    
    private System.Collections.IEnumerator CustomPatternRoutine(HapticPattern pattern)
    {
        var gamepad = UnityEngine.InputSystem.Gamepad.current;
        if (gamepad == null) yield break;
        
        float startTime = Time.time;
        int keyframeIdx = 0;
        
        while (keyframeIdx < pattern.keyframes.Length)
        {
            float elapsed = Time.time - startTime;
            var kf = pattern.keyframes[keyframeIdx];
            
            if (elapsed >= kf.time)
            {
                gamepad.SetMotorSpeeds(kf.leftIntensity, kf.rightIntensity);
                keyframeIdx++;
            }
            
            yield return null;
        }
        
        gamepad.SetMotorSpeeds(0f, 0f);
    }
    
    public void SetContinuousIntensity(float intensity)
    {
        UnityEngine.InputSystem.Gamepad.current?.SetMotorSpeeds(intensity, intensity * 0.8f);
    }
    
    public void Stop()
    {
        UnityEngine.InputSystem.Gamepad.current?.SetMotorSpeeds(0f, 0f);
    }
}
```

```csharp
using UnityEngine;
using System.Collections.Generic;

public class HapticManager : MonoBehaviour

    public static HapticManager Instance
    {

        {

            {

                DontDestroyOnLoad(go);
                _instance = go.AddComponent<HapticManager>();
            }
            return _instance;
        }
    }
    
    [Header("全局设置")]
    [SerializeField] private bool hapticsEnabled = true;
    [SerializeField, Range(0f, 1f)] private float globalIntensityMultiplier = 1f;
    
    private List<IHapticProvider> providers = new List<IHapticProvider>();
    private IHapticProvider activeProvider;
    
    private void Awake()
    {
        if (_instance != null && _instance != this)
        {
            Destroy(gameObject);
            return;
        }
        _instance = this;
        DontDestroyOnLoad(gameObject);
        
        InitializeProviders();
    }
    
    private void InitializeProviders()
    {
        // 注册所有平台提供者（优先级：手柄 > iOS > Android）
        providers.Add(new GamepadHapticProvider(this));
        providers.Add(new iOSHapticProvider());
        providers.Add(new AndroidHapticProvider());
        
        // 选择当前平台最优提供者
        foreach (var provider in providers)
        {
            if (provider.IsSupported)
            {
                activeProvider = provider;
                Debug.Log($"[Haptic] 使用触觉提供者: {provider.GetType().Name}");
                break;
            }
        }
    }
    
    // 便捷接口
    public void Impact(HapticImpactStyle style = HapticImpactStyle.Medium)
    {
        if (!hapticsEnabled || activeProvider == null) return;
        activeProvider.PlayImpact(style);
    }
    
    public void Notify(HapticNotificationType type = HapticNotificationType.Success)
    {
        if (!hapticsEnabled || activeProvider == null) return;
        activeProvider.PlayNotification(type);
    }
    
    public void Selection()
    {
        if (!hapticsEnabled || activeProvider == null) return;
        activeProvider.PlaySelection();
    }
    
    public void PlayPattern(HapticPattern pattern)
    {
        if (!hapticsEnabled || activeProvider == null) return;
        activeProvider.PlayCustomPattern(pattern);
    }

    {
        activeProvider?.Stop();
    }
```

## 六、最佳实践总结

### 6.1 触觉设计原则

1. **克制原则**：并非所有操作都需要触觉反馈，过度使用会让玩家麻木甚至关闭触觉
2. **一致性原则**：相同类型的操作（如 UI 点击）全游戏使用一致的触觉模式
3. **区分原则**：用触觉强度和模式区分不同的游戏事件（轻触 ≠ 爆炸）
4. **可关闭原则**：始终提供触觉强度设置（0~100%）及完全关闭选项
5. **平台适配原则**：不同平台的触觉能力差异巨大，必须优雅降级

### 6.2 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| iOS 触觉不工作 | 低电量模式/静音开关 | 检测 `CHHapticEngine` 可用性，静默失败 |
| Android 振动太强/太弱 | 设备马达差异 | 提供百分比调节，乘以 globalIntensityMultiplier |
| 手柄振动一直不停 | 忘记在 OnDestroy 中 Stop | 始终在 OnDestroy 调用 `SetMotorSpeeds(0,0)` |
| 不同帧率下触觉时序偏移 | 使用 Update 计时不准 | 用 `Time.realtimeSinceStartup` 或协程 WaitForSeconds |
| 主机认证触觉 API 限制 | 平台合规要求 | 参考各平台 SDK TRC/TCR 文档 |

### 6.3 性能开销参考

| 操作 | CPU 开销 | 适用频率 |
|------|---------|---------|
| iOS CoreHaptics 初始化 | ~2ms | 仅启动时一次 |
| 播放单次瞬态触觉 | ~0.1ms | 每次操作 |
| Android Vibrator.vibrate | ~0.5ms | 每次操作 |
| Gamepad.SetMotorSpeeds | ~0.05ms | 可每帧调用 |
| 自定义 AHAP Pattern 解析 | ~1ms | 预加载，运行时使用缓存 |

## 结语

触觉反馈系统是提升游戏体验的重要"隐形"设计要素。通过统一的跨平台抽象层，配合细致的触觉模式设计库，可以在 iOS、Android 和主机平台提供一致且沉浸的触觉体验。随着 PS5 DualSense 的成功，以及 Apple Vision Pro 空间触觉的出现，触觉反馈在游戏开发中的地位将越来越重要。
