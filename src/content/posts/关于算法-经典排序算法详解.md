## 快速排序实现原理

> 快速排序（Quick Sort）是一种常用的排序算法，它基于分治的思想，通过将一个无序的序列分割成两个子序列，并递归地对子序列进行排序，最终完成整个序列的排序。

### 其基本思路如下：

1. 选择数组中的一个元素作为基准（pivot）。
2. 将数组中小于等于基准的元素放在基准的左边，将大于基准的元素放在基准的右边。
3. 对基准左右两边的子数组分别重复步骤1和步骤2，直到子数组的大小为1或0（递归结束）。

### 自己整理的思路

快排的思路。首先是对于一个无序的数组，我先选取一个参考，然后将遍历剩下的元素，将数组分成两个部分，一部分比参考点小，剩下一部分比参考点大，然后再以同样的思路对切割后的段进行排序，最后递归结束的标志就是当你要排的段，只剩一个元素的时候就排结束递归了。

手搓一遍就是

```csharp
List<int> testLst = new List<int> {2,3,1,8,6,4,7,5};
Sort.QuickSort(testLst);

public class Sort
{
	public static void QuickSort(List<int> lst)
	{
		RealSort(lst, 0, lst.Count-1);
	}
	public static void RealSort(List<int> lst, int low, int high)
	{
		if(low >= high)
		{
			return;
		}
		int q = Partition(lst, low, high);
		RealSort(lst, low, q-1);
		RealSort(lst, q+1, high);
	}
	
	public static int Partition(List<int> lst, int low, int high)
	{
		int pivot = lst[high];
		int i = low;
		for(int j = low; j < high; j++)
		{
			if(lst[j] <= pivot)
			{
				Swap(lst, i, j);
				i++;
			}
		}
		Swap(lst, i, high);
		return i;
	}
	
	public static void Swap(List<int> lst, int i, int j)
	{
		int temp = lst[j];
		lst[j] = lst[i];
		lst[i]= temp;
	}
}
```
