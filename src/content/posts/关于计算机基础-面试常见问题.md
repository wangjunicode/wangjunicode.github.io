## 进制与内存

### 十六进制占内存

一个字节是 8 位，二进制范围 `00000000~11111111`（0~255）：
- **1 字节 = 8 位 = 2 个十六进制字符**
- 因为 `FF = 1111 1111 = 255`

| 进制 | 表示范围（1字节）| 举例 |
|------|----------------|------|
| 二进制 | 8 位 | `11001010` |
| 十六进制 | 2 位 | `CA` |
| 十进制 | 3 位 | `202` |

### 进制转换

```csharp
// C# 中的进制表示
int a = 0xFF;    // 十六进制：255
int b = 0b1010;  // 二进制：10
int c = 255;     // 十进制

// 十进制 → 十六进制
string hex = 255.ToString("X2");    // "FF"
string hex2 = Convert.ToString(255, 16).ToUpper();  // "FF"

// 十六进制 → 十进制
int value = Convert.ToInt32("FF", 16);  // 255
int value2 = int.Parse("FF", System.Globalization.NumberStyles.HexNumber);  // 255

// 十进制 → 二进制
string bin = Convert.ToString(255, 2);  // "11111111"
```

## 位运算

位运算是游戏开发中常用的高性能操作：

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&` | 按位与 | `6 & 3` | `2`（`110 & 011 = 010`）|
| `\|` | 按位或 | `6 \| 3` | `7`（`110 \| 011 = 111`）|
| `^` | 按位异或 | `6 ^ 3` | `5`（`110 ^ 011 = 101`）|
| `~` | 按位取反 | `~6` | `-7` |
| `<<` | 左移 | `1 << 3` | `8`（相当于 × 2³）|
| `>>` | 右移 | `8 >> 1` | `4`（相当于 ÷ 2）|

### 位运算在游戏中的应用

```csharp
// 1. 使用位掩码表示状态（Layer、状态标志）
[Flags]
public enum CharacterState
{
    None       = 0,
    Moving     = 1 << 0,  // 1
    Attacking  = 1 << 1,  // 2
    Defending  = 1 << 2,  // 4
    Stunned    = 1 << 3,  // 8
}

var state = CharacterState.Moving | CharacterState.Attacking;
bool isMoving = (state & CharacterState.Moving) != 0;  // true

// 2. 快速判断 2 的幂
bool IsPowerOfTwo(int n) => n > 0 && (n & (n - 1)) == 0;

// 3. 交换两个整数（不用临时变量）
a ^= b;
b ^= a;
a ^= b;

// 4. Unity Layer 掩码
int layerMask = LayerMask.GetMask("Ground", "Wall");
Physics.Raycast(ray, 100f, layerMask);
```

## 整数存储原理

### 原码、反码、补码

计算机使用**补码**存储有符号整数：

| 表示 | 说明 | +7 | -7 |
|------|------|----|----|
| 原码 | 符号位+绝对值 | `0111` | `1111` |
| 反码 | 原码各位取反（符号位不变）| `0111` | `1000` |
| 补码 | 反码+1 | `0111` | `1001` |

**为什么用补码**？减法可以转换为加法：`7 - 3 = 7 + (-3)`，CPU 只需要加法器即可。

### 整数溢出

```csharp
// int 的范围：-2^31 ~ 2^31-1 = -2,147,483,648 ~ 2,147,483,647
int maxInt = int.MaxValue;  // 2,147,483,647
int overflow = maxInt + 1;  // -2,147,483,648（溢出！）

// 面试题：两个正整数相加可能溢出
int a = 2_000_000_000;
int b = 2_000_000_000;
int sum = a + b;  // 溢出，结果为负数！

// 安全做法
long safeSum = (long)a + b;  // 用 long 避免溢出
```

## 字符编码

| 编码 | 特点 | 中文占字节 |
|------|------|-----------|
| ASCII | 7位，仅英文 | 不支持 |
| GB2312/GBK | 中文编码 | 2 字节 |
| Unicode | 统一字符集，仅规范 | — |
| UTF-8 | Unicode 可变长实现 | 3 字节 |
| UTF-16 | Unicode 固定 2/4 字节 | 2 字节 |

```csharp
// C# 字符串默认使用 UTF-16 编码
string s = "你好";
byte[] utf8Bytes = Encoding.UTF8.GetBytes(s);   // 6 字节（每个中文 3 字节）
byte[] utf16Bytes = Encoding.Unicode.GetBytes(s); // 4 字节（每个中文 2 字节）
```

## 大端与小端

**大端（Big Endian）**：高字节存低地址  
**小端（Little Endian）**：低字节存低地址（x86/x64 CPU 使用）

```csharp
// 检查当前系统字节序
bool isLittleEndian = BitConverter.IsLittleEndian;  // true（x86）

// 网络字节序是大端，需要转换
int networkOrder = IPAddress.HostToNetworkOrder(localValue);
```
