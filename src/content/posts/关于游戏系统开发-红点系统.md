## 红点系统概述

红点系统是游戏中常见的消息提示 UI 组件，用于提示玩家有未处理的内容（新消息、新道具、可领取奖励等）。

**设计目标**：
- 数据驱动，叶节点变化自动传播到父节点
- 低耦合，业务模块只需修改对应叶节点
- 高性能，变更时只刷新受影响的节点

## 三层架构

系统分为三个独立层面：

```
┌─────────────────────────────────┐
│        表现层（View Layer）       │  UI 红点显示/隐藏
├─────────────────────────────────┤
│        驱动层（Logic Layer）      │  节点变更通知传播
├─────────────────────────────────┤
│        结构层（Data Layer）       │  树形结构 + 红点数据
└─────────────────────────────────┘
```

## 树形结构设计

将游戏中的各系统分层构造成一棵树，每个红点作为一个节点，最终提供数据的节点为叶子节点：

```
Root（根）
├── Main（主界面）
│   ├── Bag（背包）        ← 叶节点，有新道具时=1
│   ├── Quest（任务）      ← 叶节点，有新任务时=N
│   └── Mail（邮件）       ← 叶节点，有新邮件时=N
├── Shop（商城）
│   ├── Daily（日常商店）  ← 叶节点
│   └── Limited（限时商店）← 叶节点
└── Activity（活动）       ← 汇总子节点
```

## 核心实现

```csharp
public class RedDotNode
{
    public string Key { get; private set; }
    public int Count { get; private set; }
    
    private RedDotNode _parent;
    private readonly List<RedDotNode> _children = new();
    
    // 变更回调（用于通知表现层）
    public event Action<int> OnCountChanged;

    public RedDotNode(string key)
    {
        Key = key;
    }

    public void AddChild(RedDotNode child)
    {
        child._parent = this;
        _children.Add(child);
    }

    /// <summary>
    /// 设置叶节点的数量（只有叶节点才能主动设置）
    /// </summary>
    public void SetCount(int count)
    {
        if (_children.Count > 0)
        {
            Debug.LogWarning($"[RedDot] {Key} 不是叶节点，不能直接设置 count");
            return;
        }
        
        int oldCount = Count;
        Count = count;
        
        if (oldCount != Count)
        {
            NotifyChange();
        }
    }

    /// <summary>
    /// 向上传播变更
    /// </summary>
    private void NotifyChange()
    {
        OnCountChanged?.Invoke(Count);
        _parent?.RefreshFromChildren();
    }

    /// <summary>
    /// 父节点从子节点汇总数量
    /// </summary>
    private void RefreshFromChildren()
    {
        int total = 0;
        foreach (var child in _children)
        {
            total += child.Count;
        }
        
        int oldCount = Count;
        Count = total;
        
        if (oldCount != Count)
        {
            OnCountChanged?.Invoke(Count);
            _parent?.RefreshFromChildren();
        }
    }
}
```

## 管理器

```csharp
public class RedDotManager
{
    private static RedDotManager _instance;
    public static RedDotManager Instance => _instance ??= new RedDotManager();

    private readonly Dictionary<string, RedDotNode> _nodes = new();

    public void RegisterNode(string key, string parentKey = null)
    {
        var node = new RedDotNode(key);
        _nodes[key] = node;

        if (parentKey != null && _nodes.TryGetValue(parentKey, out var parent))
        {
            parent.AddChild(node);
        }
    }

    public void SetCount(string key, int count)
    {
        if (_nodes.TryGetValue(key, out var node))
        {
            node.SetCount(count);
        }
    }

    public int GetCount(string key)
    {
        return _nodes.TryGetValue(key, out var node) ? node.Count : 0;
    }

    public void Listen(string key, Action<int> callback)
    {
        if (_nodes.TryGetValue(key, out var node))
        {
            node.OnCountChanged += callback;
        }
    }
}
```

## 表现层绑定

```csharp
public class RedDotView : MonoBehaviour
{
    [SerializeField] private string redDotKey;
    [SerializeField] private GameObject dotObject;     // 红点图标
    [SerializeField] private TextMeshProUGUI countText; // 数字

    void OnEnable()
    {
        RedDotManager.Instance.Listen(redDotKey, OnCountChanged);
        // 初始化当前状态
        OnCountChanged(RedDotManager.Instance.GetCount(redDotKey));
    }

    void OnDisable()
    {
        // 注意：实际项目需要支持取消监听
    }

    private void OnCountChanged(int count)
    {
        dotObject.SetActive(count > 0);
        if (countText != null)
        {
            countText.text = count > 99 ? "99+" : count.ToString();
            countText.gameObject.SetActive(count > 1);
        }
    }
}
```

## 使用示例

```csharp
// 初始化红点树结构（通常在游戏启动时）
void InitRedDotTree()
{
    RedDotManager.Instance.RegisterNode("Root");
    RedDotManager.Instance.RegisterNode("Main", "Root");
    RedDotManager.Instance.RegisterNode("Main.Bag", "Main");
    RedDotManager.Instance.RegisterNode("Main.Quest", "Main");
    RedDotManager.Instance.RegisterNode("Main.Mail", "Main");
}

// 业务逻辑中更新红点
void OnNewMailReceived(int mailCount)
{
    RedDotManager.Instance.SetCount("Main.Mail", mailCount);
    // Main 节点和 Root 节点会自动汇总更新
}
```
