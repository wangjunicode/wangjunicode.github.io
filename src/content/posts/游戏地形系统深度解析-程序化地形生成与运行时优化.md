# 游戏地形系统深度解析：程序化地形生成与运行时优化

## 前言

开放世界游戏中，地形系统是最核心也最复杂的技术模块之一。它需要同时解决三个核心矛盾：
- **视觉质量** vs **渲染性能**（多层纹理 vs DrawCall数量）
- **精度细节** vs **内存占用**（高分辨率高度图 vs 内存/显存）
- **实时修改** vs **流式加载**（动态地形 vs 大世界管理）

本文将系统讲解从程序化生成到运行时优化的完整地形技术栈。

---

## 一、地形高度图原理与数据结构

### 1.1 高度图（Heightmap）

地形的核心数据是高度图：一张灰度图，每个像素的灰度值代表该坐标的地面高度。

```
高度图分辨率: 1025×1025（Unity要求 2^N+1）
世界尺寸:     1000m × 1000m
高度范围:     0 ~ 600m

每个像素覆盖: 1000 / 1024 ≈ 0.98m
```

Unity 地形使用 `float[,]` 存储归一化高度（0~1），对应世界高度 = 值 × terrainData.size.y。

### 1.2 地形分块管理器

```csharp
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 地形分块管理器：按玩家位置动态加载/卸载地形块
/// 适用于大型开放世界场景
/// </summary>
public class TerrainChunkManager : MonoBehaviour
{
    [Header("地形配置")]
    public TerrainData terrainDataTemplate;    // 地形数据模板
    public Material terrainMaterial;           // 地形材质
    public int chunkSize = 500;                // 每块地形世界尺寸(m)
    public int viewDistance = 3;               // 可见范围(块数)

    [Header("地形分辨率")]
    public int heightmapResolution = 513;      // 高度图分辨率(2^N+1)
    public int alphamapResolution = 512;       // 混合贴图分辨率
    public int detailResolution = 512;         // 细节分辨率

    private Dictionary<Vector2Int, TerrainChunk> _chunks = new();
    private Transform _viewer;
    private Vector2Int _lastViewerChunk = new Vector2Int(int.MaxValue, int.MaxValue);

    // 地形生成器
    private TerrainGenerator _generator;

    void Start()
    {
        _viewer    = Camera.main?.transform ?? transform;
        _generator = GetComponent<TerrainGenerator>();
        UpdateVisibleChunks();
    }

    void Update()
    {
        Vector2Int viewerChunk = WorldToChunkCoord(_viewer.position);
        if (viewerChunk != _lastViewerChunk)
        {
            _lastViewerChunk = viewerChunk;
            UpdateVisibleChunks();
        }
    }

    void UpdateVisibleChunks()
    {
        // 标记所有块为不可见
        foreach (var chunk in _chunks.Values)
            chunk.SetVisible(false);

        // 激活视野内的块
        for (int x = -viewDistance; x <= viewDistance; x++)
        {
            for (int z = -viewDistance; z <= viewDistance; z++)
            {
                Vector2Int coord = _lastViewerChunk + new Vector2Int(x, z);

                if (!_chunks.TryGetValue(coord, out TerrainChunk chunk))
                {
                    // 异步创建新地形块
                    chunk = new TerrainChunk(coord, chunkSize, heightmapResolution,
                        alphamapResolution, terrainMaterial, transform, _generator);
                    _chunks[coord] = chunk;
                }

                chunk.SetVisible(true);
            }
        }
    }

    Vector2Int WorldToChunkCoord(Vector3 worldPos)
    {
        return new Vector2Int(
            Mathf.FloorToInt(worldPos.x / chunkSize),
            Mathf.FloorToInt(worldPos.z / chunkSize));
    }

    void OnDestroy()
    {
        foreach (var chunk in _chunks.Values)
            chunk.Destroy();
    }
}

/// <summary>
/// 单个地形块
/// </summary>
public class TerrainChunk
{
    public Terrain terrain;
    private GameObject _gameObject;
    private Vector2Int _coord;
    private int _chunkSize;
    private TerrainGenerator _generator;

    public TerrainChunk(Vector2Int coord, int size, int heightmapRes, int alphamapRes,
        Material material, Transform parent, TerrainGenerator generator)
    {
        _coord     = coord;
        _chunkSize = size;
        _generator = generator;

        // 创建 Terrain GameObject
        _gameObject = new GameObject($"Terrain_{coord.x}_{coord.y}");
        _gameObject.transform.SetParent(parent);
        _gameObject.transform.position = new Vector3(
            coord.x * size, 0f, coord.y * size);

        // 创建 TerrainData
        var terrainData = new TerrainData();
        terrainData.heightmapResolution = heightmapRes;
        terrainData.size = new Vector3(size, 600f, size);
        terrainData.alphamapResolution = alphamapRes;

        // 生成高度图
        float[,] heights = generator.GenerateHeightmap(
            coord.x * size, coord.y * size, heightmapRes, size);
        terrainData.SetHeights(0, 0, heights);

        // 生成 Alpha 混合图
        float[,,] alphamap = generator.GenerateAlphamap(
            coord.x * size, coord.y * size, heights, alphamapRes);
        terrainData.SetAlphamaps(0, 0, alphamap);

        // 添加 Terrain 组件
        terrain = _gameObject.AddComponent<Terrain>();
        terrain.terrainData = terrainData;
        terrain.materialTemplate = material;
        terrain.heightmapPixelError = 8f; // LOD 精度

        // 添加碰撞体
        var collider = _gameObject.AddComponent<TerrainCollider>();
        collider.terrainData = terrainData;
    }

    public void SetVisible(bool visible) => _gameObject.SetActive(visible);

    public void Destroy()
    {
        if (_gameObject != null)
            Object.Destroy(_gameObject);
    }
}
```

## 二、程序化地形生成：噪声函数详解

### 2.1 Perlin Noise 基础

```csharp
using UnityEngine;

/// <summary>
/// 地形高度图生成器
/// 支持多层叠加噪声（FBM：分形布朗运动）
/// </summary>
public class TerrainGenerator : MonoBehaviour
{
    [Header("基础设置")]
    public int seed = 42;
    public float globalScale = 0.003f;   // 整体缩放（越小越平缓）

    [Header("FBM 参数")]
    public int octaves = 6;               // 叠加层数（越多细节越丰富）
    [Range(0f, 1f)]
    public float persistence = 0.5f;      // 每层振幅衰减率
    public float lacunarity = 2f;         // 每层频率倍增率

    [Header("地形分层")]
    public AnimationCurve heightCurve;    // 高度重映射曲线（塑造山峰/平原）
    public float mountainThreshold = 0.6f;
    public float waterLevel = 0.1f;

    private Vector2 _offset; // 由种子决定的随机偏移

    void Awake()
    {
        Random.InitState(seed);
        _offset = new Vector2(Random.Range(-10000f, 10000f),
                              Random.Range(-10000f, 10000f));
    }

    /// <summary>
    /// 生成高度图（归一化 0~1）
    /// </summary>
    public float[,] GenerateHeightmap(float worldX, float worldZ,
        int resolution, int chunkSize)
    {
        float[,] heights = new float[resolution, resolution];
        float cellSize = (float)chunkSize / (resolution - 1);

        for (int z = 0; z < resolution; z++)
        {
            for (int x = 0; x < resolution; x++)
            {
                float wx = worldX + x * cellSize;
                float wz = worldZ + z * cellSize;

                float h = SampleFBM(wx, wz);
                heights[z, x] = h;
            }
        }

        return heights;
    }

    /// <summary>
    /// FBM（分形布朗运动）噪声采样
    /// </summary>
    float SampleFBM(float wx, float wz)
    {
        float value = 0f;
        float amplitude = 1f;
        float frequency = globalScale;
        float maxValue = 0f;

        for (int i = 0; i < octaves; i++)
        {
            float nx = wx * frequency + _offset.x;
            float nz = wz * frequency + _offset.y;

            // Unity 内置 Perlin Noise（范围 0~1）
            float perlin = Mathf.PerlinNoise(nx, nz);
            // 映射到 -1~1 使其有正有负，增加起伏感
            value += (perlin * 2f - 1f) * amplitude;

            maxValue += amplitude;
            amplitude *= persistence;
            frequency *= lacunarity;
        }

        // 归一化到 0~1
        float normalized = (value / maxValue + 1f) * 0.5f;

        // 通过曲线重映射（制造山峰效果）
        return heightCurve != null
            ? Mathf.Clamp01(heightCurve.Evaluate(normalized))
            : Mathf.Clamp01(normalized);
    }

    /// <summary>
    /// 域翘曲（Domain Warping）：让地形更自然，避免网格感
    /// </summary>
    float SampleDomainWarped(float wx, float wz)
    {
        // 第一次采样得到偏移向量
        float warpX = SampleFBM(wx + 1.7f, wz + 9.2f) * 2f - 1f;
        float warpZ = SampleFBM(wx + 8.3f, wz + 2.8f) * 2f - 1f;

        float warpStrength = 50f;
        // 第二次采样使用偏移后的坐标
        return SampleFBM(wx + warpX * warpStrength, wz + warpZ * warpStrength);
    }

    /// <summary>
    /// 生成 Alpha 混合贴图（根据高度和坡度决定纹理）
    /// layers: [0]=草地, [1]=泥土, [2]=岩石, [3]=雪地
    /// </summary>
    public float[,,] GenerateAlphamap(float worldX, float worldZ,
        float[,] heights, int resolution)
    {
        int layerCount = 4; // 需要与 TerrainData.terrainLayers 数量一致
        float[,,] alphamap = new float[resolution, resolution, layerCount];

        for (int z = 0; z < resolution; z++)
        {
            for (int x = 0; x < resolution; x++)
            {
                // 将 alphamap 坐标映射到 heightmap 坐标
                int hx = Mathf.RoundToInt((float)x / resolution * (heights.GetLength(1) - 1));
                int hz = Mathf.RoundToInt((float)z / resolution * (heights.GetLength(0) - 1));
                float height = heights[hz, hx];

                // 计算坡度（使用相邻像素高度差估算）
                float slope = CalculateSlope(heights, hz, hx);

                // 按高度和坡度分配层权重
                float grass = 0f, dirt = 0f, rock = 0f, snow = 0f;

                if (height < waterLevel + 0.05f)
                {
                    // 水边/沙滩：泥土为主
                    dirt  = 1f;
                }
                else if (slope > 0.5f)
                {
                    // 陡坡：岩石
                    rock = 1f;
                    dirt = Mathf.Clamp01((0.7f - slope) * 5f);
                    rock = 1f - dirt;
                }
                else if (height > mountainThreshold)
                {
                    // 高山：雪地
                    float snowBlend = Mathf.InverseLerp(mountainThreshold, mountainThreshold + 0.1f, height);
                    snow = snowBlend;
                    rock = 1f - snowBlend;
                }
                else
                {
                    // 中等高度：草地+泥土混合
                    float dirtBlend = Mathf.Clamp01(slope * 3f);
                    grass = 1f - dirtBlend;
                    dirt  = dirtBlend;
                }

                alphamap[z, x, 0] = grass;
                alphamap[z, x, 1] = dirt;
                alphamap[z, x, 2] = rock;
                alphamap[z, x, 3] = snow;
            }
        }

        return alphamap;
    }

    float CalculateSlope(float[,] heights, int z, int x)
    {
        int rows = heights.GetLength(0);
        int cols = heights.GetLength(1);
        float h = heights[z, x];

        float dz = 0f, dx = 0f;
        if (z > 0 && z < rows - 1)
            dz = (heights[z + 1, x] - heights[z - 1, x]) * 0.5f;
        if (x > 0 && x < cols - 1)
            dx = (heights[z, x + 1] - heights[z, x - 1]) * 0.5f;

        return Mathf.Sqrt(dx * dx + dz * dz);
    }
}
```

### 2.2 高级：侵蚀模拟

水力侵蚀让程序化地形更自然，模拟雨水冲刷形成的山谷和沟壑：

```csharp
public class HydraulicErosion
{
    struct ErosionParams
    {
        public int   dropletCount;       // 水滴数量
        public int   maxSteps;           // 最大步数
        public float inertia;            // 惯性（水流方向延续性）
        public float capacity;           // 最大携沙量
        public float deposition;         // 沉积速率
        public float erosion;            // 侵蚀速率
        public float evaporation;        // 蒸发速率
        public float gravity;            // 重力
        public int   radius;             // 影响半径
    }

    public static float[,] Erode(float[,] heightmap, ErosionParams p)
    {
        int size = heightmap.GetLength(0);
        float[,] h = (float[,])heightmap.Clone();
        System.Random rng = new System.Random(42);

        for (int drop = 0; drop < p.dropletCount; drop++)
        {
            // 随机生成水滴起始位置
            float posX = (float)rng.NextDouble() * (size - 2);
            float posY = (float)rng.NextDouble() * (size - 2);
            float dirX = 0f, dirY = 0f;
            float speed = 1f, water = 1f, sediment = 0f;

            for (int step = 0; step < p.maxSteps; step++)
            {
                int nodeX = Mathf.FloorToInt(posX);
                int nodeY = Mathf.FloorToInt(posY);
                float cellOffsetX = posX - nodeX;
                float cellOffsetY = posY - nodeY;

                // 双线性插值获取当前高度和梯度
                float heightNW = h[nodeY,     nodeX    ];
                float heightNE = h[nodeY,     nodeX + 1];
                float heightSW = h[nodeY + 1, nodeX    ];
                float heightSE = h[nodeY + 1, nodeX + 1];

                float gradX = (heightNE - heightNW) * (1 - cellOffsetY)
                            + (heightSE - heightSW) * cellOffsetY;
                float gradY = (heightSW - heightNW) * (1 - cellOffsetX)
                            + (heightSE - heightNE) * cellOffsetX;

                // 更新方向（惯性 + 重力梯度）
                dirX = dirX * p.inertia - gradX * (1 - p.inertia);
                dirY = dirY * p.inertia - gradY * (1 - p.inertia);

                float len = Mathf.Sqrt(dirX * dirX + dirY * dirY);
                if (len <= 1e-6f) break;
                dirX /= len; dirY /= len;

                posX += dirX; posY += dirY;

                // 越界检测
                if (posX < 1 || posX >= size - 1 || posY < 1 || posY >= size - 1)
                    break;

                // 计算高度变化（下坡则侵蚀，上坡则沉积）
                float currentHeight = BilinearSample(h, posX, posY);
                float prevHeight    = BilinearSample(h, posX - dirX, posY - dirY);
                float deltaHeight   = currentHeight - prevHeight;

                float sedimentCapacity = Mathf.Max(-deltaHeight * speed * water * p.capacity, 0.001f);

                if (sediment > sedimentCapacity || deltaHeight > 0f)
                {
                    // 沉积
                    float depositAmount = deltaHeight > 0f
                        ? Mathf.Min(deltaHeight, sediment)
                        : (sediment - sedimentCapacity) * p.deposition;

                    sediment -= depositAmount;
                    DepositAt(h, posX - dirX, posY - dirY, depositAmount);
                }
                else
                {
                    // 侵蚀
                    float erodeAmount = Mathf.Min(
                        (sedimentCapacity - sediment) * p.erosion, -deltaHeight);
                    erodeAmount = Mathf.Max(0f, erodeAmount);

                    sediment += erodeAmount;
                    ErodeAt(h, posX, posY, erodeAmount, p.radius, size);
                }

                speed = Mathf.Sqrt(Mathf.Max(0f, speed * speed + deltaHeight * p.gravity));
                water *= (1f - p.evaporation);

                if (water < 0.01f) break;
            }
        }

        return h;
    }

    static float BilinearSample(float[,] h, float x, float y)
    {
        int ix = Mathf.FloorToInt(x), iy = Mathf.FloorToInt(y);
        float fx = x - ix, fy = y - iy;
        int size = h.GetLength(0);
        ix = Mathf.Clamp(ix, 0, size - 2);
        iy = Mathf.Clamp(iy, 0, size - 2);

        return h[iy, ix]     * (1-fx) * (1-fy)
             + h[iy, ix+1]   * fx     * (1-fy)
             + h[iy+1, ix]   * (1-fx) * fy
             + h[iy+1, ix+1] * fx     * fy;
    }

    static void DepositAt(float[,] h, float x, float y, float amount)
    {
        int ix = Mathf.FloorToInt(x), iy = Mathf.FloorToInt(y);
        float fx = x - ix, fy = y - iy;
        int size = h.GetLength(0);
        ix = Mathf.Clamp(ix, 0, size - 2);
        iy = Mathf.Clamp(iy, 0, size - 2);

        h[iy,   ix  ] += amount * (1-fx) * (1-fy);
        h[iy,   ix+1] += amount * fx     * (1-fy);
        h[iy+1, ix  ] += amount * (1-fx) * fy;
        h[iy+1, ix+1] += amount * fx     * fy;
    }

    static void ErodeAt(float[,] h, float x, float y, float amount, int radius, int size)
    {
        // 在影响半径内按距离权重侵蚀
        float totalWeight = 0f;
        for (int dy = -radius; dy <= radius; dy++)
            for (int dx = -radius; dx <= radius; dx++)
            {
                float dist = Mathf.Sqrt(dx*dx + dy*dy);
                if (dist < radius)
                    totalWeight += (radius - dist);
            }

        for (int dy = -radius; dy <= radius; dy++)
        {
            for (int dx = -radius; dx <= radius; dx++)
            {
                int nx = Mathf.RoundToInt(x) + dx;
                int ny = Mathf.RoundToInt(y) + dy;
                if (nx < 0 || nx >= size || ny < 0 || ny >= size) continue;

                float dist = Mathf.Sqrt(dx*dx + dy*dy);
                if (dist < radius)
                {
                    float w = (radius - dist) / totalWeight;
                    h[ny, nx] = Mathf.Max(0f, h[ny, nx] - amount * w);
                }
            }
        }
    }
}
```

## 三、地形纹理混合 Shader

### 3.1 基于高度/坡度的四层混合 Shader（URP）

```hlsl
Shader "Custom/TerrainBlend"
{
    Properties
    {
        // 四层纹理
        _GrassAlbedo   ("Grass Albedo",  2D) = "white" {}
        _DirtAlbedo    ("Dirt Albedo",   2D) = "white" {}
        _RockAlbedo    ("Rock Albedo",   2D) = "grey"  {}
        _SnowAlbedo    ("Snow Albedo",   2D) = "white" {}

        _GrassNormal   ("Grass Normal",  2D) = "bump"  {}
        _DirtNormal    ("Dirt Normal",   2D) = "bump"  {}
        _RockNormal    ("Rock Normal",   2D) = "bump"  {}
        _SnowNormal    ("Snow Normal",   2D) = "bump"  {}

        _TextureTiling ("Texture Tiling", Float) = 50.0

        // 混合控制（也可从 SplatMap 读取）
        _SnowHeight    ("Snow Start Height", Range(0,1)) = 0.7
        _SnowBlend     ("Snow Blend Width",  Range(0,0.2)) = 0.05
    }

    SubShader
    {
        Tags { "RenderType"="Opaque" "RenderPipeline"="UniversalRenderPipeline" }

        Pass
        {
            Name "ForwardLit"
            Tags { "LightMode" = "UniversalForward" }

            HLSLPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile _ _MAIN_LIGHT_SHADOWS
            #pragma multi_compile _ _SHADOWS_SOFT

            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Lighting.hlsl"

            TEXTURE2D(_GrassAlbedo);  SAMPLER(sampler_GrassAlbedo);
            TEXTURE2D(_DirtAlbedo);   SAMPLER(sampler_DirtAlbedo);
            TEXTURE2D(_RockAlbedo);   SAMPLER(sampler_RockAlbedo);
            TEXTURE2D(_SnowAlbedo);   SAMPLER(sampler_SnowAlbedo);
            TEXTURE2D(_GrassNormal);  SAMPLER(sampler_GrassNormal);
            TEXTURE2D(_RockNormal);   SAMPLER(sampler_RockNormal);

            // SplatMap：R=草地, G=泥土, B=岩石, A=雪地
            TEXTURE2D(_Control);      SAMPLER(sampler_Control);

            CBUFFER_START(UnityPerMaterial)
                float4 _GrassAlbedo_ST;
                float  _TextureTiling;
                float  _SnowHeight;
                float  _SnowBlend;
            CBUFFER_END

            struct Attributes
            {
                float4 positionOS : POSITION;
                float3 normalOS   : NORMAL;
                float4 tangentOS  : TANGENT;
                float2 uv         : TEXCOORD0;
            };

            struct Varyings
            {
                float4 positionHCS : SV_POSITION;
                float2 uv          : TEXCOORD0;
                float3 positionWS  : TEXCOORD1;
                float3 normalWS    : TEXCOORD2;
                float3 tangentWS   : TEXCOORD3;
                float3 bitangentWS : TEXCOORD4;
            };

            Varyings vert(Attributes IN)
            {
                Varyings OUT;
                OUT.positionWS  = TransformObjectToWorld(IN.positionOS.xyz);
                OUT.positionHCS = TransformWorldToHClip(OUT.positionWS);
                OUT.uv          = IN.uv;

                VertexNormalInputs normalInput = GetVertexNormalInputs(IN.normalOS, IN.tangentOS);
                OUT.normalWS    = normalInput.normalWS;
                OUT.tangentWS   = normalInput.tangentWS;
                OUT.bitangentWS = normalInput.bitangentWS;
                return OUT;
            }

            // 按高度混合（带抗锯齿平滑过渡）
            float4 HeightBlend(float4 a, float ha, float4 b, float hb, float blend)
            {
                // 基于高度的混合：更真实的纹理过渡
                float d = 0.2f;
                float ma = max(ha + (1-blend), hb + blend) - d;
                float b1 = max(ha + (1-blend) - ma, 0);
                float b2 = max(hb + blend     - ma, 0);
                return (a * b1 + b * b2) / (b1 + b2 + 1e-5);
            }

            half4 frag(Varyings IN) : SV_Target
            {
                float2 uv = IN.positionWS.xz / _TextureTiling;

                // 读取 Splatmap 混合权重
                float4 splat = SAMPLE_TEXTURE2D(_Control, sampler_Control, IN.uv);
                float wGrass = splat.r;
                float wDirt  = splat.g;
                float wRock  = splat.b;
                float wSnow  = splat.a;

                // 采样各层纹理
                half4 grassCol = SAMPLE_TEXTURE2D(_GrassAlbedo, sampler_GrassAlbedo, uv);
                half4 dirtCol  = SAMPLE_TEXTURE2D(_DirtAlbedo,  sampler_DirtAlbedo,  uv);
                half4 rockCol  = SAMPLE_TEXTURE2D(_RockAlbedo,  sampler_RockAlbedo,  uv);
                half4 snowCol  = SAMPLE_TEXTURE2D(_SnowAlbedo,  sampler_SnowAlbedo,  uv);

                // 混合颜色
                half4 albedo = grassCol * wGrass
                             + dirtCol  * wDirt
                             + rockCol  * wRock
                             + snowCol  * wSnow;

                // 法线混合
                float3 grassN = UnpackNormal(SAMPLE_TEXTURE2D(_GrassNormal, sampler_GrassNormal, uv));
                float3 rockN  = UnpackNormal(SAMPLE_TEXTURE2D(_RockNormal,  sampler_RockNormal,  uv));
                float3 blendN = normalize(grassN * (wGrass + wDirt + wSnow) + rockN * wRock);

                // 切线空间法线转世界空间
                float3x3 TBN = float3x3(IN.tangentWS, IN.bitangentWS, IN.normalWS);
                float3 normalWS = normalize(mul(blendN, TBN));

                // URP 光照计算
                InputData inputData;
                ZERO_INITIALIZE(InputData, inputData);
                inputData.positionWS     = IN.positionWS;
                inputData.normalWS       = normalWS;
                inputData.viewDirectionWS = GetWorldSpaceNormalizeViewDir(IN.positionWS);
                inputData.shadowCoord    = TransformWorldToShadowCoord(IN.positionWS);

                SurfaceData surfaceData;
                ZERO_INITIALIZE(SurfaceData, surfaceData);
                surfaceData.albedo     = albedo.rgb;
                surfaceData.smoothness = wSnow * 0.3 + wRock * 0.1;
                surfaceData.metallic   = 0;
                surfaceData.alpha      = 1;

                return UniversalFragmentPBR(inputData, surfaceData);
            }
            ENDHLSL
        }
    }
}
```

## 四、植被系统：GPU Instancing 大规模渲染

### 4.1 植被散布器

```csharp
using UnityEngine;
using System.Collections.Generic;

/// <summary>
/// 基于 GPU Instancing 的植被系统
/// 支持数万棵树/草同屏渲染
/// </summary>
public class VegetationSystem : MonoBehaviour
{
    [Header("植被配置")]
    public Mesh grassMesh;
    public Material grassMaterial;    // 需支持 GPU Instancing
    public int maxInstances = 50000;
    public float density = 0.5f;      // 0~1，散布密度

    [Header("生长约束")]
    public float minHeight = 0.05f;   // 最低生长高度（水面以上）
    public float maxSlope = 30f;      // 最大坡度（度）
    public float minScale = 0.8f;
    public float maxScale = 1.5f;

    [Header("LOD 设置")]
    public float drawDistance = 100f;
    public float fadeDistance = 20f;  // 距离 drawDistance 多近开始淡出

    private List<Matrix4x4> _instanceMatrices = new List<Matrix4x4>();
    private Matrix4x4[] _batchBuffer;           // GPU Instancing 批次缓冲（最大1023）
    private const int MAX_BATCH_SIZE = 1023;

    public void GenerateVegetation(Terrain terrain)
    {
        _instanceMatrices.Clear();

        TerrainData td = terrain.terrainData;
        Vector3 terrainPos = terrain.transform.position;
        int resolution = td.heightmapResolution;

        System.Random rng = new System.Random(12345);

        // 按密度随机散布
        int attemptCount = Mathf.RoundToInt(maxInstances / density);
        for (int i = 0; i < attemptCount && _instanceMatrices.Count < maxInstances; i++)
        {
            // 归一化坐标 (0~1)
            float nx = (float)rng.NextDouble();
            float nz = (float)rng.NextDouble();

            // 获取地形高度和法线
            float height = td.GetInterpolatedHeight(nx, nz);
            Vector3 normal = td.GetInterpolatedNormal(nx, nz);

            // 高度过滤：不在水面以下生长
            float normalizedHeight = height / td.size.y;
            if (normalizedHeight < minHeight) continue;

            // 坡度过滤
            float slope = Vector3.Angle(Vector3.up, normal);
            if (slope > maxSlope) continue;

            // 读取 Splatmap 权重（只在草地层生长）
            float[,,] alphamap = td.GetAlphamaps(
                Mathf.RoundToInt(nx * (td.alphamapWidth - 1)), 1,
                Mathf.RoundToInt(nz * (td.alphamapHeight - 1)), 1);
            float grassWeight = alphamap[0, 0, 0]; // Layer 0 = 草地
            if (grassWeight < 0.3f) continue;      // 草地权重不足则跳过

            // 世界坐标
            Vector3 worldPos = terrainPos + new Vector3(
                nx * td.size.x, height, nz * td.size.z);

            // 随机旋转和缩放
            float rotY  = (float)rng.NextDouble() * 360f;
            float scale = Mathf.Lerp(minScale, maxScale, (float)rng.NextDouble());

            // 让草垂直于地面法线（或直接 Y 轴朝上）
            Quaternion rotation = Quaternion.Euler(0f, rotY, 0f);

            _instanceMatrices.Add(Matrix4x4.TRS(worldPos, rotation, Vector3.one * scale));
        }

        Debug.Log($"[VegetationSystem] 生成 {_instanceMatrices.Count} 个植被实例");
    }

    void Update()
    {
        if (_instanceMatrices.Count == 0) return;
        RenderVegetation();
    }

    void RenderVegetation()
    {
        Camera cam = Camera.main;
        if (cam == null) return;

        // 分批渲染（GPU Instancing 每批最多 1023 个）
        int total = _instanceMatrices.Count;
        int batchCount = Mathf.CeilToInt((float)total / MAX_BATCH_SIZE);

        for (int b = 0; b < batchCount; b++)
        {
            int start = b * MAX_BATCH_SIZE;
            int count = Mathf.Min(MAX_BATCH_SIZE, total - start);

            if (_batchBuffer == null || _batchBuffer.Length < count)
                _batchBuffer = new Matrix4x4[count];

            // 填充批次缓冲
            _instanceMatrices.CopyTo(start, _batchBuffer, 0, count);

            Graphics.DrawMeshInstanced(
                grassMesh,
                0,              // submesh index
                grassMaterial,
                _batchBuffer,
                count,
                null,           // MaterialPropertyBlock
                UnityEngine.Rendering.ShadowCastingMode.Off,
                false,          // receiveShadows
                gameObject.layer);
        }
    }
}
```

## 五、地形 LOD 与流式加载

### 5.1 地形流式加载策略

```csharp
using UnityEngine;
using System.Collections;

public class TerrainStreamer : MonoBehaviour
{
    [Header("LOD 设置")]
    public float pixelError_Near   = 5f;    // 近距离精度（误差越小越精细）
    public float pixelError_Medium = 15f;
    public float pixelError_Far    = 50f;

    public float distance_Near   = 200f;
    public float distance_Medium = 500f;

    private Transform _player;

    void Start()
    {
        _player = Camera.main?.transform;
        StartCoroutine(UpdateLOD());
    }

    IEnumerator UpdateLOD()
    {
        while (true)
        {
            yield return new WaitForSeconds(0.5f); // 每0.5s更新一次LOD

            var terrains = FindObjectsOfType<Terrain>();
            foreach (var t in terrains)
            {
                float dist = Vector3.Distance(_player.position, t.transform.position
                    + t.terrainData.size * 0.5f);

                if (dist < distance_Near)
                    t.heightmapPixelError = pixelError_Near;
                else if (dist < distance_Medium)
                    t.heightmapPixelError = pixelError_Medium;
                else
                    t.heightmapPixelError = pixelError_Far;

                // 远距离关闭植被细节
                t.detailObjectDistance = dist < distance_Near ? 80f : 0f;
                t.treeBillboardDistance = Mathf.Lerp(200f, 50f,
                    Mathf.InverseLerp(distance_Near, distance_Medium, dist));
            }
        }
    }
}
```

## 六、运行时地形修改（挖坑/堆叠）

```csharp
public class TerrainModifier : MonoBehaviour
{
    public Terrain terrain;

    /// <summary>
    /// 在世界位置挖一个球形坑
    /// </summary>
    public void Excavate(Vector3 worldPos, float radius, float depth)
    {
        TerrainData td = terrain.terrainData;
        Vector3 terrainPos = terrain.transform.position;

        // 转换为地形 UV 坐标
        float nx = (worldPos.x - terrainPos.x) / td.size.x;
        float nz = (worldPos.z - terrainPos.z) / td.size.z;

        int resolution = td.heightmapResolution;
        int brushRadius = Mathf.RoundToInt(radius / td.size.x * resolution);
        int centerX = Mathf.RoundToInt(nx * (resolution - 1));
        int centerZ = Mathf.RoundToInt(nz * (resolution - 1));

        int x0 = Mathf.Clamp(centerX - brushRadius, 0, resolution - 1);
        int z0 = Mathf.Clamp(centerZ - brushRadius, 0, resolution - 1);
        int x1 = Mathf.Clamp(centerX + brushRadius, 0, resolution - 1);
        int z1 = Mathf.Clamp(centerZ + brushRadius, 0, resolution - 1);

        int width  = x1 - x0 + 1;
        int height = z1 - z0 + 1;

        float[,] heights = td.GetHeights(x0, z0, width, height);

        float depthNormalized = depth / td.size.y;
        float radiusSq = (float)(brushRadius * brushRadius);

        for (int z = 0; z < height; z++)
        {
            for (int x = 0; x < width; x++)
            {
                float dx = x + x0 - centerX;
                float dz = z + z0 - centerZ;
                float distSq = dx * dx + dz * dz;

                if (distSq < radiusSq)
                {
                    // 用 smoothstep 衰减：中心深，边缘浅
                    float t = 1f - Mathf.Sqrt(distSq / radiusSq);
                    float smooth = t * t * (3f - 2f * t);
                    heights[z, x] = Mathf.Max(0f, heights[z, x] - depthNormalized * smooth);
                }
            }
        }

        td.SetHeights(x0, z0, heights);
    }
}
```

## 七、性能优化检查清单

### 7.1 关键性能指标

| 优化项目 | 目标值 | 实现方案 |
|---|---|---|
| 地形 DrawCall | ≤ 4 | 使用 Terrain.groupingID 合并相邻地形 |
| 植被 DrawCall | ≤ 10 | GPU Instancing 批次渲染 |
| 高度图内存 | ≤ 64MB | 1025分辨率 × 4byte × 块数 |
| 地形碰撞精度 | 可配置 | 远距离降低 TerrainCollider 分辨率 |
| 植被可见数 | ≤ 50000 | LOD + 视锥剔除 |

### 7.2 Unity Terrain 性能 Tips

```csharp
// 1. 合并相邻地形以减少 DrawCall
terrain1.groupingID = 1;
terrain2.groupingID = 1; // 相同 groupingID 的地形会自动合并 DrawCall

// 2. 降低 Pixel Error 以减少三角形数量（移动端建议 15~25）
terrain.heightmapPixelError = 20f;

// 3. 关闭不可见地形的渲染
terrain.drawHeightmap = false;
terrain.drawInstanced = true; // 开启 GPU Instancing 模式（Unity 2019+）

// 错误做法：循环中每次 SetHeight 都触发刷新
// 正确做法：
float[,] heights = terrainData.GetHeights(x, y, w, h);
// ... 修改 heights 数组 ...
terrainData.SetHeights(x, y, heights); // 一次性提交

// 5. 使用 Terrain.SetNeighbors 消除相邻地形接缝
terrain1.SetNeighbors(left: null, top: terrain2, right: null, bottom: null);
```

## 八、最佳实践总结

### ✅ 设计原则

1. **分块设计**：大世界地形必须分块，单块建议 500m×500m，超出视距的块卸载
2. **异步生成**：地形高度图/Alphamap 生成放在协程或 Job System 中，避免主线程卡顿
3. **侵蚀后处理**：程序化地形生成后加一轮水力侵蚀，视觉质量提升明显
4. **植被专用系统**：不要用 Terrain 自带的 Detail/Tree 系统，超过 5000 棵会卡；改用 GPU Instancing 自定义实现
5. **地形 Shader 层数控制**：纹理层数超过 4 层时 Unity 会额外增加 Pass，建议控制在 4 层以内

### ❌ 常见误区

1. **单张超大地形**：1000m 以上的单块地形在移动端无法接受，必须分块
2. **实时修改不缓存**：每帧调用 SetHeights 会导致 GPU 上传卡顿
3. **法线贴图分辨率不匹配**：纹理 Tiling 错误导致法线与 Albedo 频率不一致
4. **忽略碰撞优化**：TerrainCollider 会生成巨大的 Mesh，远距离可降低精度或用简化碰撞代替
