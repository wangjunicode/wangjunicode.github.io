# C# Source Generator在游戏开发中的实战应用：零反射的元编程方案

## 一、传统反射的性能代价

在游戏开发中，反射（Reflection）被广泛用于：
- 组件自动注册、依赖注入
- 序列化/反序列化
- 消息分发系统
- 配置表绑定

但反射的代价是沉重的：

```csharp
// 运行时反射：每次调用约 100~1000ns（含缓存后约 10~50ns）
var method  = type.GetMethod("Init", BindingFlags.Public | BindingFlags.Instance);
var invoke  = method.Invoke(instance, args); // 装箱、IL 调用开销

// 在 IL2CPP AOT 环境下更糟糕：
// - 某些反射操作在 IL2CPP 下完全不可用
// - 代码裁剪（Code Stripping）可能移除反射依赖的类型信息
```

**C# Source Generator** 是 C# 9.0 引入的编译期代码生成能力，在编译阶段扫描代码并生成额外的 `.cs` 文件，从根本上消除运行时反射需求。

## 三、实战场景一：组件自动注册

### 3.1 问题定义

游戏框架常见写法——手动注册每个组件：

```csharp
// ❌ 手动维护，容易漏注册
void RegisterAllComponents()
{
    container.Register<PlayerController>();
    container.Register<EnemyAI>();
    container.Register<WeaponSystem>();
    // 新增组件后必须来这里手动加一行...
}
```

### 3.2 定义触发特性

```csharp
// 放在 Assets/Scripts/ 下，作为触发生成器的标记
namespace GameFramework
{
    [System.AttributeUsage(System.AttributeTargets.Class,
                            AllowMultiple = false, Inherited = false)]
    public sealed class AutoRegisterAttribute : System.Attribute
    {
        public bool IsSingleton { get; }
        public AutoRegisterAttribute(bool singleton = false)
            => IsSingleton = singleton;
    }
}
```

### 3.3 Source Generator 实现

```csharp
using Microsoft.CodeAnalysis;
using Microsoft.CodeAnalysis.CSharp.Syntax;
using Microsoft.CodeAnalysis.Text;
using System.Collections.Generic;
using System.Linq;
using System.Text;

[Generator]
public class ComponentRegistryGenerator : IIncrementalGenerator
{
    public void Initialize(IncrementalGeneratorInitializationContext context)
    {
        // 1. 筛选出带 [AutoRegister] 特性的类
        var classDeclarations = context.SyntaxProvider
            .CreateSyntaxProvider(
                predicate: static (node, _) =>
                    node is ClassDeclarationSyntax cls &&
                    cls.AttributeLists.Count > 0,

                transform: static (ctx, _) =>
                    GetClassWithAutoRegister(ctx)
            )
            .Where(static m => m is not null);

        // 2. 聚合所有匹配类，生成注册代码
        context.RegisterSourceOutput(
            classDeclarations.Collect(),
            static (spc, classes) => Execute(spc, classes!)
        );
    }

    private static ClassInfo? GetClassWithAutoRegister(
        GeneratorSyntaxContext context)
    {
        var classDecl = (ClassDeclarationSyntax)context.Node;
        var symbol    = context.SemanticModel
                               .GetDeclaredSymbol(classDecl) as INamedTypeSymbol;

        if (symbol == null) return null;

        var attr = symbol.GetAttributes().FirstOrDefault(a =>
            a.AttributeClass?.Name == "AutoRegisterAttribute");

        if (attr == null) return null;

        bool isSingleton = false;
        if (attr.ConstructorArguments.Length > 0)
            isSingleton = (bool)attr.ConstructorArguments[0].Value!;

        return new ClassInfo
        {
            FullName    = symbol.ToDisplayString(),
            IsSingleton = isSingleton,
        };
    }

    private static void Execute(
        SourceProductionContext context,
        IReadOnlyList<ClassInfo> classes)
    {
        if (classes.Count == 0) return;

        var sb = new StringBuilder();
        sb.AppendLine("// <auto-generated/>");
        sb.AppendLine("// 由 ComponentRegistryGenerator 自动生成，请勿手动编辑");
        sb.AppendLine();
        sb.AppendLine("namespace GameFramework");
        sb.AppendLine("{");
        sb.AppendLine("    public static partial class AutoRegistry");
        sb.AppendLine("    {");
        sb.AppendLine("        public static void RegisterAll(IContainer container)");
        sb.AppendLine("        {");

        foreach (var cls in classes)
        {
            if (cls.IsSingleton)
                sb.AppendLine($"            container.RegisterSingleton<{cls.FullName}>();");
            else
                sb.AppendLine($"            container.Register<{cls.FullName}>();");
        }

        sb.AppendLine("        }");
        sb.AppendLine("    }");
        sb.AppendLine("}");

        context.AddSource("AutoRegistry.g.cs",
            SourceText.From(sb.ToString(), Encoding.UTF8));
    }

    private class ClassInfo
    {
        public string FullName    { get; set; } = "";
        public bool   IsSingleton { get; set; }
    }
}
```

## 四、实战场景二：网络消息分发器生成

### 4.1 目标：零反射的消息路由

```csharp
// 手动维护的消息分发（❌ 每加一个消息类型都要改这里）
switch (msgId)
{
    case 1001: handler.OnLoginResponse((LoginResponse)msg);   break;
    case 1002: handler.OnMoveSync((MoveSync)msg);             break;
    // ...几百个 case...
}

// ✅ 目标：用 Source Generator 自动生成 switch 分支
```

### 4.2 特性定义

```csharp
[System.AttributeUsage(System.AttributeTargets.Method)]
public sealed class MessageHandlerAttribute : System.Attribute
{
    public int MessageId { get; }
    public MessageHandlerAttribute(int messageId) => MessageId = messageId;
}
```

### 4.3 消息分发器生成器

```csharp
[Generator]
public class MessageDispatcherGenerator : IIncrementalGenerator
{
    public void Initialize(IncrementalGeneratorInitializationContext context)
    {
        var methods = context.SyntaxProvider
            .CreateSyntaxProvider(
                predicate: static (node, _) =>
                    node is MethodDeclarationSyntax m &&
                    m.AttributeLists.Count > 0,
                transform: GetHandlerMethod
            )
            .Where(m => m is not null);

        context.RegisterSourceOutput(methods.Collect(), GenerateDispatcher);
    }

    private static HandlerInfo? GetHandlerMethod(GeneratorSyntaxContext ctx)
    {
        var methodDecl = (MethodDeclarationSyntax)ctx.Node;
        var symbol     = ctx.SemanticModel
                            .GetDeclaredSymbol(methodDecl) as IMethodSymbol;
        if (symbol == null) return null;

        var attr = symbol.GetAttributes().FirstOrDefault(a =>
            a.AttributeClass?.Name == "MessageHandlerAttribute");
        if (attr == null) return null;

        int msgId = (int)attr.ConstructorArguments[0].Value!;

        // 获取参数类型（消息类型）
        if (symbol.Parameters.Length != 1) return null;
        var paramType = symbol.Parameters[0].Type.ToDisplayString();
        var className = symbol.ContainingType.ToDisplayString();

        return new HandlerInfo
        {
            MessageId  = msgId,
            MethodName = symbol.Name,
            ParamType  = paramType,
            ClassName  = className,
        };
    }

    private static void GenerateDispatcher(
        SourceProductionContext context,
        IReadOnlyList<HandlerInfo> handlers)
    {
        if (handlers.Count == 0) return;

        // 按类分组
        var byClass = handlers
            .GroupBy(h => h.ClassName)
            .ToList();

        var sb = new StringBuilder();
        sb.AppendLine("// <auto-generated/>");
        sb.AppendLine("using System.Collections.Generic;");
        sb.AppendLine();

        foreach (var group in byClass)
        {
            // 为每个含 [MessageHandler] 的类生成 partial 分发方法
            var parts = group.Key.Split('.');
            var ns    = string.Join(".", parts.Take(parts.Length - 1));
            var cls   = parts.Last();

            if (!string.IsNullOrEmpty(ns))
            {
                sb.AppendLine($"namespace {ns}");
                sb.AppendLine("{");
            }

            sb.AppendLine($"    public partial class {cls}");
            sb.AppendLine("    {");
            sb.AppendLine("        // 自动生成的消息分发方法");
            sb.AppendLine("        public void DispatchMessage(int msgId, object msg)");
            sb.AppendLine("        {");
            sb.AppendLine("            switch (msgId)");
            sb.AppendLine("            {");

            foreach (var h in group)
            {
                sb.AppendLine($"                case {h.MessageId}:");
                sb.AppendLine($"                    {h.MethodName}(({h.ParamType})msg);");
                sb.AppendLine($"                    break;");
            }

            sb.AppendLine("                default:");
            sb.AppendLine("                    UnityEngine.Debug.LogWarning($\"未注册的消息ID: {msgId}\");");
            sb.AppendLine("                    break;");
            sb.AppendLine("            }");
            sb.AppendLine("        }");
            sb.AppendLine("    }");

            if (!string.IsNullOrEmpty(ns))
                sb.AppendLine("}");
            sb.AppendLine();
        }

        context.AddSource("MessageDispatchers.g.cs",
            SourceText.From(sb.ToString(), Encoding.UTF8));
    }

    private class HandlerInfo
    {
        public int    MessageId  { get; set; }
        public string MethodName { get; set; } = "";
        public string ParamType  { get; set; } = "";
        public string ClassName  { get; set; } = "";
    }
}
```

## 五、实战场景三：序列化代码生成

### 5.1 为游戏存档生成序列化代码

```csharp
[System.AttributeUsage(System.AttributeTargets.Class)]
public sealed class GameSaveDataAttribute : System.Attribute { }

// 标注需要序列化的存档数据类
[GameSaveData]
public class PlayerSaveData
{
    public int    Level      { get; set; }
    public float  Exp        { get; set; }
    public string PlayerName { get; set; }
    public int[]  Inventory  { get; set; }
}
```

### 5.2 生成二进制序列化代码

```csharp
[Generator]
public class SaveDataSerializerGenerator : IIncrementalGenerator
{
    public void Initialize(IncrementalGeneratorInitializationContext context)
    {
        var classes = context.SyntaxProvider
            .CreateSyntaxProvider(
                predicate: (n, _) => n is ClassDeclarationSyntax c &&
                                     c.AttributeLists.Count > 0,
                transform: GetSaveDataClass
            )
            .Where(c => c is not null);

        context.RegisterSourceOutput(classes.Collect(), GenerateSerializer);
    }

    private static SaveDataInfo? GetSaveDataClass(GeneratorSyntaxContext ctx)
    {
        var classDecl = (ClassDeclarationSyntax)ctx.Node;
        var symbol    = ctx.SemanticModel
                           .GetDeclaredSymbol(classDecl) as INamedTypeSymbol;
        if (symbol == null) return null;

        bool hasAttr = symbol.GetAttributes().Any(a =>
            a.AttributeClass?.Name == "GameSaveDataAttribute");
        if (!hasAttr) return null;

        var props = symbol.GetMembers()
            .OfType<IPropertySymbol>()
            .Where(p => p.SetMethod != null)
            .Select(p => new PropertyInfo
            {
                Name     = p.Name,
                TypeName = p.Type.ToDisplayString(),
            })
            .ToList();

        return new SaveDataInfo
        {
            ClassName  = symbol.Name,
            Namespace  = symbol.ContainingNamespace?.ToDisplayString() ?? "",
            Properties = props,
        };
    }

    private static void GenerateSerializer(
        SourceProductionContext context,
        IReadOnlyList<SaveDataInfo> classes)
    {
        foreach (var cls in classes)
            GenerateForClass(context, cls);
    }

    private static void GenerateForClass(
        SourceProductionContext context,
        SaveDataInfo cls)
    {
        var sb = new StringBuilder();
        sb.AppendLine("// <auto-generated/>");
        sb.AppendLine("using System.IO;");
        sb.AppendLine();

        if (!string.IsNullOrEmpty(cls.Namespace))
        {
            sb.AppendLine($"namespace {cls.Namespace}");
            sb.AppendLine("{");
        }

        sb.AppendLine($"    public partial class {cls.ClassName}");
        sb.AppendLine("    {");

        // 生成 Serialize 方法
        sb.AppendLine("        public void Serialize(BinaryWriter writer)");
        sb.AppendLine("        {");
        foreach (var prop in cls.Properties)
        {
            var writeCall = GetWriteCall(prop.TypeName, prop.Name);
            sb.AppendLine($"            {writeCall}");
        }
        sb.AppendLine("        }");
        sb.AppendLine();

        // 生成 Deserialize 方法
        sb.AppendLine("        public void Deserialize(BinaryReader reader)");
        sb.AppendLine("        {");
        foreach (var prop in cls.Properties)
        {
            var readCall = GetReadCall(prop.TypeName, prop.Name);
            sb.AppendLine($"            {readCall}");
        }
        sb.AppendLine("        }");
        sb.AppendLine("    }");

        if (!string.IsNullOrEmpty(cls.Namespace))
            sb.AppendLine("}");

        context.AddSource($"{cls.ClassName}.Serializer.g.cs",
            SourceText.From(sb.ToString(), Encoding.UTF8));
    }

    private static string GetWriteCall(string typeName, string propName)
    {
        return typeName switch
        {
            "int"    => $"writer.Write({propName});",
            "float"  => $"writer.Write({propName});",
            "bool"   => $"writer.Write({propName});",
            "string" => $"writer.Write({propName} ?? string.Empty);",
            "int[]"  => $"writer.Write({propName}?.Length ?? 0); " +
                        $"if ({propName} != null) foreach(var v in {propName}) writer.Write(v);",
            _        => $"// TODO: 不支持类型 {typeName}",
        };
    }

    private static string GetReadCall(string typeName, string propName)
    {
        return typeName switch
        {
            "int"    => $"{propName} = reader.ReadInt32();",
            "float"  => $"{propName} = reader.ReadSingle();",
            "bool"   => $"{propName} = reader.ReadBoolean();",
            "string" => $"{propName} = reader.ReadString();",
            "int[]"  => $"{{ int len = reader.ReadInt32(); {propName} = new int[len]; " +
                        $"for(int i=0;i<len;i++) {propName}[i] = reader.ReadInt32(); }}",
            _        => $"// TODO: 不支持类型 {typeName}",
        };
    }

    private class SaveDataInfo
    {
        public string ClassName  { get; set; } = "";
        public string Namespace  { get; set; } = "";
        public List<PropertyInfo> Properties { get; set; } = new();
    }

    private class PropertyInfo
    {
        public string Name     { get; set; } = "";
        public string TypeName { get; set; } = "";
    }
}
```

## 八、Source Generator vs 其他方案对比

| 方案 | 运行时开销 | IL2CPP 兼容 | 代码可读性 | 调试难度 |
|------|-----------|------------|-----------|--------|
| 运行时反射 | 高（100ns+）| 部分不兼容 | 好 | 低 |
| Fody IL 织入 | 零 | 兼容 | 一般 | 高 |
| T4 模板 | 零 | 兼容 | 差（模板语法）| 中 |
| **Source Generator** | **零** | **完全兼容** | **好（生成标准C#）** | **中** |
| 手写代码 | 零 | 兼容 | 最好 | 最低 |

## 九、最佳实践总结

1. **增量生成**：使用 `IIncrementalGenerator`（而非旧的 `ISourceGenerator`），避免每次按键都触发全量生成
2. **等值比较**：生成器输入的 `SyntaxProvider` 中，用不可变 record 作为传递对象，确保增量缓存有效
3. **语义模型优先**：不要直接解析语法树字符串，通过 `SemanticModel` 获取准确的类型信息
4. **只生成 partial class**：避免覆盖用户代码，始终生成 partial 扩展
5. **注释生成代码**：在生成文件头部加 `// <auto-generated/>` 注释，IDE 和工具链会正确识别并隐藏
6. **避免复杂逻辑**：生成器在 IDE 实时运行，过重的逻辑会影响编辑器响应速度
7. **IL2CPP 友好**：生成的代码完全是普通 C#，天然适配 IL2CPP AOT 编译，无需额外配置
8. **配合 Attribute 驱动**：用自定义 Attribute 作为触发标记，保持侵入性最小
9. **单元测试生成器**：使用 `Microsoft.CodeAnalysis.CSharp.Testing` 库对生成器本身进行单元测试

## 十、小结

C# Source Generator 是游戏开发中消除运行时反射的终极方案。相比传统的 IL 织入、T4 模板或运行时反射，它在编译期生成纯净的 C# 代码，完全兼容 Unity IL2CPP 的 AOT 编译，生成结果可调试、可读、零运行时开销。通过组件自动注册、消息分发器生成、序列化代码生成等典型场景的实践，开发者可以系统性地将项目中的"反射债务"转化为编译期确定的高效代码，大幅提升游戏在低端设备上的运行性能。
