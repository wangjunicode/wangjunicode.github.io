# 游戏玩家行为分析与数据埋点系统完全指南

## 一、数据埋点体系概述

### 1.1 为什么需要埋点系统

游戏产品决策如果只依赖主观判断，往往存在严重盲区：

| 问题类型 | 无埋点时的困境 | 有埋点后的决策依据 |
|---------|--------------|-----------------|
| 关卡设计 | 不知道玩家卡在哪里 | 精确的关卡失败率、死亡热力图 |
| 付费转化 | 猜测付费门槛 | 商城浏览→购买漏斗分析 |
| 功能使用 | 不知道哪些功能无人用 | 功能点击率、使用频次 |
| 留存问题 | 不知道为什么流失 | 流失前行为路径溯源 |
| 性能体验 | 只有崩溃日志 | 卡顿率/帧率分布/设备分级 |

### 1.2 埋点分类体系

```
数据埋点体系
├── 自动采集（Auto-Track）
│   ├── 应用生命周期（启动/前台/后台/崩溃）
│   ├── 页面PV/UV（Scene加载/卸载）
│   └── 性能指标（FPS/内存/网络RTT）
├── 代码埋点（Manual-Track）
│   ├── 业务事件（关卡开始/结束/付费/分享）
│   ├── 用户交互（按钮点击/手势/滑动）
│   └── 游戏行为（战斗/技能/道具使用）
└── 可视化埋点（Visual-Track）
    ├── UI点击热力图
    └── 无需发版的动态配置埋点
```

## 二、埋点SDK架构设计

### 2.1 核心架构

```csharp
// 整体架构：事件 → 过滤 → 序列化 → 本地缓存 → 批量上报
//
// TrackEvent API
//     ↓
// EventValidator（合法性校验）
//     ↓
// EventEnricher（自动附加公共属性）
//     ↓
// EventQueue（内存队列）
//     ↓
// BatchSender（批量发送策略）
//     ↓
// LocalStorage（持久化缓冲区，防丢失）
//     ↓
// HttpReporter → 数据服务器

using System;
using System.Collections.Generic;
using System.Text;
using UnityEngine;
using Newtonsoft.Json;

/// <summary>
/// 游戏埋点SDK核心类
/// 线程安全，支持离线缓存与批量上报
/// </summary>
public class GameTracker : MonoBehaviour
{
    public static GameTracker Instance { get; private set; }

    [Header("服务端配置")]
    [SerializeField] private string reportUrl = "https://analytics.example.com/track";
    [SerializeField] private string appKey    = "YOUR_APP_KEY";
    
    [Header("批量策略")]
    [SerializeField] private int    batchSize    = 20;   // 积攒20条再上报
    [SerializeField] private float  batchInterval = 15f; // 或15秒上报一次
    [SerializeField] private int    maxQueueSize  = 500; // 内存队列上限
    
    [Header("隐私合规")]
    [SerializeField] private bool enableTracking = true; // GDPR/PIPL开关

    // 公共属性（每个事件自动携带）
    private Dictionary<string, object> commonProperties;
    
    // 内存事件队列
    private readonly Queue<TrackEvent> eventQueue = new();
    private float nextBatchTime;
    
    // 本地持久化路径
    private string localCachePath;

    void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
        
        localCachePath = Application.persistentDataPath + "/tracker_cache.json";
        InitCommonProperties();
        LoadCachedEvents(); // 加载上次未发送的事件
    }

    void Start()
    {
        nextBatchTime = Time.time + batchInterval;
        
        // 自动埋点：应用启动
        Track("app_launch", new Dictionary<string, object>
        {
            { "launch_type", "cold" },
            { "install_date", PlayerPrefs.GetString("install_date", "") }
        });
    }

    void Update()
    {
        if (Time.time >= nextBatchTime)
        {
            nextBatchTime = Time.time + batchInterval;
            SendBatch();
        }
    }

    void OnApplicationPause(bool pauseStatus)
    {
        if (pauseStatus)
        {
            Track("app_background", null);
            // 立即发送，避免数据丢失
            SendBatch(immediate: true);
            SaveCacheToLocal();
        }
        else
        {
            Track("app_foreground", null);
        }
    }

    void OnApplicationQuit()
    {
        Track("app_quit", new Dictionary<string, object>
        {
            { "session_duration", Time.realtimeSinceStartup }
        });
        SendBatch(immediate: true);
        SaveCacheToLocal();
    }

    /// <summary>初始化公共属性（每个事件自动携带）</summary>
    private void InitCommonProperties()
    {
        commonProperties = new Dictionary<string, object>
        {
            // 设备信息
            { "device_id",       SystemInfo.deviceUniqueIdentifier },
            { "device_model",    SystemInfo.deviceModel },
            { "os",              SystemInfo.operatingSystem },
            { "os_version",      SystemInfo.operatingSystemFamily.ToString() },
            { "screen_width",    Screen.width },
            { "screen_height",   Screen.height },
            { "screen_dpi",      Screen.dpi },
            
            // 应用信息
            { "app_version",     Application.version },
            { "unity_version",   Application.unityVersion },
            { "platform",        Application.platform.ToString() },
            
            // GPU信息（用于设备分级）
            { "gpu_name",        SystemInfo.graphicsDeviceName },
            { "gpu_memory_mb",   SystemInfo.graphicsMemorySize },
            
            // 用户标识（需符合隐私规范）
            { "user_id",         GetOrCreateUserId() },
        };
    }

    private string GetOrCreateUserId()
    {
        const string key = "tracker_uid";
        if (!PlayerPrefs.HasKey(key))
            PlayerPrefs.SetString(key, Guid.NewGuid().ToString("N"));
        return PlayerPrefs.GetString(key);
    }

    /// <summary>核心埋点方法</summary>
    public void Track(string eventName, Dictionary<string, object> properties = null)
    {
        if (!enableTracking) return;
        if (string.IsNullOrEmpty(eventName)) return;

        var allProps = new Dictionary<string, object>(commonProperties);
        
        // 追加时间和会话信息
        allProps["event_time"]     = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds();
        allProps["session_id"]     = SessionManager.CurrentSessionId;
        allProps["game_time"]      = (int)Time.realtimeSinceStartup;
        allProps["scene_name"]     = UnityEngine.SceneManagement.SceneManager.GetActiveScene().name;
        
        // 合并业务属性（业务属性可覆盖公共属性）
        if (properties != null)
            foreach (var kv in properties)
                allProps[kv.Key] = kv.Value;

        var evt = new TrackEvent
        {
            EventName  = eventName,
            Properties = allProps
        };

        lock (eventQueue)
        {
            if (eventQueue.Count >= maxQueueSize)
            {
                Debug.LogWarning("[Tracker] 队列已满，丢弃旧事件");
                eventQueue.Dequeue(); // 丢弃最老的
            }
            eventQueue.Enqueue(evt);
        }

        // 超过batchSize立即发送
        if (eventQueue.Count >= batchSize)
            SendBatch();
    }

    private async void SendBatch(bool immediate = false)
    {
        List<TrackEvent> batch;
        lock (eventQueue)
        {
            if (eventQueue.Count == 0) return;
            batch = new List<TrackEvent>();
            int count = Mathf.Min(eventQueue.Count, batchSize * 2);
            for (int i = 0; i < count; i++)
                batch.Add(eventQueue.Dequeue());
        }

        try
        {
            string json = JsonConvert.SerializeObject(new { app_key = appKey, events = batch });
            byte[] body = Encoding.UTF8.GetBytes(json);
            
            using var request = new UnityEngine.Networking.UnityWebRequest(reportUrl, "POST");
            request.uploadHandler   = new UnityEngine.Networking.UploadHandlerRaw(body);
            request.downloadHandler = new UnityEngine.Networking.DownloadHandlerBuffer();
            request.SetRequestHeader("Content-Type", "application/json");
            
            await request.SendWebRequest();
            
            if (request.result != UnityEngine.Networking.UnityWebRequest.Result.Success)
            {
                Debug.LogWarning($"[Tracker] 上报失败: {request.error}，重新入队");
                // 上报失败，重新入队
                lock (eventQueue)
                    foreach (var e in batch)
                        eventQueue.Enqueue(e);
            }
        }
        catch (Exception ex)
        {
            Debug.LogError($"[Tracker] 上报异常: {ex.Message}");
        }
    }

    private void SaveCacheToLocal()
    {
        try
        {
            List<TrackEvent> pending;
            lock (eventQueue)
                pending = new List<TrackEvent>(eventQueue);
            
            if (pending.Count > 0)
                System.IO.File.WriteAllText(
                    localCachePath, JsonConvert.SerializeObject(pending));
        }
        catch (Exception ex) { Debug.LogError($"[Tracker] 缓存保存失败: {ex}"); }
    }

    private void LoadCachedEvents()
    {
        try
        {
            if (!System.IO.File.Exists(localCachePath)) return;
            string json = System.IO.File.ReadAllText(localCachePath);
            var cached = JsonConvert.DeserializeObject<List<TrackEvent>>(json);
            if (cached == null) return;
            
            lock (eventQueue)
                foreach (var e in cached)
                    eventQueue.Enqueue(e);
            
            System.IO.File.Delete(localCachePath); // 加载后删除缓存
            Debug.Log($"[Tracker] 恢复 {cached.Count} 条缓存事件");
        }
        catch (Exception ex) { Debug.LogError($"[Tracker] 缓存加载失败: {ex}"); }
    }

    [Serializable]
    private class TrackEvent
    {
        [JsonProperty("event")] public string EventName { get; set; }
        [JsonProperty("props")] public Dictionary<string, object> Properties { get; set; }
    }
}
```

```csharp
/// <summary>
/// 关卡生命周期埋点
/// 追踪玩家在每个关卡的完整行为数据
/// </summary>
public class LevelTracker : MonoBehaviour
{
    private int    currentLevel;
    private float  levelStartTime;
    private int    deathCount;
    private int    retryCount;
    private float  maxProgressReached;
    private string levelStartScene;

    // 关卡开始
    public void TrackLevelStart(int levelId, string levelName, int playerLevel)
    {
        currentLevel     = levelId;
        levelStartTime   = Time.realtimeSinceStartup;
        deathCount       = 0;
        retryCount       = 0;
        maxProgressReached = 0f;
        levelStartScene  = levelName;
        
        GameTracker.Instance.Track("level_start", new Dictionary<string, object>
        {
            { "level_id",     levelId },
            { "level_name",   levelName },
            { "player_level", playerLevel },
            { "entry_source", "map_select" }, // 从哪里进入
        });
    }

    // 玩家死亡
    public void TrackPlayerDeath(Vector3 deathPosition, string causeOfDeath, float progressAtDeath)
    {
        deathCount++;
        maxProgressReached = Mathf.Max(maxProgressReached, progressAtDeath);
        
        GameTracker.Instance.Track("player_death", new Dictionary<string, object>
        {
            { "level_id",          currentLevel },
            { "death_x",           Mathf.RoundToInt(deathPosition.x) },
            { "death_y",           Mathf.RoundToInt(deathPosition.y) },
            { "death_z",           Mathf.RoundToInt(deathPosition.z) },
            { "cause",             causeOfDeath },
            { "progress",          Mathf.RoundToInt(progressAtDeath * 100) }, // 百分比
            { "death_count",       deathCount },
            { "time_survived",     Time.realtimeSinceStartup - levelStartTime },
        });
    }

    // 关卡完成
    public void TrackLevelComplete(int stars, float completionTime, Dictionary<string, int> itemsCollected)
    {
        float totalTime = Time.realtimeSinceStartup - levelStartTime;
        
        GameTracker.Instance.Track("level_complete", new Dictionary<string, object>
        {
            { "level_id",        currentLevel },
            { "stars",           stars },
            { "completion_time", Mathf.RoundToInt(completionTime) },
            { "total_time",      Mathf.RoundToInt(totalTime) },
            { "death_count",     deathCount },
            { "retry_count",     retryCount },
            { "items_collected", itemsCollected },
            { "is_first_clear",  retryCount == 0 && deathCount == 0 },
        });
    }

    // 关卡放弃
    public void TrackLevelAbandon(string reason = "player_quit")
    {
        GameTracker.Instance.Track("level_abandon", new Dictionary<string, object>
        {
            { "level_id",        currentLevel },
            { "progress",        Mathf.RoundToInt(maxProgressReached * 100) },
            { "time_spent",      Mathf.RoundToInt(Time.realtimeSinceStartup - levelStartTime) },
            { "death_count",     deathCount },
            { "reason",          reason },
        });
    }
}
```

## 四、UI 交互热力图系统

```csharp
/// <summary>
/// UI点击热力图数据收集器
/// 记录玩家在各UI界面的点击分布，用于优化UI布局
/// </summary>
public class UIHeatmapTracker : MonoBehaviour
{
    [SerializeField] private Canvas targetCanvas;
    [SerializeField] private string screenName; // 界面名称标识
    [SerializeField] private int    resolutionX = 100; // 热力图分辨率（横向格数）
    [SerializeField] private int    resolutionY = 50;  // 热力图分辨率（纵向格数）
    
    // 点击分布二维数组
    private int[,] clickGrid;
    private int    totalClicks;

    void Awake()
    {
        clickGrid = new int[resolutionX, resolutionY];
    }

    void OnEnable()
    {
        // 注册全局UI事件监听
        // 实际项目中通过EventSystem的PointerClick监听所有UI点击
    }

    /// <summary>记录一次UI点击（在EventSystem的PointerClick回调中调用）</summary>
    public void RecordClick(Vector2 screenPosition, string elementName)
    {
        // 将屏幕坐标归一化到[0,1]
        float normalizedX = screenPosition.x / Screen.width;
        float normalizedY = screenPosition.y / Screen.height;
        
        // 映射到格子坐标
        int gridX = Mathf.Clamp((int)(normalizedX * resolutionX), 0, resolutionX - 1);
        int gridY = Mathf.Clamp((int)(normalizedY * resolutionY), 0, resolutionY - 1);
        
        clickGrid[gridX, gridY]++;
        totalClicks++;

        // 单次点击埋点（精确位置）
        GameTracker.Instance.Track("ui_click", new Dictionary<string, object>
        {
            { "screen",       screenName },
            { "element",      elementName },
            { "pos_x",        Mathf.RoundToInt(normalizedX * 1000) }, // 0~1000
            { "pos_y",        Mathf.RoundToInt(normalizedY * 1000) },
            { "grid_x",       gridX },
            { "grid_y",       gridY },
        });
    }

    /// <summary>界面关闭时上报聚合热力图数据</summary>
    void OnDisable()
    {
        if (totalClicks < 5) return; // 点击太少不上报
        
        // 压缩热力图数据：只上报非零格子
        var hotspots = new List<Dictionary<string, int>>();
        for (int x = 0; x < resolutionX; x++)
        for (int y = 0; y < resolutionY; y++)
        {
            if (clickGrid[x, y] > 0)
                hotspots.Add(new Dictionary<string, int>
                    { { "x", x }, { "y", y }, { "c", clickGrid[x, y] } });
        }
        
        GameTracker.Instance.Track("ui_heatmap", new Dictionary<string, object>
        {
            { "screen",       screenName },
            { "total_clicks", totalClicks },
            { "hotspots",     hotspots },
            { "res_x",        resolutionX },
            { "res_y",        resolutionY },
        });
    }
}
```

```csharp
/// <summary>
/// 会话与留存分析管理器
/// 追踪用户日留存、周留存、次日留存等核心指标
/// </summary>
public static class SessionManager
{
    public static string CurrentSessionId { get; private set; }
    private static float sessionStartTime;
    private static int   sessionEventCount;

    // 初始化（游戏启动时调用）
    public static void Initialize()
    {
        CurrentSessionId = Guid.NewGuid().ToString("N").Substring(0, 16);
        sessionStartTime = Time.realtimeSinceStartup;
        sessionEventCount = 0;
        
        string today = DateTime.UtcNow.ToString("yyyy-MM-dd");
        string firstLaunchDate = PlayerPrefs.GetString("first_launch_date", "");
        string lastActiveDate  = PlayerPrefs.GetString("last_active_date", "");
        int    totalSessions   = PlayerPrefs.GetInt("total_sessions", 0) + 1;

        if (string.IsNullOrEmpty(firstLaunchDate))
        {
            firstLaunchDate = today;
            PlayerPrefs.SetString("first_launch_date", firstLaunchDate);
            PlayerPrefs.SetString("install_date", today);
        }

        PlayerPrefs.SetString("last_active_date", today);
        PlayerPrefs.SetInt("total_sessions", totalSessions);

        // 计算留存天数（注册后第N天登录）
        if (DateTime.TryParse(firstLaunchDate, out DateTime firstDate) &&
            DateTime.TryParse(today, out DateTime todayDate))
        {
            int daysAfterInstall = (todayDate - firstDate).Days;
            GameTracker.Instance.Track("session_start", new Dictionary<string, object>
            {
                { "session_id",          CurrentSessionId },
                { "session_number",      totalSessions },
                { "days_after_install",  daysAfterInstall },
                { "last_active_date",    lastActiveDate },
                { "is_new_day",          lastActiveDate != today },
                { "is_day1_retention",   daysAfterInstall == 1 },
                { "is_day7_retention",   daysAfterInstall == 7 },
                { "is_day30_retention",  daysAfterInstall == 30 },
            });
        }
    }

    public static void EndSession()
    {
        float sessionDuration = Time.realtimeSinceStartup - sessionStartTime;
        GameTracker.Instance.Track("session_end", new Dictionary<string, object>
        {
            { "session_id",       CurrentSessionId },
            { "duration_seconds", Mathf.RoundToInt(sessionDuration) },
            { "event_count",      sessionEventCount },
        });
    }

    public static void IncrementEventCount() => sessionEventCount++;
}
```

## 六、性能数据自动采集

```csharp
/// <summary>
/// 性能数据自动采集器
/// 按固定间隔收集FPS、内存、网络等指标
/// </summary>
public class PerformanceAutoTracker : MonoBehaviour
{
    [SerializeField] private float sampleInterval = 60f; // 每60秒采集一次
    
    private float sampleTimer;
    private float fpsAccum;
    private int   fpsSampleCount;
    private float minFps = float.MaxValue;
    private float maxFps = 0f;

    void Update()
    {
        // 帧率采样
        float fps = 1f / Time.unscaledDeltaTime;
        fpsAccum += fps;
        fpsSampleCount++;
        minFps = Mathf.Min(minFps, fps);
        maxFps = Mathf.Max(maxFps, fps);
        
        sampleTimer += Time.unscaledDeltaTime;
        if (sampleTimer >= sampleInterval)
        {
            sampleTimer = 0f;
            ReportPerformanceSnapshot();
        }
    }

    void ReportPerformanceSnapshot()
    {
        if (fpsSampleCount == 0) return;
        
        float avgFps = fpsAccum / fpsSampleCount;
        
        GameTracker.Instance.Track("perf_snapshot", new Dictionary<string, object>
        {
            // 帧率指标
            { "fps_avg",       Mathf.RoundToInt(avgFps) },
            { "fps_min",       Mathf.RoundToInt(minFps) },
            { "fps_max",       Mathf.RoundToInt(maxFps) },
            { "fps_bucket",    GetFpsBucket(avgFps) },  // "60+", "45-60", "30-45", "<30"
            
            // 内存指标
            { "mem_total_mb",  Mathf.RoundToInt(
                UnityEngine.Profiling.Profiler.GetTotalAllocatedMemoryLong() / 1048576f) },
            { "mem_mono_mb",   Mathf.RoundToInt(
                UnityEngine.Profiling.Profiler.GetMonoUsedSizeLong() / 1048576f) },
            { "mem_reserved_mb", Mathf.RoundToInt(
                UnityEngine.Profiling.Profiler.GetTotalReservedMemoryLong() / 1048576f) },
            
            // 设备状态
            { "battery_level", Mathf.RoundToInt(SystemInfo.batteryLevel * 100) },
            { "battery_status",SystemInfo.batteryStatus.ToString() },
        });
        
        // 重置采样
        fpsAccum = 0f;
        fpsSampleCount = 0;
        minFps = float.MaxValue;
        maxFps = 0f;
    }

    private string GetFpsBucket(float fps)
    {
        if (fps >= 60f) return "60+";
        if (fps >= 45f) return "45-60";
        if (fps >= 30f) return "30-45";
        return "<30";
    }
}
```

## 七、隐私合规设计

### 7.1 GDPR / PIPL 合规框架

```csharp
/// <summary>
/// 隐私合规管理器
/// 支持GDPR（欧盟）和PIPL（中国个人信息保护法）
/// </summary>
public class PrivacyComplianceManager : MonoBehaviour
{
    private const string CONSENT_KEY      = "privacy_consent";
    private const string CONSENT_DATE_KEY = "privacy_consent_date";
    
    public static bool HasUserConsent => PlayerPrefs.GetInt(CONSENT_KEY, 0) == 1;
    
    public static void GrantConsent()
    {
        PlayerPrefs.SetInt(CONSENT_KEY, 1);
        PlayerPrefs.SetString(CONSENT_DATE_KEY, DateTime.UtcNow.ToString("O"));
        
        // 用户授权后才开启追踪
        if (GameTracker.Instance != null)
            GameTracker.Instance.Track("privacy_consent_granted", new Dictionary<string, object>
            {
                { "consent_version", "v1.2" },
                { "consent_type",    "explicit" },
            });
    }

    public static void RevokeConsent()
    {
        PlayerPrefs.SetInt(CONSENT_KEY, 0);
        // 同时删除本地缓存数据
        string cachePath = Application.persistentDataPath + "/tracker_cache.json";
        if (System.IO.File.Exists(cachePath))
            System.IO.File.Delete(cachePath);
        
        // 通知服务端删除用户数据（右被遗忘权）
        GameTracker.Instance.Track("privacy_consent_revoked", null);
    }

    // 数据最小化：根据合规要求过滤敏感字段
    public static Dictionary<string, object> FilterSensitiveData(
        Dictionary<string, object> properties)
    {
        var filtered = new Dictionary<string, object>(properties);
        
        // 在部分地区不收集精确设备ID
        if (IsEURegion())
            filtered.Remove("device_id");
        
        return filtered;
    }

    private static bool IsEURegion()
    {
        // 根据语言/时区等推断地区（不使用GPS）
        return Application.systemLanguage == SystemLanguage.German ||
               Application.systemLanguage == SystemLanguage.French ||
               Application.systemLanguage == SystemLanguage.Italian;
    }
}
```

## 八、最佳实践总结

### 8.1 埋点质量标准

| 维度 | 标准 |
|------|------|
| **完整性** | 关键漏斗每个环节都有事件覆盖 |
| **一致性** | 相同业务含义使用统一的事件名和属性名 |
| **准确性** | 事件触发时机与业务语义严格对应 |
| **性能** | 批量上报，单次上报不超过50KB |
| **可靠性** | 本地缓存防丢失，失败自动重试 |
| **合规性** | 用户授权前不收集敏感数据 |

### 8.2 事件命名规范

```
// 推荐格式：{模块}_{动作}_{对象}
level_start          // 关卡开始
level_complete       // 关卡完成  
player_death         // 玩家死亡
purchase_success     // 购买成功
ui_click             // UI点击
perf_snapshot        // 性能快照

// 避免：
click_button         // 太模糊
l_s                  // 缩写不可读
LevelStart           // 大驼峰（不统一）
```

### 8.3 核心指标体系

```
留存指标
├── 次日留存（Day1 Retention）
├── 第7日留存（Day7 Retention）
└── 第30日留存（Day30 Retention）

付费指标
├── 付费率（Paying User Rate）
├── ARPU（每用户平均收入）
├── ARPPU（每付费用户平均收入）
└── LTV（用户生命周期价值）

游戏深度指标
├── 关卡通过率（Level Pass Rate）
├── 关卡失败分布（Loss Distribution）
├── 平均局时长（Avg Session Duration）
└── 功能使用深度（Feature Depth）
```

数据埋点系统是游戏产品持续进化的基础设施。通过科学的埋点体系，团队能够以数据为驱动做出更精准的产品决策，大幅提升留存率和付费转化率，实现游戏产品的持续健康增长。
