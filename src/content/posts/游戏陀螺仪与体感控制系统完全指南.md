# 游戏陀螺仪与体感控制系统完全指南

## 一、移动端传感器概述

### 1.1 三大核心传感器

| 传感器 | 测量内容 | 单位 | 频率 | 游戏应用场景 |
|--------|---------|------|------|-------------|
| **加速度计** (Accelerometer) | 线性加速度 + 重力 | m/s² | 50~200Hz | 摇晃检测、倾斜控制 |
| **陀螺仪** (Gyroscope) | 角速度 | rad/s | 100~1000Hz | 旋转控制、VR头追 |
| **磁力计** (Magnetometer) | 磁场强度 | μT | 10~50Hz | 绝对方向、指南针 |

```
传感器特性对比：
┌─────────────────┬─────────────────────┬──────────────────────┐
│  加速度计        │  陀螺仪             │  融合后              │
│  ✓ 绝对姿态参考  │  ✓ 短期精度高       │  ✓ 稳定 + 精确       │
│  ✓ 无漂移        │  ✓ 响应速度快       │  ✓ 低延迟            │
│  ✗ 抖动噪声大    │  ✗ 长期积分漂移    │                      │
│  ✗ 运动时不准    │  ✗ 绝对角度误差积累 │                      │
└─────────────────┴─────────────────────┴──────────────────────┘
```

### 1.2 Unity 传感器接口

```csharp
using UnityEngine;
using UnityEngine.InputSystem;

/// <summary>
/// 检查设备传感器支持情况
/// </summary>
public static class SensorCapability
{
    public static bool HasAccelerometer => SystemInfo.supportsAccelerometer;
    public static bool HasGyroscope => SystemInfo.supportsGyroscope;
    public static bool HasMagnetometer
    {
        get
        {
#if UNITY_IOS
            // iOS通过CMMotionManager检测
            return true; // iOS设备基本都有磁力计
#elif UNITY_ANDROID
            // Android需要通过输入系统检测
            return Attitude.current != null;
#else
            return false;
#endif
        }
    }

    /// <summary>
    /// 启用所有传感器并设置采样频率
    /// </summary>
    public static void EnableSensors(float targetHz = 100f)
    {
        // 旧版API（Unity 2019及以前）
        if (SystemInfo.supportsGyroscope)
        {
            Input.gyro.enabled = true;
            Input.gyro.updateInterval = 1f / targetHz;
        }

        // 新版 Input System API（推荐）
        InputSystem.EnableDevice(Accelerometer.current);
        InputSystem.EnableDevice(Gyroscope.current);
        InputSystem.EnableDevice(GravitySensor.current);
        InputSystem.EnableDevice(AttitudeSensor.current);

        // 设置采样频率
        if (Accelerometer.current != null)
            InputSystem.SetPollingFrequency(targetHz);
    }
}
```

## 二、原始数据处理与噪声滤波

### 2.1 低通滤波（去除高频抖动）

```csharp
using UnityEngine;

/// <summary>
/// 传感器数据低通滤波器
/// 适用于：去除手持抖动的高频噪声
/// </summary>
public class LowPassFilter
{
    private Vector3 _filteredValue;
    private float _alpha; // 0~1：越大响应越快，越小越平滑

    /// <summary>
    /// 构造低通滤波器
    /// </summary>
    /// <param name="cutoffHz">截止频率（Hz），如5Hz截止意味着高于5Hz的抖动被滤除</param>
    /// <param name="sampleRateHz">传感器采样率（Hz）</param>
    public LowPassFilter(float cutoffHz = 5f, float sampleRateHz = 100f)
    {
        float rc = 1f / (2f * Mathf.PI * cutoffHz);
        float dt = 1f / sampleRateHz;
        _alpha = dt / (rc + dt);
    }

    public Vector3 Filter(Vector3 raw)
    {
        _filteredValue = Vector3.Lerp(_filteredValue, raw, _alpha);
        return _filteredValue;
    }

    public void Reset(Vector3 initialValue)
    {
        _filteredValue = initialValue;
    }
}

/// <summary>
/// 高通滤波器
/// 适用于：从加速度计数据中分离动态加速度（去除静态重力分量）
/// </summary>
public class HighPassFilter
{
    private Vector3 _lastRaw;
    private Vector3 _lastFiltered;
    private float _alpha;

    public HighPassFilter(float cutoffHz = 2f, float sampleRateHz = 100f)
    {
        float rc = 1f / (2f * Mathf.PI * cutoffHz);
        float dt = 1f / sampleRateHz;
        _alpha = rc / (rc + dt);
    }

    public Vector3 Filter(Vector3 raw)
    {
        _lastFiltered = _alpha * (_lastFiltered + raw - _lastRaw);
        _lastRaw = raw;
        return _lastFiltered;
    }
}

/// <summary>
/// 指数加权移动平均（EWMA）- 兼顾响应速度与平滑度
/// 比固定alpha的低通滤波器更智能：剧烈运动时快响应，静止时强平滑
/// </summary>
public class AdaptiveLowPassFilter
{
    private Vector3 _filtered;
    private float _baseAlpha;
    private float _dynamicThreshold;

    public AdaptiveLowPassFilter(float baseAlpha = 0.1f, float dynamicThreshold = 2f)
    {
        _baseAlpha = baseAlpha;
        _dynamicThreshold = dynamicThreshold;
    }

    public Vector3 Filter(Vector3 raw)
    {
        float magnitude = (raw - _filtered).magnitude;
        // 运动量越大，alpha越大（响应越快）
        float adaptiveAlpha = Mathf.Clamp01(_baseAlpha + magnitude / _dynamicThreshold);
        _filtered = Vector3.Lerp(_filtered, raw, adaptiveAlpha);
        return _filtered;
    }
}
```

### 2.2 中值滤波（去除野值/突刺噪声）

```csharp
/// <summary>
/// 滑动窗口中值滤波器
/// 适用于：消除传感器偶发的野值（如磁干扰导致的突刺）
/// </summary>
public class MedianFilter
{
    private readonly int _windowSize;
    private readonly Queue<Vector3> _window;
    private readonly float[] _tempX, _tempY, _tempZ;

    public MedianFilter(int windowSize = 5)
    {
        _windowSize = windowSize;
        _window = new Queue<Vector3>(windowSize);
        _tempX = new float[windowSize];
        _tempY = new float[windowSize];
        _tempZ = new float[windowSize];
    }

    public Vector3 Filter(Vector3 raw)
    {
        _window.Enqueue(raw);
        if (_window.Count > _windowSize)
            _window.Dequeue();

        int count = 0;
        foreach (var v in _window)
        {
            _tempX[count] = v.x;
            _tempY[count] = v.y;
            _tempZ[count] = v.z;
            count++;
        }

        System.Array.Sort(_tempX, 0, count);
        System.Array.Sort(_tempY, 0, count);
        System.Array.Sort(_tempZ, 0, count);

        int mid = count / 2;
        return new Vector3(_tempX[mid], _tempY[mid], _tempZ[mid]);
    }
}
```

## 三、姿态估计算法（传感器融合）

### 3.1 互补滤波（Complementary Filter）

最简单的传感器融合方法，适合性能敏感场景：

```csharp
/// <summary>
/// 互补滤波器
/// 融合策略：
///   姿态 = α × (陀螺仪积分) + (1-α) × (加速度计计算的绝对倾角)
/// 参数：
///   α 接近1 → 更相信陀螺仪（响应快，但会漂移）
///   α 接近0 → 更相信加速度计（稳定，但运动时噪声大）
///   典型值：α = 0.98
/// </summary>
public class ComplementaryFilter
{
    private float _alpha;
    private Vector3 _eulerAngles; // Roll, Pitch, Yaw (degrees)

    public ComplementaryFilter(float alpha = 0.98f)
    {
        _alpha = alpha;
    }

    public Vector3 Update(Vector3 gyroRadsPerSec, Vector3 accelMs2, float deltaTime)
    {
        // 1. 陀螺仪积分（短期精确，但会漂移）
        Vector3 gyroAngularDelta = gyroRadsPerSec * Mathf.Rad2Deg * deltaTime;
        _eulerAngles += gyroAngularDelta;

        // 2. 加速度计计算绝对倾角（稳定参考，但有噪声）
        float accelPitch = Mathf.Atan2(-accelMs2.x,
            Mathf.Sqrt(accelMs2.y * accelMs2.y + accelMs2.z * accelMs2.z)) * Mathf.Rad2Deg;
        float accelRoll = Mathf.Atan2(accelMs2.y, accelMs2.z) * Mathf.Rad2Deg;

        // 3. 融合：高频用陀螺仪，低频用加速度计
        _eulerAngles.x = _alpha * _eulerAngles.x + (1f - _alpha) * accelPitch; // Pitch
        _eulerAngles.y = _alpha * _eulerAngles.y + (1f - _alpha) * accelRoll;  // Roll
        // Yaw(偏航)无法通过加速度计修正，仅用陀螺仪积分（会漂移）

        return _eulerAngles;
    }
}
```

### 3.2 Madgwick 算法（工业级精度）

```csharp
/// <summary>
/// Madgwick AHRS算法
/// 使用梯度下降法最小化传感器误差，精度远高于互补滤波
/// 参考论文：Madgwick, S. et al. "Estimation of IMU and MARG orientation using a gradient descent algorithm"
/// </summary>
public class MadgwickFilter
{
    // 姿态四元数（w,x,y,z）
    private float _q0 = 1f, _q1 = 0f, _q2 = 0f, _q3 = 0f;

    // 算法增益：β越大收敛越快，但噪声敏感性增加
    // 动态场景推荐：0.01~0.1；静态场景推荐：0.001~0.01
    private float _beta;

    public MadgwickFilter(float beta = 0.033f)
    {
        _beta = beta;
    }

    /// <summary>
    /// 更新姿态估计（仅使用加速度计+陀螺仪，无磁力计）
    /// </summary>
    public Quaternion UpdateIMU(Vector3 gyro, Vector3 accel, float deltaTime)
    {
        float gx = gyro.x, gy = gyro.y, gz = gyro.z;
        float ax = accel.x, ay = accel.y, az = accel.z;

        float recipNorm;
        float s0, s1, s2, s3;
        float qDot0, qDot1, qDot2, qDot3;
        float _2q0, _2q1, _2q2, _2q3, _4q0, _4q1, _4q2, _8q1, _8q2, q0q0, q1q1, q2q2, q3q3;

        // 1. 陀螺仪速率四元数导数
        qDot0 = 0.5f * (-_q1 * gx - _q2 * gy - _q3 * gz);
        qDot1 = 0.5f * (_q0 * gx + _q2 * gz - _q3 * gy);
        qDot2 = 0.5f * (_q0 * gy - _q1 * gz + _q3 * gx);
        qDot3 = 0.5f * (_q0 * gz + _q1 * gy - _q2 * gx);

        // 2. 加速度计有效时计算梯度修正
        if (!((ax == 0f) && (ay == 0f) && (az == 0f)))
        {
            // 归一化加速度计数据
            recipNorm = InvSqrt(ax * ax + ay * ay + az * az);
            ax *= recipNorm;
            ay *= recipNorm;
            az *= recipNorm;

            _2q0 = 2f * _q0; _2q1 = 2f * _q1; _2q2 = 2f * _q2; _2q3 = 2f * _q3;
            _4q0 = 4f * _q0; _4q1 = 4f * _q1; _4q2 = 4f * _q2;
            _8q1 = 8f * _q1; _8q2 = 8f * _q2;
            q0q0 = _q0 * _q0; q1q1 = _q1 * _q1; q2q2 = _q2 * _q2; q3q3 = _q3 * _q3;

            // 梯度（目标函数：g的预测方向与测量方向之差）
            s0 = _4q0 * q2q2 + _2q2 * ax + _4q0 * q1q1 - _2q1 * ay;
            s1 = _4q1 * q3q3 - _2q3 * ax + 4f * q0q0 * _q1 - _2q0 * ay - _4q1 + _8q1 * q1q1 + _8q1 * q2q2 + _4q1 * az;
            s2 = 4f * q0q0 * _q2 + _2q0 * ax + _4q2 * q3q3 - _2q3 * ay - _4q2 + _8q2 * q1q1 + _8q2 * q2q2 + _4q2 * az;
            s3 = 4f * q1q1 * _q3 - _2q1 * ax + 4f * q2q2 * _q3 - _2q2 * ay;

            recipNorm = InvSqrt(s0 * s0 + s1 * s1 + s2 * s2 + s3 * s3);
            s0 *= recipNorm; s1 *= recipNorm; s2 *= recipNorm; s3 *= recipNorm;

            // 梯度修正
            qDot0 -= _beta * s0;
            qDot1 -= _beta * s1;
            qDot2 -= _beta * s2;
            qDot3 -= _beta * s3;
        }

        // 3. 积分更新四元数
        _q0 += qDot0 * deltaTime;
        _q1 += qDot1 * deltaTime;
        _q2 += qDot2 * deltaTime;
        _q3 += qDot3 * deltaTime;

        // 4. 归一化
        recipNorm = InvSqrt(_q0 * _q0 + _q1 * _q1 + _q2 * _q2 + _q3 * _q3);
        _q0 *= recipNorm; _q1 *= recipNorm; _q2 *= recipNorm; _q3 *= recipNorm;

        return new Quaternion(_q1, _q2, _q3, _q0); // Unity四元数格式(x,y,z,w)
    }

    // 快速逆平方根（Quake III 算法移植）
    private static float InvSqrt(float x)
    {
        return 1f / Mathf.Sqrt(x);
        // 注：在IL2CPP下原版位操作可能不如直接调用快，此处使用标准实现
    }

    /// <summary>
    /// 重置姿态到初始状态
    /// </summary>
    public void Reset()
    {
        _q0 = 1f; _q1 = 0f; _q2 = 0f; _q3 = 0f;
    }

    /// <summary>
    /// 获取当前欧拉角（Roll/Pitch/Yaw，角度）
    /// </summary>
    public Vector3 GetEulerAngles()
    {
        float roll  = Mathf.Atan2(2f*(_q0*_q1 + _q2*_q3), 1f - 2f*(_q1*_q1 + _q2*_q2)) * Mathf.Rad2Deg;
        float pitch = Mathf.Asin(Mathf.Clamp(2f*(_q0*_q2 - _q3*_q1), -1f, 1f)) * Mathf.Rad2Deg;
        float yaw   = Mathf.Atan2(2f*(_q0*_q3 + _q1*_q2), 1f - 2f*(_q2*_q2 + _q3*_q3)) * Mathf.Rad2Deg;
        return new Vector3(pitch, yaw, roll);
    }
}
```

## 四、游戏控制器实现

### 4.1 赛车游戏倾斜转向控制器

```csharp
using UnityEngine;
using UnityEngine.InputSystem;

/// <summary>
/// 赛车游戏体感转向控制器
/// 特性：
///   - 自适应零点校准（消除设备握持角度差异）
///   - 死区设置（消除手持微抖导致的误输入）
///   - 灵敏度曲线（线性/二次方/S形曲线可配置）
///   - 倾斜角度钳位（防止极端姿态下的过度转向）
/// </summary>
public class TiltSteeringController : MonoBehaviour
{
    [Header("灵敏度设置")]
    [SerializeField, Range(0.5f, 3f)] private float _sensitivity = 1.5f;
    [SerializeField, Range(0f, 0.15f)] private float _deadZone = 0.05f;   // 死区（归一化）
    [SerializeField, Range(10f, 60f)] private float _maxTiltDegrees = 35f; // 最大有效倾角

    [Header("平滑设置")]
    [SerializeField, Range(0f, 1f)] private float _smoothing = 0.3f;       // 输入平滑量
    [SerializeField] private AnimationCurve _responseCurve =
        AnimationCurve.EaseInOut(0, 0, 1, 1);                              // 响应曲线

    // 传感器
    private MadgwickFilter _madgwick;
    private LowPassFilter _accelFilter;
    private LowPassFilter _gyroFilter;

    // 零点参考（校准基准）
    private Quaternion _calibrationOffset = Quaternion.identity;

    // 输出值
    private float _currentSteerInput;  // -1~1
    private float _targetSteerInput;
    private bool _isCalibrated = false;

    // 陀螺仪漂移补偿
    private Vector3 _gyroCalibrationBias;
    private int _calibrationSamples = 0;
    private const int GYRO_CALIBRATION_FRAMES = 60; // 静止1秒采样校准陀螺仪零偏

    void Awake()
    {
        _madgwick = new MadgwickFilter(beta: 0.05f);
        _accelFilter = new LowPassFilter(cutoffHz: 10f, sampleRateHz: 100f);
        _gyroFilter = new LowPassFilter(cutoffHz: 20f, sampleRateHz: 100f);

        SensorCapability.EnableSensors(100f);
    }

    void OnEnable()
    {
        // 开启时自动开始陀螺仪零偏校准
        StartCoroutine(CalibrateGyroBias());
    }

    private System.Collections.IEnumerator CalibrateGyroBias()
    {
        Vector3 biasSum = Vector3.zero;
        int samples = 0;

        Debug.Log("[TiltSteering] 正在校准陀螺仪零偏，请保持设备静止...");

        while (samples < GYRO_CALIBRATION_FRAMES)
        {
            yield return new WaitForFixedUpdate();
            var gyroData = GetRawGyro();
            biasSum += gyroData;
            samples++;
        }

        _gyroCalibrationBias = biasSum / samples;
        Debug.Log($"[TiltSteering] 陀螺仪零偏校准完成: {_gyroCalibrationBias}");
    }

    void FixedUpdate()
    {
        if (!_isCalibrated) return;

        // 1. 读取原始传感器数据
        Vector3 rawAccel = GetRawAccelerometer();
        Vector3 rawGyro  = GetRawGyro() - _gyroCalibrationBias;

        // 2. 滤波
        Vector3 filteredAccel = _accelFilter.Filter(rawAccel);
        Vector3 filteredGyro  = _gyroFilter.Filter(rawGyro);

        // 3. Madgwick融合估计姿态
        Quaternion attitude = _madgwick.UpdateIMU(filteredGyro, filteredAccel, Time.fixedDeltaTime);

        // 4. 相对于校准零点计算相对姿态
        Quaternion relativeAttitude = Quaternion.Inverse(_calibrationOffset) * attitude;

        // 5. 提取Roll（左右倾斜角度）
        Vector3 euler = relativeAttitude.eulerAngles;
        float rollDegrees = euler.z > 180f ? euler.z - 360f : euler.z; // 映射到 -180~180

        // 6. 归一化并应用死区
        float normalizedRoll = Mathf.Clamp(rollDegrees / _maxTiltDegrees, -1f, 1f);
        normalizedRoll = ApplyDeadZone(normalizedRoll, _deadZone);

        // 7. 应用响应曲线（非线性映射）
        float sign = Mathf.Sign(normalizedRoll);
        float curvedInput = sign * _responseCurve.Evaluate(Mathf.Abs(normalizedRoll));

        // 8. 应用灵敏度
        _targetSteerInput = Mathf.Clamp(curvedInput * _sensitivity, -1f, 1f);

        // 9. 输入平滑（减少颠簸感）
        _currentSteerInput = Mathf.Lerp(_currentSteerInput, _targetSteerInput,
            1f - _smoothing * (1f - Time.fixedDeltaTime * 60f));
    }

    /// <summary>
    /// 手动校准：将当前姿态设为零点
    /// 玩家按下校准按钮时调用
    /// </summary>
    public void Calibrate()
    {
        var accel = _accelFilter.Filter(GetRawAccelerometer());
        var gyro  = _gyroFilter.Filter(GetRawGyro() - _gyroCalibrationBias);
        _calibrationOffset = _madgwick.UpdateIMU(gyro, accel, Time.deltaTime);
        _isCalibrated = true;
        Debug.Log("[TiltSteering] 零点校准完成");
    }

    /// <summary>
    /// 获取当前转向输入值 (-1 左转 ~ 1 右转)
    /// </summary>
    public float GetSteerInput() => _currentSteerInput;

    private float ApplyDeadZone(float value, float deadZone)
    {
        if (Mathf.Abs(value) < deadZone) return 0f;
        // 死区外线性重映射，保证死区边缘不突变
        return Mathf.Sign(value) * (Mathf.Abs(value) - deadZone) / (1f - deadZone);
    }

    private Vector3 GetRawAccelerometer()
    {
        if (Accelerometer.current != null)
            return Accelerometer.current.acceleration.ReadValue();
        return Input.acceleration;
    }

    private Vector3 GetRawGyro()
    {
        if (UnityEngine.InputSystem.Gyroscope.current != null)
            return UnityEngine.InputSystem.Gyroscope.current.angularVelocity.ReadValue();
        if (Input.gyro.enabled)
            return Input.gyro.rotationRateUnbiased;
        return Vector3.zero;
    }
}
```

### 4.2 FPS游戏陀螺仪瞄准辅助

```csharp
/// <summary>
/// FPS陀螺仪辅助瞄准系统
/// 主流手游实现方案（和平精英/荒野行动同类设计）：
///   - 陀螺仪微调：大范围转向用摇杆，精确瞄准用陀螺仪
///   - 激活条件：开镜时自动激活，未开镜时关闭
///   - 速度映射：陀螺仪角速度直接映射为镜头速度
/// </summary>
public class GyroAimAssist : MonoBehaviour
{
    [Header("灵敏度")]
    [SerializeField] private float _horizontalSensitivity = 2.0f;
    [SerializeField] private float _verticalSensitivity = 1.5f;

    [Header("行为设置")]
    [SerializeField] private float _activationThreshold = 0.02f; // 激活陀螺仪的最小角速度 (rad/s)
    [SerializeField] private bool _invertVertical = false;
    [SerializeField] private bool _onlyActiveWhenAiming = true;  // 只在开镜时激活

    // 输出增量
    public float DeltaYaw   { get; private set; }
    public float DeltaPitch { get; private set; }

    private bool _isAiming = false;
    private LowPassFilter _gyroSmooth;
    private Vector3 _previousGyro;

    void Awake()
    {
        _gyroSmooth = new LowPassFilter(cutoffHz: 15f, sampleRateHz: 100f);
        SensorCapability.EnableSensors(100f);
    }

    public void SetAiming(bool isAiming)
    {
        _isAiming = isAiming;
        if (!isAiming)
        {
            DeltaYaw = 0f;
            DeltaPitch = 0f;
        }
    }

    void Update()
    {
        if (_onlyActiveWhenAiming && !_isAiming) return;
        if (!SystemInfo.supportsGyroscope) return;

        // 读取陀螺仪角速度（rad/s）
        Vector3 gyroRate = Vector3.zero;
        if (UnityEngine.InputSystem.Gyroscope.current != null)
            gyroRate = UnityEngine.InputSystem.Gyroscope.current.angularVelocity.ReadValue();
        else if (Input.gyro.enabled)
            gyroRate = Input.gyro.rotationRateUnbiased;

        // 平滑处理
        gyroRate = _gyroSmooth.Filter(gyroRate);

        // 应用激活阈值（静止时不产生输入）
        float yawRate   = Mathf.Abs(gyroRate.y) > _activationThreshold ? gyroRate.y : 0f;
        float pitchRate = Mathf.Abs(gyroRate.x) > _activationThreshold ? gyroRate.x : 0f;

        // 转换为相机旋转增量（角速度 * 时间 * 灵敏度）
        DeltaYaw   = -yawRate   * _horizontalSensitivity * Time.deltaTime * Mathf.Rad2Deg;
        DeltaPitch = (_invertVertical ? 1f : -1f) * pitchRate * _verticalSensitivity * Time.deltaTime * Mathf.Rad2Deg;
    }
}
```

### 4.3 摇晃/敲击手势检测

```csharp
/// <summary>
/// 加速度计手势识别器
/// 支持手势：摇晃、敲击、翻转
/// </summary>
public class AccelerometerGestureDetector : MonoBehaviour
{
    public event System.Action OnShake;        // 摇晃事件
    public event System.Action OnDoubleTap;    // 双击事件
    public event System.Action OnFlipOver;     // 翻转事件（屏幕向下）

    [Header("摇晃检测")]
    [SerializeField] private float _shakeThreshold = 15f;      // 摇晃阈值 (m/s²)
    [SerializeField] private float _shakeCooldown = 0.5f;       // 摇晃冷却时间
    [SerializeField] private int _shakeRequiredCount = 3;        // 连续峰值次数

    [Header("双击检测")]
    [SerializeField] private float _tapThreshold = 20f;
    [SerializeField] private float _doubleTapInterval = 0.3f;

    private HighPassFilter _highPass;   // 去除重力，只保留动态加速度
    private float _lastShakeTime;
    private float _lastTapTime;
    private int _shakePeakCount;
    private float _lastPeakMagnitude;
    private Vector3 _previousFilteredAccel;

    void Awake()
    {
        _highPass = new HighPassFilter(cutoffHz: 2f, sampleRateHz: 100f);
        SensorCapability.EnableSensors(100f);
    }

    void FixedUpdate()
    {
        Vector3 rawAccel = Input.acceleration * 9.81f; // 归一化到 m/s²
        Vector3 dynamicAccel = _highPass.Filter(rawAccel);
        float magnitude = dynamicAccel.magnitude;

        DetectShake(magnitude);
        DetectTap(magnitude);
        DetectFlipOver(rawAccel);

        _previousFilteredAccel = dynamicAccel;
    }

    private void DetectShake(float magnitude)
    {
        if (Time.time - _lastShakeTime < _shakeCooldown) return;

        bool wasPeak = _lastPeakMagnitude > _shakeThreshold;
        bool isPeak  = magnitude > _shakeThreshold;

        // 检测到新的峰值（上升沿）
        if (isPeak && !wasPeak)
        {
            _shakePeakCount++;
            if (_shakePeakCount >= _shakeRequiredCount)
            {
                _shakePeakCount = 0;
                _lastShakeTime = Time.time;
                OnShake?.Invoke();
                Debug.Log("[Gesture] 摇晃检测到！");
            }
        }

        // 超时重置峰值计数
        if (Time.time - _lastShakeTime > 1.0f)
            _shakePeakCount = 0;

        _lastPeakMagnitude = magnitude;
    }

    private void DetectTap(float magnitude)
    {
        if (magnitude > _tapThreshold && _lastPeakMagnitude <= _tapThreshold)
        {
            float now = Time.time;
            if (now - _lastTapTime < _doubleTapInterval)
            {
                OnDoubleTap?.Invoke();
                Debug.Log("[Gesture] 双击检测到！");
            }
            _lastTapTime = now;
        }
    }

    private void DetectFlipOver(Vector3 rawAccel)
    {
        // Y轴重力分量为负表示屏幕朝下
        if (rawAccel.y < -8f) // ~-9.8 m/s²
            OnFlipOver?.Invoke();
    }
}
```

```csharp
/// <summary>
/// 跨平台传感器适配层
/// 统一处理iOS/Android的坐标系差异与陀螺仪方向差异
/// </summary>
public static class SensorPlatformAdapter
{
    /// <summary>
    /// Unity坐标系（左手系）中加速度计的重力方向：Y轴向上为正
    ///
    /// iOS: 加速度计输出直接使用（Unity已做坐标系转换）
    /// Android: 需要根据屏幕方向旋转坐标系
    /// </summary>
    public static Vector3 GetNormalizedAccelerometer()
    {
        Vector3 accel = Input.acceleration;

#if UNITY_ANDROID && !UNITY_EDITOR
        // Android Portrait模式下坐标系修正
        ScreenOrientation orientation = Screen.orientation;
        switch (orientation)
        {
            case ScreenOrientation.LandscapeLeft:
                return new Vector3(-accel.y, accel.x, accel.z);
            case ScreenOrientation.LandscapeRight:
                return new Vector3(accel.y, -accel.x, accel.z);
            case ScreenOrientation.PortraitUpsideDown:
                return new Vector3(-accel.x, -accel.y, accel.z);
            default: // Portrait
                return accel;
        }
#else
        return accel;
#endif
    }

    /// <summary>
    /// 陀螺仪坐标系修正
    /// </summary>
    public static Vector3 GetNormalizedGyroRate()
    {
        if (!Input.gyro.enabled) return Vector3.zero;
        Vector3 rate = Input.gyro.rotationRateUnbiased;

#if UNITY_IOS && !UNITY_EDITOR
        // iOS陀螺仪使用右手坐标系，Unity使用左手系，需要翻转Z轴
        return new Vector3(rate.x, rate.y, -rate.z);
#else
        return rate;
#endif
    }

    /// <summary>
    /// 检测设备是否横屏持握（赛车游戏专用）
    /// </summary>
    public static bool IsLandscapeHeld()
    {
        return Screen.width > Screen.height;
    }
}
```

## 六、性能优化策略

```csharp
/// <summary>
/// 传感器轮询节流器
/// 问题：陀螺仪可达1000Hz，但游戏逻辑通常只需60Hz，频繁读取浪费CPU
/// 解决：独立线程采集，主线程以固定频率消费
/// </summary>
public class SensorThrottler : MonoBehaviour
{
    private volatile Vector3 _latestAccel;
    private volatile Vector3 _latestGyro;
    private System.Threading.Thread _sensorThread;
    private volatile bool _running = false;
    private float _targetRateHz;

    public void StartSampling(float targetRateHz = 100f)
    {
        _targetRateHz = targetRateHz;
        _running = true;
        _sensorThread = new System.Threading.Thread(SensorReadLoop)
        {
            IsBackground = true,
            Priority = System.Threading.ThreadPriority.AboveNormal
        };
        _sensorThread.Start();
    }

    private void SensorReadLoop()
    {
        int sleepMs = (int)(1000f / _targetRateHz);
        while (_running)
        {
            // 注意：Unity的Input类不能在非主线程访问！
            // 实际项目应使用 Input System 的 IInputStateChangeMonitor 接口
            // 或使用平台原生传感器API（Java/ObjC）通过桥接读取
            System.Threading.Thread.Sleep(sleepMs);
        }
    }

    void OnDestroy()
    {
        _running = false;
        _sensorThread?.Join(1000);
    }

    // 主线程安全读取
    public Vector3 GetAccelerometer() => _latestAccel;
    public Vector3 GetGyroRate()      => _latestGyro;
}
```

## 七、最佳实践总结

### 7.1 游戏类型与传感器方案选型

| 游戏类型 | 推荐方案 | 关键参数 |
|---------|---------|---------|
| **赛车游戏** | Madgwick融合 + 倾斜映射 | β=0.05, α_smooth=0.98, 死区15% |
| **FPS辅助瞄准** | 陀螺仪直接映射（无需融合） | 高通滤波3Hz, 灵敏度0.5~2可配置 |
| **体感节奏游戏** | 加速度计峰值检测 | 中值滤波5帧, 阈值动态调整 |
| **AR地平线对齐** | 完整AHRS（含磁力计） | β=0.01（静态精度优先） |
| **飞行游戏** | 互补滤波（轻量） | α=0.98, 截止频率8Hz |

### 7.2 常见问题排查

```
问题1：角度漂移（Drift）
  原因：陀螺仪积分误差累积
  解决：
    ① 增大Madgwick β值（加速加速度计权重）
    ② 启用零速更新（ZUPT）：静止时自动修正
    ③ 定期提示用户重新校准

问题2：抖动（Jitter）
  原因：高频噪声未充分滤除
  解决：
    ① 降低LowPassFilter截止频率
    ② 增大输入平滑系数
    ③ 扩大死区范围

问题3：响应延迟
  原因：过度滤波导致的相位滞后
  解决：
    ① 使用自适应滤波器（低速高平滑，高速低平滑）
    ② 预测补偿（Kalman预测步骤）

问题4：iOS/Android表现差异
  原因：底层传感器硬件与驱动差异
  解决：
    ① 每个平台独立调参
    ② 提供用户可调节的灵敏度选项
    ③ 必要时屏蔽特定低端机型的陀螺仪功能
```

### 7.3 用户体验设计原则

- **必须提供退出选项**：始终保留虚拟摇杆备用输入方式
- **灵敏度可调节**：至少提供低/中/高三档预设
- **一键校准**：游戏暂停界面提供重新校准入口
- **低电量降级**：电量低于20%时自动降低传感器采样频率
- **横竖屏自适应**：坐标系转换随屏幕方向自动更新
- **热身提示**：陀螺仪冷启动的前2~3秒数据不稳定，延迟启用体感控制
