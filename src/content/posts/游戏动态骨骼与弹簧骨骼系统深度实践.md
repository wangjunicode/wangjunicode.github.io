# 游戏动态骨骼与弹簧骨骼系统深度实践

## 一、为什么需要动态骨骼？

在 3A 游戏和手游中，角色头发飘动、裙摆晃动、耳坠/装饰品惯性摆动是提升角色质感的关键表现。这类效果依赖**动态骨骼（Dynamic Bone / Spring Bone）系统**——它不是靠美术手工 K 帧，而是在运行时用**物理弹簧模型**实时驱动骨骼链，让骨骼随角色运动自然响应惯性、重力和碰撞。

### 1.1 静态骨骼 vs 动态骨骼

| 维度 | 静态骨骼（手 K 动画） | 动态骨骼（物理驱动） |
|------|----------------------|----------------------|
| 表现 | 固定，不受运动影响 | 自然惯性响应 |
| 制作成本 | 高，每个动作都需处理 | 低，参数调好一劳永逸 |
| 多样性 | 有限 | 角色不同速度下表现不同 |
| 性能 | 极低 | 需 CPU/GPU 计算，需优化 |

### 1.2 常见应用场景

- 角色头发（分层骨骼链）
- 裙摆、披风、尾巴（多链并行）
- 耳环、饰品（短链，强弹性）
- 布甲下摆、法袍（宽幅多链）
- 尾部特效骨骼（宠物、坐骑附件）

## 二、弹簧骨骼物理原理

### 2.1 质点弹簧模型

每根动态骨骼链可建模为一串**质点**（Particle）由弹簧连接：

```
Root (固定) --弹簧-- P1 --弹簧-- P2 --弹簧-- P3 (末端)
```

每帧对每个质点执行：

```
加速度 = (弹簧力 + 重力 + 阻力) / 质量
速度  += 加速度 * dt
位置  += 速度 * dt
```

### 2.2 约束求解

原始物理模拟会导致骨骼伸长/压缩，需要对骨骼长度进行**距离约束**（Distance Constraint）：

```csharp
// 约束：保持骨骼长度不变
float currentLen = Vector3.Distance(child.position, parent.position);
float targetLen   = restLength; // 美术设置的静止长度
float correction  = (currentLen - targetLen) / currentLen;
child.position   -= (child.position - parent.position) * correction * 0.5f;
parent.position  += (child.position - parent.position) * correction * 0.5f;
```

### 2.3 位置约束（Verlet 积分）

更稳定的方案是 **Verlet 积分**，直接存储上一帧位置而非速度：

```csharp
Vector3 velocity = currentPos - prevPos;         // 隐式速度
Vector3 nextPos  = currentPos
                 + velocity * (1f - damping)      // 阻尼
                 + gravity * dt * dt;             // 重力
prevPos = currentPos;
currentPos = nextPos;
```

Verlet 积分天然稳定，不会因大步长发散，是弹簧骨骼的首选积分方法。

## 三、DynamicBone 核心算法解析

Unity Asset Store 上的 DynamicBone 是最广泛使用的动态骨骼插件，其核心流程：

### 3.1 数据结构

```csharp
[System.Serializable]
public class Particle
{
    public Transform transform;
    public int       parentIndex;
    
    public Vector3 position;       // 当前世界坐标
    public Vector3 prevPosition;   // 上一帧世界坐标（Verlet）
    public Vector3 initLocalPos;   // 静止局部坐标
    public Quaternion initLocalRot; // 静止局部旋转
    
    public float damping;    // 阻尼 [0,1]
    public float elasticity; // 弹性/刚度 [0,1]
    public float stiffness;  // 硬度：向静止姿态回弹 [0,1]
    public float inert;      // 惯性 [0,1]
    public float radius;     // 碰撞半径
    public float boneLength; // 与父节点距离
}
```

### 3.2 每帧更新流程

```csharp
void UpdateParticles1(float timeVar)
{
    // 1. 计算本帧根骨骼的移动量（用于惯性传递）
    Vector3 move = m_RootBone.position - m_PrevRootPos;
    
    foreach (var p in m_Particles)
    {
        if (p.parentIndex >= 0)
        {
            // Verlet 积分
            Vector3 v = p.position - p.prevPosition;
            Vector3 rmove = move * p.inert;          // 根运动惯性
            
            p.prevPosition = p.position + rmove;
            
            // 重力 + 外力
            Vector3 force = m_Gravity;
            force         += m_Force;
            
            p.position += v * (1f - p.damping) + force * timeVar;
        }
        else
        {
            // 根粒子：直接跟随骨骼
            p.prevPosition = p.position;
            p.position     = p.transform.position;
        }
    }
}

void UpdateParticles2(float timeVar)
{
    for (int i = 1; i < m_Particles.Count; i++)
    {
        var p  = m_Particles[i];
        var p0 = m_Particles[p.parentIndex];
        
        // 刚度：向初始局部旋转回弹
        if (p.stiffness > 0)
        {
            Matrix4x4 pm = p0.transform.localToWorldMatrix;
            // 计算弹性目标点（父骨骼局部空间的初始位置）
            Vector3 restPos = pm.MultiplyPoint3x4(p.initLocalPos);
            p.position      = Vector3.Lerp(p.position, restPos, p.stiffness * timeVar);
        }
        
        // 距离约束：维持骨骼长度
        Vector3 d = p0.position - p.position;
        float   len = d.magnitude;
        if (len > 0)
            p.position += d * (1f - p.boneLength / len);
        
        // 碰撞检测（球/胶囊碰撞体）
        foreach (var c in m_Colliders)
            c.Collide(ref p.position, p.radius);
    }
}
```

## 四、自研轻量级弹簧骨骼框架

生产环境中，DynamicBone 有以下痛点：
- 价格较高，不开源
- 大量角色同屏时性能开销不可控
- 缺少 GPU 加速和 Burst/Job 支持

下面实现一个支持 **Unity Job System + Burst** 的轻量级弹簧骨骼系统。

### 4.1 核心数据结构（NativeArray 友好）

```csharp
// 粒子状态，使用 Blittable 结构体以支持 Burst

public struct SpringParticle
{
    public float3 position;
    public float3 prevPosition;
    public float3 initLocalPos;   // 静止局部坐标（相对父节点）
    public float  damping;
    public float  stiffness;
    public float  gravity;
    public float  boneLength;
    public int    parentIndex;    // -1 = 根节点
}

// 骨骼链定义（不可变）
public struct SpringChainDef
{
    public int startIndex;  // 在粒子数组中的起始下标
    public int length;      // 链的粒子数
}
```

### 4.2 Burst Job 实现物理积分

```csharp
using Unity.Burst;
using Unity.Collections;
using Unity.Jobs;
using Unity.Mathematics;

[BurstCompile]
public struct SpringPhysicsJob : IJobParallelFor
{
    public NativeArray<SpringParticle> Particles;
    
    [ReadOnly] public NativeArray<float3> RootPositions;   // 根骨骼当前世界坐标
    [ReadOnly] public NativeArray<float3> RootPrevPos;     // 根骨骼上一帧坐标
    
    public float DeltaTime;
    public float3 GlobalGravity;
    public float3 ExternalForce;
    
    // IJobParallelFor：每个粒子独立计算 Verlet 积分（不含约束）
    public void Execute(int i)
    {
        var p = Particles[i];
        if (p.parentIndex < 0)
        {
            // 根粒子直接跟踪骨骼
            p.prevPosition = p.position;
            p.position     = RootPositions[i];
            Particles[i]   = p;
            return;
        }
        
        // 惯性：跟随根骨骼移动
        float3 rootMove = RootPositions[0] - RootPrevPos[0];
        p.prevPosition += rootMove * 0.8f; // inert = 0.8
        
        // Verlet 积分
        float3 velocity = p.position - p.prevPosition;
        float3 force    = (GlobalGravity + ExternalForce) * DeltaTime * DeltaTime;
        
        p.prevPosition = p.position;
        p.position     = p.position
                        + velocity * (1f - p.damping)
                        + force;
        
        Particles[i] = p;
    }
}
```

### 4.3 约束修正（串行，需要父子顺序）

```csharp
// 约束修正必须串行，因为存在父子依赖
public struct SpringConstraintJob : IJob
{
    public NativeArray<SpringParticle> Particles;
    [ReadOnly] public NativeArray<float4x4> ParentWorldMatrices; // 父骨骼世界矩阵
    public float DeltaTime;
    
    public void Execute()
    {
        for (int i = 1; i < Particles.Length; i++)
        {
            var p  = Particles[i];
            var p0 = Particles[p.parentIndex];
            
            // ① 刚度：向静止姿态回弹
            if (p.stiffness > 0.001f)
            {
                // 父骨骼世界空间中的静止位置
                float3 restWorld = math.transform(ParentWorldMatrices[p.parentIndex], p.initLocalPos);
                p.position = math.lerp(p.position, restWorld, p.stiffness * DeltaTime * 60f);
            }
            
            // ② 距离约束：保持骨骼长度
            float3 dir = p0.position - p.position;
            float  len = math.length(dir);
            if (len > 0.0001f)
            {
                p.position += dir * (1f - p.boneLength / len);
            }
            
            Particles[i] = p;
        }
    }
}
```

### 4.4 管理器：调度 Job 并回写 Transform

```csharp
using System.Collections.Generic;
using UnityEngine;
using Unity.Collections;
using Unity.Jobs;

public class SpringBoneManager : MonoBehaviour
{
    [Header("物理参数")]
    public Vector3 gravity      = new Vector3(0, -9.8f, 0);
    public Vector3 externalForce = Vector3.zero;
    
    // 运行时数据
    private NativeArray<SpringParticle> m_Particles;
    private NativeArray<float4x4>       m_ParentMatrices;
    private NativeArray<float3>         m_RootCurrPos;
    private NativeArray<float3>         m_RootPrevPos;
    
    private List<Transform> m_Transforms = new List<Transform>();
    private JobHandle        m_JobHandle;
    
    void Start() => InitializeParticles();
    
    void LateUpdate()
    {
        // 等上帧 Job 完成
        m_JobHandle.Complete();
        
        // 从 Job 结果写回 Transform（主线程）
        ApplyParticlesToBones();
        
        // 更新父骨骼矩阵（每帧刷新）
        for (int i = 0; i < m_Transforms.Count; i++)
        {
            m_ParentMatrices[i] = m_Transforms[i] != null
                ? (float4x4)m_Transforms[i].localToWorldMatrix
                : float4x4.identity;
        }
        
        // 调度本帧物理
        float dt = Time.deltaTime;
        
        var physJob = new SpringPhysicsJob
        {
            Particles      = m_Particles,
            RootPositions  = m_RootCurrPos,
            RootPrevPos    = m_RootPrevPos,
            DeltaTime      = dt,
            GlobalGravity  = gravity,
            ExternalForce  = externalForce
        };
        
        var constraintJob = new SpringConstraintJob
        {
            Particles           = m_Particles,
            ParentWorldMatrices = m_ParentMatrices,
            DeltaTime           = dt
        };
        
        // 物理积分并行，约束串行等待积分完成
        var physHandle       = physJob.Schedule(m_Particles.Length, 32);
        m_JobHandle          = constraintJob.Schedule(physHandle);
    }
    
    private void ApplyParticlesToBones()
    {
        for (int i = 0; i < m_Transforms.Count; i++)
        {
            if (m_Transforms[i] == null) continue;
            var p = m_Particles[i];
            
            // 通过与父节点的方向差来反推旋转
            int parentIdx = p.parentIndex;
            if (parentIdx >= 0)
            {
                var parent = m_Transforms[parentIdx];
                Vector3 targetDir = ((Vector3)p.position - parent.position).normalized;
                Vector3 restDir   = parent.TransformDirection(m_Transforms[i].localPosition.normalized);
                
                Quaternion rot = Quaternion.FromToRotation(restDir, targetDir);
                parent.rotation = rot * parent.rotation;
            }
        }
    }
    
    private void InitializeParticles()
    {
        // 遍历子骨骼，构建粒子数组
        // 实际项目中需递归收集并建立父子索引
        // 此处为示意，省略完整收集逻辑
        m_Particles       = new NativeArray<SpringParticle>(m_Transforms.Count, Allocator.Persistent);
        m_ParentMatrices  = new NativeArray<float4x4>(m_Transforms.Count, Allocator.Persistent);
        m_RootCurrPos     = new NativeArray<float3>(m_Transforms.Count, Allocator.Persistent);
        m_RootPrevPos     = new NativeArray<float3>(m_Transforms.Count, Allocator.Persistent);
    }
    
    void OnDestroy()
    {
        m_JobHandle.Complete();
        if (m_Particles.IsCreated)     m_Particles.Dispose();
        if (m_ParentMatrices.IsCreated) m_ParentMatrices.Dispose();
        if (m_RootCurrPos.IsCreated)   m_RootCurrPos.Dispose();
        if (m_RootPrevPos.IsCreated)   m_RootPrevPos.Dispose();
    }
}
```

## 五、碰撞体系统

弹簧骨骼若不处理碰撞，头发会穿透身体。常用碰撞体：

### 5.1 球形碰撞体（最常用）

```csharp
public static bool SphereCollide(ref Vector3 particlePos, float particleRadius,
                                  Vector3 sphereCenter, float sphereRadius)
{
    Vector3 delta  = particlePos - sphereCenter;
    float   dist   = delta.magnitude;
    float   minDist = particleRadius + sphereRadius;
    
    if (dist < minDist && dist > 0.0001f)
    {
        // 将粒子推出碰撞球表面
        particlePos = sphereCenter + delta / dist * minDist;
        return true;
    }
    return false;
}
```

### 5.2 胶囊碰撞体

```csharp
public static bool CapsuleCollide(ref Vector3 particlePos, float particleRadius,
                                   Vector3 capsuleA, Vector3 capsuleB, float capsuleRadius)
{
    // 找粒子在线段上的最近点
    Vector3 ab      = capsuleB - capsuleA;
    float   t       = Vector3.Dot(particlePos - capsuleA, ab) / ab.sqrMagnitude;
    t = Mathf.Clamp01(t);
    Vector3 closest = capsuleA + ab * t;
    
    return SphereCollide(ref particlePos, particleRadius, closest, capsuleRadius);
}
```

### 5.3 平面碰撞体（防止穿地）

```csharp
public static bool PlaneCollide(ref Vector3 particlePos, float particleRadius,
                                 Vector3 planeNormal, float planeDistance)
{
    float dist = Vector3.Dot(particlePos, planeNormal) - planeDistance;
    if (dist < particleRadius)
    {
        particlePos += planeNormal * (particleRadius - dist);
        return true;
    }
    return false;
}
```

## 六、性能优化策略

### 6.1 距离 LOD：按角色距离降频

```csharp
public class SpringBoneLOD : MonoBehaviour
{
    [Header("LOD 距离配置")]
    public float distanceLOD1 = 10f;  // 10m 内：60fps
    public float distanceLOD2 = 20f;  // 20m 内：30fps
    public float distanceLOD3 = 40f;  // 40m 内：15fps（可关闭模拟）
    
    private Camera   m_MainCamera;
    private float    m_UpdateInterval;
    private float    m_Timer;
    
    void Start()
    {
        m_MainCamera = Camera.main;
    }
    
    void Update()
    {
        float dist = Vector3.Distance(transform.position, m_MainCamera.transform.position);
        
        if      (dist < distanceLOD1) m_UpdateInterval = 0f;          // 每帧
        else if (dist < distanceLOD2) m_UpdateInterval = 1f / 30f;
        else if (dist < distanceLOD3) m_UpdateInterval = 1f / 15f;
        else                          m_UpdateInterval = -1f;          // 禁用
        
        // 控制 SpringBoneManager enabled 状态
        GetComponent<SpringBoneManager>().enabled = (m_UpdateInterval >= 0f);
    }
}
```

### 6.3 固定步长：避免帧率抖动

```csharp
// 使用固定物理步长（类似 FixedUpdate），避免帧率不稳定导致抖动
private const float FIXED_STEP = 1f / 60f;
private float m_Accumulator = 0f;

void LateUpdate()
{
    m_Accumulator += Time.deltaTime;
    
    int steps = 0;
    while (m_Accumulator >= FIXED_STEP && steps < 3) // 最多补帧3次
    {
        SimulateStep(FIXED_STEP);
        m_Accumulator -= FIXED_STEP;
        steps++;
    }
    
    // 超出限制时丢弃累积时间，防止螺旋下降
    if (steps >= 3)
        m_Accumulator = 0f;
}
```

## 七、Wind 外力系统

```csharp
[CreateAssetMenu(menuName = "Game/Spring Bone Wind Config")]
public class WindConfig : ScriptableObject
{
    public float strength    = 1f;
    public float frequency   = 0.5f;    // 风速频率
    public float turbulence  = 0.3f;    // 随机扰动
    public Vector3 direction = Vector3.right;
}

public class SpringBoneWindDriver : MonoBehaviour
{
    public WindConfig config;
    
    private float m_TimeOffset;
    
    void Start()
    {
        m_TimeOffset = Random.value * 100f; // 错开不同角色的相位
    }
    
    public Vector3 GetWindForce(Vector3 particleWorldPos)
    {
        float t = Time.time * config.frequency + m_TimeOffset;
        
        // 使用 Perlin 噪声模拟自然风的不规则性
        float noise = Mathf.PerlinNoise(
            particleWorldPos.x * 0.1f + t,
            particleWorldPos.z * 0.1f + t * 0.7f
        );
        
        float windStrength = config.strength * (0.5f + noise * config.turbulence);
        return config.direction.normalized * windStrength;
    }
}
```

## 十、最佳实践总结

### 10.1 参数调节指南

| 参数 | 头发 | 裙摆 | 耳饰 | 尾巴 |
|------|------|------|------|------|
| Damping（阻尼） | 0.2 | 0.15 | 0.3 | 0.1 |
| Stiffness（刚度） | 0.1 | 0.05 | 0.2 | 0.08 |
| Gravity（重力缩放） | 1.0 | 0.8 | 0.5 | 1.2 |
| Elasticity（弹性） | 0.1 | 0.2 | 0.3 | 0.15 |

### 10.2 常见问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 骨骼抖动发散 | 步长过大，刚度过高 | 减小步长，使用固定 60fps 子步 |
| 头发穿透身体 | 碰撞体数量不足 | 在胸部/肩部添加碰撞球 |
| 运动突然停止时骨骼飞出 | 惯性参数过大 | 降低 inert 参数（< 0.5） |
| 静止时姿态偏移 | 重力与刚度不平衡 | 增大 stiffness，或调整 gravity dir |
| 多角色性能问题 | 每帧 Transform 回写 | 使用距离 LOD + Job System 批处理 |

### 10.3 架构建议

```
SpringBoneSystem（全局单例）
├── 统一 NativeArray 粒子池
├── Job 调度器（Burst 加速）
├── LOD 管理器（距离/可见性）
├── WindDriver（全局风场）
└── ColliderPool（碰撞体池）
    ├── SphereCollider
    ├── CapsuleCollider
    └── PlaneCollider
```

> **性能目标**：100 个角色各 30 粒子（3000 粒子总量），使用 Burst Job 后在中端手机应控制在 **0.3ms 以内**。

---

## 结语

动态骨骼是提升角色表现力性价比极高的技术。从最简单的 Verlet 积分到 Job System Burst 加速，再到合理的 LOD 策略，一个完善的弹簧骨骼系统能让角色在各种运动状态下都拥有自然流畅的二级动画效果。建议在项目初期就完成基础框架搭建，为后续角色扩展打好基础。
