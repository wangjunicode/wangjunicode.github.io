# 游戏客户端性能压测与自动化分析框架：从基准测试到线上监控完全指南

## 一、为什么需要系统化性能测试

### 1.1 手工测试的局限性

```
传统性能测试痛点：
❌ 依赖人工目测帧率，主观误差大
❌ 每次版本迭代重复测同一场景，浪费人力
❌ 发现性能退化时已经历数十次提交，难以定位
❌ 不同测试设备/时间/环境结果不可比较
❌ 线上真实设备性能与开发机差异巨大，测试不代表真实
❌ 发热/降频问题难以在短期测试中暴露

系统化性能测试目标：
✅ 自动化：CI/CD流水线中自动运行性能测试
✅ 可比较：每次构建的性能数据可横向对比
✅ 可追踪：性能退化精确到具体提交/PR
✅ 多维度：FPS / CPU / GPU / 内存 / 温度全覆盖
✅ 真实设备：在目标机型上自动化运行
✅ 报警机制：性能低于基线自动阻断合入
```

### 1.2 性能测试金字塔

```
        ▲ 线上真实监控（Firebase/AppFlyer数据）
       ▲ ▲ 真实设备自动化测试（Android Farm/iOS Farm）
      ▲ ▲ ▲ Editor内集成测试（Unity Test Runner + Profiler）
     ▲ ▲ ▲ ▲ 单元性能基准（microbenchmark，Burst/Math）
```

```csharp
// PerformanceBenchmarkBase.cs
using System;
using System.Collections;
using System.Collections.Generic;
using NUnit.Framework;
using Unity.PerformanceTesting;
using UnityEngine;
using UnityEngine.TestTools;

/// <summary>
/// 性能基准测试基类，所有性能测试继承此类
/// </summary>
public abstract class PerformanceBenchmarkBase
{
    protected const int WARMUP_FRAMES = 30;   // 预热帧数
    protected const int MEASURE_FRAMES = 120; // 测量帧数（2秒@60fps）

    /// <summary>
    /// 测量一段代码的执行耗时（微秒级）
    /// </summary>
    protected IEnumerator MeasureFrameTime(string testName, Action setup = null, Action teardown = null)
    {
        setup?.Invoke();

        // 预热阶段
        for (int i = 0; i < WARMUP_FRAMES; i++)
            yield return null;

        // 正式测量
        using (Measure.Frames()
            .WarmupCount(WARMUP_FRAMES)
            .MeasurementCount(MEASURE_FRAMES)
            .SampleGroup(new SampleGroup(testName, SampleUnit.Millisecond))
            .Scope())
        {
            for (int i = 0; i < MEASURE_FRAMES; i++)
                yield return null;
        }

        teardown?.Invoke();
    }

    /// <summary>
    /// 测量内存分配情况
    /// </summary>
    protected IEnumerator MeasureMemoryAllocation(string testName, Action workload)
    {
        yield return null;

        using (Measure.Method(workload)
            .WarmupCount(5)
            .IterationsPerMeasurement(100)
            .MeasurementCount(10)
            .GC()
            .SampleGroup(new SampleGroup(testName + "_GC", SampleUnit.Byte))
            .Run())
        {
        }
    }

    /// <summary>
    /// 自定义指标记录
    /// </summary>
    protected void RecordCustomMetric(string groupName, double value, SampleUnit unit = SampleUnit.Millisecond)
    {
        Measure.Custom(new SampleGroup(groupName, unit), value);
    }
}
```

### 2.2 自动化性能测试流程管理器

```csharp
// PerformanceTestRunner.cs - 自动化运行框架
using System;
using System.Collections;
using System.Collections.Generic;
using System.IO;
using UnityEngine;
using Newtonsoft.Json;

[System.Serializable]
public class PerformanceResult
{
    public string testName;
    public string buildVersion;
    public string deviceModel;
    public string gpuName;
    public int ramMB;
    public DateTime timestamp;

    // 帧率指标
    public float avgFPS;
    public float minFPS;
    public float p5FPS;  // 5th percentile（低帧率保证）
    public float p95FPS; // 95th percentile

    // 帧时间指标
    public float avgFrameTimeMs;
    public float maxFrameTimeMs;
    public float stutterRatio; // 帧时间 > 2*avgFrameTime 的比例

    // 内存指标
    public float peakRamMB;
    public float avgRamMB;
    public int gcCollections;
    public float gcAllocMB;

    // GPU指标（如果可用）
    public float gpuFrameTimeMs;
    public float gpuMemoryMB;

    // 温度（Android）
    public float cpuTemperature;
    public bool thermalThrottled;
}

public class PerformanceTestRunner : MonoBehaviour
{
    [Header("测试配置")]
    public List<PerformanceTestScene> testScenes;
    public int warmupSeconds = 10;
    public int measureSeconds = 60;
    public string outputDirectory = "PerformanceResults";

    [Header("基线配置")]
    public TextAsset performanceBaseline; // 基线JSON

    private List<PerformanceResult> _results = new List<PerformanceResult>();
    private FrameTimeSampler _frameSampler;

    void Start()
    {
        _frameSampler = new FrameTimeSampler();
        StartCoroutine(RunAllTests());
    }

    IEnumerator RunAllTests()
    {
        Debug.Log("=== 开始性能压测 ===");
        Debug.Log($"设备: {SystemInfo.deviceModel} | GPU: {SystemInfo.graphicsDeviceName}");

        foreach (var testScene in testScenes)
        {
            yield return RunSingleTest(testScene);
        }

        // 生成报告
        GenerateReport();

        // 对比基线
        bool passed = CompareWithBaseline();

        // CI模式下，退出并返回状态码
#if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
#else
        Application.Quit(passed ? 0 : 1);
#endif
    }

    IEnumerator RunSingleTest(PerformanceTestScene testScene)
    {
        Debug.Log($"测试场景: {testScene.sceneName}");

        // 加载场景
        yield return UnityEngine.SceneManagement.SceneManager.LoadSceneAsync(testScene.sceneName);
        yield return null;

        // 预热
        Debug.Log($"预热中 ({warmupSeconds}s)...");
        _frameSampler.Reset();
        float warmupEnd = Time.time + warmupSeconds;
        while (Time.time < warmupEnd) yield return null;

        // 正式测量
        Debug.Log($"正式测量中 ({measureSeconds}s)...");
        _frameSampler.Reset();
        _frameSampler.StartRecording();

        float measureEnd = Time.time + measureSeconds;
        while (Time.time < measureEnd)
        {
            _frameSampler.Update();
            yield return null;
        }

        _frameSampler.StopRecording();

        // 收集结果
        PerformanceResult result = _frameSampler.GetResult();
        result.testName = testScene.sceneName;
        result.buildVersion = Application.version;
        result.deviceModel = SystemInfo.deviceModel;
        result.gpuName = SystemInfo.graphicsDeviceName;
        result.ramMB = SystemInfo.systemMemorySize;
        result.timestamp = DateTime.Now;

        // 采集内存
        result.peakRamMB = UnityEngine.Profiling.Profiler.GetTotalReservedMemoryLong() / (1024f * 1024f);
        result.gcAllocMB = UnityEngine.Profiling.Profiler.GetMonoUsedSizeLong() / (1024f * 1024f);

        // 采集温度（Android）
#if UNITY_ANDROID && !UNITY_EDITOR
        result.cpuTemperature = GetAndroidCpuTemperature();
        result.thermalThrottled = IsThermalThrottled();
#endif

        _results.Add(result);
        Debug.Log($"场景 {testScene.sceneName} 测试完成: 平均FPS={result.avgFPS:F1}, P5={result.p5FPS:F1}");
    }

    void GenerateReport()
    {
        string json = JsonConvert.SerializeObject(_results, Formatting.Indented);
        string path = Path.Combine(Application.persistentDataPath, outputDirectory,
                                   $"perf_{DateTime.Now:yyyyMMdd_HHmmss}.json");

        Directory.CreateDirectory(Path.GetDirectoryName(path));
        File.WriteAllText(path, json);

        // 同时生成Markdown报告
        GenerateMarkdownReport(path.Replace(".json", ".md"));

        Debug.Log($"性能报告已保存: {path}");
    }

    void GenerateMarkdownReport(string path)
    {
        using (var sw = new StreamWriter(path))
        {
            sw.WriteLine("# 性能压测报告");
            sw.WriteLine($"**生成时间**: {DateTime.Now:yyyy-MM-dd HH:mm:ss}");
            sw.WriteLine($"**设备**: {SystemInfo.deviceModel}");
            sw.WriteLine($"**GPU**: {SystemInfo.graphicsDeviceName}");
            sw.WriteLine();

            sw.WriteLine("| 场景 | 均值FPS | P5 FPS | 最低FPS | 卡顿率 | 峰值内存 |");
            sw.WriteLine("|------|---------|--------|---------|--------|---------|");

            foreach (var r in _results)
            {
                sw.WriteLine($"| {r.testName} | {r.avgFPS:F1} | {r.p5FPS:F1} | {r.minFPS:F1} | {r.stutterRatio:P1} | {r.peakRamMB:F0}MB |");
            }
        }
    }

    bool CompareWithBaseline()
    {
        if (performanceBaseline == null) return true;

        var baseline = JsonConvert.DeserializeObject<List<PerformanceResult>>(performanceBaseline.text);
        bool allPassed = true;

        foreach (var result in _results)
        {
            var baselineResult = baseline.Find(b => b.testName == result.testName);
            if (baselineResult == null) continue;

            // 检查性能退化
            float fpsDegradation = (baselineResult.avgFPS - result.avgFPS) / baselineResult.avgFPS;
            float memoryIncrease = (result.peakRamMB - baselineResult.peakRamMB) / baselineResult.peakRamMB;

            if (fpsDegradation > 0.05f) // FPS下降超过5%
            {
                Debug.LogError($"[性能退化] {result.testName}: 均值FPS从{baselineResult.avgFPS:F1}降至{result.avgFPS:F1}（退化{fpsDegradation:P1}）");
                allPassed = false;
            }

            if (result.p5FPS < baselineResult.p5FPS * 0.9f) // P5 FPS下降超过10%
            {
                Debug.LogError($"[性能退化] {result.testName}: P5 FPS从{baselineResult.p5FPS:F1}降至{result.p5FPS:F1}");
                allPassed = false;
            }

            if (memoryIncrease > 0.1f) // 内存增长超过10%
            {
                Debug.LogWarning($"[内存警告] {result.testName}: 峰值内存从{baselineResult.peakRamMB:F0}MB增至{result.peakRamMB:F0}MB");
            }
        }

        return allPassed;
    }

#if UNITY_ANDROID && !UNITY_EDITOR
    float GetAndroidCpuTemperature()
    {
        try
        {
            using (var file = File.OpenText("/sys/class/thermal/thermal_zone0/temp"))
            {
                float temp = float.Parse(file.ReadToEnd().Trim());
                return temp / 1000f; // millidegrees to degrees
            }
        }
        catch { return -1f; }
    }

    bool IsThermalThrottled()
    {
        // 通过读取CPU频率判断是否降频
        try
        {
            using (var maxFile = File.OpenText("/sys/devices/system/cpu/cpu0/cpufreq/cpuinfo_max_freq"))
            using (var curFile = File.OpenText("/sys/devices/system/cpu/cpu0/cpufreq/scaling_cur_freq"))
            {
                float maxFreq = float.Parse(maxFile.ReadToEnd().Trim());
                float curFreq = float.Parse(curFile.ReadToEnd().Trim());
                return curFreq < maxFreq * 0.8f;
            }
        }
        catch { return false; }
    }
#endif
}

// 帧时间采样器
public class FrameTimeSampler
{
    private List<float> _frameTimes = new List<float>(10000);
    private float _lastTime;
    private bool _recording;

    public void Reset()
    {
        _frameTimes.Clear();
    }

    public void StartRecording()
    {
        _recording = true;
        _lastTime = Time.realtimeSinceStartup;
    }

    public void StopRecording()
    {
        _recording = false;
    }

    public void Update()
    {
        if (!_recording) return;
        float now = Time.realtimeSinceStartup;
        _frameTimes.Add((now - _lastTime) * 1000f);
        _lastTime = now;
    }

    public PerformanceResult GetResult()
    {
        _frameTimes.Sort();
        int n = _frameTimes.Count;

        float avgFrameTime = 0;
        foreach (var ft in _frameTimes) avgFrameTime += ft;
        avgFrameTime /= n;

        // 计算卡顿率（帧时间 > 2倍平均值）
        int stutterFrames = 0;
        foreach (var ft in _frameTimes)
            if (ft > avgFrameTime * 2f) stutterFrames++;

        return new PerformanceResult
        {
            avgFrameTimeMs = avgFrameTime,
            maxFrameTimeMs = _frameTimes[n - 1],
            avgFPS = 1000f / avgFrameTime,
            minFPS = 1000f / _frameTimes[n - 1],
            p5FPS  = 1000f / _frameTimes[(int)(n * 0.95f)],  // P5对应帧时间的P95
            p95FPS = 1000f / _frameTimes[(int)(n * 0.05f)],  // P95对应帧时间的P5
            stutterRatio = (float)stutterFrames / n
        };
    }
}
```

### 3.1 Android性能数据采集脚本

```python
# android_perf_collector.py
# 通过ADB采集Android设备性能数据

import subprocess
import time
import json
import threading
from dataclasses import dataclass, asdict
from datetime import datetime
from typing import List

@dataclass
class AndroidPerfSample:
    timestamp: float
    fps: float
    cpu_total: float       # 总CPU占用率
    cpu_per_core: list     # 各核心占用
    ram_mb: float          # RSS内存（MB）
    gpu_freq_mhz: float    # GPU频率
    cpu_temp: float        # CPU温度（摄氏度）
    battery_temp: float    # 电池温度
    frame_time_ms: float   # 帧时间

class AndroidPerfCollector:
    def __init__(self, package_name: str, device_serial: str = None):
        self.package_name = package_name
        self.device_serial = device_serial
        self.adb = f"adb {'-s ' + device_serial if device_serial else ''}"
        self.samples: List[AndroidPerfSample] = []
        self._running = False

    def adb_cmd(self, cmd: str) -> str:
        full_cmd = f"{self.adb} shell {cmd}"
        try:
            return subprocess.check_output(full_cmd, shell=True, text=True, timeout=5).strip()
        except Exception as e:
            return ""

    def get_pid(self) -> int:
        output = self.adb_cmd(f"pidof {self.package_name}")
        if output:
            return int(output.split()[0])
        return -1

    def get_cpu_usage(self, pid: int) -> tuple:
        # 读取/proc/stat计算CPU占用
        stat1 = self.adb_cmd("cat /proc/stat").split('\n')
        time.sleep(0.1)
        stat2 = self.adb_cmd("cat /proc/stat").split('\n')

        def parse_stat(line):
            parts = line.split()[1:]
            return [int(x) for x in parts[:8]]

        total_cpu = 0.0
        per_core = []

        for i, (s1, s2) in enumerate(zip(stat1, stat2)):
            if not s1.startswith('cpu'): break
            v1 = parse_stat(s1)
            v2 = parse_stat(s2)
            idle_diff = v2[3] - v1[3]
            total_diff = sum(v2) - sum(v1)
            usage = 1.0 - (idle_diff / max(total_diff, 1))
            if i == 0:
                total_cpu = usage * 100
            else:
                per_core.append(round(usage * 100, 1))

        # 进程CPU占用
        proc_stat = self.adb_cmd(f"cat /proc/{pid}/stat").split()

        return total_cpu, per_core

    def get_memory_mb(self, pid: int) -> float:
        output = self.adb_cmd(f"cat /proc/{pid}/status | grep VmRSS")
        if output:
            kb = int(output.split()[1])
            return kb / 1024.0
        return 0.0

    def get_temperatures(self) -> dict:
        temps = {}
        # CPU温度
        for zone in range(10):
            path = f"/sys/class/thermal/thermal_zone{zone}/temp"
            val = self.adb_cmd(f"cat {path} 2>/dev/null")
            if val:
                try:
                    temps[f"zone{zone}"] = int(val) / 1000.0
                except:
                    pass
        return temps

    def get_gpu_frequency(self) -> float:
        # Qualcomm Adreno
        paths = [
            "/sys/class/kgsl/kgsl-3d0/gpuclk",
            "/sys/class/kgsl/kgsl-3d0/devfreq/cur_freq",
        ]
        for path in paths:
            val = self.adb_cmd(f"cat {path} 2>/dev/null")
            if val:
                try:
                    return int(val) / 1_000_000  # Hz to MHz
                except:
                    pass
        return 0.0

    def get_frame_stats(self) -> float:
        """通过SurfaceFlinger获取帧时间"""
        output = self.adb_cmd(f"dumpsys SurfaceFlinger --latency SurfaceView[{self.package_name}]")
        if not output:
            return 0.0

        lines = output.strip().split('\n')
        if len(lines) < 3:
            return 0.0

        # 取最近10帧的帧时间
        frame_times = []
        for line in lines[1:]:
            parts = line.strip().split()
            if len(parts) >= 3:
                try:
                    present = int(parts[2])
                    queue = int(parts[0])
                    frame_time_ns = present - queue
                    if 0 < frame_time_ns < 100_000_000:  # 合理范围：0~100ms
                        frame_times.append(frame_time_ns / 1_000_000)  # ns to ms
                except:
                    pass

        return sum(frame_times) / len(frame_times) if frame_times else 0.0

    def start_collection(self, interval_sec: float = 1.0):
        self._running = True
        pid = self.get_pid()
        if pid < 0:
            raise RuntimeError(f"找不到进程: {self.package_name}")

        print(f"开始采集 PID={pid}，间隔={interval_sec}s")

        def collect_loop():
            while self._running:
                ts = time.time()
                cpu_total, cpu_cores = self.get_cpu_usage(pid)
                ram = self.get_memory_mb(pid)
                temps = self.get_temperatures()
                gpu_freq = self.get_gpu_frequency()
                frame_time = self.get_frame_stats()

                sample = AndroidPerfSample(
                    timestamp=ts,
                    fps=1000.0 / frame_time if frame_time > 0 else 0,
                    cpu_total=cpu_total,
                    cpu_per_core=cpu_cores,
                    ram_mb=ram,
                    gpu_freq_mhz=gpu_freq,
                    cpu_temp=temps.get('zone0', 0),
                    battery_temp=temps.get('zone1', 0),
                    frame_time_ms=frame_time
                )
                self.samples.append(sample)
                time.sleep(interval_sec)

        self._thread = threading.Thread(target=collect_loop, daemon=True)
        self._thread.start()

    def stop_collection(self) -> dict:
        self._running = False
        if hasattr(self, '_thread'):
            self._thread.join(timeout=5)

        return self.analyze()

    def analyze(self) -> dict:
        if not self.samples:
            return {}

        fps_list = [s.fps for s in self.samples if s.fps > 0]
        cpu_list = [s.cpu_total for s in self.samples]
        ram_list = [s.ram_mb for s in self.samples]
        temp_list = [s.cpu_temp for s in self.samples if s.cpu_temp > 0]

        fps_sorted = sorted(fps_list)
        n = len(fps_sorted)

        report = {
            "summary": {
                "avg_fps": sum(fps_list) / len(fps_list),
                "min_fps": min(fps_list),
                "p1_fps": fps_sorted[max(0, int(n * 0.01))],
                "p5_fps": fps_sorted[max(0, int(n * 0.05))],
                "avg_cpu_pct": sum(cpu_list) / len(cpu_list),
                "max_cpu_pct": max(cpu_list),
                "avg_ram_mb": sum(ram_list) / len(ram_list),
                "peak_ram_mb": max(ram_list),
                "max_cpu_temp": max(temp_list) if temp_list else 0,
                "thermal_throttle_detected": max(temp_list) > 85 if temp_list else False,
            },
            "samples": [asdict(s) for s in self.samples],
            "device_info": {
                "package": self.package_name,
                "timestamp": datetime.now().isoformat(),
            }
        }

        return report

    time.sleep(10)

    collector.start_collection(interval_sec=0.5)

    print("测试运行中（60秒）...")
    time.sleep(60)

    report = collector.stop_collection()

    # 保存报告
    with open(f"perf_report_{datetime.now():%Y%m%d_%H%M%S}.json", 'w') as f:
        json.dump(report, f, indent=2)

    summary = report["summary"]
    print(f"\n=== 性能分析报告 ===")
    print(f"平均FPS: {summary['avg_fps']:.1f}")
    print(f"P1 FPS:  {summary['p1_fps']:.1f}")
    print(f"P5 FPS:  {summary['p5_fps']:.1f}")
    print(f"CPU占用: {summary['avg_cpu_pct']:.1f}%")
    print(f"内存峰值: {summary['peak_ram_mb']:.0f}MB")
    print(f"CPU最高温度: {summary['max_cpu_temp']:.1f}°C")
    print(f"发热降频: {'是' if summary['thermal_throttle_detected'] else '否'}")
```

### 4.2 性能门控检查脚本

```python
# perf_gate_check.py
import json
import sys
import argparse

def check_performance_gate(result_path, baseline_path, fps_threshold, memory_threshold):
    with open(result_path) as f:
        result = json.load(f)["summary"]
    with open(baseline_path) as f:
        baseline = json.load(f)["summary"]

    failures = []
    warnings = []

    # FPS检查
    avg_fps_degradation = (baseline["avg_fps"] - result["avg_fps"]) / baseline["avg_fps"]
    p5_fps_degradation = (baseline["p5_fps"] - result["p5_fps"]) / baseline["p5_fps"]

    if avg_fps_degradation > fps_threshold:
        failures.append(f"❌ 均值FPS退化 {avg_fps_degradation:.1%}（基线:{baseline['avg_fps']:.1f} → 当前:{result['avg_fps']:.1f}）")

    if p5_fps_degradation > fps_threshold * 1.5:
        failures.append(f"❌ P5 FPS退化 {p5_fps_degradation:.1%}（基线:{baseline['p5_fps']:.1f} → 当前:{result['p5_fps']:.1f}）")

    # 内存检查
    memory_increase = (result["peak_ram_mb"] - baseline["peak_ram_mb"]) / baseline["peak_ram_mb"]
    if memory_increase > memory_threshold:
        failures.append(f"❌ 内存增长 {memory_increase:.1%}（基线:{baseline['peak_ram_mb']:.0f}MB → 当前:{result['peak_ram_mb']:.0f}MB）")

    # 温度检查
    if result.get("thermal_throttle_detected"):
        warnings.append(f"⚠️ 检测到热降频！CPU最高温度：{result['max_cpu_temp']:.1f}°C")

    # 输出结果
    print("\n=== 性能门控检查结果 ===")
    if failures:
        print("\n【失败项】")
        for f in failures:
            print(f)
    if warnings:
        print("\n【警告项】")
        for w in warnings:
            print(w)

    if not failures and not warnings:
        print("✅ 所有性能指标通过检查！")

    if failures:
        print(f"\n门控结果：FAIL（{len(failures)}项未通过）")
        sys.exit(1)
    else:
        print(f"\n门控结果：PASS")
        sys.exit(0)

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--result", required=True)
    parser.add_argument("--baseline", required=True)
    parser.add_argument("--fps-threshold", type=float, default=0.05)
    parser.add_argument("--memory-threshold", type=float, default=0.10)
    args = parser.parse_args()

    check_performance_gate(args.result, args.baseline, args.fps_threshold, args.memory_threshold)
```

## 五、线上性能监控看板

### 5.1 运行时性能上报SDK

```csharp
// RuntimePerfReporter.cs - 集成到游戏中的性能上报SDK
using System;
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.Networking;

public class RuntimePerfReporter : MonoBehaviour
{
    public static RuntimePerfReporter Instance { get; private set; }

    [Header("上报配置")]
    public string reportEndpoint = "https://your-analytics.com/perf";
    public float reportIntervalSeconds = 30f;
    public bool enableInProduction = true;

    // 性能采样窗口
    private CircularBuffer<float> _frameTimeBuffer = new CircularBuffer<float>(300);
    private long _gcCollectionCount;
    private float _peakMemoryMB;

    void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);

        if (enableInProduction || Application.isEditor)
            StartCoroutine(PeriodicReport());
    }

    void Update()
    {
        float frameTimeMs = Time.unscaledDeltaTime * 1000f;
        _frameTimeBuffer.Push(frameTimeMs);

        float currentMemMB = UnityEngine.Profiling.Profiler.GetTotalReservedMemoryLong() / (1024f * 1024f);
        _peakMemoryMB = Mathf.Max(_peakMemoryMB, currentMemMB);
    }

    IEnumerator PeriodicReport()
    {
        while (true)
        {
            yield return new WaitForSecondsRealtime(reportIntervalSeconds);
            yield return StartCoroutine(SendReport());
        }
    }

    IEnumerator SendReport()
    {
        float[] samples = _frameTimeBuffer.ToArray();
        if (samples.Length == 0) yield break;

        System.Array.Sort(samples);
        int n = samples.Length;
        float avg = 0; foreach (var s in samples) avg += s; avg /= n;

        var payload = new Dictionary<string, object>
        {
            ["device_model"] = SystemInfo.deviceModel,
            ["gpu"] = SystemInfo.graphicsDeviceName,
            ["os_version"] = SystemInfo.operatingSystem,
            ["app_version"] = Application.version,
            ["scene"] = UnityEngine.SceneManagement.SceneManager.GetActiveScene().name,
            ["avg_fps"] = 1000f / avg,
            ["p1_fps"] = 1000f / samples[(int)(n * 0.99f)],
            ["p5_fps"] = 1000f / samples[(int)(n * 0.95f)],
            ["avg_frame_ms"] = avg,
            ["max_frame_ms"] = samples[n - 1],
            ["peak_memory_mb"] = _peakMemoryMB,
            ["session_id"] = PlayerPrefs.GetString("session_id", Guid.NewGuid().ToString()),
            ["timestamp"] = DateTimeOffset.UtcNow.ToUnixTimeSeconds()
        };

        string jsonBody = Newtonsoft.Json.JsonConvert.SerializeObject(payload);
        byte[] bodyRaw = System.Text.Encoding.UTF8.GetBytes(jsonBody);

        using (var request = new UnityWebRequest(reportEndpoint, "POST"))
        {
            request.uploadHandler = new UploadHandlerRaw(bodyRaw);
            request.downloadHandler = new DownloadHandlerBuffer();
            request.SetRequestHeader("Content-Type", "application/json");
            yield return request.SendWebRequest();

            if (request.result != UnityWebRequest.Result.Success)
                Debug.LogWarning($"性能上报失败: {request.error}");
        }

        // 重置峰值内存
        _peakMemoryMB = 0;
    }
}

// 轻量级环形缓冲（无GC分配）
public class CircularBuffer<T>
{
    private T[] _buffer;
    private int _head, _count;

    public CircularBuffer(int capacity)
    {
        _buffer = new T[capacity];
    }

    public void Push(T value)
    {
        _buffer[_head % _buffer.Length] = value;
        _head = (_head + 1) % _buffer.Length;
        _count = Mathf.Min(_count + 1, _buffer.Length);
    }

    public T[] ToArray()
    {
        T[] result = new T[_count];
        for (int i = 0; i < _count; i++)
            result[i] = _buffer[(_head - _count + i + _buffer.Length) % _buffer.Length];
        return result;
    }
}
```

## 六、最佳实践总结

### 6.1 性能测试场景设计原则

| 原则 | 说明 |
|------|------|
| **确定性** | 场景随机种子固定，每次复现相同行为 |
| **代表性** | 覆盖游戏最重场景（大量角色、特效、UI） |
| **独立性** | 测试间无互相依赖，资源完全清理 |
| **可重复性** | 同一机器多次运行误差 < 2% |
| **快速性** | 单个测试 < 3分钟，全套 < 30分钟 |

### 6.2 性能基线管理策略

```
基线版本管理：
1. 每个Sprint结束时更新基线（锁定为该Sprint最优性能）
2. 重大架构变更后重新建立基线
3. 基线按设备分档（高中低端各一份）
4. 基线数据存储在版本库中，与代码一起Review

性能门控配置建议：
- 开发分支：FPS退化>10% 警告，>20% 阻断
- 主干分支：FPS退化>5% 警告，>10% 阻断
- 发布分支：FPS退化>3% 阻断，任何退化都需要说明
```

### 6.3 常见性能问题定位流程

```
问题：某次提交后P5 FPS从55→42

定位步骤：
1. git bisect 自动二分查找引入问题的提交
2. 对比两个版本的Profiler数据（CPU/GPU时间线）
3. 重点关注：
   - DrawCall数量变化
   - GC Alloc峰值
   - 物理计算时间
   - 脚本Update耗时（Profiler.BeginSample标记）
4. 使用RenderDoc对比GPU帧捕获
5. 确认修复后更新基线
```
