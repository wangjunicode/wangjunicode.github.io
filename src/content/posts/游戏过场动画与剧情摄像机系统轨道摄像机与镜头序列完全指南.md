# 游戏过场动画与剧情摄像机系统：轨道摄像机与镜头序列完全指南

## 一、过场动画摄像机系统概述

过场动画（Cutscene）是游戏叙事的重要手段。一个完整的过场动画摄像机系统需要：

- **多镜头序列**：按时间轴切换不同摄像机视角
- **平滑过渡**：摄像机切换时的渐入渐出、CrossFade
- **轨道运动**：摄像机沿贝塞尔曲线/样条轨道平滑移动
- **注视跟踪**：自动跟踪目标角色的视线朝向
- **DOF/景深联动**：剧情场景自动调整焦距突出主角
- **与游戏玩法摄像机无缝切换**：过场结束后恢复到游戏视角

本文设计一套**不依赖 Cinemachine** 的轻量自研方案，适合有深度定制需求的项目。

## 四、贝塞尔曲线轨道插值

过场动画中摄像机运动要丝滑，需要贝塞尔或 Catmull-Rom 样条插值：

```csharp
/// <summary>
/// 样条插值工具类
/// </summary>
public static class SplineUtils
{
    /// <summary>
    /// Catmull-Rom 样条插值（通过所有控制点的平滑曲线）
    /// </summary>
    public static Vector3 CatmullRom(
        Vector3 p0, Vector3 p1, Vector3 p2, Vector3 p3,
        float t)
    {
        float t2 = t * t;
        float t3 = t2 * t;

        return 0.5f * (
            (2f * p1) +
            (-p0 + p2) * t +
            (2f * p0 - 5f * p1 + 4f * p2 - p3) * t2 +
            (-p0 + 3f * p1 - 3f * p2 + p3) * t3
        );
    }

    /// <summary>
    /// 在一组控制点构成的 Catmull-Rom 样条上采样位置
    /// </summary>
    /// <param name="points">控制点列表（至少4个）</param>
    /// <param name="t">归一化进度 [0, 1]</param>
    public static Vector3 SampleCatmullRom(List<Vector3> points, float t)
    {
        if (points.Count < 2)
            return points.Count == 1 ? points[0] : Vector3.zero;

        // 添加首尾虚拟控制点（延长曲线使端点平滑）
        var pts = new Vector3[points.Count + 2];
        pts[0] = points[0] + (points[0] - points[1]);
        for (int i = 0; i < points.Count; i++) pts[i + 1] = points[i];
        pts[pts.Length - 1] = points[^1] + (points[^1] - points[^2]);

        // 段数
        int segments = points.Count - 1;
        float scaled = t * segments;
        int segIdx = Mathf.Clamp((int)scaled, 0, segments - 1);
        float localT = scaled - segIdx;

        return CatmullRom(
            pts[segIdx],
            pts[segIdx + 1],
            pts[segIdx + 2],
            pts[segIdx + 3],
            localT
        );
    }

    /// <summary>
    /// 三次贝塞尔曲线（带控制柄）
    /// </summary>
    public static Vector3 Bezier3(
        Vector3 p0, Vector3 p1, Vector3 p2, Vector3 p3, float t)
    {
        float u = 1f - t;
        return u * u * u * p0
             + 3f * u * u * t * p1
             + 3f * u * t * t * p2
             + t * t * t * p3;
    }

    /// <summary>
    /// 缓动函数
    /// </summary>
    public static float Ease(float t, EasingType type)
    {
        return type switch
        {
            EasingType.Linear    => t,
            EasingType.EaseIn    => t * t,
            EasingType.EaseOut   => t * (2f - t),
            EasingType.EaseInOut => t < 0.5f ? 2f * t * t : -1f + (4f - 2f * t) * t,
            EasingType.Bounce    => BounceEase(t),
            EasingType.Spring    => SpringEase(t),
            _                    => t,
        };
    }

    private static float BounceEase(float t)
    {
        if (t < 1f / 2.75f)     return 7.5625f * t * t;
        if (t < 2f / 2.75f)     { t -= 1.5f / 2.75f;   return 7.5625f * t * t + 0.75f;   }
        if (t < 2.5f / 2.75f)   { t -= 2.25f / 2.75f;  return 7.5625f * t * t + 0.9375f; }
        t -= 2.625f / 2.75f;    return 7.5625f * t * t + 0.984375f;
    }

    private static float SpringEase(float t)
    {
        return Mathf.Pow(2f, -10f * t) * Mathf.Sin((t - 0.075f) * (2f * Mathf.PI) / 0.3f) + 1f;
    }
}
```

## 五、镜头行为执行器

每种 ShotType 对应一个行为执行器，运行时驱动摄像机：

```csharp
using System.Collections;
using UnityEngine;
using UnityEngine.Rendering.Universal;

/// <summary>
/// 镜头执行基类
/// </summary>
public abstract class ShotBehavior
{
    protected Camera Camera;
    protected CameraShotData Data;

    public virtual void Init(Camera cam, CameraShotData data)
    {
        Camera = cam;
        Data = data;
    }

    /// <summary>
    /// 每帧驱动摄像机，t = 归一化进度 [0, 1]
    /// </summary>
    public abstract void Tick(float t);

    /// <summary>
    /// 镜头开始时调用一次
    /// </summary>
    public virtual void OnEnter() { }

    /// <summary>
    /// 镜头结束时调用一次
    /// </summary>
    public virtual void OnExit() { }
}

/// <summary>
/// 静态镜头：固定在指定位置朝向目标
/// </summary>
public class StaticShotBehavior : ShotBehavior
{
    public override void Tick(float t)
    {
        Camera.transform.position = Data.position;
        Camera.transform.rotation = Data.rotation;
        Camera.fieldOfView = Data.fieldOfView;

        // 如果有注视目标，实时朝向目标
        if (Data.lookAtTarget != null)
        {
            Vector3 targetPos = Data.lookAtTarget.position + Data.lookAtOffset;
            Camera.transform.LookAt(targetPos);
        }
    }
}

/// <summary>
/// 轨道镜头：沿 Catmull-Rom 样条运动，同时注视目标
/// </summary>
public class TrackShotBehavior : ShotBehavior
{
    public override void Tick(float t)
    {
        float easedT = SplineUtils.Ease(t, Data.trackEasing);

        // 沿轨道采样位置
        if (Data.trackPoints != null && Data.trackPoints.Count >= 2)
        {
            Camera.transform.position = SplineUtils.SampleCatmullRom(
                Data.trackPoints, easedT);
        }

        // FOV 变化（可选，用于推近效果）
        Camera.fieldOfView = Data.fieldOfView;

        // 注视目标
        if (Data.lookAtTarget != null)
        {
            Vector3 targetPos = Data.lookAtTarget.position + Data.lookAtOffset;
            Vector3 dir = (targetPos - Camera.transform.position).normalized;
            if (dir != Vector3.zero)
                Camera.transform.rotation = Quaternion.LookRotation(dir);
        }
        else
        {
            // 朝向轨道切线方向（前进方向）
            float nextT = Mathf.Min(t + 0.01f, 1f);
            Vector3 nextPos = SplineUtils.SampleCatmullRom(Data.trackPoints, nextT);
            Vector3 dir = (nextPos - Camera.transform.position).normalized;
            if (dir != Vector3.zero)
                Camera.transform.rotation = Quaternion.LookRotation(dir);
        }
    }
}

/// <summary>
/// 环绕镜头：绕目标旋转
/// </summary>
public class OrbitShotBehavior : ShotBehavior
{
    public override void Tick(float t)
    {
        if (Data.lookAtTarget == null) return;

        float easedT = SplineUtils.Ease(t, EasingType.Linear);
        float angle = Mathf.Lerp(Data.orbitAngleStart, Data.orbitAngleEnd, easedT);

        // 在水平面绕目标环绕
        Vector3 center = Data.lookAtTarget.position + Data.lookAtOffset;
        float rad = angle * Mathf.Deg2Rad;
        Vector3 offset = new Vector3(
            Mathf.Sin(rad) * Data.followDistance,
            0f,
            Mathf.Cos(rad) * Data.followDistance
        );

        Camera.transform.position = center + offset;
        Camera.transform.LookAt(center);
        Camera.fieldOfView = Data.fieldOfView;
    }
}

/// <summary>
/// 镜头行为工厂
/// </summary>
public static class ShotBehaviorFactory
{
    public static ShotBehavior Create(ShotType type) => type switch
    {
        ShotType.Static => new StaticShotBehavior(),
        ShotType.Track  => new TrackShotBehavior(),
        ShotType.Orbit  => new OrbitShotBehavior(),
        ShotType.Follow => new StaticShotBehavior(), // Follow 复用 Static，实时 LookAt
        _               => new StaticShotBehavior(),
    };
}
```

## 六、过渡效果控制器

镜头切换时的 FadeBlack / CrossFade 效果：

```csharp
using System.Collections;
using UnityEngine;
using UnityEngine.UI;

/// <summary>
/// 过渡效果控制器（需要场景中有一个全屏黑色 Image）
/// </summary>
public class CutsceneTransitionController : MonoBehaviour
{
    [SerializeField] private Image fadePanel;

    private Coroutine _currentTransition;

    private void Awake()
    {
        if (fadePanel != null)
            fadePanel.color = new Color(0, 0, 0, 0);
    }

    /// <summary>
    /// 黑屏淡出（进入过场）
    /// </summary>
    public IEnumerator FadeToBlack(float duration)
    {
        if (fadePanel == null) yield break;
        float elapsed = 0;
        while (elapsed < duration)
        {
            elapsed += Time.unscaledDeltaTime;
            float a = Mathf.Clamp01(elapsed / duration);
            fadePanel.color = new Color(0, 0, 0, a);
            yield return null;
        }
        fadePanel.color = new Color(0, 0, 0, 1);
    }

    /// <summary>
    /// 从黑屏淡入
    /// </summary>
    public IEnumerator FadeFromBlack(float duration)
    {
        if (fadePanel == null) yield break;
        float elapsed = 0;
        while (elapsed < duration)
        {
            elapsed += Time.unscaledDeltaTime;
            float a = 1f - Mathf.Clamp01(elapsed / duration);
            fadePanel.color = new Color(0, 0, 0, a);
            yield return null;
        }
        fadePanel.color = new Color(0, 0, 0, 0);
    }

    /// <summary>
    /// 闪白效果（用于动作冲击、闪光）
    /// </summary>
    public IEnumerator FlashWhite(float holdDuration, float fadeDuration)
    {
        if (fadePanel == null) yield break;
        fadePanel.color = Color.white;
        yield return new WaitForSecondsRealtime(holdDuration);

        float elapsed = 0;
        while (elapsed < fadeDuration)
        {
            elapsed += Time.unscaledDeltaTime;
            float a = 1f - Mathf.Clamp01(elapsed / fadeDuration);
            fadePanel.color = new Color(1, 1, 1, a);
            yield return null;
        }
        fadePanel.color = new Color(1, 1, 1, 0);
    }
}
```

## 七、景深联动控制器

过场动画中常用 DOF 景深突出焦点角色：

```csharp
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

/// <summary>
/// URP 景深（DOF）过场联动控制器
/// </summary>
public class CutsceneDofController
{
    private DepthOfField _dof;
    private bool _wasEnabled;

    public void Init(Volume volume)
    {
        volume?.sharedProfile.TryGet(out _dof);
    }

    /// <summary>
    /// 进入过场，开启 DOF 并设置参数
    /// </summary>
    public void OnShotEnter(CameraShotData data)
    {
        if (_dof == null) return;
        _wasEnabled = _dof.active;
        _dof.active = data.enableDof;

        if (data.enableDof)
        {
            _dof.focalLength.Override(data.focalLength);
            _dof.aperture.Override(data.aperture);
        }
    }

    /// <summary>
    /// 过场结束，恢复原始 DOF 状态
    /// </summary>
    public void OnCutsceneEnd()
    {
        if (_dof == null) return;
        _dof.active = _wasEnabled;
        _dof.focalLength.Override(50f); // 恢复默认
        _dof.aperture.Override(5.6f);
    }

    /// <summary>
    /// 动态更新焦点距离（对准角色）
    /// </summary>
    public void UpdateFocusDistance(Camera cam, Transform target)
    {
        if (_dof == null || target == null) return;
        float dist = Vector3.Distance(cam.transform.position, target.position);
        _dof.focusDistance.Override(dist);
    }
}
```

## 八、过场动画导演（核心驱动）

```csharp
using System;
using System.Collections;
using UnityEngine;

/// <summary>
/// 过场动画导演 - 核心调度器
/// 驱动镜头序列播放、过渡、回调
/// </summary>
public class CutsceneDirector : MonoBehaviour
{
    [Header("过场专用摄像机")]
    [SerializeField] private Camera cutsceneCamera;

    [Header("主游戏摄像机")]
    [SerializeField] private Camera mainCamera;

    [Header("过渡控制器")]
    [SerializeField] private CutsceneTransitionController transition;

    [Header("全局后处理 Volume")]
    [SerializeField] private Volume globalVolume;

    // 事件回调
    public event Action OnCutsceneStart;
    public event Action OnCutsceneEnd;
    public event Action<int> OnShotBegin; // 参数：镜头索引

    private CutsceneSequenceAsset _currentSequence;
    private bool _isPlaying;
    private bool _skipRequested;

    private CutsceneDofController _dofController;
    private Coroutine _playCoroutine;

    private void Awake()
    {
        _dofController = new CutsceneDofController();
        _dofController.Init(globalVolume);

        // 初始化：过场摄像机关闭
        cutsceneCamera.enabled = false;
    }

    /// <summary>
    /// 播放指定过场序列
    /// </summary>
    public void Play(CutsceneSequenceAsset sequence)
    {
        if (_isPlaying) Stop();
        _currentSequence = sequence;
        _playCoroutine = StartCoroutine(PlaySequence());
    }

    public void Stop()
    {
        if (_playCoroutine != null)
        {
            StopCoroutine(_playCoroutine);
            _playCoroutine = null;
        }
        EndCutscene();
    }

    /// <summary>
    /// 请求跳过（在下一个镜头切换点生效）
    /// </summary>
    public void RequestSkip()
    {
        if (_currentSequence != null && _currentSequence.skipEnabled)
            _skipRequested = true;
    }

    private IEnumerator PlaySequence()
    {
        _isPlaying = true;
        _skipRequested = false;

        // 切换到过场摄像机
        mainCamera.enabled   = false;
        cutsceneCamera.enabled = true;

        OnCutsceneStart?.Invoke();

        var shots = _currentSequence.shots;
        for (int i = 0; i < shots.Count; i++)
        {
            if (_skipRequested) break;

            var shotData = shots[i];
            OnShotBegin?.Invoke(i);

            // 进入过渡
            yield return StartCoroutine(HandleTransitionIn(shotData));

            // 执行镜头
            yield return StartCoroutine(ExecuteShot(shotData));

            // 退出过渡（下一镜头的进入过渡会处理）
        }

        EndCutscene();
    }

    private IEnumerator HandleTransitionIn(CameraShotData data)
    {
        float dur = data.transitionDuration;
        switch (data.transitionIn)
        {
            case TransitionType.Cut:
                // 无过渡
                break;

            case TransitionType.FadeBlack:
                yield return transition.FadeToBlack(dur * 0.5f);
                // 在黑屏中完成位置切换，然后淡入
                yield return new WaitForSecondsRealtime(0.1f);
                yield return transition.FadeFromBlack(dur * 0.5f);
                break;

            case TransitionType.Blend:
                // Blend 模式：位置/朝向插值到目标
                yield return BlendToShot(data, dur);
                break;
        }
    }

    private IEnumerator ExecuteShot(CameraShotData data)
    {
        // 创建并初始化行为
        var behavior = ShotBehaviorFactory.Create(data.shotType);
        behavior.Init(cutsceneCamera, data);
        behavior.OnEnter();

        // 景深联动
        _dofController.OnShotEnter(data);

        float elapsed = 0f;
        float duration = data.duration;

        while (elapsed < duration && !_skipRequested)
        {
            float t = elapsed / duration;
            behavior.Tick(t);

            // 实时更新焦点距离
            if (data.enableDof && data.lookAtTarget != null)
                _dofController.UpdateFocusDistance(cutsceneCamera, data.lookAtTarget);

            elapsed += Time.unscaledDeltaTime;
            yield return null;
        }

        behavior.Tick(1f); // 确保最后一帧到达终点
        behavior.OnExit();
    }

    private IEnumerator BlendToShot(CameraShotData targetData, float duration)
    {
        Vector3    startPos = cutsceneCamera.transform.position;
        Quaternion startRot = cutsceneCamera.transform.rotation;
        float      startFov = cutsceneCamera.fieldOfView;

        float elapsed = 0;
        while (elapsed < duration)
        {
            float t = SplineUtils.Ease(elapsed / duration, EasingType.EaseInOut);
            cutsceneCamera.transform.position = Vector3.Lerp(startPos, targetData.position, t);
            cutsceneCamera.transform.rotation = Quaternion.Slerp(startRot, targetData.rotation, t);
            cutsceneCamera.fieldOfView = Mathf.Lerp(startFov, targetData.fieldOfView, t);
            elapsed += Time.unscaledDeltaTime;
            yield return null;
        }
    }

    private void EndCutscene()
    {
        _isPlaying = false;

        // 恢复主摄像机
        cutsceneCamera.enabled = false;
        mainCamera.enabled     = true;

        // 恢复 DOF
        _dofController.OnCutsceneEnd();

        OnCutsceneEnd?.Invoke();
    }
}
```

```csharp
        DrawDefaultInspector();
        var path = (CameraTrackPath)target;

        if (GUILayout.Button("添加控制点（相机当前位置）"))
        {
            var cam = SceneView.lastActiveSceneView?.camera;
            Vector3 pos = cam != null ? cam.transform.position : Vector3.zero;
            Undo.RecordObject(target, "添加轨道点");
            path.Points.Add(pos);
            EditorUtility.SetDirty(target);
        }

        if (_selectedPoint >= 0 && _selectedPoint < path.Points.Count)
        {
            if (GUILayout.Button("删除选中点"))
            {
                Undo.RecordObject(target, "删除轨道点");
                path.Points.RemoveAt(_selectedPoint);
                _selectedPoint = -1;
                EditorUtility.SetDirty(target);
            }
        }
    }

    private void OnSceneGUI()
    {
        var path = (CameraTrackPath)target;
        if (path.Points == null || path.Points.Count < 2) return;

        Handles.color = Color.yellow;

        // 绘制曲线
        int steps = 50;
        Vector3 prev = SplineUtils.SampleCatmullRom(path.Points, 0);
        for (int i = 1; i <= steps; i++)
        {
            float t = i / (float)steps;
            Vector3 next = SplineUtils.SampleCatmullRom(path.Points, t);
            Handles.DrawLine(prev, next);
            prev = next;
        }

        // 绘制控制点 Handle
        for (int i = 0; i < path.Points.Count; i++)
        {
            Handles.color = i == _selectedPoint ? Color.red : Color.cyan;
            float size = HandleUtility.GetHandleSize(path.Points[i]) * 0.1f;

            if (Handles.Button(path.Points[i], Quaternion.identity, size, size, Handles.SphereHandleCap))
                _selectedPoint = i;

            if (i == _selectedPoint)
            {
                EditorGUI.BeginChangeCheck();
                Vector3 newPos = Handles.PositionHandle(path.Points[i], Quaternion.identity);
                if (EditorGUI.EndChangeCheck())
                {
                    Undo.RecordObject(target, "移动轨道点");
                    path.Points[i] = newPos;
                    EditorUtility.SetDirty(target);
                }
            }

            // 绘制序号标签
            Handles.Label(path.Points[i] + Vector3.up * 0.3f, $"P{i}");
        }
    }
}
```

## 十、与 Timeline 集成

如果项目使用 Unity Timeline，可以将自研镜头系统作为 PlayableAsset 接入：

```csharp
using UnityEngine;
using UnityEngine.Playables;
using UnityEngine.Timeline;

/// <summary>
/// Timeline Clip：驱动单个镜头
/// </summary>
[System.Serializable]
public class CameraShotClip : PlayableAsset, ITimelineClipAsset
{
    public CameraShotData ShotData = new CameraShotData();

    public ClipCaps clipCaps => ClipCaps.Blending;

    public override Playable CreatePlayable(PlayableGraph graph, GameObject owner)
    {
        var playable = ScriptPlayable<CameraShotPlayableBehaviour>.Create(graph);
        playable.GetBehaviour().ShotData = ShotData;
        return playable;
    }
}

public class CameraShotPlayableBehaviour : PlayableBehaviour
{
    public CameraShotData ShotData;

    private ShotBehavior _behavior;
    private Camera _cam;

    public override void OnGraphStart(Playable playable)
    {
        _cam = Camera.main;
        _behavior = ShotBehaviorFactory.Create(ShotData.shotType);
        _behavior.Init(_cam, ShotData);
    }

    public override void ProcessFrame(Playable playable, FrameData info, object playerData)
    {
        float duration = (float)playable.GetDuration();
        float time     = (float)playable.GetTime();
        float t = duration > 0 ? time / duration : 0;
        _behavior?.Tick(t);
    }
}
```

## 十一、最佳实践总结

### 1. 过场期间暂停游戏逻辑
```csharp
void StartCutscene()
{
    Time.timeScale = 0; // 暂停游戏物理/AI
    // 过场摄像机使用 unscaledDeltaTime
}
```

### 2. 镜头数量控制
- 单段过场动画建议不超过 **8~12 个镜头**
- 每个镜头时长控制在 **2~6 秒**，太短会让玩家看不清，太长会无聊

### 3. 轨道点密度
- Catmull-Rom 轨道控制点间距建议 **1~3 个单位**
- 控制点过密会导致曲线震荡（Runge 现象）

### 4. FOV 变化策略
| 场景 | 推荐 FOV |
|------|---------|
| 宽景/环境展示 | 75~90° |
| 人物对话特写 | 35~50° |
| 动作追击场景 | 85~100°（宽角产生速度感）|
| 情感表达近景 | 25~35°（长焦压缩空间）|

### 5. 摄像机运动原则
- **落幅优先**：摄像机需要在目标位置稳定 0.5s 再切换
- **避免高频抖动**：快速切换 > 3 个镜头/秒会让玩家眩晕
- **摄像机运动方向与角色动作一致**：角色向右走，镜头微微向右 Pan

### 6. 性能注意事项
- 过场期间可关闭主摄像机和 Culling Mask 中不需要渲染的层
- 高质量过场可临时提升 `QualitySettings.shadowDistance`
- 过场结束后一定要恢复所有临时修改的设置

## 总结

| 模块 | 职责 |
|------|------|
| CutsceneSequenceAsset | 数据容器，存储镜头序列的 ScriptableObject |
| ShotBehavior | 各类镜头的运动逻辑（静态/轨道/环绕）|
| SplineUtils | Catmull-Rom / 贝塞尔曲线采样工具 |
| CutsceneTransitionController | 淡入淡出、闪白等过渡效果 |
| CutsceneDofController | 景深效果联动，突出焦点角色 |
| CutsceneDirector | 总调度，驱动镜头序列播放与切换 |
| CameraTrackPathEditor | Scene 视图可视化轨道编辑工具 |

通过数据驱动的镜头序列 + 插件化的行为执行器，系统具备良好的可扩展性。策划可以通过 ScriptableObject 配置数据调整过场，程序只需维护执行器逻辑，职责清晰、迭代高效。
