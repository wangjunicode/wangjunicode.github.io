# 游戏多语言字体渲染与文本排版系统深度解析

## 前言

"把游戏翻译成多语言"听起来只是换文本，但在工程层面却是冰山一角下的深水区。中文需要数千个字形，阿拉伯语从右到左书写且字母根据上下文变形，泰语没有单词间距但换行规则复杂，日语竖排横排并存，韩语音节块组合……每一种语言都对字体渲染系统提出独特挑战。本文将系统性地梳理游戏多语言文本系统的完整技术链路。

## 一、Unicode 基础与字符处理

### 1.1 Unicode 编码层次

```
Unicode 术语层次：
┌─────────────────────────────────────────┐
│  码点 (Code Point)                       │
│  如 U+4E2D = "中"                        │
├─────────────────────────────────────────┤
│  字形 (Glyph) = 码点的具体渲染形状         │
│  同一码点在不同字体中形状不同              │
├─────────────────────────────────────────┤
│  字素簇 (Grapheme Cluster)               │
│  用户感知的"一个字符"                     │
│  如 "é" = U+0065 (e) + U+0301 (重音符)  │
└─────────────────────────────────────────┘

UTF-8 编码规则（变长）：
  U+0000~007F  : 1字节 (ASCII兼容)
  U+0080~07FF  : 2字节
  U+0800~FFFF  : 3字节 (中文区: U+4E00~9FFF)
  U+10000~10FFFF: 4字节 (Emoji, 古文字)
```

### 1.2 C# 字符串处理陷阱

```csharp
// ⚠️ 错误示范：直接用 string.Length 统计字符数
string text = "Hello 🎮";
Debug.Log(text.Length); // 输出 9，不是 8！
// 因为 🎮 (U+1F3AE) 是代理对，占2个char

// ✅ 正确：使用 StringInfo 获取文本元素数
using System.Globalization;
var info = new StringInfo(text);
Debug.Log(info.LengthInTextElements); // 输出 8

// ✅ 正确遍历字素簇
var enumerator = StringInfo.GetTextElementEnumerator(text);
while (enumerator.MoveNext())
{
    string element = (string)enumerator.Current;
    Debug.Log($"字素: {element}, 码点: U+{char.ConvertToUtf32(element, 0):X4}");
}

// ⚠️ 代理对（Surrogate Pairs）处理
string emoji = "🎮";
char high = emoji[0]; // 高代理: D83C
char low  = emoji[1]; // 低代理: DFAE
int codePoint = char.ConvertToUtf32(high, low); // 正确获取码点

// 工具方法：安全截断字符串（不破坏代理对和组合字符）
public static string SafeSubstring(string text, int maxLength)
{
    if (text.Length <= maxLength) return text;
    
    // 确保不在代理对中间截断
    int cutPoint = maxLength;
    while (cutPoint > 0 && char.IsLowSurrogate(text[cutPoint]))
        cutPoint--;
    
    return text.Substring(0, cutPoint);
}
```

## 二、SDF 字体渲染技术

### 2.1 SDF vs 传统位图字体

| 对比维度 | 传统位图字体 | SDF字体 |
|---------|------------|---------|
| 缩放质量 | 放大模糊锯齿 | 任意缩放清晰 |
| 描边/发光 | 需单独图集 | Shader参数直接实现 |
| 内存占用 | 每字号一套图集 | 一套图集复用所有字号 |
| 渲染性能 | 简单采样 | 多一次比较操作（极小开销） |
| 极小字号 | 可接受 | 细节丢失（MSDF改善） |

### 2.2 SDF Shader 实现

```hlsl
// SDFFont.shader - 自定义SDF字体渲染

Shader "Custom/SDFFont"
{
    Properties
    {
        _MainTex ("SDF Atlas", 2D) = "white" {}
        _FaceColor ("字体颜色", Color) = (1,1,1,1)
        _OutlineColor ("描边颜色", Color) = (0,0,0,1)
        _OutlineWidth ("描边宽度", Range(0, 0.5)) = 0
        _ShadowColor ("阴影颜色", Color) = (0,0,0,0.5)
        _ShadowOffset ("阴影偏移", Vector) = (0.002, -0.002, 0, 0)
        _Softness ("边缘柔和度", Range(0, 0.5)) = 0.02
        _GlowColor ("发光颜色", Color) = (1,1,0,0)
        _GlowRange ("发光范围", Range(0, 0.5)) = 0.2
        _GlowPower ("发光强度", Range(0, 1)) = 0.5
    }
    
    SubShader
    {
        Tags { "Queue"="Transparent" "RenderType"="Transparent" }
        Blend SrcAlpha OneMinusSrcAlpha
        ZWrite Off
        Cull Off
        
        Pass
        {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #pragma multi_compile _ SDF_OUTLINE SDF_GLOW SDF_SHADOW
            #include "UnityCG.cginc"
            
            struct appdata
            {
                float4 vertex : POSITION;
                float2 uv     : TEXCOORD0;
                float4 color  : COLOR;
            };
            
            struct v2f
            {
                float4 pos    : SV_POSITION;
                float2 uv     : TEXCOORD0;
                float4 color  : COLOR;
            };
            
            sampler2D _MainTex;
            float4 _MainTex_TexelSize;
            float4 _FaceColor;
            float4 _OutlineColor;
            float  _OutlineWidth;
            float4 _ShadowColor;
            float4 _ShadowOffset;
            float  _Softness;
            float4 _GlowColor;
            float  _GlowRange;
            float  _GlowPower;
            
            v2f vert(appdata v)
            {
                v2f o;
                o.pos   = UnityObjectToClipPos(v.vertex);
                o.uv    = v.uv;
                o.color = v.color * _FaceColor;
                return o;
            }
            
            fixed4 frag(v2f i) : SV_Target
            {
                // SDF值：0.5 = 字形边界，>0.5 = 内部，<0.5 = 外部
                float sdf = tex2D(_MainTex, i.uv).a;
                
                // 边缘平滑（抗锯齿）
                // fwidth 计算屏幕空间导数，自动适配不同缩放
                float dist = fwidth(sdf);
                float softness = max(_Softness, dist);
                
                // 字体主体
                float face = smoothstep(0.5 - softness, 0.5 + softness, sdf);
                
                fixed4 col = i.color;
                col.a *= face;
                
#ifdef SDF_OUTLINE
                if (_OutlineWidth > 0)
                {
                    float outline = smoothstep(
                        0.5 - _OutlineWidth - softness,
                        0.5 - _OutlineWidth + softness,
                        sdf
                    );
                    // 混合描边：在字形外但在描边范围内
                    col = lerp(_OutlineColor * float4(1,1,1,outline), col, face);
                }
#endif

#ifdef SDF_GLOW
                float glowDist = 0.5 - _GlowRange;
                float glow = smoothstep(glowDist - softness, glowDist + softness, sdf) * (1 - face);
                glow = pow(glow, 1.0 / max(0.01, _GlowPower));
                col = lerp(col, _GlowColor, glow * _GlowColor.a);
#endif

#ifdef SDF_SHADOW
                float shadowSDF = tex2D(_MainTex, i.uv - _ShadowOffset.xy).a;
                float shadow = smoothstep(0.5 - softness, 0.5 + softness, shadowSDF);
                fixed4 shadowCol = _ShadowColor * float4(1,1,1,shadow * (1 - face));
                col = lerp(shadowCol, col, face + (1 - face) * col.a);
#endif
                
                return col;
            }
            ENDCG
        }
    }
}
```

```csharp
// MSDF 生成工具封装（使用 msdfgen 库）
// MSDF 通过R/G/B三通道编码符号距离场，改善小字号的锐利度

public static class MSDFGenerator
{
    [MenuItem("Tools/Font/Generate MSDF Atlas")]
    public static void GenerateMSDFAtlas()
    {
        // 调用 msdf-atlas-gen 命令行工具
        string fontPath = EditorUtility.OpenFilePanel("选择字体文件", "", "ttf,otf");
        if (string.IsNullOrEmpty(fontPath)) return;
        
        string outputPath = Path.Combine(Application.dataPath, "Fonts/MSDF");
        Directory.CreateDirectory(outputPath);
        
        string atlasName = Path.GetFileNameWithoutExtension(fontPath);
        
        // 执行命令：生成中英文混合图集
        var args = $"-font \"{fontPath}\" " +
                   $"-type msdf " +
                   $"-format png " +
                   $"-size 4096 " +        // 图集尺寸
                   $"-pxrange 8 " +         // SDF范围（像素）
                   $"-charset {BuildCharsetFile()} " +
                   $"-imageout \"{outputPath}/{atlasName}_msdf.png\" " +
                   $"-json \"{outputPath}/{atlasName}_msdf.json\"";
        
        var process = new System.Diagnostics.Process
        {
            StartInfo = new System.Diagnostics.ProcessStartInfo
            {
                FileName = "msdf-atlas-gen",
                Arguments = args,
                UseShellExecute = false,
                RedirectStandardOutput = true,
                CreateNoWindow = true
            }
        };
        
        process.Start();
        string output = process.StandardOutput.ReadToEnd();
        process.WaitForExit();
        
        Debug.Log($"MSDF生成完成: {output}");
        AssetDatabase.Refresh();
    }
    
    private static string BuildCharsetFile()
    {
        // 收集项目中所有文本资源的字符集
        var charset = new HashSet<char>();
        
        // 基础ASCII
        for (char c = ' '; c <= '~'; c++) charset.Add(c);
        
        // 扫描本地化文本文件
        string[] textFiles = Directory.GetFiles(
            Path.Combine(Application.dataPath, "Localization"),
            "*.json", SearchOption.AllDirectories
        );
        
        foreach (var file in textFiles)
        {
            string content = File.ReadAllText(file);
            foreach (char c in content)
            {
                if (c > 127) charset.Add(c); // 非ASCII字符
            }
        }
        
        // 写入字符集文件
        string charsetPath = Path.Combine(Path.GetTempPath(), "font_charset.txt");
        File.WriteAllText(charsetPath, new string(charset.ToArray()), System.Text.Encoding.UTF8);
        return charsetPath;
    }
}
```

## 三、动态字体图集系统

### 3.1 运行时动态字形加载

```csharp
using TMPro;
using System.Collections.Generic;

/// <summary>
/// 动态字体图集管理器
/// 解决CJK字符集过大、图集无法预生成的问题
/// </summary>
public class DynamicFontAtlasManager : MonoBehaviour
{
    private static DynamicFontAtlasManager _instance;
    public static DynamicFontAtlasManager Instance => _instance;
    
    [Header("字体配置")]
    [SerializeField] private TMP_FontAsset _baseFontAsset;
    [SerializeField] private TMP_FontAsset[] _fallbackFonts; // 备用字体链
    [SerializeField] private int _atlasWidth = 2048;
    [SerializeField] private int _atlasHeight = 2048;
    [SerializeField] private int _atlasGrowSize = 512; // 图集不够时增长大小
    
    // 字形缓存：码点 -> 字形是否已加载
    private HashSet<uint> _loadedGlyphs = new();
    // 待加载队列
    private HashSet<uint> _pendingGlyphs = new();
    
    private float _pendingUpdateTimer;
    private const float UpdateInterval = 0.1f; // 每0.1秒批量更新一次图集
    
    private void Awake()
    {
        _instance = this;
    }
    
    private void Update()
    {
        if (_pendingGlyphs.Count == 0) return;
        
        _pendingUpdateTimer += Time.deltaTime;
        if (_pendingUpdateTimer < UpdateInterval) return;
        _pendingUpdateTimer = 0;
        
        FlushPendingGlyphs();
    }
    
    /// <summary>
    /// 预加载指定文本的所有字形（在文本显示前调用）
    /// </summary>
    public void PreloadText(string text, System.Action onComplete = null)
    {
        var missingGlyphs = new List<uint>();
        
        var enumerator = StringInfo.GetTextElementEnumerator(text);
        while (enumerator.MoveNext())
        {
            string element = (string)enumerator.Current;
            uint codePoint = (uint)char.ConvertToUtf32(element, 0);
            
            if (!_loadedGlyphs.Contains(codePoint))
                missingGlyphs.Add(codePoint);
        }
        
        if (missingGlyphs.Count == 0)
        {
            onComplete?.Invoke();
            return;
        }
        
        foreach (var glyph in missingGlyphs)
            _pendingGlyphs.Add(glyph);
        
        // 立即加载（不等待定时）
        StartCoroutine(LoadGlyphsAndCallback(missingGlyphs, onComplete));
    }
    
    private IEnumerator LoadGlyphsAndCallback(List<uint> glyphs, System.Action callback)
    {
        yield return null; // 等一帧让UI渲染完当前帧
        
        TryAddGlyphsToAtlas(glyphs);
        
        yield return null; // 等图集更新传播到GPU
        
        callback?.Invoke();
    }
    
    private void FlushPendingGlyphs()
    {
        var glyphs = new List<uint>(_pendingGlyphs);
        _pendingGlyphs.Clear();
        TryAddGlyphsToAtlas(glyphs);
    }
    
    private void TryAddGlyphsToAtlas(List<uint> codePoints)
    {
        if (_baseFontAsset == null) return;
        
        // TMP 动态字体：直接请求字形更新
        bool success = _baseFontAsset.TryAddCharacters(codePoints.ToArray());
        
        if (!success)
        {
            // 图集空间不足，尝试扩展或回退到备用字体
            Debug.LogWarning($"字体图集空间不足，正在扩展图集...");
            ExpandAtlas();
            _baseFontAsset.TryAddCharacters(codePoints.ToArray());
        }
        
        foreach (var cp in codePoints)
            _loadedGlyphs.Add(cp);
    }
    
    private void ExpandAtlas()
    {
        // TMP 的 AtlasPopulationMode 为 Dynamic 时支持扩展
        _atlasWidth  = Mathf.Min(_atlasWidth  + _atlasGrowSize, 8192);
        _atlasHeight = Mathf.Min(_atlasHeight + _atlasGrowSize, 8192);
        
        // 更新图集尺寸（需要重新生成）
        var newAtlas = new Texture2D(_atlasWidth, _atlasHeight, TextureFormat.Alpha8, false);
        newAtlas.filterMode = FilterMode.Bilinear;
        // 实际项目需要复制旧图集数据并更新TMP内部引用
    }
}
```

### 3.2 字体回退链（Font Fallback）

```csharp
// 字体回退链配置：处理单一字体无法覆盖所有语言的情况
[CreateAssetMenu(fileName = "FontFallbackConfig", menuName = "Game/Font Fallback Config")]
public class FontFallbackConfig : ScriptableObject
{
    [System.Serializable]
    public class LanguageFontGroup
    {
        public string languageCode;          // "zh-CN", "ar", "th", "ko"...
        public TMP_FontAsset primaryFont;
        public TMP_FontAsset[] fallbackChain;
        public bool enableRTL;               // 是否启用RTL排版
        public float lineHeightMultiplier = 1.0f;
        public float letterSpacingMultiplier = 1.0f;
    }
    
    [SerializeField] private TMP_FontAsset _defaultFont;
    [SerializeField] private List<LanguageFontGroup> _languageGroups;
    
    private Dictionary<string, LanguageFontGroup> _groupMap;
    
    public void Initialize()
    {
        _groupMap = new Dictionary<string, LanguageFontGroup>();
        foreach (var group in _languageGroups)
            _groupMap[group.languageCode] = group;
    }
    
    public LanguageFontGroup GetGroupForLanguage(string langCode)
    {
        // 精确匹配
        if (_groupMap.TryGetValue(langCode, out var group)) return group;
        
        // 降级匹配：zh-TW → zh
        string baseCode = langCode.Split('-')[0];
        if (_groupMap.TryGetValue(baseCode, out group)) return group;
        
        return null;
    }
    
    public void ApplyToText(TMP_Text text, string langCode)
    {
        var group = GetGroupForLanguage(langCode);
        if (group == null)
        {
            text.font = _defaultFont;
            return;
        }
        
        text.font = group.primaryFont;
        
        // 设置回退链
        if (group.fallbackChain != null && group.fallbackChain.Length > 0)
        {
            group.primaryFont.fallbackFontAssetTable.Clear();
            group.primaryFont.fallbackFontAssetTable.AddRange(group.fallbackChain);
        }
        
        // 语言专属排版参数
        text.lineSpacing = (group.lineHeightMultiplier - 1.0f) * 100f;
        text.characterSpacing = (group.letterSpacingMultiplier - 1.0f) * 5f;
        
        // RTL支持
        if (group.enableRTL)
        {
            text.isRightToLeftText = true;
        }
    }
}
```

## 四、双向文本（BiDi）排版

### 4.1 Unicode 双向算法基础

```
BiDi 算法处理场景：
  阿拉伯文（RTL） + 数字（LTR） + 英文（LTR） 混排
  
  逻辑顺序：    م ر ح ب ا   Hello   123
  视觉顺序：    123   Hello   ا ب ح ر م
  
  Unicode 方向控制字符：
  U+200E  LRM  左向标记（Left-to-Right Mark）
  U+200F  RLM  右向标记（Right-to-Left Mark）
  U+202A  LRE  左向嵌入开始
  U+202B  RLE  右向嵌入开始
  U+202C  PDF  弹出方向格式
  U+2066  LRI  左向隔离（推荐使用）
  U+2067  RLI  右向隔离
  U+2069  PDI  弹出方向隔离
```

### 4.2 RTL 文本处理工具

```csharp
using System.Text;
using System.Globalization;

public static class BiDiTextProcessor
{
    // 方向控制字符
    private const char LRM = '\u200E';
    private const char RLM = '\u200F';
    private const char LRI = '\u2066';
    private const char RLI = '\u2067';
    private const char PDI = '\u2069';
    
    // 判断字符是否是RTL字符
    public static bool IsRTLChar(char c)
    {
        var category = CharUnicodeInfo.GetUnicodeCategory(c);
        
        // 快速范围检查
        // 阿拉伯语: U+0600-06FF, U+0750-077F, U+FB50-FDFF, U+FE70-FEFF
        // 希伯来语: U+0590-05FF
        // 波斯语等其他RTL语言
        return (c >= '\u0590' && c <= '\u05FF') || // 希伯来语
               (c >= '\u0600' && c <= '\u06FF') || // 阿拉伯语
               (c >= '\u0750' && c <= '\u077F') || // 阿拉伯语补充
               (c >= '\uFB50' && c <= '\uFDFF') || // 阿拉伯语表示形式A
               (c >= '\uFE70' && c <= '\uFEFF');   // 阿拉伯语表示形式B
    }
    
    // 检测文本的主方向
    public static bool IsRTLText(string text)
    {
        int rtlCount = 0, ltrCount = 0;
        foreach (char c in text)
        {
            if (IsRTLChar(c)) rtlCount++;
            else if (char.IsLetter(c)) ltrCount++;
        }
        return rtlCount > ltrCount;
    }
    
    // 为混合方向文本插入方向标记
    public static string ProcessMixedDirectionText(string text, bool baseIsRTL)
    {
        if (string.IsNullOrEmpty(text)) return text;
        
        var sb = new StringBuilder();
        bool currentIsRTL = baseIsRTL;
        bool inEmbedding = false;
        
        for (int i = 0; i < text.Length; i++)
        {
            char c = text[i];
            bool charIsRTL = IsRTLChar(c);
            
            // 方向发生变化
            if (i > 0 && charIsRTL != currentIsRTL)
            {
                if (inEmbedding)
                {
                    sb.Append(PDI);   // 关闭上一个嵌入
                    inEmbedding = false;
                }
                
                // 开启新的方向嵌入
                sb.Append(charIsRTL ? RLI : LRI);
                inEmbedding = true;
                currentIsRTL = charIsRTL;
            }
            
            sb.Append(c);
        }
        
        if (inEmbedding) sb.Append(PDI);
        
        return sb.ToString();
    }
    
    // 阿拉伯字母连字处理（将独立形式转为连写形式）
    private static readonly Dictionary<char, char[]> ArabicForms = new()
    {
        // 示例：ب (U+0628) 的四种形式：独立/词首/词中/词尾
        ['\u0628'] = new char[] { '\uFE8F', '\uFE91', '\uFE92', '\uFE90' }
    };
    
    public static string ApplyArabicShaping(string text)
    {
        // 实际项目推荐使用 ICU4C 或 HarfBuzz 库处理
        // 这里仅作为简单示例
        return text; // 占位
    }
}

// TextMeshPro RTL 扩展组件
[RequireComponent(typeof(TMP_Text))]
public class RTLTextAdapter : MonoBehaviour
{
    private TMP_Text _text;
    private string _rawText;
    
    private void Awake()

        _text = GetComponent<TMP_Text>();
    }

        _rawText = text;

            // 阿拉伯语等需要字形重塑

        }
        else
        {
            _text.text = text;
        }

    {
        // RTL 语言默认右对齐
        if (isRTL)
        {
            _text.alignment = _text.alignment switch
            {
                TextAlignmentOptions.Left    => TextAlignmentOptions.Right,
                TextAlignmentOptions.TopLeft => TextAlignmentOptions.TopRight,
                TextAlignmentOptions.BottomLeft => TextAlignmentOptions.BottomRight,
                _ => _text.alignment
            };
        }
    }
    
    private static bool IsRTLLanguage(string langCode)
    {
        return langCode is "ar" or "ar-SA" or "ar-EG" or "he" or "fa" or "ur";
    }
}
```

## 五、CJK 字体专项优化

### 5.1 CJK 字符集分层策略

```csharp
// CJK 字符覆盖优先级分层
public static class CJKCharsetStrategy
{
    // 第1层：高频3500字（覆盖日常95%文本）
    // 国标GB2312一级字符集
    
    // 第2层：7000字（覆盖99%文本）
    // GB2312全部
    
    // 第3层：动态加载（剩余生僻字）
    // 实时从字体文件按需提取
    
    public enum FontPackagePriority
    {
        CoreCJK_3500,    // 3500常用字，预加载，约4MB图集
        ExtendedCJK_7000, // 扩展常用字，异步加载
        FullCJK_27000,   // 全字集，动态图集
    }
    
    // 判断字符属于哪一层
    public static FontPackagePriority GetPriority(char c)
    {
        if (IsCoreCJK(c)) return FontPackagePriority.CoreCJK_3500;
        if (IsExtendedCJK(c)) return FontPackagePriority.ExtendedCJK_7000;
        return FontPackagePriority.FullCJK_27000;
    }
    
    private static bool IsCoreCJK(char c)
    {
        // 国标一级汉字范围（简化示例）
        return c >= '\u4E00' && c <= '\u9FFF';
    }
    
    private static bool IsExtendedCJK(char c)
    {
        return (c >= '\u3400' && c <= '\u4DBF') || // CJK扩展A
               (c >= '\u20000' && c <= '\u2A6DF');  // CJK扩展B（代理对）
    }
}

// 运行时CJK字体图集构建器
public class CJKAtlasBuilder
{
    private TMP_FontAsset _fontAsset;
    
    // 根据当前场景文本预构建图集
    public IEnumerator PreBuildForScene(string sceneId)
    {
        // 1. 收集场景所有文本
        var allText = LocalizationManager.GetAllTextForScene(sceneId);
        var requiredChars = ExtractUniqueChars(allText);
        
        // 2. 按优先级分组
        var coreChars = requiredChars.Where(c => CJKCharsetStrategy.GetPriority(c) 
            == CJKCharsetStrategy.FontPackagePriority.CoreCJK_3500).ToList();
        var extChars = requiredChars.Except(coreChars).ToList();
        
        // 3. 立即加载核心字符
        _fontAsset.TryAddCharacters(coreChars.ToArray());
        yield return null;
        
        // 4. 异步加载扩展字符（不阻塞）
        if (extChars.Count > 0)
        {
            yield return LoadCharactersAsync(extChars);
        }
        
        Debug.Log($"CJK图集构建完成: 共 {requiredChars.Count} 个字符");
    }
    
    private IEnumerator LoadCharactersAsync(List<uint> chars)
    {
        const int BatchSize = 100;
        for (int i = 0; i < chars.Count; i += BatchSize)
        {
            var batch = chars.GetRange(i, Mathf.Min(BatchSize, chars.Count - i));
            _fontAsset.TryAddCharacters(batch.ToArray());
            yield return null; // 每批次让出一帧
        }
    }
    
    private List<uint> ExtractUniqueChars(string[] texts)
    {
        var set = new HashSet<uint>();
        foreach (var text in texts)
        {
            var e = StringInfo.GetTextElementEnumerator(text);
            while (e.MoveNext())
            {
                string elem = (string)e.Current;
                set.Add((uint)char.ConvertToUtf32(elem, 0));
            }
        }
        return set.ToList();
    }
}
```

```csharp
// 扩展TMP富文本标签：支持游戏专属格式
// 如 <key>A</key> → 渲染为游戏手柄/键盘按键图标

[RequireComponent(typeof(TMP_Text))]
public class GameRichTextParser : MonoBehaviour
{
    [Serializable]
    public class TagConfig
    {
        public string tagName;
        public Sprite iconSprite;
        public Color tintColor = Color.white;
    }
    
    [SerializeField] private List<TagConfig> _customTags;
    [SerializeField] private TMP_SpriteAsset _spriteAsset;
    
    private TMP_Text _text;
    private Dictionary<string, TagConfig> _tagMap;
    
    private void Awake()
    {
        _text = GetComponent<TMP_Text>();
        _tagMap = new Dictionary<string, TagConfig>();
        foreach (var tag in _customTags)
            _tagMap[tag.tagName] = tag;
    }
    
    public void SetGameText(string rawText)
    {
        string processed = ProcessCustomTags(rawText);
        _text.SetText(processed);
    }
    
    private string ProcessCustomTags(string text)
    {
        // <key>A</key> → <sprite name="key_A">
        // <color_scheme>primary</color_scheme> → <color=#FF6B35>
        // <num_format>1234567</num_format> → 1,234,567 (按当前语言格式化)
        
        var sb = new StringBuilder(text);
        
        foreach (var tag in _customTags)
        {
            string openTag  = $"<{tag.tagName}>";
            string closeTag = $"</{tag.tagName}>";
            
            int startIdx;
            while ((startIdx = sb.ToString().IndexOf(openTag)) >= 0)
            {
                int endIdx = sb.ToString().IndexOf(closeTag, startIdx);
                if (endIdx < 0) break;
                
                int contentStart = startIdx + openTag.Length;
                int contentLen = endIdx - contentStart;
                string content = sb.ToString().Substring(contentStart, contentLen);
                
                // 替换为精灵标签
                string replacement = $"<sprite name=\"{tag.tagName}_{content}\">";
                
                sb.Remove(startIdx, endIdx + closeTag.Length - startIdx);
                sb.Insert(startIdx, replacement);
            }
        }
        
        // 数字格式化
        text = FormatNumbers(sb.ToString());
        
        return text;
    }
    
    private string FormatNumbers(string text)
    {
        // 根据当前语言格式化数字
        // 阿拉伯语使用阿拉伯-印度数字: ٠١٢٣٤٥٦٧٨٩
        string langCode = LocalizationManager.CurrentLanguage;
        if (langCode.StartsWith("ar"))
        {
            return System.Text.RegularExpressions.Regex.Replace(text, @"\d+", m =>
            {
                var arabic = new string(m.Value.Select(c => (char)(c - '0' + '\u0660')).ToArray());
                return arabic;
            });
        }
        return text;
    }
}
```

```csharp
// TMP 文本渲染批处理优化
public static class TMPBatchOptimizer
{
    // 问题：大量动态文本导致Draw Call暴增
    // 原因：每个TMP_Text是独立的Draw Call
    // 方案：使用TMP_Text共享材质 + Canvas合并

    public static void OptimizeTextCanvas(Canvas canvas)
    {
        // 1. 确保使用 CanvasRenderer 而非 WorldSpace Canvas
        canvas.renderMode = RenderMode.ScreenSpaceCamera;
        canvas.additionalShaderChannels = AdditionalCanvasShaderChannels.TexCoord1
            | AdditionalCanvasShaderChannels.Color;
        
        // 2. 合并使用相同字体的文本到同一字体材质
        var textComponents = canvas.GetComponentsInChildren<TMP_Text>();
        var fontGroups = textComponents.GroupBy(t => t.font);
        
        foreach (var group in fontGroups)
        {
            var font = group.Key;
            if (font == null) continue;
            
            // 创建共享材质实例（避免每个文本独立材质）
            var sharedMaterial = new Material(font.material);
            
            foreach (var text in group)
            {
                text.fontSharedMaterial = sharedMaterial;
            }
        }
    }
    
    // 分析当前场景文本DrawCall情况
    [MenuItem("Tools/Font/Analyze Text DrawCalls")]
    public static void AnalyzeDrawCalls()
    {
        var allTexts = FindObjectsOfType<TMP_Text>();
        var materialGroups = allTexts.GroupBy(t => t.fontMaterial?.GetInstanceID() ?? 0);
        
        Debug.Log($"=== TMP DrawCall 分析 ===");
        Debug.Log($"文本组件总数: {allTexts.Length}");
        Debug.Log($"独立材质数(≈DrawCall数): {materialGroups.Count()}");
        
        foreach (var group in materialGroups.OrderByDescending(g => g.Count()))
        {
            var sample = group.First();
            Debug.Log($"  材质: {sample.fontMaterial?.name ?? "null"} - {group.Count()} 个文本");
        }
    }
```

## 七、字体打包与热更新

```csharp
// 字体资源热更新方案
public class FontHotUpdateManager : MonoBehaviour
{
    [SerializeField] private string _fontCDNBase = "https://cdn.example.com/fonts/";
    
    private Dictionary<string, TMP_FontAsset> _loadedFonts = new();
    
    // 按语言按需下载字体
    public IEnumerator EnsureFontLoaded(string languageCode)
    {
        string fontKey = GetFontKeyForLanguage(languageCode);
        
        if (_loadedFonts.ContainsKey(fontKey))
            yield break;
        
        // 检查本地缓存
        string localPath = GetLocalFontPath(fontKey);
        if (File.Exists(localPath))
        {
            yield return LoadFontFromLocal(fontKey, localPath);
        }
        else
        {
            // 从CDN下载
            string url = _fontCDNBase + fontKey + ".bundle";
            yield return DownloadAndCacheFont(fontKey, url, localPath);
        }
    }
    
    private IEnumerator DownloadAndCacheFont(string key, string url, string localPath)
    {
        using var request = UnityEngine.Networking.UnityWebRequest.Get(url);
        yield return request.SendWebRequest();
        
        if (request.result != UnityEngine.Networking.UnityWebRequest.Result.Success)
        {
            Debug.LogError($"字体下载失败: {url} - {request.error}");
            yield break;
        }
        
        // 保存到本地
        byte[] data = request.downloadHandler.data;
        File.WriteAllBytes(localPath, data);
        
        yield return LoadFontFromBundle(key, localPath);
    }
    
    private IEnumerator LoadFontFromBundle(string key, string path)
    {
        var bundleRequest = AssetBundle.LoadFromFileAsync(path);
        yield return bundleRequest;
        
        var bundle = bundleRequest.assetBundle;
        if (bundle == null) yield break;
        
        var fontRequest = bundle.LoadAssetAsync<TMP_FontAsset>(key);
        yield return fontRequest;
        
        var font = fontRequest.asset as TMP_FontAsset;
        if (font != null)
        {
            _loadedFonts[key] = font;
            // 注册到TMP全局字体库
            TMP_Settings.defaultFontAsset = font;
        }
        
        bundle.Unload(false);
    }
    
    private string GetFontKeyForLanguage(string langCode) =>
        langCode switch
        {
            "zh-CN" => "NotoSansSC",
            "zh-TW" => "NotoSansTC",
            "ja"    => "NotoSansJP",
            "ko"    => "NotoSansKR",
            "ar"    => "NotoSansArabic",
            "th"    => "NotoSansThai",
            _       => "NotoSans"
        };
    
    private string GetLocalFontPath(string key) =>
        Path.Combine(Application.persistentDataPath, "Fonts", key + ".bundle");
    
    private IEnumerator LoadFontFromLocal(string key, string path)
    {
        yield return LoadFontFromBundle(key, path);
    }
}
```

## 八、最佳实践总结

### 8.1 多语言字体技术选型

| 场景 | 推荐方案 | 理由 |
|-----|---------|------|
| 英文/欧文 | SDF预生成图集 | 字符集小，质量高 |
| CJK（中日韩） | 分层动态图集 | 字符集过大，动态加载 |
| 阿拉伯/希伯来 | BiDi处理+RTL排版 | 需要双向算法 |
| 泰语/缅甸语 | HarfBuzz整形 | 复杂字形组合 |
| 多语言混排 | 字体回退链 | 单字体无法全覆盖 |

### 8.2 工程规范

```
多语言字体工程清单：

✅ 字符集管理
  - 扫描所有本地化文本，提取实际用到的字符集
  - 按语言分别维护字符集文件，纳入版本管理
  - CI检查：新增文本中的字符是否都在图集中

✅ 性能控制
  - CJK核心3500字预加载，其余动态加载
  - 使用字体回退链避免字体膨胀
  - 共享TMP材质减少DrawCall
  - 在语言切换时及时释放旧语言字体图集

✅ 质量保障
  - 自动化截图测试：每个语言的UI截图与基准对比
  - 文本溢出检测：所有UI组件在最长语言下的截断测试
  - RTL排版自动化验证脚本

✅ 运营支持
  - 字体热更新通道（新语言无需更新客户端）
  - 字体CDN分发策略（按地区就近分发）
```

多语言字体系统是本地化工程中最容易被低估的技术挑战。从早期架构设计就做好字符集管理、字体分层、回退机制，能够避免后期大量的返工与线上问题。
