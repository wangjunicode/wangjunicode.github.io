# 游戏角色换装与外观定制系统：从装备换肤到捏脸工程完全指南

## 一、换装系统概述

角色外观定制是现代游戏中提升玩家粘性的核心功能，技术实现涉及多个层面：

```
换装系统架构
├── 装备部位管理（头盔/身体/手/腿/武器）
├── 骨骼共享机制（Shared Skeleton）
├── 网格合并优化（Mesh Combine）
├── 材质染色系统（Dye/Tint）
├── 捏脸 BlendShape 参数系统
├── 外观数据序列化（存档/网络同步）
└── 换装预览（UI 虚拟相机）
```

### 核心挑战

| 挑战 | 表现 | 解决方案 |
|------|------|---------|
| DrawCall 爆炸 | 每件装备独立 Renderer | 网格合并或 GPU Instancing |
| 骨骼不匹配 | 换装后动画错乱 | 骨骼重映射/共享骨架 |
| 内存峰值 | 预加载所有装备 | Addressables 按需加载 |
| 捏脸实时性 | BlendShape 更新卡顿 | Burst Job 批量更新 |
| 网络同步 | 外观数据过大 | 位压缩编码 |

## 二、骨骼共享换装方案

### 2.1 原理

所有部位共享同一套骨骼（`bones[]` 数组），每个 `SkinnedMeshRenderer` 引用相同的骨骼 Transform 数组，从而保证：
- 动画统一驱动所有部位
- 可任意替换任何部位的 Mesh
- 无需重新绑定蒙皮权重

### 2.3 骨骼共享换装核心逻辑

```csharp
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.AddressableAssets;
using UnityEngine.ResourceManagement.AsyncOperations;
using Cysharp.Threading.Tasks;

/// <summary>
/// 角色换装管理器：骨骼共享方案
/// </summary>
public class CharacterEquipmentManager : MonoBehaviour
{
    [Header("骨骼根节点")]
    public Transform skeletonRoot;

    [Header("默认身体 SkinnedMeshRenderer（用于获取骨骼引用）")]
    public SkinnedMeshRenderer bodyRenderer;

    // 每个部位当前装备的 SkinnedMeshRenderer
    private readonly Dictionary<EquipSlot, SkinnedMeshRenderer> _slotRenderers = new();
    
    // 骨骼名称 -> Transform 映射（加速骨骼查找）
    private Dictionary<string, Transform> _boneMap;

    // 当前外观状态
    private CharacterAppearance _currentAppearance = new();

    // 已加载的 Addressables Handle（用于释放）
    private readonly Dictionary<EquipSlot, AsyncOperationHandle<Mesh>> _loadHandles = new();

    void Awake()
    {
        BuildBoneMap();
    }

    void BuildBoneMap()
    {
        _boneMap = new Dictionary<string, Transform>();
        BuildBoneMapRecursive(skeletonRoot);
    }

    void BuildBoneMapRecursive(Transform bone)
    {
        if (bone == null) return;
        _boneMap[bone.name] = bone;
        foreach (Transform child in bone)
            BuildBoneMapRecursive(child);
    }

    /// <summary>
    /// 异步换装：加载指定部位的装备
    /// </summary>
    public async UniTask EquipAsync(EquipSlot slot, EquipmentConfig config)
    {
        if (config == null)
        {
            Unequip(slot);
            return;
        }

        // 1. 异步加载 Mesh（Addressables）
        Mesh mesh = null;
        if (!string.IsNullOrEmpty(config.meshAssetKey))
        {
            // 释放旧的加载句柄
            if (_loadHandles.TryGetValue(slot, out var oldHandle) && oldHandle.IsValid())
                Addressables.Release(oldHandle);

            var handle = Addressables.LoadAssetAsync<Mesh>(config.meshAssetKey);
            _loadHandles[slot] = handle;
            mesh = await handle;
        }

        // 2. 获取或创建该部位的 SkinnedMeshRenderer
        var smr = GetOrCreateSlotRenderer(slot);

        // 3. 重新绑定骨骼（骨骼共享关键步骤）
        if (mesh != null)
        {
            smr.sharedMesh = mesh;
            RebindBones(smr, mesh);
        }

        // 4. 设置材质
        if (config.materials?.Length > 0)
            smr.sharedMaterials = config.materials;

        // 5. 更新外观记录
        _currentAppearance.SetSlot(slot, config.equipId);
        
        // 6. 应用染色
        if (config.supportDye)
            ApplyDye(smr, _currentAppearance.GetDyeColor(slot, 0), _currentAppearance.GetDyeColor(slot, 1));

        Debug.Log($"[Equip] Slot={slot} equipped: {config.displayName}");
    }

    /// <summary>
    /// 骨骼重绑定：将 Mesh 的骨骼名称映射到当前骨架
    /// </summary>
    void RebindBones(SkinnedMeshRenderer smr, Mesh mesh)
    {
        // 从 Mesh 的骨骼名称重新查找 Transform（需要 Mesh 存储了 bindposes 和 bone names）
        // 注意：Unity 原生 Mesh 不直接存储骨骼名称，通常需要自定义数据或约定命名

        // 方案一：从参考 Renderer 复制骨骼数组（推荐：所有装备骨骼顺序一致）
        if (bodyRenderer != null && bodyRenderer.bones.Length > 0)
        {
            smr.bones = bodyRenderer.bones;
            smr.rootBone = bodyRenderer.rootBone;
            return;
        }

        // 方案二：按骨骼名称逐个查找（需要 Mesh 附带骨骼名称元数据）
        // 此方案性能略低，适合骨骼数量不多的场景
        var boneNames = GetBoneNamesFromMesh(mesh);
        if (boneNames != null)
        {
            var bones = new Transform[boneNames.Length];
            for (int i = 0; i < boneNames.Length; i++)
            {
                if (_boneMap.TryGetValue(boneNames[i], out var bone))
                    bones[i] = bone;
                else
                    Debug.LogWarning($"[Equip] 骨骼未找到: {boneNames[i]}");
            }
            smr.bones = bones;
        }
    }

    // 从自定义 Mesh 元数据获取骨骼名称（需要配合美术管线约定）
    string[] GetBoneNamesFromMesh(Mesh mesh)
    {
        // 实际项目中可以用 Mesh.name + 外部JSON/ScriptableObject 存储骨骼名称表
        return null;
    }

    SkinnedMeshRenderer GetOrCreateSlotRenderer(EquipSlot slot)
    {
        if (_slotRenderers.TryGetValue(slot, out var existing) && existing != null)
            return existing;

        var go = new GameObject($"Equip_{slot}");
        go.transform.SetParent(transform, false);
        var smr = go.AddComponent<SkinnedMeshRenderer>();
        _slotRenderers[slot] = smr;
        return smr;
    }

    public void Unequip(EquipSlot slot)
    {
        if (_slotRenderers.TryGetValue(slot, out var smr) && smr != null)
            smr.enabled = false;
        _currentAppearance.ClearSlot(slot);
    }

    void ApplyDye(SkinnedMeshRenderer smr, Color primary, Color secondary)
    {
        // 复制材质实例（避免修改共享材质）
        var mats = smr.materials; // 这会自动创建实例
        foreach (var mat in mats)
        {
            if (mat.HasProperty("_PrimaryColor")) mat.SetColor("_PrimaryColor", primary);
            if (mat.HasProperty("_SecondaryColor")) mat.SetColor("_SecondaryColor", secondary);
        }
        smr.materials = mats;
    }
}
```

## 三、网格合并优化

### 3.1 为什么需要网格合并

每个 `SkinnedMeshRenderer` 产生一个 DrawCall。5件装备 = 5个 DrawCall。  
**合并后：1个 DrawCall**，适合移动端等对 DrawCall 敏感的平台。

### 3.2 动态合并实现

```csharp
using System.Collections.Generic;
using System.Linq;
using UnityEngine;

/// <summary>
/// SkinnedMeshRenderer 动态合并工具
/// 注意：合并后无法单独更换部位，需要脱装时重新合并
/// </summary>
public static class SkinnedMeshCombiner
{
    public struct CombineInput
    {
        public SkinnedMeshRenderer renderer;
        public Matrix4x4[] bindPoses;
    }

    /// <summary>
    /// 合并多个 SkinnedMeshRenderer 为一个（使用 Atlas 图集共享材质）
    /// </summary>
    public static SkinnedMeshRenderer Combine(
        List<SkinnedMeshRenderer> renderers,
        Transform skeletonRoot,
        out Material combinedMaterial,
        int atlasSize = 2048)
    {
        // 1. 收集所有骨骼（去重）
        var allBones = new List<Transform>();
        var boneIndexMap = new Dictionary<Transform, int>();
        foreach (var smr in renderers)
        {
            foreach (var bone in smr.bones)
            {
                if (bone != null && !boneIndexMap.ContainsKey(bone))
                {
                    boneIndexMap[bone] = allBones.Count;
                    allBones.Add(bone);
                }
            }
        }

        // 2. 构建图集（将所有贴图合并到一张大贴图）
        var textures = renderers.SelectMany(r => r.sharedMaterials)
                                .Select(m => m?.mainTexture as Texture2D)
                                .Where(t => t != null)
                                .Distinct()
                                .ToArray();

        var atlas = new Texture2D(atlasSize, atlasSize, TextureFormat.RGBA32, false);
        var uvRects = atlas.PackTextures(textures, 2, atlasSize);
        atlas.Apply();

        // 3. 重新映射 UV 坐标 + 合并 CombineInstance
        var combineInstances = new List<CombineInstance>();
        int texIdx = 0;

        foreach (var smr in renderers)
        {
            if (smr.sharedMesh == null) continue;

            var mesh = smr.sharedMesh;
            var newMesh = UnityEngine.Object.Instantiate(mesh);

            // 重新映射骨骼索引
            var boneWeights = mesh.boneWeights;
            var newBoneWeights = new BoneWeight[boneWeights.Length];
            for (int i = 0; i < boneWeights.Length; i++)
            {
                var bw = boneWeights[i];
                // 将原骨骼索引映射到合并后的索引
                int b0 = bw.boneIndex0 < smr.bones.Length ? boneIndexMap.GetValueOrDefault(smr.bones[bw.boneIndex0], bw.boneIndex0) : bw.boneIndex0;
                int b1 = bw.boneIndex1 < smr.bones.Length ? boneIndexMap.GetValueOrDefault(smr.bones[bw.boneIndex1], bw.boneIndex1) : bw.boneIndex1;
                int b2 = bw.boneIndex2 < smr.bones.Length ? boneIndexMap.GetValueOrDefault(smr.bones[bw.boneIndex2], bw.boneIndex2) : bw.boneIndex2;
                int b3 = bw.boneIndex3 < smr.bones.Length ? boneIndexMap.GetValueOrDefault(smr.bones[bw.boneIndex3], bw.boneIndex3) : bw.boneIndex3;
                newBoneWeights[i] = new BoneWeight { boneIndex0 = b0, boneIndex1 = b1, boneIndex2 = b2, boneIndex3 = b3,
                    weight0 = bw.weight0, weight1 = bw.weight1, weight2 = bw.weight2, weight3 = bw.weight3 };
            }
            newMesh.boneWeights = newBoneWeights;

            // 重新映射 UV（映射到图集对应区域）
            if (texIdx < uvRects.Length)
            {
                var rect = uvRects[texIdx++];
                var uvs = mesh.uv;
                for (int i = 0; i < uvs.Length; i++)
                    uvs[i] = new Vector2(rect.x + uvs[i].x * rect.width, rect.y + uvs[i].y * rect.height);
                newMesh.uv = uvs;
            }

            for (int sub = 0; sub < newMesh.subMeshCount; sub++)
            {
                combineInstances.Add(new CombineInstance
                {
                    mesh = newMesh,
                    subMeshIndex = sub,
                    transform = smr.transform.localToWorldMatrix
                });
            }
        }

        // 4. 合并 Mesh
        var combinedMesh = new Mesh { name = "CombinedCharacterMesh" };
        combinedMesh.CombineMeshes(combineInstances.ToArray(), true, false);

        // 5. 设置 bindposes
        var bindPoses = new Matrix4x4[allBones.Count];
        for (int i = 0; i < allBones.Count; i++)
            bindPoses[i] = allBones[i].worldToLocalMatrix * skeletonRoot.localToWorldMatrix;
        combinedMesh.bindposes = bindPoses;

        // 6. 创建合并后的 SkinnedMeshRenderer
        var result = new GameObject("CombinedRenderer").AddComponent<SkinnedMeshRenderer>();
        result.sharedMesh = combinedMesh;
        result.bones = allBones.ToArray();
        result.rootBone = skeletonRoot;

        // 7. 创建使用图集的材质
        combinedMaterial = new Material(Shader.Find("Universal Render Pipeline/Lit"));
        combinedMaterial.mainTexture = atlas;
        result.sharedMaterial = combinedMaterial;

        return result;
    }
}
```

## 四、捏脸（BlendShape）参数系统

### 4.1 捏脸参数定义

```csharp
using System;
using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 捏脸参数配置（映射 BlendShape 名称到语义化参数）
/// </summary>
[CreateAssetMenu(menuName = "Character/Face Customization Config")]
public class FaceCustomizationConfig : ScriptableObject
{
    [Serializable]
    public class FaceParameter
    {
        public string paramId;       // 逻辑 ID，如 "eye_size"
        public string displayName;   // 显示名，如 "眼睛大小"
        public string category;      // 分类，如 "眼睛"
        
        [Range(0, 100)]
        public float defaultValue = 50f;
        
        // 可能影响多个 BlendShape（有时候一个语义参数对应多个形态）
        public BlendShapeMapping[] mappings;
    }

    [Serializable]
    public class BlendShapeMapping
    {
        public string blendShapeName;
        public AnimationCurve curve; // 参数值 0-100 → BlendShape weight 0-100 的映射曲线
        public SkinnedMeshRendererTarget target; // 哪个 Renderer 上的 BlendShape
    }

    [Serializable]
    public enum SkinnedMeshRendererTarget { Head, Body, Eyes, Brows }

    public FaceParameter[] parameters;
}

/// <summary>
/// 捏脸控制器
/// </summary>
public class FaceCustomizationController : MonoBehaviour
{
    public FaceCustomizationConfig config;

    [Header("目标 SkinnedMeshRenderer")]
    public SkinnedMeshRenderer headRenderer;
    public SkinnedMeshRenderer eyesRenderer;

    // 当前参数值字典（参数ID → 值 0-100）
    private Dictionary<string, float> _paramValues = new();

    // BlendShape 名称 → 索引缓存
    private Dictionary<string, int> _headBlendShapeIndex = new();
    private Dictionary<string, int> _eyesBlendShapeIndex = new();

    void Awake()
    {
        BuildBlendShapeIndexCache();
        ResetToDefault();
    }

    void BuildBlendShapeIndexCache()
    {
        if (headRenderer?.sharedMesh != null)
        {
            var mesh = headRenderer.sharedMesh;
            for (int i = 0; i < mesh.blendShapeCount; i++)
                _headBlendShapeIndex[mesh.GetBlendShapeName(i)] = i;
        }

        if (eyesRenderer?.sharedMesh != null)
        {
            var mesh = eyesRenderer.sharedMesh;
            for (int i = 0; i < mesh.blendShapeCount; i++)
                _eyesBlendShapeIndex[mesh.GetBlendShapeName(i)] = i;
        }
    }

    public void ResetToDefault()
    {
        foreach (var param in config.parameters)
            SetParameter(param.paramId, param.defaultValue);
    }

    /// <summary>
    /// 设置捏脸参数（0-100）
    /// </summary>
    public void SetParameter(string paramId, float value)
    {
        value = Mathf.Clamp(value, 0f, 100f);
        _paramValues[paramId] = value;

        var paramDef = System.Array.Find(config.parameters, p => p.paramId == paramId);
        if (paramDef == null) return;

        foreach (var mapping in paramDef.mappings)
        {
            float blendWeight = mapping.curve?.Evaluate(value / 100f) * 100f ?? value;
            ApplyBlendShape(mapping.target, mapping.blendShapeName, blendWeight);
        }
    }

    void ApplyBlendShape(FaceCustomizationConfig.SkinnedMeshRendererTarget target, string shapeName, float weight)
    {
        SkinnedMeshRenderer smr;
        Dictionary<string, int> indexCache;

        switch (target)
        {
            case FaceCustomizationConfig.SkinnedMeshRendererTarget.Head:
                smr = headRenderer; indexCache = _headBlendShapeIndex; break;
            case FaceCustomizationConfig.SkinnedMeshRendererTarget.Eyes:
                smr = eyesRenderer; indexCache = _eyesBlendShapeIndex; break;
            default: return;
        }

        if (smr == null) return;
        if (indexCache.TryGetValue(shapeName, out int idx))
            smr.SetBlendShapeWeight(idx, weight);
        else
            Debug.LogWarning($"[FaceCustom] BlendShape 未找到: {shapeName}");
    }

    public float GetParameter(string paramId) =>
        _paramValues.TryGetValue(paramId, out float v) ? v : 0f;

    /// <summary>
    /// 序列化当前捏脸数据
    /// </summary>
    public FaceAppearanceData Serialize()
    {
        var data = new FaceAppearanceData();
        foreach (var kv in _paramValues)
            data.parameters[kv.Key] = (byte)(kv.Value); // 压缩为 0-100 byte
        return data;
    }

    public void Deserialize(FaceAppearanceData data)
    {
        foreach (var kv in data.parameters)
            SetParameter(kv.Key, kv.Value);
    }
}

[Serializable]
public class FaceAppearanceData
{
    public Dictionary<string, byte> parameters = new();
}
```

## 五、外观染色系统

### 5.1 Shader 染色实现（URP）

```hlsl
// CharacterDye.shader
Shader "Custom/CharacterDye"
{
    Properties
    {
        _MainTex ("Albedo", 2D) = "white" {}
        _MaskTex ("Dye Mask (R=Primary, G=Secondary)", 2D) = "black" {}
        _PrimaryColor ("Primary Dye Color", Color) = (1,1,1,1)
        _SecondaryColor ("Secondary Dye Color", Color) = (0.5,0.5,0.5,1)
        _DyeStrength ("Dye Strength", Range(0,1)) = 1.0
    }
    SubShader
    {
        Tags { "RenderPipeline"="UniversalPipeline" "RenderType"="Opaque" }
        
        HLSLPROGRAM
        // ... URP Pass 设置省略
        
        half4 frag(Varyings input) : SV_Target
        {
            half4 albedo = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, input.uv);
            half4 mask   = SAMPLE_TEXTURE2D(_MaskTex, sampler_MaskTex, input.uv);
            
            // mask.r = 主染色区域权重，mask.g = 副染色区域权重
            // 使用 HSV 混合：保留原始亮度，替换色调
            half3 dyedPrimary = ApplyDyeColor(albedo.rgb, _PrimaryColor.rgb, mask.r * _DyeStrength);
            half3 dyedFinal   = ApplyDyeColor(dyedPrimary, _SecondaryColor.rgb, mask.g * _DyeStrength);
            
            return half4(dyedFinal, albedo.a);
        }
        
        // HSV 染色函数：保留亮度，替换色调和饱和度
        half3 ApplyDyeColor(half3 original, half3 dyeColor, half strength)
        {
            if (strength <= 0) return original;
            
            half origLuminance = dot(original, half3(0.299, 0.587, 0.114));
            half3 tinted = dyeColor * origLuminance * 2.0;
            return lerp(original, tinted, strength);
        }
        
        ENDHLSL
    }
}
```

### 5.2 运行时染色管理器

```csharp
using System.Collections.Generic;
using UnityEngine;

public class CharacterDyeManager : MonoBehaviour
{
    // 每个部位的独立 MaterialPropertyBlock（避免创建 Material 实例）
    private Dictionary<SkinnedMeshRenderer, MaterialPropertyBlock> _propBlocks = new();

    private static readonly int PrimaryColorID   = Shader.PropertyToID("_PrimaryColor");
    private static readonly int SecondaryColorID = Shader.PropertyToID("_SecondaryColor");
    private static readonly int DyeStrengthID    = Shader.PropertyToID("_DyeStrength");

    /// <summary>
    /// 应用染色（使用 MaterialPropertyBlock，零 GC 分配）
    /// </summary>
    public void ApplyDye(SkinnedMeshRenderer smr, Color primary, Color secondary, float strength = 1f)
    {
        if (smr == null) return;

        if (!_propBlocks.TryGetValue(smr, out var block))
        {
            block = new MaterialPropertyBlock();
            _propBlocks[smr] = block;
        }

        smr.GetPropertyBlock(block);
        block.SetColor(PrimaryColorID, primary);
        block.SetColor(SecondaryColorID, secondary);
        block.SetFloat(DyeStrengthID, strength);
        smr.SetPropertyBlock(block);
    }

    public void ResetDye(SkinnedMeshRenderer smr)
    {
        if (smr == null) return;
        smr.SetPropertyBlock(null);
        _propBlocks.Remove(smr);
    }
}
```

## 六、外观数据序列化与存储

### 6.1 外观数据结构

```csharp
using System;
using System.Collections.Generic;

/// <summary>
/// 完整的角色外观数据，用于存档/网络同步

[Serializable]
public class CharacterAppearance

    // 染色数据：部位 → (主色, 副色)
    public Dictionary<EquipSlot, SerializableColor2> dyeColors = new();
    
    // 捏脸参数
    public FaceAppearanceData faceData = new();
    
    // 性别/体型
    public int bodyType = 0; // 0=默认, 1=纤细, 2=健壮
    
    public void SetSlot(EquipSlot slot, string equipId)
        => equipSlots[slot] = equipId;

    public void ClearSlot(EquipSlot slot)
        => equipSlots.Remove(slot);

    public string GetSlot(EquipSlot slot)
        => equipSlots.TryGetValue(slot, out var id) ? id : null;

    public Color GetDyeColor(EquipSlot slot, int channel)
    {
        if (!dyeColors.TryGetValue(slot, out var c)) return Color.white;
        return channel == 0 ? c.primary : c.secondary;
    }

    public void SetDyeColor(EquipSlot slot, Color primary, Color secondary)
        => dyeColors[slot] = new SerializableColor2 { primary = primary, secondary = secondary };
}

[Serializable]
public struct SerializableColor2
{
    public Color primary;
    public Color secondary;
}
```

### 6.2 紧凑编码（网络同步优化）

```csharp
using System.Collections.Generic;
using System.IO;
using UnityEngine;

/// <summary>
/// 外观数据紧凑二进制编码（用于网络同步，减少带宽）
/// </summary>
public static class AppearanceEncoder
{
    // 颜色压缩：RGBA → 4字节（各通道 8bit）
    static uint PackColor(Color c)
    {
        uint r = (uint)(c.r * 255) & 0xFF;
        uint g = (uint)(c.g * 255) & 0xFF;
        uint b = (uint)(c.b * 255) & 0xFF;
        return (r << 16) | (g << 8) | b;
    }

    static Color UnpackColor(uint packed)
    {
        float r = ((packed >> 16) & 0xFF) / 255f;
        float g = ((packed >> 8)  & 0xFF) / 255f;
        float b = ( packed        & 0xFF) / 255f;
        return new Color(r, g, b);
    }

    public static byte[] Encode(CharacterAppearance appearance, Dictionary<string, ushort> equipIdTable)
    {
        using var ms = new MemoryStream();
        using var writer = new BinaryWriter(ms);

        writer.Write((byte)appearance.version);
        writer.Write((byte)appearance.bodyType);

        // 装备槽（每个部位用 2 字节 equipId 索引）
        writer.Write((byte)appearance.equipSlots.Count);
        foreach (var kv in appearance.equipSlots)
        {
            writer.Write((byte)kv.Key);
            ushort idIdx = equipIdTable.TryGetValue(kv.Value, out var idx) ? idx : ushort.MaxValue;
            writer.Write(idIdx);
        }

        // 染色数据（每个部位 7 字节：1字节slot + 3字节主色 + 3字节副色）
        writer.Write((byte)appearance.dyeColors.Count);
        foreach (var kv in appearance.dyeColors)
        {
            writer.Write((byte)kv.Key);
            writer.Write((byte)(kv.Value.primary.r * 255));
            writer.Write((byte)(kv.Value.primary.g * 255));
            writer.Write((byte)(kv.Value.primary.b * 255));
            writer.Write((byte)(kv.Value.secondary.r * 255));
            writer.Write((byte)(kv.Value.secondary.g * 255));
            writer.Write((byte)(kv.Value.secondary.b * 255));
        }

        // 捏脸参数（每个参数：n字节ID哈希 + 1字节值）
        var faceParams = appearance.faceData.parameters;
        writer.Write((byte)faceParams.Count);
        foreach (var kv in faceParams)
        {
            writer.Write((uint)kv.Key.GetHashCode()); // 4字节哈希
            writer.Write(kv.Value);
        }

        return ms.ToArray();
    }

    // Decode 方法对称实现（省略）
}
```

## 七、换装预览系统

```csharp
using UnityEngine;
using UnityEngine.UI;
using Cysharp.Threading.Tasks;

/// <summary>
/// 换装预览相机控制器
/// </summary>
public class EquipPreviewController : MonoBehaviour
{
    [Header("预览相机")]
    public Camera previewCamera;
    public RenderTexture previewRT;
    public RawImage previewImage;

    [Header("旋转控制")]
    public Transform characterPivot;
    public float rotateSpeed = 90f;
    public float zoomSpeed = 0.5f;
    [Range(1f, 5f)] public float cameraDistance = 3f;

    [Header("预览角色")]
    public CharacterEquipmentManager previewCharacter;

    private bool _isDragging;
    private Vector2 _lastMousePos;
    private float _currentYaw = 0;

    void Start()
    {
        // 设置预览 RT
        if (previewRT == null)
            previewRT = new RenderTexture(512, 512, 24);
        previewCamera.targetTexture = previewRT;
        previewImage.texture = previewRT;
    }

    void Update()
    {
        HandleInput();
    }

    void HandleInput()
    {
        // 拖动旋转
        if (Input.GetMouseButtonDown(0))
        {
            _isDragging = true;
            _lastMousePos = Input.mousePosition;
        }
        if (Input.GetMouseButtonUp(0)) _isDragging = false;

        if (_isDragging && characterPivot != null)
        {
            Vector2 delta = (Vector2)Input.mousePosition - _lastMousePos;
            _currentYaw -= delta.x * rotateSpeed * Time.deltaTime;
            characterPivot.localRotation = Quaternion.Euler(0, _currentYaw, 0);
            _lastMousePos = Input.mousePosition;
        }

        // 滚轮缩放
        float scroll = Input.GetAxis("Mouse ScrollWheel");
        cameraDistance = Mathf.Clamp(cameraDistance - scroll * zoomSpeed, 1f, 8f);
        if (previewCamera != null)
        {
            var pos = previewCamera.transform.localPosition;
            previewCamera.transform.localPosition = new Vector3(pos.x, pos.y, -cameraDistance);
        }
    }

    /// <summary>
    /// 在预览角色上试穿装备（不影响主角色）
    /// </summary>
    public async UniTask TryEquip(EquipSlot slot, EquipmentConfig config)
    {
        if (previewCharacter != null)
            await previewCharacter.EquipAsync(slot, config);
    }

    /// <summary>
    /// 截取当前预览图（用于存档头像）
    /// </summary>
    public Texture2D CapturePreview()
    {
        var prev = RenderTexture.active;
        RenderTexture.active = previewRT;
        var tex = new Texture2D(previewRT.width, previewRT.height, TextureFormat.RGB24, false);
        tex.ReadPixels(new Rect(0, 0, previewRT.width, previewRT.height), 0, 0);
        tex.Apply();
        RenderTexture.active = prev;
        return tex;
    }
}
```

## 八、最佳实践总结

### 架构设计

1. **骨骼共享是基础**：所有角色装备必须基于相同骨架制作，美术规范需在立项时确定
2. **按需加载（Addressables）**：不要预加载所有装备资源，按部位异步加载，降低内存峰值
3. **MaterialPropertyBlock 优于 Material 实例**：前者不额外创建 Material 对象，无内存碎片

### 性能优化

| 场景 | 策略 |
|------|------|
| 大量玩家同屏 | GPU Instancing + 简化 LOD 外观 |
| 移动端 | Mesh 合并减少 DrawCall |
| 高频捏脸调整 | 异步 BlendShape 更新（Burst Job） |
| 网络同步外观 | 紧凑编码（<64 字节/角色） |

### 美术规范要求

- **骨骼命名统一**：所有装备美术资产使用相同的骨骼命名约定（如 `Bip001_L_Hand`）
- **UV 分区约定**：主要换装部位放在独立 UV 通道，便于图集打包
- **BlendShape 命名约定**：如 `BS_EyeSize_Open`, `BS_FaceWidth_Wide`，保证跨模型一致
- **LOD 层级**：远景角色只渲染合并后的低模，节省渲染开销

### 扩展方向

- **动态骨骼换装**：斗篷、发饰等使用 `DynamicBone` 的装备，换装后需重新初始化物理链
- **换装动画过渡**：穿脱动画（配合 Timeline）提升代入感
- **AR 试衣**：对接摄像头姿态估计，实现现实中试穿效果
- **AI 配装推荐**：基于玩家历史行为训练配装推荐模型
