## 为什么游戏开发需要数学

```
"数学不好能做游戏开发吗？"

做普通功能：可以（大多数功能不需要复杂数学）
做好性能优化：难（理解矩阵运算才能优化渲染）
做渲染特效：必须（Shader 大量使用向量数学）
做物理系统：必须（碰撞检测、力学计算）
做战斗系统：需要（技能范围、弹道计算）
```

不需要成为数学家，但要掌握游戏开发常用的数学工具。

## 一、向量（Vector）

### 1.1 向量基础

```csharp
// 向量的本质：表示方向和大小（或者纯粹的位移）
Vector3 v = new Vector3(1, 0, 0);  // 向右的单位向量

// 关键操作
Vector3 a = new Vector3(1, 2, 3);
Vector3 b = new Vector3(4, 5, 6);

// 加法：位移叠加
Vector3 sum = a + b;  // (5, 7, 9)

// 减法：从 b 指向 a 的方向（及距离）
Vector3 diff = a - b;  // (-3, -3, -3)

// 数乘：缩放
Vector3 scaled = a * 2f;  // (2, 4, 6)

// 长度（模）
float length = a.magnitude;  // sqrt(1+4+9) = sqrt(14) ≈ 3.74

// 归一化（方向向量）
Vector3 dir = a.normalized;  // 长度为 1 的方向向量
```

### 1.2 点积（Dot Product）

```csharp
// 点积：a·b = |a||b|cos(θ)
// 几何意义：衡量两个向量的"对齐程度"
//   结果 > 0：同向（夹角 < 90°）
//   结果 = 0：垂直（夹角 = 90°）
//   结果 < 0：反向（夹角 > 90°）

float dot = Vector3.Dot(a, b); // Unity 的点积

// 实际应用 1：检测敌人是否在前方
bool IsInFront(Transform viewer, Vector3 target)
{
    Vector3 toTarget = (target - viewer.position).normalized;
    float dot = Vector3.Dot(viewer.forward, toTarget);
    return dot > 0; // dot > 0 意味着在前半球
}

// 实际应用 2：视野检测（60度视角）
bool IsInFieldOfView(Transform viewer, Vector3 target, float fovDegrees)
{
    Vector3 toTarget = (target - viewer.position).normalized;
    float dot = Vector3.Dot(viewer.forward, toTarget);
    float threshold = Mathf.Cos(fovDegrees * 0.5f * Mathf.Deg2Rad);
    return dot > threshold;
}

// 实际应用 3：光照计算（Lambert 漫反射）
// intensity = max(0, N·L)  其中 N=法线，L=光照方向
float diffuse = Mathf.Max(0, Vector3.Dot(normal, lightDir));

// 实际应用 4：投影
// 把 a 投影到 b 方向上
Vector3 projection = Vector3.Dot(a, b.normalized) * b.normalized;
```

### 1.3 叉积（Cross Product）

```csharp
// 叉积：a×b = 垂直于 a 和 b 的向量（长度 = |a||b|sin(θ)）
// 遵循右手定则
// 注意：叉积不满足交换律，a×b = -(b×a)

Vector3 cross = Vector3.Cross(a, b);

// 实际应用 1：计算法线（给三角形）
Vector3 CalculateTriangleNormal(Vector3 v0, Vector3 v1, Vector3 v2)
{
    Vector3 edge1 = v1 - v0;
    Vector3 edge2 = v2 - v0;
    return Vector3.Cross(edge1, edge2).normalized; // 三角形法线
}

// 实际应用 2：判断点在直线的哪一侧（2D）
float CrossProduct2D(Vector2 a, Vector2 b)
{
    return a.x * b.y - a.y * b.x; // 2D 叉积的 z 分量
}

bool IsPointOnLeftSide(Vector2 lineStart, Vector2 lineEnd, Vector2 point)
{
    Vector2 lineDir = lineEnd - lineStart;
    Vector2 toPoint = point - lineStart;
    return CrossProduct2D(lineDir, toPoint) > 0;
}

// 实际应用 3：获取与前方垂直的右方向
Vector3 GetRightVector(Vector3 forward, Vector3 worldUp)
{
    return Vector3.Cross(worldUp, forward).normalized;
}
```

## 二、矩阵（Matrix）

### 2.1 变换矩阵

```csharp
// 4×4 矩阵用于表示 3D 变换（平移+旋转+缩放）
// Unity 的 Transform 内部就是一个矩阵

// 获取变换矩阵
Matrix4x4 trs = Matrix4x4.TRS(
    position,   // 平移
    rotation,   // 旋转（四元数）
    scale       // 缩放
);

// 变换一个点（受平移影响）
Vector3 transformedPoint = trs.MultiplyPoint3x4(localPoint);

// 变换一个方向（不受平移影响）
Vector3 transformedDir = trs.MultiplyVector(localDirection);

// 逆矩阵（反变换：从世界空间到本地空间）
Matrix4x4 inverse = trs.inverse;
Vector3 localPoint = inverse.MultiplyPoint3x4(worldPoint);
```

### 2.2 矩阵变换的顺序问题

```csharp
// 矩阵乘法不满足交换律！顺序很重要

// 错误直觉：
Matrix4x4 wrong = scaleMatrix * rotateMatrix * translateMatrix;
// 这会先平移，再旋转，再缩放（从右到左执行）

// Unity 的变换顺序：缩放 → 旋转 → 平移（SRT）
Matrix4x4 correct = translateMatrix * rotateMatrix * scaleMatrix;
// 从右到左：先缩放，再旋转，再平移

// 实际场景：旋转绕世界轴 vs 绕本地轴
void RotateAroundWorldAxis()
{
    // 先移到新位置，再旋转 → 绕世界原点旋转
    transform.rotation = Quaternion.AngleAxis(angle, Vector3.up) * transform.rotation;
}

void RotateAroundLocalAxis()
{
    // 在本地空间中旋转 → 绕自身轴旋转
    transform.rotation = transform.rotation * Quaternion.AngleAxis(angle, Vector3.up);
}
```

### 2.3 坐标空间变换

```
游戏中的坐标空间：
  模型空间（Local Space）：以物体自身为中心
  世界空间（World Space）：游戏世界的绝对坐标
  观察空间（View Space）：以摄像机为中心
  裁剪空间（Clip Space）：GPU 处理的标准化空间
  屏幕空间（Screen Space）：最终显示的像素坐标

变换流水线：
  模型空间 → [Model Matrix] → 世界空间
  世界空间 → [View Matrix] → 观察空间
  观察空间 → [Projection Matrix] → 裁剪空间
  裁剪空间 → [NDC/Viewport变换] → 屏幕空间

在 Shader 中：
  float4 clipPos = mul(UNITY_MATRIX_MVP, positionOS); // 一步到位
  // MVP = Model × View × Projection
```

```csharp
// 常用的空间变换（Unity C#）
// 世界空间 ↔ 本地空间
Vector3 worldPos = transform.TransformPoint(localPos);      // 本地 → 世界
Vector3 localPos = transform.InverseTransformPoint(worldPos); // 世界 → 本地

// 方向（不受位移影响）
Vector3 worldDir = transform.TransformDirection(localDir);      // 本地方向 → 世界方向
Vector3 localDir = transform.InverseTransformDirection(worldDir); // 世界方向 → 本地方向

// 屏幕空间 ↔ 世界空间
Vector3 worldPos = Camera.main.ScreenToWorldPoint(screenPos);  // 屏幕 → 世界
Vector3 screenPos = Camera.main.WorldToScreenPoint(worldPos);  // 世界 → 屏幕
```

## 三、四元数（Quaternion）

### 3.1 为什么需要四元数

```
欧拉角（Euler Angles）的问题：
  - 万向锁（Gimbal Lock）：当某轴旋转 90° 时，另外两轴重合，失去一个自由度
  - 插值不自然：两个旋转之间线性插值欧拉角会走奇怪的路径
  - 顺序依赖：X-Y-Z 旋转和 Y-X-Z 旋转结果不同

四元数（Quaternion）：q = w + xi + yj + zk
  - 没有万向锁
  - 可以做球面线性插值（Slerp），结果自然
  - 旋转合成只需一次乘法
```

### 3.2 四元数实用操作

```csharp
// ✅ 创建旋转
Quaternion rotY90 = Quaternion.AngleAxis(90f, Vector3.up);    // 绕 Y 轴转 90°
Quaternion lookAt = Quaternion.LookRotation(direction);        // 朝向某方向
Quaternion fromTo = Quaternion.FromToRotation(from, to);       // 从一个方向转到另一个方向

// ✅ 插值（用于平滑旋转）
// Slerp：球面线性插值（等速度旋转）
Quaternion result = Quaternion.Slerp(startRot, endRot, t);

// Lerp：线性插值（不等速，但更快）
Quaternion result = Quaternion.Lerp(startRot, endRot, t);

// RotateTowards：限制每帧最大旋转角度
Quaternion result = Quaternion.RotateTowards(
    transform.rotation, 
    targetRotation, 
    maxDegPerFrame
);

// ✅ 合并旋转（先旋转 a，再旋转 b）
Quaternion combined = b * a; // 注意：从右到左

// ✅ 用四元数旋转向量
Vector3 rotatedVector = rotation * vector;

// ✅ 获取欧拉角（用于显示，不建议用于运算）
Vector3 euler = quaternion.eulerAngles;

// ❌ 常见错误：直接比较四元数
bool same = q1 == q2; // 可能因为浮点精度失败

// ✅ 正确做法
bool same = Quaternion.Angle(q1, q2) < 0.01f;
// 或：
bool same = Mathf.Abs(Quaternion.Dot(q1, q2)) > 0.9999f;
```

### 3.3 实用场景：平滑跟踪目标

```csharp
public class SmoothLookAt : MonoBehaviour
{
    [SerializeField] private Transform _target;
    [SerializeField] private float _rotateSpeed = 5f;
    
    void Update()
    {
        Vector3 direction = (_target.position - transform.position).normalized;
        
        if (direction.sqrMagnitude < 0.001f) return;
        
        Quaternion targetRotation = Quaternion.LookRotation(direction);
        
        // 平滑旋转（每帧最多旋转 rotateSpeed * deltaTime 度）
        transform.rotation = Quaternion.RotateTowards(
            transform.rotation, 
            targetRotation, 
            _rotateSpeed * Time.deltaTime
        );
    }
}

// 炮台旋转：只允许在 XZ 平面旋转（不仰头/低头）
public class TurretRotation : MonoBehaviour
{
    void AimAt(Vector3 worldTarget)
    {
        Vector3 direction = worldTarget - transform.position;
        
        // 消除 Y 分量（只在 XZ 平面旋转）
        direction.y = 0;
        
        if (direction.sqrMagnitude < 0.001f) return;
        
        Quaternion targetRot = Quaternion.LookRotation(direction);
        transform.rotation = Quaternion.Slerp(
            transform.rotation, targetRot, Time.deltaTime * 10f);
    }
}
```

## 四、插值：让游戏动作流畅

### 4.1 线性插值（Lerp）

```csharp
// Lerp(a, b, t)：在 a 和 b 之间插值
// t = 0 → a，t = 1 → b，t = 0.5 → 中间值

// 帧率无关的平滑移动
void SmoothFollow()
{
    // ❌ 错误：帧率相关
    transform.position = Vector3.Lerp(transform.position, target.position, 0.1f);
    
    // ✅ 正确：帧率无关
    // 原理：每帧乘以 (1 - factor * dt)，等效于指数衰减
    float factor = 1f - Mathf.Pow(0.01f, Time.deltaTime); // 0.01 = 阻尼系数
    transform.position = Vector3.Lerp(transform.position, target.position, factor);
    
    // 更简单的近似（高帧率下也能工作）
    transform.position = Vector3.Lerp(
        transform.position, 
        target.position, 
        1f - Mathf.Exp(-10f * Time.deltaTime) // 10 = 响应速度
    );
}
```

### 4.2 缓动函数（Easing Functions）

```csharp
public static class Easing
{
    // 线性（无缓动）
    public static float Linear(float t) => t;
    
    // 缓入：开始慢，结束快
    public static float EaseIn(float t) => t * t;
    public static float EaseInCubic(float t) => t * t * t;
    
    // 缓出：开始快，结束慢
    public static float EaseOut(float t) => t * (2f - t);
    public static float EaseOutCubic(float t) => 1f - Mathf.Pow(1f - t, 3f);
    
    // 缓入缓出：两端慢，中间快
    public static float EaseInOut(float t) => t < 0.5f 
        ? 2f * t * t 
        : -1f + (4f - 2f * t) * t;
    
    // 弹簧效果（超过目标后回弹）
    public static float EaseOutBack(float t)
    {
        const float c1 = 1.70158f;
        const float c3 = c1 + 1f;
        return 1f + c3 * Mathf.Pow(t - 1f, 3f) + c1 * Mathf.Pow(t - 1f, 2f);
    }
    
    // 弹跳效果
    public static float EaseOutBounce(float t)
    {
        const float n1 = 7.5625f;
        const float d1 = 2.75f;
        
        if (t < 1f / d1)       return n1 * t * t;
        else if (t < 2f / d1)  return n1 * (t -= 1.5f / d1) * t + 0.75f;
        else if (t < 2.5f / d1) return n1 * (t -= 2.25f / d1) * t + 0.9375f;
        else                    return n1 * (t -= 2.625f / d1) * t + 0.984375f;
    }
}

// 应用示例：技能图标冷却显示
public class SkillIconAnimator : MonoBehaviour
{
    private Image _icon;
    
    public IEnumerator AnimateCooldownEnd()
    {
        float elapsed = 0f;
        float duration = 0.3f;
        Vector3 originalScale = Vector3.one;
        Vector3 bounceScale = Vector3.one * 1.3f;
        
        while (elapsed < duration)
        {
            elapsed += Time.deltaTime;
            float t = elapsed / duration;
            
            // 使用弹簧效果：先放大再回弹到正常大小
            float easedT = Easing.EaseOutBack(t);
            _icon.transform.localScale = Vector3.Lerp(bounceScale, originalScale, easedT);
            
            yield return null;
        }
        
        _icon.transform.localScale = originalScale;
    }
}
```

## 五、实用数学算法

### 5.1 贝塞尔曲线（子弹/技能弧线轨迹）

```csharp
public static class BezierCurve
{
    // 二次贝塞尔曲线（一个控制点）
    public static Vector3 Quadratic(Vector3 p0, Vector3 p1, Vector3 p2, float t)
    {
        float oneMinusT = 1f - t;
        return oneMinusT * oneMinusT * p0 
             + 2f * oneMinusT * t * p1 
             + t * t * p2;
    }
    
    // 三次贝塞尔曲线（两个控制点）
    public static Vector3 Cubic(Vector3 p0, Vector3 p1, Vector3 p2, Vector3 p3, float t)
    {
        float oneMinusT = 1f - t;
        return Mathf.Pow(oneMinusT, 3) * p0
             + 3f * Mathf.Pow(oneMinusT, 2) * t * p1
             + 3f * oneMinusT * t * t * p2
             + t * t * t * p3;
    }
}

// 技能抛物线飞行
public class ProjectileMovement : MonoBehaviour
{
    private Vector3 _start, _end, _control;
    private float _elapsed, _duration;
    
    public void Launch(Vector3 start, Vector3 end, float height, float duration)
    {
        _start = start;
        _end = end;
        _duration = duration;
        
        // 控制点：在起始和结束的中点上方
        Vector3 mid = (start + end) * 0.5f;
        _control = mid + Vector3.up * height;
        
        _elapsed = 0f;
    }
    
    void Update()
    {
        _elapsed += Time.deltaTime;
        float t = Mathf.Clamp01(_elapsed / _duration);
        
        transform.position = BezierCurve.Quadratic(_start, _control, _end, t);
        
        // 朝向运动方向
        if (t < 0.99f)
        {
            Vector3 nextPos = BezierCurve.Quadratic(_start, _control, _end, t + 0.01f);
            transform.rotation = Quaternion.LookRotation(nextPos - transform.position);
        }
        
        if (t >= 1f) OnArrived();
    }
    
    void OnArrived() { /* 到达目标 */ }
}
```

### 5.2 射线与圆/球体相交检测

```csharp
// 射线与球体相交检测（不使用 Physics.Raycast）
// 适合：帧同步场景中需要精确控制的碰撞检测
public static bool RaySphereIntersect(
    Vector3 rayOrigin, Vector3 rayDir,    // 射线
    Vector3 sphereCenter, float radius,   // 球体
    out float hitDistance)
{
    hitDistance = 0;
    
    Vector3 oc = rayOrigin - sphereCenter;
    float a = Vector3.Dot(rayDir, rayDir);         // = 1（如果 rayDir 已归一化）
    float b = 2f * Vector3.Dot(oc, rayDir);
    float c = Vector3.Dot(oc, oc) - radius * radius;
    
    float discriminant = b * b - 4f * a * c;
    
    if (discriminant < 0) return false; // 不相交
    
    hitDistance = (-b - Mathf.Sqrt(discriminant)) / (2f * a);
    return hitDistance >= 0; // 负值表示射线在球体内
}

// 使用
if (RaySphereIntersect(
    transform.position, transform.forward,
    enemy.position, 1.5f,
    out float dist))
{
    Debug.Log($"命中！距离：{dist}");
}
```
