# 游戏世界分区与流式场景管理系统：大世界加载、分区切割与无缝切换完全指南

## 概述

开放世界游戏（Open World Game）对场景管理提出了极端挑战：单一场景可能覆盖数十平方公里，涵盖数百万个游戏对象。如果将整个世界一次性加载到内存，一款中型开放世界游戏的场景资源往往超过 10GB——这对移动端设备完全不可行。

流式场景管理（Streaming Scene Management）是解决这一问题的核心技术：将世界切割成若干分区（Chunk），根据玩家位置动态加载/卸载相邻区块，使内存中始终只保留玩家可见范围附近的数据。

---

## 1. 世界分区基础概念

### 1.1 核心术语

| 术语 | 含义 |
|------|------|
| Chunk（区块） | 世界的最小分区单元，通常为正方形格子 |
| Active Zone | 当前已加载且可交互的区块集合 |
| Buffer Zone | 已加载但尚未激活（预热缓冲）的区块 |
| Load Distance | 以玩家为中心触发加载的距离阈值 |
| Unload Distance | 超出该距离触发卸载的阈值（≥ Load Distance，防抖） |
| Seam（缝合线） | 相邻区块边界的物理/视觉连续性处理 |
| Pivot Offset | 区块内对象相对于区块原点的局部坐标 |

### 1.2 分区策略对比

```
格子分区（Grid-based）          六边形分区（Hex-based）
┌──┬──┬──┬──┐                  /\  /\  /\
│  │  │  │  │                 /  \/  \/  \
├──┼──┼──┼──┤                 \  /\  /\  /
│  │██│  │  │                  \/  \/  \/
├──┼──┼──┼──┤                 /\  /\  /\
│  │  │  │  │                /  \/  \/  \
└──┴──┴──┴──┘                \  /\  /\  /
  ██=玩家位置                  \/  \/  \/

优点：实现简单，坐标运算直观      优点：各方向等距，无对角线远近差异
缺点：对角方向距离偏大           缺点：坐标系复杂（需用Cube坐标）
```

## 2. 核心分区系统实现

### 2.1 区块坐标系统

```csharp
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.SceneManagement;

/// <summary>
/// 区块坐标：用整数网格坐标唯一标识一个区块
/// </summary>
public readonly struct ChunkCoord : System.IEquatable<ChunkCoord>
{
    public readonly int X;
    public readonly int Z;

    public ChunkCoord(int x, int z) { X = x; Z = z; }

    /// <summary>
    /// 从世界坐标计算所属区块坐标
    /// </summary>
    public static ChunkCoord FromWorldPos(Vector3 worldPos, float chunkSize)
    {
        return new ChunkCoord(
            Mathf.FloorToInt(worldPos.x / chunkSize),
            Mathf.FloorToInt(worldPos.z / chunkSize));
    }

    /// <summary>
    /// 区块中心的世界坐标
    /// </summary>
    public Vector3 ToWorldCenter(float chunkSize)
    {
        return new Vector3(
            (X + 0.5f) * chunkSize,
            0f,
            (Z + 0.5f) * chunkSize);
    }

    /// <summary>
    /// 获取以自身为中心、指定半径内的所有区块坐标（菱形范围）
    /// </summary>
    public IEnumerable<ChunkCoord> GetNeighbors(int radius)
    {
        for (int dz = -radius; dz <= radius; dz++)
        for (int dx = -radius; dx <= radius; dx++)
        {
            // 使用切比雪夫距离（正方形范围）
            if (Mathf.Abs(dx) <= radius && Mathf.Abs(dz) <= radius)
                yield return new ChunkCoord(X + dx, Z + dz);
        }
    }

    /// <summary>
    /// 切比雪夫距离（用于正方形加载区域判断）
    /// </summary>
    public int ChebyshevDistance(ChunkCoord other)
        => Mathf.Max(Mathf.Abs(X - other.X), Mathf.Abs(Z - other.Z));

    public bool Equals(ChunkCoord other) => X == other.X && Z == other.Z;
    public override bool Equals(object obj) => obj is ChunkCoord c && Equals(c);
    public override int GetHashCode() => HashCode.Combine(X, Z);
    public static bool operator ==(ChunkCoord a, ChunkCoord b) => a.Equals(b);
    public static bool operator !=(ChunkCoord a, ChunkCoord b) => !a.Equals(b);
    public override string ToString() => $"Chunk({X},{Z})";
}
```

### 2.2 区块状态机

```csharp
/// <summary>
/// 区块生命周期状态
/// </summary>
public enum ChunkState
{
    Unloaded,        // 未加载
    Queued,          // 已入加载队列
    Loading,         // 异步加载中
    Loaded,          // 场景已加载，对象已激活
    Unloading,       // 异步卸载中
}

/// <summary>
/// 区块数据描述
/// </summary>
public class ChunkData
{
    public ChunkCoord Coord;
    public ChunkState State = ChunkState.Unloaded;
    public string SceneName;               // 对应的 Unity Scene 名称
    public Scene LoadedScene;              // 加载后的 Scene 引用
    public AsyncOperation LoadOperation;   // 异步操作句柄
    public float LastAccessTime;           // 最后访问时间（用于 LRU 卸载）
    public int Priority;                   // 加载优先级（越近优先级越高）
    public Bounds WorldBounds;             // 区块在世界中的 AABB

    public bool IsFullyLoaded => State == ChunkState.Loaded;
    public bool IsTransitioning => State == ChunkState.Loading || State == ChunkState.Unloading;
}
```

### 2.3 世界流管理器

```csharp
/// <summary>
/// 世界流管理器：核心调度系统
/// 负责根据玩家位置，动态管理区块的加载/卸载
/// </summary>
public class WorldStreamingManager : MonoBehaviour
{
    [Header("区块配置")]
    [SerializeField] private float _chunkSize = 200f;       // 每个区块的边长（米）
    [SerializeField] private int _loadRadius = 3;           // 加载半径（区块数）
    [SerializeField] private int _unloadRadius = 5;         // 卸载半径（比加载大，防抖）
    [SerializeField] private int _maxConcurrentLoads = 3;   // 最大并发加载数

    [Header("内存预算")]
    [SerializeField] private long _memoryBudgetMB = 512;    // 加载区块总内存上限

    [Header("调试")]
    [SerializeField] private bool _showDebugGizmos = true;

    private Transform _player;
    private ChunkCoord _lastPlayerChunk;
    private readonly Dictionary<ChunkCoord, ChunkData> _chunks = new();
    private readonly Queue<ChunkCoord> _loadQueue = new();
    private readonly HashSet<ChunkCoord> _inQueue = new();
    private int _activeLoadCount;

    // 区块名称规则：Scene_ChunkX_ChunkZ（需在 Build Settings 中注册）
    private string GetSceneName(ChunkCoord coord) => $"World_Chunk_{coord.X}_{coord.Z}";

    private void Awake()
    {
        _player = GameObject.FindGameObjectWithTag("Player").transform;
    }

    private void Start()
    {
        _lastPlayerChunk = ChunkCoord.FromWorldPos(_player.position, _chunkSize);
        // 初始加载：立即同步加载玩家所在区块，异步加载周围区块
        ForceLoadChunkSync(_lastPlayerChunk);
        UpdateLoadedChunks();
    }

    private void Update()
    {
        ChunkCoord currentChunk = ChunkCoord.FromWorldPos(_player.position, _chunkSize);

        // 玩家跨越区块边界时触发重算
        if (currentChunk != _lastPlayerChunk)
        {
            _lastPlayerChunk = currentChunk;
            UpdateLoadedChunks();
        }

        // 每帧处理加载队列（限制并发数）
        ProcessLoadQueue();
    }

    /// <summary>
    /// 更新加载/卸载决策
    /// </summary>
    private void UpdateLoadedChunks()
    {
        var playerChunk = _lastPlayerChunk;

        // 1. 标记需要加载的区块
        var shouldBeLoaded = new HashSet<ChunkCoord>(
            playerChunk.GetNeighbors(_loadRadius));

        // 2. 计算优先级（距离越近优先级越高）
        foreach (var coord in shouldBeLoaded)
        {
            int dist = playerChunk.ChebyshevDistance(coord);
            EnsureChunkQueued(coord, _loadRadius - dist + 1);
        }

        // 3. 卸载超出范围的区块
        var toUnload = new List<ChunkCoord>();
        foreach (var (coord, chunk) in _chunks)
        {
            if (playerChunk.ChebyshevDistance(coord) > _unloadRadius
                && chunk.IsFullyLoaded)
            {
                toUnload.Add(coord);
            }
        }

        // 按 LRU 策略卸载（最久未访问的优先卸载）
        toUnload.Sort((a, b) =>
            _chunks[a].LastAccessTime.CompareTo(_chunks[b].LastAccessTime));

        foreach (var coord in toUnload)
        {
            StartCoroutine(UnloadChunkAsync(coord));
        }
    }

    private void EnsureChunkQueued(ChunkCoord coord, int priority)
    {
        if (_chunks.TryGetValue(coord, out var existing))
        {
            existing.LastAccessTime = Time.time;
            if (existing.IsFullyLoaded || existing.IsTransitioning) return;
        }

        if (_inQueue.Contains(coord)) return;

        // 创建区块数据
        if (!_chunks.ContainsKey(coord))
        {
            _chunks[coord] = new ChunkData
            {
                Coord = coord,
                State = ChunkState.Queued,
                SceneName = GetSceneName(coord),
                Priority = priority,
                LastAccessTime = Time.time,
                WorldBounds = new Bounds(
                    coord.ToWorldCenter(_chunkSize),
                    Vector3.one * _chunkSize)
            };
        }

        _loadQueue.Enqueue(coord);
        _inQueue.Add(coord);
    }

    private void ProcessLoadQueue()
    {
        while (_activeLoadCount < _maxConcurrentLoads && _loadQueue.Count > 0)
        {
            var coord = _loadQueue.Dequeue();
            _inQueue.Remove(coord);

            if (!_chunks.TryGetValue(coord, out var chunk)) continue;
            if (chunk.State != ChunkState.Queued) continue;

            // 内存预算检查
            if (EstimateMemoryUsageMB() >= _memoryBudgetMB)
            {
                Debug.LogWarning($"[WorldStream] 内存预算已满，暂停加载 {coord}");
                break;
            }

            StartCoroutine(LoadChunkAsync(chunk));
        }
    }

    private System.Collections.IEnumerator LoadChunkAsync(ChunkData chunk)
    {
        chunk.State = ChunkState.Loading;
        _activeLoadCount++;

        // 检查 Scene 是否在 Build Settings 中
        string sceneName = chunk.SceneName;
        if (!Application.CanStreamedLevelBeLoaded(sceneName))
        {
            Debug.LogWarning($"[WorldStream] 场景 {sceneName} 未注册，跳过");
            chunk.State = ChunkState.Unloaded;
            _chunks.Remove(chunk.Coord);
            _activeLoadCount--;
            yield break;
        }

        var op = SceneManager.LoadSceneAsync(sceneName, LoadSceneMode.Additive);
        op.allowSceneActivation = false;
        chunk.LoadOperation = op;

        // 等待加载完成（90%进度时暂停，等待手动激活）
        while (op.progress < 0.9f) yield return null;

        // 激活场景
        op.allowSceneActivation = true;
        yield return op;

        // 获取加载后的 Scene
        chunk.LoadedScene = SceneManager.GetSceneByName(sceneName);
        chunk.State = ChunkState.Loaded;
        chunk.LastAccessTime = Time.time;
        _activeLoadCount--;

        // 触发区块加载完成事件
        OnChunkLoaded(chunk);
        Debug.Log($"[WorldStream] 区块 {chunk.Coord} 加载完成");
    }

    private System.Collections.IEnumerator UnloadChunkAsync(ChunkCoord coord)
    {
        if (!_chunks.TryGetValue(coord, out var chunk)) yield break;
        if (!chunk.IsFullyLoaded) yield break;

        chunk.State = ChunkState.Unloading;

        // 卸载前触发事件（允许其他系统做清理）
        OnChunkUnloading(chunk);

        var op = SceneManager.UnloadSceneAsync(chunk.LoadedScene);
        if (op == null)
        {
            _chunks.Remove(coord);
            yield break;
        }

        yield return op;
        _chunks.Remove(coord);
        Debug.Log($"[WorldStream] 区块 {coord} 已卸载");
    }

    private void ForceLoadChunkSync(ChunkCoord coord)
    {
        string sceneName = GetSceneName(coord);
        if (!Application.CanStreamedLevelBeLoaded(sceneName)) return;

        SceneManager.LoadScene(sceneName, LoadSceneMode.Additive);
        var scene = SceneManager.GetSceneByName(sceneName);
        _chunks[coord] = new ChunkData
        {
            Coord = coord,
            State = ChunkState.Loaded,
            SceneName = sceneName,
            LoadedScene = scene,
            LastAccessTime = Time.time,
        };
    }

    // 事件回调（供外部系统监听）
    private void OnChunkLoaded(ChunkData chunk) { }
    private void OnChunkUnloading(ChunkData chunk) { }

    private long EstimateMemoryUsageMB()
    {
        // 简单估算：每个已加载区块约占用固定内存
        int loadedCount = 0;
        foreach (var c in _chunks.Values)
            if (c.IsFullyLoaded) loadedCount++;
        return loadedCount * 30L; // 每个区块约 30MB（依实际资源调整）
    }

#if UNITY_EDITOR
    private void OnDrawGizmos()
    {
        if (!_showDebugGizmos || _chunks == null) return;
        foreach (var (coord, chunk) in _chunks)
        {
            Gizmos.color = chunk.State switch
            {
                ChunkState.Loaded => new Color(0, 1, 0, 0.15f),
                ChunkState.Loading => new Color(1, 1, 0, 0.15f),
                ChunkState.Queued => new Color(0, 0.5f, 1f, 0.1f),
                _ => new Color(1, 0, 0, 0.1f)
            };
            Gizmos.DrawCube(coord.ToWorldCenter(_chunkSize), Vector3.one * _chunkSize * 0.95f);
            Gizmos.color = Color.white;
            Gizmos.DrawWireCube(coord.ToWorldCenter(_chunkSize), Vector3.one * _chunkSize);
        }
    }
#endif
}
```

## 3. 边界缝合：消除区块接缝

### 3.1 地形高度缝合

```csharp
/// <summary>
/// 区块边界地形缝合器
/// 确保相邻区块边界处高度连续，无裂缝
/// </summary>
public class ChunkSeamStitcher : MonoBehaviour
{
    [SerializeField] private Terrain _terrain;
    [SerializeField] private int _seamWidth = 3; // 边界过渡宽度（像素）

    /// <summary>
    /// 与右侧相邻区块缝合（East 方向）
    /// </summary>
    public void StitchEast(Terrain eastTerrain)
    {
        if (_terrain == null || eastTerrain == null) return;

        var data = _terrain.terrainData;
        var eastData = eastTerrain.terrainData;
        int res = data.heightmapResolution;

        float[,] heights = data.GetHeights(0, 0, res, res);
        float[,] eastHeights = eastData.GetHeights(0, 0, res, res);

        // 在右边界做线性插值过渡
        for (int z = 0; z < res; z++)
        {
            float myEdge = heights[z, res - 1];
            float eastEdge = eastHeights[z, 0];

            for (int x = 0; x < _seamWidth; x++)
            {
                float t = (float)x / _seamWidth;
                float blended = Mathf.Lerp(myEdge, eastEdge, t * 0.5f + 0.5f);
                heights[z, res - _seamWidth + x] = blended;
            }
        }

        data.SetHeights(0, 0, heights);
        Debug.Log($"[SeamStitch] 东向缝合完成");
    }

    /// <summary>
    /// 批量缝合所有已加载区块的边界
    /// </summary>
    public static void StitchAllLoadedChunks(
        Dictionary<ChunkCoord, Terrain> terrainMap,
        float chunkSize)
    {
        foreach (var (coord, terrain) in terrainMap)
        {
            var eastCoord = new ChunkCoord(coord.X + 1, coord.Z);
            var northCoord = new ChunkCoord(coord.X, coord.Z + 1);

            if (terrainMap.TryGetValue(eastCoord, out var eastTerrain))
            {
                terrain.SetNeighbors(null, null, eastTerrain, null);
                eastTerrain.SetNeighbors(terrain, null, null, null);
            }
            if (terrainMap.TryGetValue(northCoord, out var northTerrain))
            {
                terrain.SetNeighbors(null, northTerrain, null, null);
                northTerrain.SetNeighbors(null, null, null, terrain);
            }
        }
    }
}
```

## 4. 无缝传送系统

### 4.1 跨区块传送

```csharp
/// <summary>
/// 无缝传送管理器
/// 处理玩家从一个区块跨越到另一个区块时的无缝体验
/// </summary>
public class SeamlessTransitionManager : MonoBehaviour
{
    [SerializeField] private WorldStreamingManager _streamingManager;
    [SerializeField] private float _preloadDistance = 50f; // 预加载触发距离

    private Transform _player;
    private ChunkCoord _lastChunk;

    private void Update()
    {
        ChunkCoord current = ChunkCoord.FromWorldPos(_player.position, 200f);
        if (current != _lastChunk)
        {
            OnPlayerCrossedChunkBorder(_lastChunk, current);
            _lastChunk = current;
        }
    }

    private void OnPlayerCrossedChunkBorder(ChunkCoord from, ChunkCoord to)
    {
        Debug.Log($"[Transition] 玩家从区块 {from} 进入 {to}");

        // 通知各系统：区块切换
        var args = new ChunkTransitionArgs { From = from, To = to };
        EventBus.Emit("OnChunkTransition", args);

        // 物理世界重新中心化（防止浮点精度问题）
        if (NeedsOriginShift(to))
        {
            PerformOriginShift(to);
        }
    }

    /// <summary>
    /// 浮点原点偏移（Floating Origin）
    /// 当玩家远离世界原点时，将整个世界坐标系平移，使玩家始终在原点附近
    /// </summary>
    private void PerformOriginShift(ChunkCoord playerChunk)
    {
        Vector3 chunkCenter = playerChunk.ToWorldCenter(200f);
        Vector3 offset = -chunkCenter;

        // 移动所有已加载场景中的物体
        foreach (var scene in GetAllLoadedScenes())
        {
            foreach (var root in scene.GetRootGameObjects())
            {
                root.transform.position += offset;
            }
        }

        // 重置玩家位置
        _player.position += offset;

        // 更新物理世界（防止碰撞抖动）
        Physics.SyncTransforms();

        Debug.Log($"[OriginShift] 世界原点平移: {offset}");
    }

    private bool NeedsOriginShift(ChunkCoord coord)
    {
        // 当玩家距世界原点超过 1000m 时触发原点偏移
        float dist = coord.ToWorldCenter(200f).magnitude;
        return dist > 1000f;
    }

    private IEnumerable<Scene> GetAllLoadedScenes()
    {
        for (int i = 0; i < SceneManager.sceneCount; i++)
            yield return SceneManager.GetSceneAt(i);
    }
}

public struct ChunkTransitionArgs
{
    public ChunkCoord From;
    public ChunkCoord To;
}
```

## 5. 大世界 LOD 分级策略

### 5.1 基于距离的 LOD 调度

```csharp
/// <summary>
/// 大世界 LOD 管理器
/// 根据玩家距离动态切换区块内物体的 LOD 级别
/// </summary>
public class WorldLODManager : MonoBehaviour
{
    [System.Serializable]
    public struct LODThreshold
    {
        public float Distance;    // 触发距离（米）
        public int LODLevel;      // 目标 LOD 级别（0=最高质量）
        public bool DisableNavMesh; // 是否禁用 NavMesh 代理
        public bool DisablePhysics; // 是否禁用物理碰撞
    }

    [SerializeField] private LODThreshold[] _thresholds = new[]
    {
        new LODThreshold { Distance = 100f,  LODLevel = 0, DisableNavMesh = false, DisablePhysics = false },
        new LODThreshold { Distance = 300f,  LODLevel = 1, DisableNavMesh = true,  DisablePhysics = false },
        new LODThreshold { Distance = 600f,  LODLevel = 2, DisableNavMesh = true,  DisablePhysics = true  },
        new LODThreshold { Distance = 1000f, LODLevel = 3, DisableNavMesh = true,  DisablePhysics = true  },
    };

    private Transform _player;
    // 注册的 LOD 对象：区块 → 对象列表
    private readonly Dictionary<ChunkCoord, List<WorldLODObject>> _registeredObjects = new();

    public void RegisterChunkObjects(ChunkCoord coord, Scene scene)
    {
        var objects = new List<WorldLODObject>();
        foreach (var root in scene.GetRootGameObjects())
        {
            var lodObjs = root.GetComponentsInChildren<WorldLODObject>(true);
            objects.AddRange(lodObjs);
        }
        _registeredObjects[coord] = objects;
    }

    public void UnregisterChunk(ChunkCoord coord)
    {
        _registeredObjects.Remove(coord);
    }

    private void Update()
    {
        Vector3 playerPos = _player.position;
        foreach (var (coord, objects) in _registeredObjects)
        {
            foreach (var obj in objects)
            {
                if (obj == null) continue;
                float dist = Vector3.Distance(playerPos, obj.transform.position);
                ApplyLOD(obj, dist);
            }
        }
    }

    private void ApplyLOD(WorldLODObject obj, float distance)
    {
        // 找到匹配的 LOD 级别
        int targetLevel = _thresholds[^1].LODLevel; // 默认最低 LOD
        bool disableNav = _thresholds[^1].DisableNavMesh;
        bool disablePhys = _thresholds[^1].DisablePhysics;

        for (int i = 0; i < _thresholds.Length; i++)
        {
            if (distance <= _thresholds[i].Distance)
            {
                targetLevel = _thresholds[i].LODLevel;
                disableNav = _thresholds[i].DisableNavMesh;
                disablePhys = _thresholds[i].DisablePhysics;
                break;
            }
        }

        obj.SetLODLevel(targetLevel, disableNav, disablePhys);
    }
}

/// <summary>
/// 挂载在区块内需要 LOD 管理的游戏对象上
/// </summary>
public class WorldLODObject : MonoBehaviour
{
    private LODGroup _lodGroup;
    private UnityEngine.AI.NavMeshAgent _navAgent;
    private Collider[] _colliders;
    private int _currentLevel = -1;

    private void Awake()
    {
        _lodGroup = GetComponent<LODGroup>();
        _navAgent = GetComponent<UnityEngine.AI.NavMeshAgent>();
        _colliders = GetComponentsInChildren<Collider>();
    }

    public void SetLODLevel(int level, bool disableNav, bool disablePhys)
    {
        if (_currentLevel == level) return;
        _currentLevel = level;

        // 切换 LODGroup
        if (_lodGroup != null)
            _lodGroup.ForceLOD(level);

        // NavMesh 代理
        if (_navAgent != null)
            _navAgent.enabled = !disableNav;

        // 物理碰撞（非触发器）
        if (disablePhys)
            foreach (var col in _colliders)
                if (!col.isTrigger) col.enabled = false;
        else
            foreach (var col in _colliders)
                col.enabled = true;
    }
}
```

```csharp
        sb.AppendLine($"总对象数: {allObjects.Length}");
        sb.AppendLine($"分区数量: {chunkCount.Count}");

            sb.AppendLine($"  {coord}: {count} 个对象");

        Debug.Log(sb.ToString());
        EditorUtility.DisplayDialog("分析结果", sb.ToString(), "确定");
    }

    private void PartitionWorld()
    {
        if (!AssetDatabase.IsValidFolder(_outputPath))
            System.IO.Directory.CreateDirectory(_outputPath.Replace("Assets", Application.dataPath));

        var allObjects = FindObjectsOfType<GameObject>();
        var chunkObjects = new Dictionary<ChunkCoord, List<GameObject>>();

        // 按坐标分组
        foreach (var go in allObjects)
        {
            if (!ShouldInclude(go)) continue;
            var coord = ChunkCoord.FromWorldPos(go.transform.position, _chunkSize);
            if (!chunkObjects.ContainsKey(coord))

        foreach (var (coord, objects) in chunkObjects)

            EditorSceneManager.CloseScene(newScene, false);

                (float)processed / chunkObjects.Count);

        AssetDatabase.Refresh();
        Debug.Log($"[WorldPartition] 分区完成，共生成 {chunkObjects.Count} 个区块场景");
    }

        if (!_includeStaticObjects && go.isStatic) return false;
        if (!_includeDynamicObjects && !go.isStatic) return false;
        return true;
    }
```

## 7. 性能优化策略

### 7.1 区块预测性加载

```csharp
/// <summary>
/// 基于速度向量预测玩家下一步所在区块，提前触发加载
/// </summary>
public class PredictiveChunkLoader : MonoBehaviour
{
    [SerializeField] private float _predictionTime = 3f; // 预测未来几秒的位置

    private Rigidbody _playerRb;
    private WorldStreamingManager _streamManager;

    private void Update()
    {
        if (_playerRb.velocity.sqrMagnitude < 1f) return;

        // 预测未来位置
        Vector3 predictedPos = _playerRb.position + _playerRb.velocity * _predictionTime;
        ChunkCoord predictedChunk = ChunkCoord.FromWorldPos(predictedPos, 200f);
        ChunkCoord currentChunk = ChunkCoord.FromWorldPos(_playerRb.position, 200f);

        // 预测路径上的区块（当前→预测，取中间区块）
        var pathChunks = GetPathChunks(currentChunk, predictedChunk);
        foreach (var coord in pathChunks)
        {
            // 提前以低优先级触发加载
            _streamManager.RequestChunkPreload(coord, priority: 1);
        }
    }

    private IEnumerable<ChunkCoord> GetPathChunks(ChunkCoord from, ChunkCoord to)
    {
        // Bresenham 线段算法：获取两个区块之间经过的所有区块
        int x0 = from.X, z0 = from.Z;
        int x1 = to.X,   z1 = to.Z;
        int dx = Mathf.Abs(x1 - x0), dz = Mathf.Abs(z1 - z0);
        int sx = x0 < x1 ? 1 : -1, sz = z0 < z1 ? 1 : -1;
        int err = dx - dz;

        while (true)
        {
            yield return new ChunkCoord(x0, z0);
            if (x0 == x1 && z0 == z1) break;
            int e2 = 2 * err;
            if (e2 > -dz) { err -= dz; x0 += sx; }
            if (e2 < dx)  { err += dx; z0 += sz; }
        }
    }
}
```

## 8. 最佳实践总结

### 8.1 区块尺寸选择指南

| 游戏类型 | 推荐区块大小 | 加载半径 | 说明 |
|----------|-------------|---------|------|
| 移动端 RPG | 100m × 100m | 2 区块 | 内存受限，小区块+少半径 |
| PC/主机 RPG | 200m × 200m | 3 区块 | 标准配置 |
| MMO 大地图 | 500m × 500m | 2 区块 | 区块大，减少场景数量 |
| Battle Royale | 1000m × 1000m | 1 区块 | 地图固定，主要靠 LOD |

### 8.2 常见问题与解法

| 问题 | 根因 | 解法 |
|------|------|------|
| 区块边界处模型穿帮 | 跨区块的大型物体被切断 | 大型物体放入多区块引用，或单独维护跨区块对象层 |
| 传送点处画面闪烁 | 加载未完成就切换相机 | 新区块加载100%后再淡入相机 |
| 内存波动剧烈 | 加载/卸载频繁切换 | 增大 unloadRadius - loadRadius 差值（建议≥2）|
| 浮点精度导致抖动 | 玩家远离原点后 float 精度不足 | 实现 Floating Origin，每次跨越大阈值时重置坐标系 |
| 区块边界战斗中断 | NPC 跟随玩家跨越区块时被卸载 | 重要 NPC 使用"锚定"机制，暂时豁免卸载 |

## 总结

大世界流式场景管理是开放世界游戏的基础设施。核心设计要点：

1. **正交切割 + 整数坐标**：网格分区简单高效，坐标运算无浮点误差
2. **双阈值防抖**：加载距离 < 卸载距离，防止区块边界来回触发
3. **内存预算硬约束**：动态计算已用内存，超出预算时停止加载新区块
4. **预测性加载**：根据玩家速度向量提前加载未来区块，消除卡顿感
5. **浮点原点偏移**：大世界必备，防止远离原点后的精度问题
6. **边界缝合**：地形使用 `SetNeighbors` API，自动处理 Terrain 接缝
