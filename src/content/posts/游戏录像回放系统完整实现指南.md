# 游戏录像回放系统完整实现指南

## 前言

游戏回放系统（Replay System）在多类型游戏中扮演关键角色：MOBA/RTS类游戏的战斗复盘、格斗游戏的高光时刻分享、竞技游戏的反作弊取证、以及赛事转播的精彩回放。一个高质量的回放系统不仅需要**精确复现**游戏画面，还需要支持**任意跳转**、**多倍速播放**、以及**极致压缩**以节省存储空间。

本文将系统性介绍两种主流回放方案的设计原理，并提供完整的Unity工程实现。

---

## 一、两种核心方案对比

### 1.1 确定性重放（Deterministic Replay）

**原理**：录制所有玩家输入（而非游戏状态），回放时以相同输入驱动相同逻辑，依靠确定性仿真重现游戏画面。

```
录制：
  每帧 → 记录 [输入序列: {玩家ID, 操作, 时间戳}]
  
回放：
  读取输入序列 → 驱动确定性逻辑 → 渲染画面
```

**优点**：
- 录制体积极小（只存输入，通常几百KB）
- 完美精确，与实际游戏100%一致
- 天然支持反作弊（服务端可重放验证）

**缺点**：
- 必须保证逻辑完全确定性（禁止浮点数、随机数、时间戳影响逻辑）
- 跳转困难（必须从头开始仿真到目标帧）
- 版本耦合（旧录像在新版本可能无法播放）

### 1.2 快照回放（Snapshot Replay）

**原理**：周期性录制完整游戏状态快照，加上帧间增量，回放时直接恢复状态。

```
录制：
  每N帧 → 关键帧快照（完整状态）
  每帧  → 增量状态（仅变化部分）
  
回放：
  定位最近关键帧 → 恢复状态 → 应用增量 → 渲染
```

**优点**：
- 支持任意跳转（跳到最近关键帧再重放）
- 不依赖确定性（适合物理引擎等难以确定性的场景）

**缺点**：
- 体积较大（状态数据远大于输入数据）
- 实现复杂（需要完整的状态序列化/反序列化）

### 1.3 方案选型建议

| 游戏类型 | 推荐方案 |
|---------|---------|
| 帧同步RTS/MOBA | 确定性重放（天然确定性）|
| 物理模拟为主 | 快照回放 |
| 需要任意跳转 | 混合方案（确定性 + 定期快照）|
| 单机动作游戏 | 快照回放 |

## 二、确定性输入录制系统

### 2.1 输入记录格式设计

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.IO.Compression;
using UnityEngine;

/// <summary>
/// 输入帧记录 - 录像的基本单元
/// </summary>
[Serializable]
public struct InputFrame
{
    public int FrameIndex;          // 逻辑帧索引
    public uint InputMask;          // 按键位掩码（节省空间）
    public short AnalogX;           // 摇杆X轴（-32768~32767，定点数）
    public short AnalogY;           // 摇杆Y轴
    public short AimX;              // 瞄准X
    public short AimY;              // 瞄准Y
    public byte PlayerId;           // 玩家ID
    public byte Flags;              // 扩展标志位

    // 编码为8字节（高度压缩）
    public void Serialize(BinaryWriter writer)
    {
        writer.Write(FrameIndex);   // 4 bytes
        writer.Write(InputMask);    // 4 bytes
        writer.Write(AnalogX);      // 2 bytes
        writer.Write(AnalogY);      // 2 bytes
        writer.Write(AimX);         // 2 bytes
        writer.Write(AimY);         // 2 bytes
        writer.Write(PlayerId);     // 1 byte
        writer.Write(Flags);        // 1 byte
        // Total: 18 bytes/frame
    }

    public static InputFrame Deserialize(BinaryReader reader)
    {
        return new InputFrame
        {
            FrameIndex = reader.ReadInt32(),
            InputMask  = reader.ReadUInt32(),
            AnalogX    = reader.ReadInt16(),
            AnalogY    = reader.ReadInt16(),
            AimX       = reader.ReadInt16(),
            AimY       = reader.ReadInt16(),
            PlayerId   = reader.ReadByte(),
            Flags      = reader.ReadByte(),
        };
    }
}

/// <summary>
/// 录像文件头
/// </summary>
[Serializable]
public class ReplayHeader
{
    public const uint MagicNumber = 0x52504C59;  // "RPLY"
    public const ushort CurrentVersion = 1;

    public uint Magic;
    public ushort Version;
    public byte PlayerCount;
    public int TotalFrames;
    public long StartTimestamp;   // UTC Unix时间戳
    public string MapId;
    public string GameVersion;
    public string MatchId;
    public uint[] RandomSeeds;    // 每个玩家的随机种子

    public void Serialize(BinaryWriter writer)
    {
        writer.Write(Magic);
        writer.Write(Version);
        writer.Write(PlayerCount);
        writer.Write(TotalFrames);
        writer.Write(StartTimestamp);
        writer.Write(GameVersion ?? string.Empty);
        writer.Write(MapId ?? string.Empty);
        writer.Write(MatchId ?? string.Empty);
        writer.Write(RandomSeeds?.Length ?? 0);
        if (RandomSeeds != null)
            foreach (uint seed in RandomSeeds)
                writer.Write(seed);
    }

    public static ReplayHeader Deserialize(BinaryReader reader)
    {
        var header = new ReplayHeader
        {
            Magic          = reader.ReadUInt32(),
            Version        = reader.ReadUInt16(),
            PlayerCount    = reader.ReadByte(),
            TotalFrames    = reader.ReadInt32(),
            StartTimestamp = reader.ReadInt64(),
            GameVersion    = reader.ReadString(),
            MapId          = reader.ReadString(),
            MatchId        = reader.ReadString(),
        };
        int seedCount = reader.ReadInt32();
        header.RandomSeeds = new uint[seedCount];
        for (int i = 0; i < seedCount; i++)
            header.RandomSeeds[i] = reader.ReadUInt32();
        return header;
    }
}
```

### 2.2 录制器实现

```csharp
using System.Collections.Generic;
using System.IO;
using System.IO.Compression;
using UnityEngine;

/// <summary>
/// 录像录制器 - 在游戏运行时记录输入
/// </summary>
public class ReplayRecorder
{
    private readonly List<InputFrame>[] _inputFramesByPlayer;
    private readonly ReplayHeader _header;
    private int _currentFrame = 0;
    private bool _isRecording = false;

    // 环形缓冲区（用于实现"最近N秒"录像）
    private readonly int _bufferCapacity;
    private bool _useRingBuffer;

    public int CurrentFrame => _currentFrame;
    public bool IsRecording => _isRecording;

    public ReplayRecorder(
        int playerCount, 
        string mapId, 
        string matchId,
        uint[] randomSeeds,
        bool useRingBuffer = false,
        int ringBufferSeconds = 30,
        int fps = 20)
    {
        _inputFramesByPlayer = new List<InputFrame>[playerCount];
        for (int i = 0; i < playerCount; i++)
            _inputFramesByPlayer[i] = new List<InputFrame>(10000);

        _useRingBuffer = useRingBuffer;
        _bufferCapacity = useRingBuffer ? ringBufferSeconds * fps : int.MaxValue;

        _header = new ReplayHeader
        {
            Magic          = ReplayHeader.MagicNumber,
            Version        = ReplayHeader.CurrentVersion,
            PlayerCount    = (byte)playerCount,
            StartTimestamp = System.DateTimeOffset.UtcNow.ToUnixTimeSeconds(),
            MapId          = mapId,
            MatchId        = matchId,
            GameVersion    = Application.version,
            RandomSeeds    = randomSeeds,
        };
    }

    public void StartRecording()
    {
        _isRecording = true;
        _currentFrame = 0;
        foreach (var list in _inputFramesByPlayer)
            list.Clear();
        Debug.Log("[Replay] 开始录制");
    }

    /// <summary>
    /// 每逻辑帧调用，记录所有玩家输入
    /// </summary>
    public void RecordFrame(InputFrame[] allPlayersInput)
    {
        if (!_isRecording) return;

        for (int i = 0; i < allPlayersInput.Length && i < _inputFramesByPlayer.Length; i++)
        {
            var frame = allPlayersInput[i];
            frame.FrameIndex = _currentFrame;
            frame.PlayerId = (byte)i;

            var list = _inputFramesByPlayer[i];
            list.Add(frame);

            // 环形缓冲：超出容量则移除最旧的
            if (_useRingBuffer && list.Count > _bufferCapacity)
                list.RemoveAt(0);
        }

        _currentFrame++;
    }

    public void StopRecording()
    {
        _isRecording = false;
        _header.TotalFrames = _currentFrame;
        Debug.Log($"[Replay] 录制结束，共 {_currentFrame} 帧");
    }

    /// <summary>
    /// 将录像序列化并压缩保存
    /// </summary>
    public byte[] Serialize()
    {
        using var ms = new MemoryStream();
        using var gzip = new GZipStream(ms, CompressionLevel.Optimal);
        using var writer = new BinaryWriter(gzip);

        _header.Serialize(writer);

        // 写入所有帧数据（按帧交错存储，便于随机访问）
        for (int frame = 0; frame < _currentFrame; frame++)
        {
            for (int player = 0; player < _inputFramesByPlayer.Length; player++)
            {
                if (frame < _inputFramesByPlayer[player].Count)
                    _inputFramesByPlayer[player][frame].Serialize(writer);
            }
        }

        writer.Flush();
        gzip.Flush();
        return ms.ToArray();
    }

    /// <summary>
    /// 保存到文件
    /// </summary>
    public void SaveToFile(string filePath)
    {
        byte[] data = Serialize();
        File.WriteAllBytes(filePath, data);
        
        float sizeMB = data.Length / 1024f / 1024f;
        Debug.Log($"[Replay] 保存到 {filePath}，大小: {sizeMB:F2}MB，{_currentFrame}帧");
    }
}
```

## 三、回放播放器实现

### 3.1 播放控制器

```csharp
using System.Collections.Generic;
using System.IO;
using System.IO.Compression;
using UnityEngine;

/// <summary>
/// 录像播放器 - 控制回放的播放、暂停、跳转、倍速
/// </summary>
public class ReplayPlayer : MonoBehaviour
{
    public enum PlaybackState { Idle, Playing, Paused, Finished }

    [Header("回放配置")]
    [SerializeField] private float _defaultSpeed = 1.0f;
    [SerializeField] private float[] _speedOptions = { 0.25f, 0.5f, 1.0f, 2.0f, 4.0f, 8.0f };

    // 回放状态
    private PlaybackState _state = PlaybackState.Idle;
    private ReplayData _replayData;
    private int _currentFrame = 0;
    private float _playbackSpeed = 1.0f;
    private float _frameAccumulator = 0f;

    // 逻辑帧率（与录制时一致）
    private const int LogicFPS = 20;
    private const float LogicFrameTime = 1f / LogicFPS;

    // 回调
    public event System.Action<int, int> OnFrameChanged;      // (currentFrame, totalFrames)
    public event System.Action<PlaybackState> OnStateChanged;
    public event System.Action OnPlaybackComplete;

    // 暴露给UI的属性
    public float Progress => _replayData != null && _replayData.TotalFrames > 0 
        ? (float)_currentFrame / _replayData.TotalFrames : 0f;
    public float PlaybackSpeed => _playbackSpeed;
    public PlaybackState State => _state;
    public int TotalFrames => _replayData?.TotalFrames ?? 0;
    public int CurrentFrame => _currentFrame;

    /// <summary>
    /// 加载录像数据
    /// </summary>
    public void LoadReplay(byte[] compressedData)
    {
        _replayData = ReplayData.Deserialize(compressedData);
        _currentFrame = 0;
        SetState(PlaybackState.Idle);
        
        Debug.Log($"[ReplayPlayer] 加载录像: {_replayData.TotalFrames}帧 "
            + $"({_replayData.Header.GameVersion})");
    }

    public void LoadReplayFromFile(string filePath)
    {
        byte[] data = File.ReadAllBytes(filePath);
        LoadReplay(data);
    }

    #region 播放控制

    public void Play()
    {
        if (_replayData == null) return;
        if (_state == PlaybackState.Finished) Seek(0);
        SetState(PlaybackState.Playing);
    }

    public void Pause()
    {
        if (_state == PlaybackState.Playing)
            SetState(PlaybackState.Paused);
    }

    public void TogglePlayPause()
    {
        if (_state == PlaybackState.Playing) Pause();
        else Play();
    }

    public void StepForward(int frames = 1)
    {
        if (_state == PlaybackState.Paused)
        {
            SeekRelative(frames);
        }
    }

    public void StepBackward(int frames = 1)
    {
        if (_state == PlaybackState.Paused)
        {
            // 确定性重放：需要从快照重新仿真
            int targetFrame = Mathf.Max(0, _currentFrame - frames);
            SeekBySimulation(targetFrame);
        }
    }

    /// <summary>
    /// 直接跳转到指定帧（需要快照支持）
    /// </summary>
    public void Seek(int targetFrame)
    {
        if (_replayData == null) return;
        targetFrame = Mathf.Clamp(targetFrame, 0, _replayData.TotalFrames);

        if (targetFrame >= _currentFrame)
        {
            // 向前跳转：从当前位置继续仿真
            SeekForward(targetFrame);
        }
        else
        {
            // 向后跳转：找最近快照，重新仿真
            SeekBySimulation(targetFrame);
        }
    }

    public void SeekByProgress(float progress)
    {
        int targetFrame = Mathf.RoundToInt(progress * TotalFrames);
        Seek(targetFrame);
    }

    public void SetPlaybackSpeed(float speed)
    {
        _playbackSpeed = Mathf.Clamp(speed, 0.1f, 16f);
        Debug.Log($"[ReplayPlayer] 播放速度: {_playbackSpeed}x");
    }

    public void CycleSpeed()
    {
        int currentIndex = System.Array.FindIndex(_speedOptions, s => Mathf.Approximately(s, _playbackSpeed));
        int nextIndex = (currentIndex + 1) % _speedOptions.Length;
        SetPlaybackSpeed(_speedOptions[nextIndex]);
    }

    #endregion

    private void Update()
    {
        if (_state != PlaybackState.Playing) return;

        _frameAccumulator += Time.unscaledDeltaTime * _playbackSpeed;

        while (_frameAccumulator >= LogicFrameTime)
        {
            _frameAccumulator -= LogicFrameTime;
            AdvanceOneFrame();

            if (_currentFrame >= _replayData.TotalFrames)
            {
                SetState(PlaybackState.Finished);
                OnPlaybackComplete?.Invoke();
                return;
            }
        }
    }

    private void AdvanceOneFrame()
    {
        // 获取当前帧的所有玩家输入
        InputFrame[] inputs = _replayData.GetFrameInputs(_currentFrame);
        
        // 将输入注入游戏逻辑（替代真实输入）
        ReplayInputInjector.InjectFrame(inputs);
        
        // 推进逻辑帧
        GameLogicEngine.Instance.UpdateLogicFrame(_currentFrame);
        
        _currentFrame++;
        OnFrameChanged?.Invoke(_currentFrame, _replayData.TotalFrames);
    }

    private void SeekForward(int targetFrame)
    {
        while (_currentFrame < targetFrame)
            AdvanceOneFrame();
    }

    private void SeekBySimulation(int targetFrame)
    {
        // 找最近的快照点
        int snapshotFrame = _replayData.GetNearestSnapshotFrame(targetFrame);
        
        // 恢复快照
        GameStateSnapshot snapshot = _replayData.GetSnapshot(snapshotFrame);
        GameLogicEngine.Instance.RestoreSnapshot(snapshot);
        _currentFrame = snapshotFrame;

        // 从快照点仿真到目标帧
        SeekForward(targetFrame);
    }

    private void SeekRelative(int deltaFrames)
    {
        int targetFrame = Mathf.Clamp(_currentFrame + deltaFrames, 0, _replayData.TotalFrames);
        Seek(targetFrame);
    }

    private void SetState(PlaybackState newState)
    {
        _state = newState;
        OnStateChanged?.Invoke(newState);
    }
}
```

### 3.2 录像数据容器

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.IO.Compression;

/// <summary>
/// 录像数据容器 - 管理帧数据和快照索引
/// </summary>
public class ReplayData
{
    public ReplayHeader Header { get; private set; }
    public int TotalFrames => Header.TotalFrames;

    // 输入帧数据：[playerIndex][frameIndex] = InputFrame
    private InputFrame[][] _inputFrames;
    
    // 快照索引：帧索引 -> 快照数据
    private readonly SortedDictionary<int, GameStateSnapshot> _snapshots 
        = new SortedDictionary<int, GameStateSnapshot>();

    private const int SnapshotInterval = 300;  // 每300帧（15秒@20fps）一个快照

    public static ReplayData Deserialize(byte[] compressedData)
    {
        var replayData = new ReplayData();
        
        using var ms = new MemoryStream(compressedData);
        using var gzip = new GZipStream(ms, CompressionMode.Decompress);
        using var reader = new BinaryReader(gzip);

        replayData.Header = ReplayHeader.Deserialize(reader);
        int playerCount = replayData.Header.PlayerCount;
        int totalFrames = replayData.Header.TotalFrames;

        replayData._inputFrames = new InputFrame[playerCount][];
        for (int p = 0; p < playerCount; p++)
            replayData._inputFrames[p] = new InputFrame[totalFrames];

        for (int frame = 0; frame < totalFrames; frame++)
        {
            for (int player = 0; player < playerCount; player++)
            {
                replayData._inputFrames[player][frame] = InputFrame.Deserialize(reader);
            }
        }

        return replayData;
    }

    /// <summary>
    /// 获取指定帧所有玩家的输入
    /// </summary>
    public InputFrame[] GetFrameInputs(int frameIndex)
    {
        var inputs = new InputFrame[_inputFrames.Length];
        for (int p = 0; p < _inputFrames.Length; p++)
        {
            if (frameIndex < _inputFrames[p].Length)
                inputs[p] = _inputFrames[p][frameIndex];
        }
        return inputs;
    }

    /// <summary>
    /// 找不超过targetFrame的最近快照帧
    /// </summary>
    public int GetNearestSnapshotFrame(int targetFrame)
    {
        int best = 0;
        foreach (int snapshotFrame in _snapshots.Keys)
        {
            if (snapshotFrame <= targetFrame)
                best = snapshotFrame;
            else
                break;
        }
        return best;
    }

    public GameStateSnapshot GetSnapshot(int frameIndex)
    {
        return _snapshots.TryGetValue(frameIndex, out var snapshot) ? snapshot : null;
    }

    public void AddSnapshot(int frameIndex, GameStateSnapshot snapshot)
    {
        _snapshots[frameIndex] = snapshot;
    }

    /// <summary>
    /// 计算文件大小估算
    /// </summary>
    public int EstimatedRawSize()
    {
        // 每帧每玩家18字节
        return TotalFrames * Header.PlayerCount * 18;
    }
}
```

```csharp
using System;
using System.Collections.Generic;

public class GameStateSnapshot

    {
        public int EntityId;
        public string EntityType;
        public FixedVector3 Position;
        public long HP;
        public bool IsActive;
    }

    // 定点数向量（确保序列化精确）
    [Serializable]

    {
        public long X, Y, Z;  // 定点数，精度1/1000

        public static FixedVector3 FromUnityVector(UnityEngine.Vector3 v)
        {
            return new FixedVector3
            {
                X = (long)(v.x * 1000),
                Y = (long)(v.y * 1000),
                Z = (long)(v.z * 1000)
            };
        }

        public UnityEngine.Vector3 ToUnityVector()
        {
            return new UnityEngine.Vector3(X / 1000f, Y / 1000f, Z / 1000f);
        }
    }

    public struct FixedQuaternion
    {
        public long X, Y, Z, W;
    }

    /// <summary>
    /// 估算快照大小（字节）
    /// </summary>

    {
        int size = 16;  // 帧索引 + 时间戳
        size += (Players?.Length ?? 0) * 64;     // 每个玩家约64字节
        size += (Entities?.Length ?? 0) * 32;    // 每个实体约32字节
        return size;
    }
```

### 4.2 快照管理器

```csharp
using System.Collections.Generic;
using UnityEngine;

/// <summary>
/// 快照管理器 - 在录制过程中周期性采集快照
/// </summary>
public class SnapshotManager
{
    private const int SnapshotInterval = 300;  // 每300帧采集一次快照
    private readonly ReplayData _replayData;
    private readonly IGameStateProvider _stateProvider;

    public SnapshotManager(ReplayData replayData, IGameStateProvider stateProvider)
    {
        _replayData = replayData;
        _stateProvider = stateProvider;
    }

    /// <summary>
    /// 每帧调用，检查是否需要采集快照
    /// </summary>
    public void OnFrameEnd(int frameIndex)
    {
        if (frameIndex % SnapshotInterval == 0)
        {
            TakeSnapshot(frameIndex);
        }
    }

    private void TakeSnapshot(int frameIndex)
    {
        var snapshot = _stateProvider.CaptureState(frameIndex);
        _replayData.AddSnapshot(frameIndex, snapshot);
        
        Debug.Log($"[Snapshot] 帧{frameIndex}快照已采集，大小约{snapshot.EstimatedSize()}字节");
    }
}

/// <summary>
/// 游戏状态提供者接口 - 由游戏逻辑层实现
/// </summary>
public interface IGameStateProvider
{
    GameStateSnapshot CaptureState(int frameIndex);
    void RestoreState(GameStateSnapshot snapshot);
}
```

```csharp
using UnityEngine;
using UnityEngine.UI;

/// <summary>
/// 回放控制栏UI
/// </summary>
public class ReplayControlBar : MonoBehaviour
{
    [Header("控件绑定")]
    [SerializeField] private Button _playPauseBtn;
    [SerializeField] private Button _stepForwardBtn;
    [SerializeField] private Button _stepBackwardBtn;
    [SerializeField] private Button _speedBtn;
    [SerializeField] private Slider _progressSlider;
    [SerializeField] private Text _timeText;
    [SerializeField] private Text _speedText;
    [SerializeField] private Image _playIcon;
    [SerializeField] private Image _pauseIcon;

    [SerializeField] private ReplayPlayer _replayPlayer;

    private bool _isDraggingSlider = false;
    private const int LogicFPS = 20;

    private void Awake()
    {
        _playPauseBtn.onClick.AddListener(OnPlayPauseClicked);
        _stepForwardBtn.onClick.AddListener(() => _replayPlayer.StepForward(1));
        _stepBackwardBtn.onClick.AddListener(() => _replayPlayer.StepBackward(1));
        _speedBtn.onClick.AddListener(OnSpeedClicked);

        _progressSlider.onValueChanged.AddListener(OnSliderValueChanged);

        _replayPlayer.OnFrameChanged += OnFrameChanged;
        _replayPlayer.OnStateChanged += OnStateChanged;
    }

    private void OnPlayPauseClicked()
    {
        _replayPlayer.TogglePlayPause();
    }

    private void OnSpeedClicked()
    {
        _replayPlayer.CycleSpeed();
        UpdateSpeedDisplay();
    }

    private void OnSliderValueChanged(float value)
    {
        if (_isDraggingSlider)
        {
            _replayPlayer.SeekByProgress(value);
        }
    }

    // 开始拖动进度条
    public void OnSliderBeginDrag()
    {
        _isDraggingSlider = true;
        _replayPlayer.Pause();
    }

    // 结束拖动进度条
    public void OnSliderEndDrag()
    {
        _isDraggingSlider = false;
        _replayPlayer.Play();
    }

    private void OnFrameChanged(int currentFrame, int totalFrames)
    {
        if (!_isDraggingSlider)
            _progressSlider.SetValueWithoutNotify((float)currentFrame / totalFrames);

        UpdateTimeDisplay(currentFrame, totalFrames);
    }

    private void OnStateChanged(ReplayPlayer.PlaybackState state)
    {
        bool isPlaying = state == ReplayPlayer.PlaybackState.Playing;
        _playIcon.gameObject.SetActive(!isPlaying);
        _pauseIcon.gameObject.SetActive(isPlaying);
    }

    private void UpdateTimeDisplay(int currentFrame, int totalFrames)
    {
        float currentSec = currentFrame / (float)LogicFPS;
        float totalSec   = totalFrames  / (float)LogicFPS;

        _timeText.text = $"{FormatTime(currentSec)} / {FormatTime(totalSec)}";
    }

    private void UpdateSpeedDisplay()
    {
        _speedText.text = $"{_replayPlayer.PlaybackSpeed:0.#}x";
    }

    private string FormatTime(float seconds)
    {
        int min = (int)(seconds / 60);
        int sec = (int)(seconds % 60);
        return $"{min:00}:{sec:00}";
    }

    private void OnDestroy()
    {
        if (_replayPlayer != null)
        {
            _replayPlayer.OnFrameChanged  -= OnFrameChanged;
            _replayPlayer.OnStateChanged  -= OnStateChanged;
        }
    }
}
```

### 6.1 录像文件管理

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using UnityEngine;

/// <summary>
/// 录像文件管理器 - 本地存储与云端同步
/// </summary>
public class ReplayFileManager
{
    private const string ReplayDir = "Replays";
    private const string FileExtension = ".rply";
    private const int MaxLocalReplays = 20;     // 最多保留20个本地录像

    private string RootPath => Path.Combine(Application.persistentDataPath, ReplayDir);

    public ReplayFileManager()
    {
        Directory.CreateDirectory(RootPath);
    }

    /// <summary>
    /// 保存录像
    /// </summary>
    public string SaveReplay(ReplayRecorder recorder, string displayName = null)
    {
        string timestamp = DateTime.Now.ToString("yyyyMMdd_HHmmss");
        string filename = $"replay_{timestamp}{FileExtension}";
        string filePath = Path.Combine(RootPath, filename);

        recorder.SaveToFile(filePath);
        CleanupOldReplays();

        Debug.Log($"[ReplayFile] 保存录像: {filePath}");
        return filePath;
    }

    /// <summary>
    /// 获取本地录像列表（按时间倒序）
    /// </summary>
    public List<ReplayFileInfo> GetLocalReplays()
    {
        var files = Directory.GetFiles(RootPath, $"*{FileExtension}");
        var result = new List<ReplayFileInfo>();

        foreach (string file in files)
        {
            var info = new System.IO.FileInfo(file);
            result.Add(new ReplayFileInfo
            {
                FilePath = file,
                FileName = info.Name,
                FileSizeKB = info.Length / 1024f,
                CreateTime = info.CreationTime,
            });
        }

        result.Sort((a, b) => b.CreateTime.CompareTo(a.CreateTime));
        return result;
    }

    /// <summary>
    /// 删除旧录像，保持数量上限
    /// </summary>
    private void CleanupOldReplays()
    {
        var replays = GetLocalReplays();
        while (replays.Count > MaxLocalReplays)
        {
            File.Delete(replays[replays.Count - 1].FilePath);
            replays.RemoveAt(replays.Count - 1);
        }
    }

    public class ReplayFileInfo
    {
        public string FilePath;
        public string FileName;
        public float FileSizeKB;
        public DateTime CreateTime;
    }
}
```

## 七、与帧同步系统整合

```csharp
/// <summary>
/// 帧同步系统中集成录像功能
/// </summary>
public class LockstepEngine : MonoBehaviour
{
    private ReplayRecorder _recorder;
    private ReplayPlayer _player;
    private SnapshotManager _snapshotManager;
    
    public bool IsInReplayMode { get; private set; }

    private void StartGame(MatchConfig config)
    {
        IsInReplayMode = false;
        
        // 创建录制器
        _recorder = new ReplayRecorder(
            playerCount: config.PlayerCount,
            mapId: config.MapId,
            matchId: config.MatchId,
            randomSeeds: config.RandomSeeds
        );
        _recorder.StartRecording();
        
        // 创建快照管理器
        var replayData = new ReplayData();
        _snapshotManager = new SnapshotManager(replayData, this as IGameStateProvider);
    }

    private void OnLogicFrameEnd(int frameIndex, InputFrame[] inputs)
    {
        if (!IsInReplayMode)
        {
            // 录制模式：记录输入
            _recorder.RecordFrame(inputs);
            _snapshotManager.OnFrameEnd(frameIndex);
        }
    }

    public void StartReplay(string replayFilePath)
    {
        IsInReplayMode = true;
        
        // 停止网络逻辑
        NetworkManager.Instance.Disconnect();
        
        // 初始化回放
        _player = gameObject.AddComponent<ReplayPlayer>();
        _player.LoadReplayFromFile(replayFilePath);
        _player.Play();
    }

    public void EndGame()
    {
        if (!IsInReplayMode && _recorder != null)
        {
            _recorder.StopRecording();
            var fileManager = new ReplayFileManager();
            fileManager.SaveReplay(_recorder);
        }
    }
}
```

## 八、最佳实践总结

### 8.1 方案选择决策树

```
是否采用帧同步架构？
├─ 是 → 确定性重放方案（体积小，精确）
│   └─ 是否需要任意跳转？
│       ├─ 是 → 混合方案（确定性 + 关键帧快照）
│       └─ 否 → 纯输入录制
└─ 否 → 快照回放方案
    └─ 优化快照频率（性能 vs 跳转粒度）
```

### 8.2 性能要点

1. **录制开销极低**：输入录制每帧<1μs，对游戏逻辑影响可忽略
2. **快照异步化**：快照序列化在子线程进行，避免主线程阻塞
3. **增量快照**：只记录变化的实体，减少70%以上快照体积
4. **回放时关闭AI**：回放模式下禁用AI计算，节省CPU
5. **LOD降级**：回放时可降低粒子/阴影等视觉效果，提升多倍速流畅性

### 8.3 常见陷阱

| 陷阱 | 解决方案 |
|------|---------|
| 浮点运算不确定 | 使用定点数替代所有逻辑层浮点运算 |
| System.Random不确定 | 使用种子固定的自定义随机数生成器 |
| Time.deltaTime影响逻辑 | 逻辑层使用固定帧时间，不依赖realtime |
| 版本兼容问题 | 录像头写入GameVersion，加载时检查兼容性 |
| 快照过大 | 只记录变化的实体（增量快照）|
| 任意跳转慢 | 增加快照密度（内存换时间）|

### 8.4 工程建议

1. **单元测试**：编写录制-回放对比测试，验证确定性
2. **哈希校验**：每个快照记录状态哈希，回放时校验一致性
3. **版本管理**：录像格式版本化，保证向后兼容
4. **加密保护**：竞技游戏录像加密，防止通过分析录像作弊
5. **云端存储**：精彩录像上传云端，支持社区分享

## 结语

录像回放系统是游戏技术中颇具挑战性的功能，它要求开发者对游戏架构有深刻理解——尤其是确定性约束。**帧同步游戏天然具备确定性重放能力**，这也是为什么MOBA、RTS类游戏的录像文件往往只有几百KB。

对于非帧同步游戏，通过合理设计快照粒度和增量压缩，同样可以实现高质量的回放体验。建议在架构设计阶段就将回放系统纳入考量，而非事后补充，因为确定性改造的成本往往是颠覆性的。
