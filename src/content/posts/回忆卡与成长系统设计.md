# 回忆卡与成长系统设计

## 1. 系统概述

回忆卡（MemoryCard）是本项目独特的养成与社交机制——玩家通过与 NPC 角色互动积累好感度，解锁特殊剧情（Script），并获得实体"回忆卡"作为收藏品。回忆卡可以分享、展示，也可以作为战斗中的助战道具。

**系统核心功能：**
- **剧情解锁**：完成特定剧情 + 好感度条件后解锁对应回忆卡
- **本地保存**：回忆卡图片（PNG）存储在本地，支持离线查看
- **防篡改校验**：本地图片通过 HMAC-SHA256 签名，防止替换
- **COS 上传**：支持将回忆卡上传到腾讯云 COS，用于社交分享

### 3.1 获取回忆卡列表

```csharp
// 位置：Hotfix/Function/GamePlay/Player/MemoryCard/MemoryCardComponentSystem.cs
[FriendOf(typeof(MemoryCardComponent))]
public static class MemoryCardComponentSystem
{
    private const string MemoryCardFolder = "MemoryCard";
    // 文件命名格式：MC_{角色ID}_{剧情ID}_{卡片ID}_{时间戳}_{分辨率}.png
    private const string MemoryCardNameFormat = "MC_{0}_{1}_{2}_{3}_{4}.png";
    // 防篡改锁标识（MCDL = Memory Card Default Lock）
    private const long MemoryCardDefaultLockKey = 0x4D43444C;
    
    // 请求服务器下发所有回忆卡简要信息
    public static async ETTask<List<MemoryCardBriefData>> RequestMemoryCardList(
        this MemoryCardComponent self)
    {
        var req = new ZoneCultivatesMemoryCardBriefInfoReq();
        var nr = await self.ClientScene()
            .GetComponent<NetworkComponent>()
            .SendAsync<ZoneCultivatesMemoryCardBriefInfoResp>(
                (uint)ZoneClientCmd.ZoneCsCultivateMemoryCardBriefInfo, req);
        
        if (nr.IsCompleteSuccess)
        {
            return ServerDataConverter.ToList(
                nr.Data?.MemoryCardIndexList, 
                ServerDataConverter.GetMemoryCardBriefData);
        }
        return null;
    }
```

### 3.3 本地保存与防篡改

```csharp
    // 将回忆卡图片保存到本地 persistent 目录
    public static void SaveMemoryCardToLocal(
        this MemoryCardComponent self, 
        MemoryCardDetailData data, 
        Texture2D texture)
    {
        string folder = Path.Combine(Application.persistentDataPath, MemoryCardFolder);
        if (!Directory.Exists(folder)) Directory.CreateDirectory(folder);
        
        // 构造文件名（含时间戳和分辨率，确保唯一性）
        string fileName = ZString.Format(MemoryCardNameFormat, 
            data.CharacterId, data.ScriptId, data.CardId,
            DateTimeOffset.UtcNow.ToUnixTimeSeconds(),
            $"{texture.width}x{texture.height}");
        
        string filePath = Path.Combine(folder, fileName);
        
        // 将 Texture2D 编码为 PNG
        byte[] pngData = texture.EncodeToPNG();
        
        // 计算 HMAC-SHA256 签名（Key = MemoryCardDefaultLockKey）
        byte[] signature = ComputeHMACSHA256(pngData, MemoryCardDefaultLockKey);
        
        // 写入：[签名(32字节)] + [PNG数据]
        using var stream = new FileStream(filePath, FileMode.Create);
        stream.Write(signature, 0, signature.Length);
        stream.Write(pngData, 0, pngData.Length);
        
        Log.LogInfo($"[MemoryCard] 已保存: {filePath}");
    }
    
    private static byte[] ComputeHMACSHA256(byte[] data, long key)
    {
        byte[] keyBytes = BitConverter.GetBytes(key);
        using var hmac = new HMACSHA256(keyBytes);
        return hmac.ComputeHash(data);
    }
    
    // 读取本地回忆卡，同时校验签名（防篡改）
    public static Texture2D LoadLocalCard(string filePath)
    {
        if (!File.Exists(filePath)) return null;
        
        byte[] allData = File.ReadAllBytes(filePath);
        if (allData.Length < 32) return null;  // 至少需要签名长度
        
        // 分离签名和图片数据
        byte[] signature = allData[..32];
        byte[] pngData   = allData[32..];
        
        // 重新计算签名并比较
        byte[] expectedSig = ComputeHMACSHA256(pngData, MemoryCardDefaultLockKey);
        if (!signature.SequenceEqual(expectedSig))
        {
            Log.Warning($"[MemoryCard] 签名校验失败，文件可能被篡改: {filePath}");
            return null;
        }
        
        // 校验通过，加载图片
        var texture = new Texture2D(2, 2);
        texture.LoadImage(pngData);
        return texture;
    }
```

## 4. 好感度系统（CultivationComponent）

```csharp
// 位置：Hotfix/Function/GamePlay/Player/Cultivation/
// 好感度驱动回忆卡解锁
public static class CultivationComponentSystem
{
    // 战斗结束时更新好感度（PVP结算/主线剧情完成）
    public static void OnBattleEnd(this CultivationComponent self, BattleResult result)
    {
        int gainedIntimacy = CalculateIntimacyGain(result);
        self.AddIntimacy(result.CharacterId, gainedIntimacy);
        
        // 检查是否满足解锁条件
        CheckUnlockMemoryCard(self, result.CharacterId);
    }
    
    private static void CheckUnlockMemoryCard(
        CultivationComponent self, int characterId)
    {
        var cardConfigs = CfgManager.tables.TbMemoryCard.GetCardsByCharacter(characterId);
        foreach (var cardConf in cardConfigs)
        {
            if (self.IsCardUnlocked(cardConf.Id)) continue;  // 已解锁，跳过
            
            // 检查所有解锁条件
            bool intimacyMet = self.GetIntimacy(characterId) >= cardConf.RequiredIntimacy;
            bool storyMet = self.IsStoryCompleted(cardConf.RequiredStoryId);
            
            if (intimacyMet && storyMet)
            {
                self.UnlockMemoryCard(cardConf.Id);
                // 发布解锁事件（UI显示解锁动画）
                EventSystem.Instance.Publish(self.ClientScene(), 
                    new Evt_MemoryCardUnlocked { CardId = cardConf.Id });
            }
        }
    }
}
```

## 5. COS 上传分享

```csharp
    // 将回忆卡上传到腾讯云 COS（用于好友间分享）
    public static async ETTask<string> UploadToCOS(
        this MemoryCardComponent self, string localFilePath)
    {
        // SDK目录：Scripts/SDK/CosSDK
        var cosConfig = new CosConfig
        {
            Bucket = AppConfig.COSBucket,
            Region = AppConfig.COSRegion,
            SecretId = await self.GetTempSecretId(),    // 临时密钥（STS）
            SecretKey = await self.GetTempSecretKey(),
        };
        
        string cosKey = $"memorycard/{PlayerData.OpenId}/{Path.GetFileName(localFilePath)}";
        
        // 上传（TransferManager 支持大文件分块上传）
        var result = await CosManager.Upload(cosConfig, localFilePath, cosKey);
        if (result.IsSuccess)
        {
            return $"https://{cosConfig.Bucket}.cos.{cosConfig.Region}.myqcloud.com/{cosKey}";
        }
        return null;
    }
```

## 6. 常见问题与最佳实践

**Q: 为什么要对回忆卡图片做签名？**  
A: 防止玩家替换本地图片文件（用其他图片冒充解锁的回忆卡）。签名使用 HMAC-SHA256，Key 内嵌在代码中，普通玩家无法伪造。

**Q: 回忆卡图片很大，如何避免占用太多存储？**  
A: 配置表中设置分辨率限制（如最大 1024x1024），并对旧版本回忆卡进行定期清理（超过 N 张时删除最早的本地缓存）。

**Q: 如果服务器接口超时，回忆卡无法显示怎么办？**  
A: 优先使用本地缓存（先读 persistent 目录），本地不存在再向服务器请求。加载失败时显示默认封面图（`DefaultMemoryCard.png`）。

**Q: COS 临时密钥（STS）如何获取？**  
A: 通过游戏服务器的安全接口获取，不直接在客户端存储永久密钥。临时密钥有效期通常 30 分钟，上传前检查是否过期，过期重新申请。
