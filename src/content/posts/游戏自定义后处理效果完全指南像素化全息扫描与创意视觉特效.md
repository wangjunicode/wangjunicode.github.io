# 游戏自定义后处理效果完全指南：像素化、全息扫描与创意视觉特效

## 一、URP自定义后处理架构全景

### 1.1 URP后处理扩展点

Unity URP 提供了两种自定义后处理扩展方式：

| 方式 | API入口 | 适用场景 |
|-----|---------|---------|
| Volume Component + Custom Pass | `VolumeComponent` + `ScriptableRenderFeature` | 需要Volume参数控制的标准后处理效果 |
| Renderer Feature（全量控制） | `ScriptableRendererFeature` + `ScriptableRenderPass` | 需要精确控制注入点的复杂效果 |

**推荐架构**（两者结合）：
```
ScriptableRendererFeature（生命周期管理）
    └─ ScriptableRenderPass（具体渲染逻辑）
         └─ 读取 VolumeComponent 参数（实现Volume动画过渡）
```

### 1.2 URP后处理管线注入点

```csharp
// 常用的RenderPassEvent注入位置
RenderPassEvent.BeforeRenderingPrePasses     // 深度预通道之前
RenderPassEvent.AfterRenderingGbuffer        // G-Buffer之后（延迟渲染）
RenderPassEvent.AfterRenderingSkybox         // 天空盒之后
RenderPassEvent.BeforeRenderingTransparents  // 透明物体之前
RenderPassEvent.AfterRenderingTransparents   // 透明物体之后（可读深度）
RenderPassEvent.AfterRendering               // 最终输出之前（全屏后处理）
```

### 1.3 基础框架代码（可复用模板）

```csharp
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

/// <summary>
/// 自定义后处理 Renderer Feature 基础模板
/// 所有自定义后处理效果都应继承此模式
/// </summary>
public abstract class CustomPostProcessFeature<TSettings> : ScriptableRendererFeature
    where TSettings : VolumeComponent, IPostProcessComponent
{
    [System.Serializable]
    public class PassSettings
    {
        public RenderPassEvent renderPassEvent = RenderPassEvent.AfterRenderingTransparents;
        public Shader shader;
    }
    
    [SerializeField] protected PassSettings settings = new PassSettings();
    
    private CustomPostProcessPass _pass;
    
    public override void Create()
    {
        if (settings.shader == null) return;
        _pass = new CustomPostProcessPass(settings.shader, GetPassName());
        _pass.renderPassEvent = settings.renderPassEvent;
    }
    
    public override void AddRenderPasses(ScriptableRenderer renderer, ref RenderingData renderingData)
    {
        // 检查Camera是否需要后处理
        if (renderingData.cameraData.postProcessEnabled == false) return;
        
        // 检查Volume Component是否激活
        var stack = VolumeManager.instance.stack;
        var component = stack.GetComponent<TSettings>();
        if (component == null || !component.IsActive()) return;
        
        renderer.EnqueuePass(_pass);
    }
    
    protected abstract string GetPassName();
}
```

## 二、像素化效果（Pixelation）完整实现

### 2.1 效果原理

像素化效果将屏幕降采样到较低分辨率，然后放大还原，产生复古游戏的像素风格：

```
原始帧缓冲（1920×1080）
    ↓ 降分辨率UV采样（320×180）
像素块化图像
    ↓ 等比放大
输出（1920×1080，但内容为低分辨率像素块）
```

### 2.2 Volume Component参数定义

```csharp
using UnityEngine;
using UnityEngine.Rendering;

[System.Serializable, VolumeComponentMenu("Custom Post-process/Pixelation")]

{
    [Tooltip("像素块大小（越大像素感越强）")]
    public ClampedIntParameter pixelSize = new ClampedIntParameter(1, 1, 32);
    
    [Tooltip("颜色量化等级（模拟低位色深，0=不限制）")]
    public ClampedIntParameter colorLevels = new ClampedIntParameter(0, 0, 32);
    
    [Tooltip("启用抖动（Dithering）减轻色带")]
    public BoolParameter enableDithering = new BoolParameter(false);
    
    // IPostProcessComponent 接口：控制Volume是否激活
    public bool IsActive() => pixelSize.value > 1;
    public bool IsTileCompatible() => true;
}
```

### 2.3 像素化Shader实现

```hlsl
// Pixelation.shader（URP兼容）
Shader "CustomPostProcess/Pixelation"
{
    Properties
    {
        _MainTex("Source", 2D) = "white" {}
    }
    
    SubShader
    {
        Tags { "RenderType" = "Opaque" "RenderPipeline" = "UniversalPipeline" }
        
        Pass
        {
            ZTest Always ZWrite Off Cull Off
            
            HLSLPROGRAM
            #pragma vertex   FullscreenVert
            #pragma fragment PixelationFrag
            #pragma multi_compile_local _ _ENABLE_DITHERING
            
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            
            TEXTURE2D(_MainTex);
            SAMPLER(sampler_MainTex);
            float4 _MainTex_TexelSize; // 自动注入：xy=texelSize, zw=resolution
            
            float _PixelSize;
            float _ColorLevels;
            
            // Bayer矩阵抖动（4×4）
            static const float BayerMatrix[16] = {
                0.0/16.0,  8.0/16.0,  2.0/16.0, 10.0/16.0,
                12.0/16.0,  4.0/16.0, 14.0/16.0,  6.0/16.0,
                3.0/16.0, 11.0/16.0,  1.0/16.0,  9.0/16.0,
                15.0/16.0,  7.0/16.0, 13.0/16.0,  5.0/16.0
            };
            
            struct Attributes
            {
                float4 positionOS : POSITION;
                float2 uv         : TEXCOORD0;
            };
            
            struct Varyings
            {
                float4 positionCS : SV_POSITION;
                float2 uv         : TEXCOORD0;
            };
            
            Varyings FullscreenVert(Attributes input)
            {
                Varyings output;
                output.positionCS = TransformObjectToHClip(input.positionOS.xyz);
                output.uv = input.uv;
                return output;
            }
            
            float4 PixelationFrag(Varyings input) : SV_Target
            {
                float2 uv = input.uv;
                float2 resolution = _MainTex_TexelSize.zw;
                
                // 1. 计算像素块化UV
                float2 pixelResolution = resolution / _PixelSize;
                float2 pixelatedUV = floor(uv * pixelResolution) / pixelResolution;
                // 采样像素块中心（加半个像素偏移）
                pixelatedUV += 0.5 / pixelResolution;
                
                float4 color = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, pixelatedUV);
                
                // 2. 颜色量化（可选）
                if (_ColorLevels > 0)
                {
                    #if _ENABLE_DITHERING
                    // Bayer矩阵抖动
                    int2 pixelCoord = int2(uv * resolution) % 4;
                    float threshold = BayerMatrix[pixelCoord.y * 4 + pixelCoord.x];
                    color.rgb += (threshold - 0.5) / _ColorLevels;
                    #endif
                    
                    color.rgb = round(color.rgb * _ColorLevels) / _ColorLevels;
                }
                
                return color;
            }
            ENDHLSL
        }
    }
}
```

### 2.4 Renderer Feature实现

```csharp
using UnityEngine;
using UnityEngine.Rendering;
using UnityEngine.Rendering.Universal;

public class PixelationRendererFeature : ScriptableRendererFeature
{
    private PixelationRenderPass _pass;
    
    [SerializeField] private Shader _shader;
    
    public override void Create()
    {
        if (_shader == null)
        {
            _shader = Shader.Find("CustomPostProcess/Pixelation");
        }
        _pass = new PixelationRenderPass(_shader);
        _pass.renderPassEvent = RenderPassEvent.AfterRenderingTransparents;
    }
    
    public override void AddRenderPasses(ScriptableRenderer renderer, ref RenderingData renderingData)
    {
        if (!renderingData.cameraData.postProcessEnabled) return;
        
        var stack = VolumeManager.instance.stack;
        var pixelation = stack.GetComponent<PixelationEffect>();
        if (pixelation == null || !pixelation.IsActive()) return;
        
        _pass.Setup(pixelation);
        renderer.EnqueuePass(_pass);
    }
    
    protected override void Dispose(bool disposing)
    {
        _pass?.Dispose();
    }
}

public class PixelationRenderPass : ScriptableRenderPass, System.IDisposable
{
    private Material _material;
    private PixelationEffect _effect;
    private RTHandle _tempRT;
    
    private static readonly int ID_PixelSize   = Shader.PropertyToID("_PixelSize");
    private static readonly int ID_ColorLevels = Shader.PropertyToID("_ColorLevels");
    
    public PixelationRenderPass(Shader shader)
    {
        _material = CoreUtils.CreateEngineMaterial(shader);
        _tempRT = RTHandles.Alloc(Vector2.one, name: "_PixelationTemp");
    }
    
    public void Setup(PixelationEffect effect)
    {
        _effect = effect;
    }
    
    public override void Execute(ScriptableRenderContext context, ref RenderingData renderingData)
    {
        if (_material == null || _effect == null) return;
        
        var cmd = CommandBufferPool.Get("Pixelation Post-Process");
        
        // 设置Shader参数
        _material.SetFloat(ID_PixelSize,   _effect.pixelSize.value);
        _material.SetFloat(ID_ColorLevels, _effect.colorLevels.value);
        
        if (_effect.enableDithering.value)
            _material.EnableKeyword("_ENABLE_DITHERING");
        else
            _material.DisableKeyword("_ENABLE_DITHERING");
        
        // 获取当前相机颜色Buffer
        var cameraColorTarget = renderingData.cameraData.renderer.cameraColorTargetHandle;
        
        // Blit：当前帧 → 临时RT（应用后处理）→ 写回相机RT
        Blitter.BlitCameraTexture(cmd, cameraColorTarget, _tempRT, _material, 0);
        Blitter.BlitCameraTexture(cmd, _tempRT, cameraColorTarget);
        
        context.ExecuteCommandBuffer(cmd);
        CommandBufferPool.Release(cmd);
    }
    
    public void Dispose()
    {
        _tempRT?.Release();
        CoreUtils.Destroy(_material);
    }
}
```

## 三、全息扫描效果（Holographic Scan）

### 3.1 效果分解

全息扫描效果由三个视觉层叠加而成：
1. **扫描线**（Scan Lines）：水平/竖直重复的暗纹
2. **发光扫描带**（Scan Band）：以固定速度向上移动的亮带
3. **色偏（Chromatic Aberration）**：RGB通道的轻微偏移，产生全息感

### 3.2 全息扫描Shader

```hlsl
// HolographicScan.shader
Shader "CustomPostProcess/HolographicScan"
{
    Properties { _MainTex("Source", 2D) = "white" {} }
    
    SubShader
    {
        Tags { "RenderPipeline" = "UniversalPipeline" }
        
        Pass
        {
            ZTest Always ZWrite Off Cull Off
            
            HLSLPROGRAM
            #pragma vertex   FullscreenVert
            #pragma fragment HolographicFrag
            
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            
            TEXTURE2D(_MainTex);
            SAMPLER(sampler_MainTex);
            
            // 扫描线参数
            float _ScanLineFrequency;    // 扫描线密度（默认150）
            float _ScanLineIntensity;    // 扫描线强度（默认0.15）
            
            // 扫描带参数
            float _ScanBandSpeed;        // 扫描带移动速度（默认0.3）
            float _ScanBandWidth;        // 扫描带宽度（默认0.05）
            float _ScanBandIntensity;    // 扫描带亮度（默认0.5）
            float4 _ScanBandColor;       // 扫描带颜色（默认淡蓝色）
            
            // 色偏参数
            float _ChromaticOffset;      // 色偏强度（默认0.003）
            
            // 整体色调
            float4 _HoloTint;           // 全息色调（默认淡青色）
            float _HoloIntensity;        // 全息效果强度（0~1混合）
            
            struct Attributes { float4 pos : POSITION; float2 uv : TEXCOORD0; };
            struct Varyings   { float4 pos : SV_POSITION; float2 uv : TEXCOORD0; };
            
            Varyings FullscreenVert(Attributes v)
            {
                Varyings o;
                o.pos = TransformObjectToHClip(v.pos.xyz);
                o.uv  = v.uv;
                return o;
            }
            
            float4 HolographicFrag(Varyings i) : SV_Target
            {
                float2 uv = i.uv;
                
                // ── 1. 色差（Chromatic Aberration）──
                float2 caOffset = float2(_ChromaticOffset, 0);
                float r = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, uv + caOffset).r;
                float g = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, uv           ).g;
                float b = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, uv - caOffset).b;
                float4 srcColor = float4(r, g, b, 1.0);
                
                // ── 2. 扫描线（水平暗纹）──
                float scanLine = sin(uv.y * _ScanLineFrequency * 3.14159 * 2.0);
                float scanLineMask = (scanLine * 0.5 + 0.5); // [0, 1]
                // 扫描线使颜色略微变暗
                srcColor.rgb *= 1.0 - scanLineMask * _ScanLineIntensity;
                
                // ── 3. 移动扫描带 ──
                float scanBandPos = frac(_Time.y * _ScanBandSpeed); // 当前扫描带位置 [0,1]
                float distToScan  = abs(uv.y - scanBandPos);
                float scanBand    = smoothstep(_ScanBandWidth, 0.0, distToScan);
                // 扫描带叠加发光色
                srcColor.rgb += scanBand * _ScanBandIntensity * _ScanBandColor.rgb;
                
                // ── 4. 全息色调混合 ──
                // 将图像偏向全息色调（淡青绿色）
                float3 holoColor = srcColor.rgb * _HoloTint.rgb;
                srcColor.rgb = lerp(srcColor.rgb, holoColor, _HoloIntensity * 0.3);
                
                // ── 5. 整体亮度微调（全息感通常偏高对比度）──
                srcColor.rgb = pow(srcColor.rgb, 0.9); // 轻微提亮暗部
                
                return srcColor;
            }
            ENDHLSL
        }
    }
}
```

## 四、热浪扭曲效果（Heat Distortion）

### 4.1 效果实现（深度感知版）

热浪扭曲通过 `_DistortionMap`（Flow Map）偏移采样UV，结合深度缓冲只影响远处物体：

```hlsl
// HeatDistortion.shader
Shader "CustomPostProcess/HeatDistortion"
{
    Properties
    {
        _MainTex("Source", 2D)            = "white" {}
        _DistortionMap("Distortion Map", 2D) = "gray" {}
    }
    
    SubShader
    {
        Tags { "RenderPipeline" = "UniversalPipeline" }
        
        Pass
        {
            ZTest Always ZWrite Off Cull Off
            
            HLSLPROGRAM
            #pragma vertex   FullscreenVert
            #pragma fragment HeatFrag
            
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/Core.hlsl"
            #include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/DeclareDepthTexture.hlsl"
            
            TEXTURE2D(_MainTex);         SAMPLER(sampler_MainTex);
            TEXTURE2D(_DistortionMap);   SAMPLER(sampler_DistortionMap);
            
            float _DistortionStrength;   // 扭曲强度
            float _DistortionSpeed;      // 流动速度
            float _DistortionTiling;     // 贴图平铺
            float _DepthFalloff;         // 深度衰减（近处减弱扭曲）
            float _DepthThreshold;       // 深度阈值（低于此深度不扭曲）
            
            struct Attributes { float4 pos : POSITION; float2 uv : TEXCOORD0; };
            struct Varyings   { float4 pos : SV_POSITION; float2 uv : TEXCOORD0; };
            
            Varyings FullscreenVert(Attributes v)
            {
                Varyings o;
                o.pos = TransformObjectToHClip(v.pos.xyz);
                o.uv  = v.uv;
                return o;
            }
            
            float4 HeatFrag(Varyings i) : SV_Target
            {
                float2 uv = i.uv;
                
                // ── 1. 读取场景深度（用于深度衰减）──
                float rawDepth = SampleSceneDepth(uv);
                float linearDepth = LinearEyeDepth(rawDepth, _ZBufferParams);
                // 近处物体（深度小）扭曲弱，远处强
                float depthMask = saturate((linearDepth - _DepthThreshold) * _DepthFalloff);
                
                // ── 2. 采样扭曲贴图（Flow Map）──
                float2 flowUV1 = uv * _DistortionTiling + float2(_Time.y * _DistortionSpeed, 0);
                float2 flowUV2 = uv * _DistortionTiling + float2(0, _Time.y * _DistortionSpeed * 0.7);
                
                float2 flow1 = SAMPLE_TEXTURE2D(_DistortionMap, sampler_DistortionMap, flowUV1).rg;
                float2 flow2 = SAMPLE_TEXTURE2D(_DistortionMap, sampler_DistortionMap, flowUV2).rg;
                
                // 叠加两层扭曲，产生更自然的流体感
                float2 flowDir = (flow1 + flow2) * 0.5 - 0.5; // 转换到 [-0.5, 0.5]
                
                // ── 3. 计算偏移后UV ──
                float2 distortedUV = uv + flowDir * _DistortionStrength * depthMask;
                
                // ── 4. 采样源颜色 ──
                float4 distortedColor  = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, distortedUV);
                float4 originalColor   = SAMPLE_TEXTURE2D(_MainTex, sampler_MainTex, uv);
                
                // 边缘淡出（防止边界采样越界产生撕裂）
                float edgeFade = min(
                    min(uv.x, 1 - uv.x),
                    min(uv.y, 1 - uv.y)
                ) * 20.0;
                edgeFade = saturate(edgeFade);
                
                return lerp(originalColor, distortedColor, edgeFade);
            }
            ENDHLSL
        }
    }
}
```

## 五、边缘检测效果（Edge Detection）

### 5.1 Roberts Cross算子（轻量版）

```hlsl
// 在Full Screen Pass中使用 Roberts Cross 边缘检测
float4 EdgeDetection_RobertsCross(TEXTURE2D_PARAM(tex, samp), float2 uv, float2 texelSize)
{
    // Roberts Cross 2×2采样核
    float4 s00 = SAMPLE_TEXTURE2D(tex, samp, uv + texelSize * float2(-1, -1));
    float4 s11 = SAMPLE_TEXTURE2D(tex, samp, uv + texelSize * float2( 1,  1));
    float4 s10 = SAMPLE_TEXTURE2D(tex, samp, uv + texelSize * float2( 1, -1));
    float4 s01 = SAMPLE_TEXTURE2D(tex, samp, uv + texelSize * float2(-1,  1));
    
    float4 gX = s00 - s11; // X方向梯度
    float4 gY = s10 - s01; // Y方向梯度
    
    return sqrt(gX * gX + gY * gY); // 边缘强度
}
```

### 5.2 基于法线+深度的精确边缘检测

```hlsl
#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/DeclareNormalsTexture.hlsl"
#include "Packages/com.unity.render-pipelines.universal/ShaderLibrary/DeclareDepthTexture.hlsl"

float ComputeEdge(float2 uv, float2 texelSize)
{
    // 采样周围4个点的法线和深度
    float2 offsets[4] = {
        float2( 1, 0), float2(-1, 0),
        float2( 0, 1), float2( 0,-1)
    };
    
    float3 centerNormal = SampleSceneNormals(uv);
    float  centerDepth  = LinearEyeDepth(SampleSceneDepth(uv), _ZBufferParams);
    
    float normalEdge = 0;
    float depthEdge  = 0;
    
    for (int j = 0; j < 4; j++)
    {
        float2 neighborUV = uv + offsets[j] * texelSize;
        float3 neighborN  = SampleSceneNormals(neighborUV);
        float  neighborD  = LinearEyeDepth(SampleSceneDepth(neighborUV), _ZBufferParams);
        
        // 法线差异 → 相邻面夹角边缘
        normalEdge += saturate(1.0 - dot(centerNormal, neighborN));
        // 深度差异 → 物体轮廓边缘
        depthEdge  += abs(centerDepth - neighborD) / centerDepth;
    }
    
    return saturate(normalEdge * 0.5 + depthEdge * 0.1);
}
```

## 六、组合效果管理：多Pass后处理链

### 6.1 效果叠加顺序

```csharp
/// <summary>
/// 多个自定义后处理效果的顺序管理
/// 注意：多个Renderer Feature按添加顺序执行，但都在同一个RenderPassEvent下
/// </summary>
public class PostProcessPipelineManager : MonoBehaviour
{
    // 建议的效果叠加顺序（依赖关系决定顺序）
    // 1. 深度相关效果（需要完整深度缓冲）：热浪扭曲、景深
    // 2. 全屏色彩效果（像素化、色调映射）
    // 3. 叠加效果（扫描线、全息、UI相关）
    // 4. 抗锯齿（最后执行，对最终输出平滑处理）
    
    [SerializeField] private Volume _postProcessVolume;
    
    public void SetHolographicMode(bool enable, float transitionTime = 0.5f)
    {
        var stack = _postProcessVolume.profile;
        
        if (stack.TryGet<HolographicScanEffect>(out var holo))
        {
            // 通过协程平滑过渡效果强度
            StartCoroutine(TransitionEffect(holo.holoIntensity, enable ? 1f : 0f, transitionTime));
        }
    }
    
    private System.Collections.IEnumerator TransitionEffect(
        ClampedFloatParameter param, float target, float duration)
    {
        float start   = param.value;
        float elapsed = 0f;
        
        while (elapsed < duration)
        {
            elapsed    += Time.deltaTime;
            param.value = Mathf.Lerp(start, target, elapsed / duration);
            yield return null;
        }
        
        param.value = target;
    }
}
```

### 6.2 运行时动态切换效果

```csharp
using UnityEngine;
using UnityEngine.Rendering;

/// <summary>
/// 后处理效果的场景化管理：根据游戏状态切换视觉风格
/// 例：正常游戏→受伤闪红→全息回放→死亡灰度
/// </summary>
public class GameStateVisualController : MonoBehaviour
{
    [SerializeField] private Volume _globalVolume;
    
    private PixelationEffect     _pixelation;
    private HolographicScanEffect _holographic;
    
    private void Awake()
    {
        var profile = _globalVolume.profile;
        profile.TryGet(out _pixelation);
        profile.TryGet(out _holographic);
    }
    
    /// <summary>进入回放模式：启用全息扫描效果</summary>
    public void EnterReplayMode()
    {
        if (_holographic != null)
        {
            _holographic.active         = true;
            _holographic.holoIntensity.value = 0.8f;
            _holographic.scanBandSpeed.value = 0.5f;
        }
    }
    
    /// <summary>进入像素风格关卡</summary>
    public void EnterPixelWorld(int pixelBlockSize = 8)
    {
        if (_pixelation != null)
        {
            _pixelation.active             = true;
            _pixelation.pixelSize.value    = pixelBlockSize;
            _pixelation.colorLevels.value  = 16; // 16色限制
        }
    }
    
    /// <summary>恢复正常渲染</summary>
    public void RestoreNormalRendering()
    {
        if (_pixelation   != null) _pixelation.active   = false;
        if (_holographic  != null) _holographic.active  = false;
    }
}
```

## 七、性能优化与最佳实践

### 7.1 Blit优化：避免额外RT分配

```csharp
// ❌ 低效：每帧分配临时RT
RenderTexture temp = RenderTexture.GetTemporary(...);
Graphics.Blit(src, temp, material);
Graphics.Blit(temp, dst);
RenderTexture.ReleaseTemporary(temp);

// ✅ 高效：使用RTHandle持久化管理
// RTHandle在Feature.Create()时分配，仅尺寸变化时重新分配
private RTHandle _tempRT;
_tempRT = RTHandles.Alloc(Vector2.one, name: "_PostProcessTemp");
// Execute中使用：
Blitter.BlitCameraTexture(cmd, src, _tempRT, material, 0);
Blitter.BlitCameraTexture(cmd, _tempRT, dst);
```

### 7.2 CommandBuffer复用

```csharp
// ✅ 使用CommandBufferPool，避免每帧GC分配
var cmd = CommandBufferPool.Get("MyPostProcess");
try
{
    // ... 渲染命令
    context.ExecuteCommandBuffer(cmd);
}
finally
{
    CommandBufferPool.Release(cmd); // 必须在finally中释放，防止异常泄漏
}
```

### 7.3 移动端后处理分级

```csharp
// 根据设备性能等级动态降级后处理
public static class PostProcessQualitySettings
{
    public static void ApplyByDeviceGrade(Volume volume)
    {
        var grade = GetDeviceGrade();
        var profile = volume.profile;
        
        if (profile.TryGet<PixelationEffect>(out var pix))
        {
            // 移动端低配：关闭抖动，减少Texture Fetch
            pix.enableDithering.value = grade >= DeviceGrade.Medium;
        }
        
        if (profile.TryGet<HolographicScanEffect>(out var holo))
        {
            // 移动端低配：降低扫描线密度
            holo.scanLineFrequency.value = grade >= DeviceGrade.High ? 150f : 60f;
        }
    }
    
    private static DeviceGrade GetDeviceGrade()
    {
        // 基于GPU内存和像素填充率评估设备等级
        long gpuMem = SystemInfo.graphicsMemorySize;
        return gpuMem switch
        {
            > 4096 => DeviceGrade.High,
            > 2048 => DeviceGrade.Medium,
            _      => DeviceGrade.Low
        };
    }
    
    private enum DeviceGrade { Low, Medium, High }
}
```

### 7.4 最佳实践总结

| 类别 | 规范 |
|-----|------|
| RT管理 | 使用 RTHandles，在 Dispose 中释放 |
| CommandBuffer | 使用 CommandBufferPool，finally 中释放 |
| 精度 | 后处理 Shader 一律使用 half 精度（颜色运算不需要 float） |
| 分支 | 用 Shader Keyword 替代动态分支（`#pragma multi_compile_local`） |
| 采样次数 | 单个后处理 Pass 的 Texture Fetch 不超过 4 次（移动端标准） |
| 参数更新 | 使用 PropertyToID 预缓存，避免每帧字符串哈希 |
| 深度依赖 | 需要深度的效果使用 `DeclareDepthTexture.hlsl`，并在 Pass 中配置 `ConfigureInput(ScriptableRenderPassInput.Depth)` |
| 调试 | 使用 Frame Debugger 的 "Custom Post Process" 标签验证注入顺序 |

掌握 URP 自定义后处理框架，是游戏视觉风格化和特效制作的核心能力。从像素化的复古风格到全息界面的科幻感，再到热浪扭曲的沉浸式氛围，每种效果都遵循相同的架构模式：**Volume Component 定义参数 → Renderer Feature 管理生命周期 → ScriptableRenderPass 执行 GPU 命令**。在移动端项目中，合理的效果分级和 RT 管理是保持稳定帧率的关键。
