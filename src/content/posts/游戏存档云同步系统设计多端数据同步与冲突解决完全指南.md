# 游戏存档云同步系统设计：多端数据同步、冲突解决与差量上传完全指南

## 1. 云存档系统概述

玩家在多设备游玩、换机等场景下，**游戏进度的云端同步**是现代游戏的基础功能。设计不当的云存档系统会导致：

- 数据覆盖丢失（手机存档把PC进度覆盖）
- 同步死锁（双端修改导致无限冲突）
- 流量浪费（每次同步上传全量数据）
- 弱网下数据不一致

### 1.2 同步策略对比

| 策略 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|

### 2.2 本地存档管理器

```csharp
/// <summary>
/// 本地存档管理器
/// 支持：加密存储、版本备份、快速读写
/// </summary>
public class LocalSaveManager
{
    private const string SAVE_FILE = "save.dat";
    private const string BACKUP_FILE = "save.bak";
    private const int MAX_BACKUPS = 3;

    private readonly string _savePath;
    private readonly string _encryptionKey;
    private GameSaveData _currentSave;

    // 脏标记：记录哪些模块被修改过
    private readonly HashSet<string> _dirtyModules = new HashSet<string>();

    public LocalSaveManager(string playerId, string encryptionKey)
    {
        _savePath = Application.persistentDataPath + "/saves/" + playerId + "/";
        _encryptionKey = encryptionKey;
        System.IO.Directory.CreateDirectory(_savePath);
    }

    /// <summary>
    /// 加载本地存档
    /// </summary>
    public GameSaveData Load()
    {
        string filePath = _savePath + SAVE_FILE;

        try
        {
            if (!System.IO.File.Exists(filePath))
            {
                _currentSave = CreateNewSave();
                return _currentSave;
            }

            byte[] encrypted = System.IO.File.ReadAllBytes(filePath);
            byte[] decrypted = AesEncryption.Decrypt(encrypted, _encryptionKey);
            string json = System.Text.Encoding.UTF8.GetString(decrypted);

            var save = JsonSerializer.Deserialize<GameSaveData>(json);

            // 验证校验和
            if (!ValidateChecksum(save))
            {
                Debug.LogWarning("[Save] Checksum mismatch, trying backup...");
                save = LoadBackup();
            }

            _currentSave = save;
            return save;
        }
        catch (Exception e)
        {
            Debug.LogError($"[Save] Load failed: {e.Message}, trying backup...");
            _currentSave = LoadBackup() ?? CreateNewSave();
            return _currentSave;
        }
    }

    /// <summary>
    /// 保存存档（带原子写入）
    /// </summary>
    public void Save(GameSaveData data)
    {
        // 更新版本向量
        string deviceId = SystemInfo.deviceUniqueIdentifier;
        if (!data.VersionVector.ContainsKey(deviceId))
            data.VersionVector[deviceId] = 0;
        data.VersionVector[deviceId]++;
        data.LastModifiedUtc = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds();
        data.LastModifiedDeviceId = deviceId;

        // 计算校验和
        data.Checksum = ComputeChecksum(data);

        string json = JsonSerializer.Serialize(data);
        byte[] decrypted = System.Text.Encoding.UTF8.GetBytes(json);
        byte[] encrypted = AesEncryption.Encrypt(decrypted, _encryptionKey);

        // 原子写入：先写临时文件，再替换
        string tempPath = _savePath + "save.tmp";
        string finalPath = _savePath + SAVE_FILE;

        System.IO.File.WriteAllBytes(tempPath, encrypted);

        // 备份旧存档
        if (System.IO.File.Exists(finalPath))
        {
            RotateBackups();
            System.IO.File.Copy(finalPath, _savePath + BACKUP_FILE, overwrite: true);
        }

        System.IO.File.Move(tempPath, finalPath);
        _currentSave = data;
        _dirtyModules.Clear();
    }

    /// <summary>
    /// 标记模块为脏（需要同步）
    /// </summary>
    public void MarkDirty(string module) => _dirtyModules.Add(module);

    public IReadOnlySet<string> GetDirtyModules() => _dirtyModules;

    private bool ValidateChecksum(GameSaveData data)
    {
        string expected = data.Checksum;
        data.Checksum = null;
        string actual = ComputeChecksum(data);
        data.Checksum = expected;
        return expected == actual;
    }

    private string ComputeChecksum(GameSaveData data)
    {
        // 使用SHA256计算存档内容的校验和
        string content = JsonSerializer.Serialize(data);
        using var sha256 = System.Security.Cryptography.SHA256.Create();
        var hash = sha256.ComputeHash(System.Text.Encoding.UTF8.GetBytes(content));
        return Convert.ToBase64String(hash);
    }

    private GameSaveData LoadBackup()
    {
        string backupPath = _savePath + BACKUP_FILE;
        if (!System.IO.File.Exists(backupPath)) return null;

        try
        {
            byte[] encrypted = System.IO.File.ReadAllBytes(backupPath);
            byte[] decrypted = AesEncryption.Decrypt(encrypted, _encryptionKey);
            string json = System.Text.Encoding.UTF8.GetString(decrypted);
            return JsonSerializer.Deserialize<GameSaveData>(json);
        }
        catch { return null; }
    }

    private GameSaveData CreateNewSave()
    {
        return new GameSaveData
        {
            SaveId = Guid.NewGuid().ToString(),
            PlayerId = "local",
            LastModifiedUtc = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds(),
            LastModifiedDeviceId = SystemInfo.deviceUniqueIdentifier
        };
    }

    private void RotateBackups()
    {
        // 简单备份轮换：只保留最新的一个备份
        string backup1 = _savePath + "save.bak1";
        if (System.IO.File.Exists(_savePath + BACKUP_FILE))
            System.IO.File.Copy(_savePath + BACKUP_FILE, backup1, overwrite: true);
    }
}
```

## 3. 差量计算与压缩上传

### 3.1 存档差量计算

```csharp
/// <summary>
/// 差量计算器
/// 只上传自上次同步以来发生变化的字段
/// 可大幅减少网络传输量（通常节省80-95%流量）
/// </summary>
public class SaveDeltaCalculator
{
    /// <summary>
    /// 计算两个存档版本之间的差量
    /// </summary>
    public SaveDelta Calculate(GameSaveData baseline, GameSaveData current)
    {
        var delta = new SaveDelta
        {
            SaveId = current.SaveId,
            PlayerId = current.PlayerId,
            VersionVector = current.VersionVector,
            LastModifiedUtc = current.LastModifiedUtc,
            Timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds()
        };

        // 对比PlayerProgress
        var progressDelta = DiffPlayerProgress(baseline?.PlayerProgress, current.PlayerProgress);
        if (progressDelta != null) delta.PlayerProgressDelta = progressDelta;

        // 对比Inventory（逐项目对比）
        var inventoryDelta = DiffInventory(baseline?.Inventory, current.Inventory);
        if (inventoryDelta != null) delta.InventoryDelta = inventoryDelta;

        // Settings通常全量更新
        if (HasSettingsChanged(baseline?.Settings, current.Settings))
            delta.SettingsDelta = current.Settings;

        return delta;
    }

    private PlayerProgressDelta DiffPlayerProgress(PlayerProgress baseline, PlayerProgress current)
    {
        if (baseline == null) return new PlayerProgressDelta { FullData = current };

        var delta = new PlayerProgressDelta();
        bool hasChange = false;

        if (baseline.Level != current.Level)
        { delta.Level = current.Level; hasChange = true; }

        if (baseline.Experience != current.Experience)
        { delta.Experience = current.Experience; hasChange = true; }

        if (baseline.CurrentChapter != current.CurrentChapter)
        { delta.CurrentChapter = current.CurrentChapter; hasChange = true; }

        // 计算新完成的任务（只记录新增，不记录减少）
        var newQuests = current.CompletedQuests.Except(baseline.CompletedQuests).ToList();
        if (newQuests.Count > 0)
        { delta.NewCompletedQuests = newQuests; hasChange = true; }

        // 新解锁的成就
        var newAchievements = current.Achievements
            .Where(kv => kv.Value && !baseline.Achievements.GetValueOrDefault(kv.Key))
            .ToDictionary(kv => kv.Key, kv => kv.Value);
        if (newAchievements.Count > 0)
        { delta.NewAchievements = newAchievements; hasChange = true; }

        return hasChange ? delta : null;
    }

    private InventoryDelta DiffInventory(InventoryData baseline, InventoryData current)
    {
        if (baseline == null) return new InventoryDelta { FullData = current };

        var delta = new InventoryDelta();
        bool hasChange = false;

        if (baseline.Gold != current.Gold) { delta.Gold = current.Gold; hasChange = true; }
        if (baseline.Gems != current.Gems) { delta.Gems = current.Gems; hasChange = true; }

        // 逐个Item对比
        var modifiedItems = new Dictionary<string, ItemStack>();
        foreach (var kv in current.Items)
        {
            if (!baseline.Items.TryGetValue(kv.Key, out var baseItem) ||
                baseItem.Count != kv.Value.Count)
            {
                modifiedItems[kv.Key] = kv.Value;
                hasChange = true;
            }
        }

        // 记录被删除的Item（Count=0）
        foreach (var key in baseline.Items.Keys)
        {
            if (!current.Items.ContainsKey(key))
            {
                modifiedItems[key] = new ItemStack { ItemId = key, Count = 0 };
                hasChange = true;
            }
        }

        if (modifiedItems.Count > 0) delta.ModifiedItems = modifiedItems;

        return hasChange ? delta : null;
    }

    private bool HasSettingsChanged(SettingsData baseline, SettingsData current)
    {
        if (baseline == null) return true;
        return JsonSerializer.Serialize(baseline) != JsonSerializer.Serialize(current);
    }
}

[Serializable]
public class SaveDelta
{
    public string SaveId { get; set; }
    public string PlayerId { get; set; }
    public Dictionary<string, long> VersionVector { get; set; }
    public long LastModifiedUtc { get; set; }
    public long Timestamp { get; set; }

    public PlayerProgressDelta PlayerProgressDelta { get; set; }
    public InventoryDelta InventoryDelta { get; set; }
    public SettingsData SettingsDelta { get; set; }
}
```

## 4. 冲突检测与解决算法

### 4.1 版本向量冲突检测

```csharp
/// <summary>
/// 版本向量比较器
/// 用于检测并发修改（冲突）
/// </summary>
public static class VersionVectorComparer
{
    public enum Relation
    {
        Equal,          // 完全相同
        LocalNewer,     // 本地更新（本地可以直接覆盖云端）
        RemoteNewer,    // 云端更新（需要下载云端存档）
        Concurrent,     // 并发修改（存在真实冲突，需要合并）
    }

    /// <summary>
    /// 比较两个版本向量的关系
    /// </summary>
    public static Relation Compare(
        Dictionary<string, long> local,
        Dictionary<string, long> remote)
    {
        bool localHasNewer = false;
        bool remoteHasNewer = false;

        // 检查local中的每个设备版本
        foreach (var kv in local)
        {
            long remoteVersion = remote.GetValueOrDefault(kv.Key, 0);
            if (kv.Value > remoteVersion) localHasNewer = true;
            if (kv.Value < remoteVersion) remoteHasNewer = true;
        }

        // 检查remote中有但local没有的设备
        foreach (var kv in remote)
        {
            if (!local.ContainsKey(kv.Key) && kv.Value > 0)
                remoteHasNewer = true;
        }

        if (!localHasNewer && !remoteHasNewer) return Relation.Equal;
        if (localHasNewer && !remoteHasNewer) return Relation.LocalNewer;
        if (!localHasNewer && remoteHasNewer) return Relation.RemoteNewer;
        return Relation.Concurrent; // 两边都有更新 = 冲突
    }

    /// <summary>
    /// 合并两个版本向量（取每个设备的最大值）
    /// </summary>
    public static Dictionary<string, long> Merge(
        Dictionary<string, long> v1,
        Dictionary<string, long> v2)
    {
        var merged = new Dictionary<string, long>(v1);
        foreach (var kv in v2)
        {
            if (!merged.ContainsKey(kv.Key) || merged[kv.Key] < kv.Value)
                merged[kv.Key] = kv.Value;
        }
        return merged;
    }
}
```

### 4.2 三路合并算法

```csharp
/// <summary>
/// 三路合并算法（Three-Way Merge）
/// 需要一个共同祖先（Base）版本
/// Local = Base + 本地修改
/// Remote = Base + 远端修改
/// Merged = Base + 本地修改 + 远端修改（无交叉冲突时）
/// </summary>
public class ThreeWayMerger
{
    /// <summary>
    /// 合并存档数据
    /// </summary>
    public MergeResult<GameSaveData> Merge(
        GameSaveData baseVersion,   // 上次同步时的版本（公共祖先）
        GameSaveData localVersion,  // 本地当前版本
        GameSaveData remoteVersion) // 云端版本
    {
        var result = new MergeResult<GameSaveData>();
        var merged = new GameSaveData
        {
            SaveId = localVersion.SaveId,
            PlayerId = localVersion.PlayerId,
            VersionVector = VersionVectorComparer.Merge(
                localVersion.VersionVector,
                remoteVersion.VersionVector)
        };

        // 合并玩家进度
        var progressResult = MergePlayerProgress(
            baseVersion?.PlayerProgress,
            localVersion.PlayerProgress,
            remoteVersion.PlayerProgress);
        merged.PlayerProgress = progressResult.Data;
        result.Conflicts.AddRange(progressResult.Conflicts);

        // 合并背包（逐物品级别合并）
        var inventoryResult = MergeInventory(
            baseVersion?.Inventory,
            localVersion.Inventory,
            remoteVersion.Inventory);
        merged.Inventory = inventoryResult.Data;
        result.Conflicts.AddRange(inventoryResult.Conflicts);

        // Settings：使用最近修改时间的版本（LWW策略）
        merged.Settings = localVersion.LastModifiedUtc > remoteVersion.LastModifiedUtc
            ? localVersion.Settings
            : remoteVersion.Settings;

        merged.LastModifiedUtc = Math.Max(localVersion.LastModifiedUtc, remoteVersion.LastModifiedUtc);
        result.Data = merged;
        result.HasConflicts = result.Conflicts.Count > 0;

        return result;
    }

    private MergeResult<PlayerProgress> MergePlayerProgress(
        PlayerProgress baseP,
        PlayerProgress localP,
        PlayerProgress remoteP)
    {
        var result = new MergeResult<PlayerProgress> { Data = new PlayerProgress() };
        var merged = result.Data;

        // Level：取最大值（玩家永远不会降级）
        merged.Level = Math.Max(localP.Level, remoteP.Level);
        // Experience：取最大值
        merged.Experience = Math.Max(localP.Experience, remoteP.Experience);
        // Chapter：取最大值
        merged.CurrentChapter = Math.Max(localP.CurrentChapter, remoteP.CurrentChapter);

        // 已完成任务：取并集（任务不可撤销）
        merged.CompletedQuests = localP.CompletedQuests
            .Union(remoteP.CompletedQuests)
            .ToList();

        // 成就：取并集
        merged.Achievements = new Dictionary<string, bool>();
        foreach (var kv in localP.Achievements)
            merged.Achievements[kv.Key] = kv.Value;
        foreach (var kv in remoteP.Achievements)
        {
            if (!merged.Achievements.ContainsKey(kv.Key))
                merged.Achievements[kv.Key] = kv.Value;
            else if (kv.Value) // 成就只能解锁不能撤销
                merged.Achievements[kv.Key] = true;
        }

        return result;
    }

    private MergeResult<InventoryData> MergeInventory(
        InventoryData baseI,
        InventoryData localI,
        InventoryData remoteI)
    {
        var result = new MergeResult<InventoryData> { Data = new InventoryData() };
        var merged = result.Data;

        // 货币：使用Last-Write-Win（以最后修改时间为准）
        // 注意：这里需要配合服务端验证，防止作弊
        merged.Gold = localI.LastModifiedGold > remoteI.LastModifiedGold
            ? localI.Gold : remoteI.Gold;
        merged.Gems = localI.LastModifiedGems > remoteI.LastModifiedGems
            ? localI.Gems : remoteI.Gems;

        // 物品：逐个合并
        merged.Items = new Dictionary<string, ItemStack>();
        var allItemIds = localI.Items.Keys.Union(remoteI.Items.Keys).ToHashSet();

        foreach (var itemId in allItemIds)
        {
            localI.Items.TryGetValue(itemId, out var localItem);
            remoteI.Items.TryGetValue(itemId, out var remoteItem);
            bool baseHasItem = baseI?.Items.ContainsKey(itemId) ?? false;

            if (localItem == null && remoteItem != null)
            {
                // 只有远端有：直接使用远端
                merged.Items[itemId] = remoteItem;
            }
            else if (localItem != null && remoteItem == null)
            {
                // 只有本地有：直接使用本地
                merged.Items[itemId] = localItem;
            }
            else if (localItem != null && remoteItem != null)
            {
                // 双端都有：使用LastModified更新的版本
                if (localItem.LastModified >= remoteItem.LastModified)
                    merged.Items[itemId] = localItem;
                else
                    merged.Items[itemId] = remoteItem;

                // 如果数量差异过大，记录为冲突供人工审核
                int diff = Math.Abs(localItem.Count - remoteItem.Count);
                if (diff > 100 && baseHasItem) // 差距超过100个，可能是外挂
                {
                    result.Conflicts.Add(new MergeConflict
                    {
                        Field = $"Inventory.{itemId}",
                        LocalValue = localItem.Count.ToString(),
                        RemoteValue = remoteItem.Count.ToString(),
                        Reason = "Large quantity discrepancy"
                    });
                }
            }
        }

        return result;
    }
}

public class MergeResult<T>
{
    public T Data { get; set; }
    public bool HasConflicts { get; set; }
    public List<MergeConflict> Conflicts { get; } = new List<MergeConflict>();
}

public class MergeConflict
{
    public string Field { get; set; }
    public string LocalValue { get; set; }
    public string RemoteValue { get; set; }
    public string Reason { get; set; }
    public ConflictResolution Resolution { get; set; } = ConflictResolution.UseRemote;
}

public enum ConflictResolution { UseLocal, UseRemote, UseMerged, NeedManual }
```

```csharp
/// <summary>
/// 云存档同步管理器
/// 处理完整的上传/下载/冲突解决流程
/// </summary>
public class CloudSaveManager
{
    private readonly LocalSaveManager _localSave;
    private readonly ICloudSaveApi _cloudApi;
    private readonly ThreeWayMerger _merger = new();
    private readonly SaveDeltaCalculator _deltaCalc = new();

    // 上次同步时的基础版本（用于三路合并）
    private GameSaveData _lastSyncedVersion;

    // 同步锁（防止并发同步）
    private bool _isSyncing;

    public event Action<SyncResult> OnSyncComplete;
    public event Action<MergeConflict[]> OnConflictsFound;

    public CloudSaveManager(LocalSaveManager localSave, ICloudSaveApi cloudApi)
    {
        _localSave = localSave;
        _cloudApi = cloudApi;
    }

    /// <summary>
    /// 主同步入口
    /// </summary>
    public async UniTask<SyncResult> SyncAsync(CancellationToken ct = default)
    {
        if (_isSyncing) return new SyncResult { Status = SyncStatus.AlreadySyncing };
        _isSyncing = true;

        try
        {
            return await DoSyncAsync(ct);
        }
        finally
        {
            _isSyncing = false;
        }
    }

    private async UniTask<SyncResult> DoSyncAsync(CancellationToken ct)
    {
        var localData = _localSave.Load();

        // 1. 检查云端版本
        var cloudMeta = await _cloudApi.GetMetadataAsync(localData.PlayerId, ct);
        if (cloudMeta == null)
        {
            // 云端没有存档，直接上传
            await UploadFullAsync(localData, ct);
            _lastSyncedVersion = DeepCopy(localData);
            return new SyncResult { Status = SyncStatus.Uploaded };
        }

        // 2. 比较版本向量
        var relation = VersionVectorComparer.Compare(
            localData.VersionVector,
            cloudMeta.VersionVector);

        switch (relation)
        {
            case VersionVectorComparer.Relation.Equal:
                return new SyncResult { Status = SyncStatus.UpToDate };

            case VersionVectorComparer.Relation.LocalNewer:
                // 本地更新，计算差量上传
                await UploadDeltaAsync(localData, ct);
                _lastSyncedVersion = DeepCopy(localData);
                return new SyncResult { Status = SyncStatus.Uploaded };

            case VersionVectorComparer.Relation.RemoteNewer:
                // 云端更新，下载并应用
                var cloudData = await _cloudApi.DownloadAsync(localData.PlayerId, ct);
                _localSave.Save(cloudData);
                _lastSyncedVersion = DeepCopy(cloudData);
                return new SyncResult { Status = SyncStatus.Downloaded, NewData = cloudData };

            case VersionVectorComparer.Relation.Concurrent:
                // 冲突：需要三路合并
                return await ResolveConcurrentAsync(localData, ct);

            default:
                return new SyncResult { Status = SyncStatus.Error };
        }
    }

    private async UniTask<SyncResult> ResolveConcurrentAsync(
        GameSaveData local, CancellationToken ct)
    {
        var remote = await _cloudApi.DownloadAsync(local.PlayerId, ct);
        var baseVersion = _lastSyncedVersion; // 公共祖先

        // 执行三路合并
        var mergeResult = _merger.Merge(baseVersion, local, remote);

        if (mergeResult.HasConflicts)
        {
            // 通知上层有冲突，等待用户或自动策略解决
            OnConflictsFound?.Invoke(mergeResult.Conflicts.ToArray());
        }

        // 保存合并结果
        _localSave.Save(mergeResult.Data);
        await UploadFullAsync(mergeResult.Data, ct);
        _lastSyncedVersion = DeepCopy(mergeResult.Data);

        return new SyncResult
        {
            Status = SyncStatus.Merged,
            NewData = mergeResult.Data,
            Conflicts = mergeResult.Conflicts.ToArray()
        };
    }

    private async UniTask UploadDeltaAsync(GameSaveData current, CancellationToken ct)
    {
        var delta = _deltaCalc.Calculate(_lastSyncedVersion, current);
        string deltaJson = JsonSerializer.Serialize(delta);

        // 压缩减少流量
        byte[] compressed = GZipCompressor.Compress(
            System.Text.Encoding.UTF8.GetBytes(deltaJson));

        Debug.Log($"[CloudSave] Delta upload: {deltaJson.Length} bytes → {compressed.Length} bytes " +
                  $"({100 - compressed.Length * 100 / deltaJson.Length}% compression)");

        await _cloudApi.UploadDeltaAsync(current.PlayerId, compressed, ct);
    }

    private async UniTask UploadFullAsync(GameSaveData data, CancellationToken ct)
    {
        string json = JsonSerializer.Serialize(data);
        byte[] compressed = GZipCompressor.Compress(
            System.Text.Encoding.UTF8.GetBytes(json));
        await _cloudApi.UploadFullAsync(data.PlayerId, compressed, ct);
    }

    private GameSaveData DeepCopy(GameSaveData data)
    {
        string json = JsonSerializer.Serialize(data);
        return JsonSerializer.Deserialize<GameSaveData>(json);
    }
}

public class SyncResult
{
    public SyncStatus Status { get; set; }
    public GameSaveData NewData { get; set; }
    public MergeConflict[] Conflicts { get; set; }
}

public enum SyncStatus
{
    UpToDate,       // 已是最新，无需操作
    Uploaded,       // 上传成功
    Downloaded,     // 下载更新成功
    Merged,         // 冲突合并成功
    AlreadySyncing, // 正在同步中
    Error           // 同步失败
}
```

## 6. 离线同步与断点续传

### 6.1 离线队列

```csharp
/// <summary>
/// 离线同步队列
/// 网络不可用时，将操作记录到队列，联网后自动重放
/// </summary>
public class OfflineSyncQueue
{
    private const string QUEUE_FILE = "offline_queue.json";
    private readonly string _queuePath;
    private readonly Queue<SyncOperation> _queue = new Queue<SyncOperation>();

    public OfflineSyncQueue(string savePath)
    {
        _queuePath = savePath + QUEUE_FILE;
        LoadQueue();
    }

    public void Enqueue(SyncOperation op)
    {
        _queue.Enqueue(op);
        PersistQueue(); // 持久化到磁盘
    }

    /// <summary>
    /// 联网后重放队列
    /// </summary>
    public async UniTask<bool> ReplayAsync(CloudSaveManager cloudSave, CancellationToken ct)
    {
        while (_queue.Count > 0)
        {
            var op = _queue.Peek();
            try
            {
                await ExecuteOperation(op, cloudSave, ct);
                _queue.Dequeue();
                PersistQueue();
            }
            catch (OperationCanceledException)
            {
                return false; // 网络中断，停止重放
            }
        }
        return true;
    }

    private async UniTask ExecuteOperation(
        SyncOperation op, CloudSaveManager cloudSave, CancellationToken ct)
    {
        switch (op.Type)
        {
            case SyncOpType.FullSync:
                await cloudSave.SyncAsync(ct);
                break;
            // 可扩展其他操作类型
        }
    }

    private void LoadQueue()
    {
        if (!System.IO.File.Exists(_queuePath)) return;
        string json = System.IO.File.ReadAllText(_queuePath);
        var ops = JsonSerializer.Deserialize<List<SyncOperation>>(json);
        foreach (var op in ops) _queue.Enqueue(op);
    }

    private void PersistQueue()
    {
        string json = JsonSerializer.Serialize(_queue.ToList());
        System.IO.File.WriteAllText(_queuePath, json);
    }
}

[Serializable]
public class SyncOperation
{
    public SyncOpType Type { get; set; }
    public long Timestamp { get; set; }
    public string Payload { get; set; }
}

public enum SyncOpType { FullSync, DeltaSync, DeleteSave }
```

## 7. 最佳实践总结

### 7.1 云存档系统设计原则

1. **幂等性**：同一操作执行多次结果相同（上传失败重试不会造成数据重复）
2. **最终一致性**：允许短暂不一致，保证最终收敛到一致状态
3. **防御性保存**：关键操作前先本地备份，再上传云端
4. **乐观并发**：使用版本向量乐观锁，不阻塞玩家操作

### 7.2 同步时机策略

```csharp
public class SmartSyncTrigger
{
    private const float MIN_SYNC_INTERVAL = 30f;    // 最短30秒同步一次
    private const float IDLE_SYNC_DELAY = 5f;       // 空闲5秒后同步
    private float _lastSyncTime;
    private float _lastOperationTime;

    public bool ShouldSync()
    {
        float now = Time.realtimeSinceStartup;

        // 距离上次同步超过30秒，且当前空闲（最近5秒没有操作）
        bool enoughTime = now - _lastSyncTime >= MIN_SYNC_INTERVAL;
        bool isIdle = now - _lastOperationTime >= IDLE_SYNC_DELAY;

        return enoughTime && isIdle;
    }

    // 在关键节点强制同步
    public void RegisterOperation() => _lastOperationTime = Time.realtimeSinceStartup;

    // 建议的触发点：退出游戏、切后台、完成关卡、购买物品
}
```

### 7.3 数据安全策略

| 威胁 | 防护措施 |
|------|---------|
| 本地存档篡改 | AES加密 + SHA256校验和 |
| 中间人攻击 | HTTPS + 服务端签名验证 |
| 回滚攻击 | 服务端版本号递增检查 |
| 货币修改 | 服务端验证交易日志 |
| 多设备作弊 | 异常差量检测 + 人工审核 |

### 7.4 性能优化建议

- **差量上传**可节省 80-95% 流量
- **Gzip 压缩**再节省 60-80% 流量
- **合理的同步频率**（空闲触发，不要定时轮询）
- **版本元数据**与**存档内容**分开请求（先比较版本，需要时再下载内容）
- 使用 **Content Delivery Network（CDN）** 加速存档下载
