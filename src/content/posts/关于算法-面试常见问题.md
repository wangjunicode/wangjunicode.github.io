关于算法-面试常见的算法问题

# 斐波那契数列递归算法优化

## 概述

斐波那契数列，就是1，1，2，3，5.... 从第三个数开始的规律为，下个数为前两个数的和，现在请设计一个算法，计算第n个数的值。

## 基本思路

```csharp
public class Func
{
	public static int Fib(int n)
	{
		if(n == 1 || n == 2)
		{
			return 1;
		}
		return Fib(n-2) + Fib(n-1);
	}
}
```

## 优化分析

该算法的时间复杂为多少，时间复杂度代表算法随数量级变化的一个量级描述。向上面这个，它的时间复杂度就是，最后return递归是两次，n越大，这里就是2的n次幂; 关于空间复杂度，就是算法执行所需的临时空间随着变量n的变化情况，上面这个空间复杂度为O(n)

## 优化方案

```csharp
public class Func
{
	public static int Fib(int n)
	{
		int[] fib = new int[n];
		fib[0] = 1;
		fib[1] = 1;
		for(int i = 2; i < n ; i++)
		{
			fib[i] = fib[i-2] + fib[i-1];
		}
		return fib[n-1];
	}
}
// 这里时间复杂度变成了O(n), 空间复杂度为O(n), 那还可以继续优化


//因为实际只用记住三个数，所以用三个变量代替数组，这里时间复杂度依然O(n), 但空间复杂度为O(1)
public class Func
{
    public static int Fib(int n)
    {
        int first = 1;
        int second = 1;
        int third = first + second;
        for(int i = 3; i <= n; i++)
        {
            third = first + second;
            first = second;
            second = third;
        }
        return third;
    }
}

public class Func
{
    public static int Fib(int n, int first, int second)
    {
        if(n <= 1)
        {
            return first;
        }
        else
        {
            return Fib(n-1, second, first+second);
        }
    }
}

//尾递归法：通过两个变量保存计算值，传递给下一次进行计算，递归的过程中也是根据n值变化逐步重复运算，和循环差不多，时间复杂度和空间复杂度也都一样，但是比循环更简洁，更优雅
```
