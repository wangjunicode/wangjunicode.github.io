# 游戏新手引导系统完全指南：从高亮遮罩到剧情融合

## 一、为什么新手引导至关重要

新手引导（Onboarding Tutorial）是游戏留存率的第一道门槛。数据显示，超过 **40%** 的玩家会在首次进入游戏的 **5 分钟内流失**，而一套流畅、不打断节奏的引导系统可以将次日留存提升 15%~30%。

引导系统面临的核心矛盾：
- **功能完整性** vs **学习曲线平缓**
- **强制引导** vs **玩家自主探索**
- **工程实现复杂度** vs **策划灵活配置**

本文将从零构建一套**数据驱动、可组合、支持断点续导**的工业级新手引导系统。

```
         ▲                    ▲
         │                    │
┌────────────────┐   ┌────────────────┐

│  (JSON/SO)      │   │ (游戏事件适配)  │
└────────────────┘   └────────────────┘
```

**核心原则**：
1. **数据驱动**：所有引导步骤由配置表（ScriptableObject 或 JSON）定义，策划可自由编辑
2. **事件解耦**：引导触发通过游戏事件总线驱动，不侵入业务逻辑
3. **可组合**：每个步骤由多个独立的"呈现模块"组合而成
4. **断点续导**：记录完成进度，重进游戏可续接

## 四、遮罩高亮系统

遮罩系统是引导最核心的视觉元素，实现"高亮指定区域、模糊其余内容"的效果。

### 4.1 Shader：镂空遮罩

```glsl
// TutorialMask.shader
Shader "Tutorial/MaskOverlay"
{
    Properties
    {
        _Color ("Mask Color", Color) = (0, 0, 0, 0.7)
        _HoleRect ("Hole Rect (xMin, yMin, xMax, yMax)", Vector) = (0, 0, 1, 1)
        _Radius ("Corner Radius", Float) = 0.0
        _EdgeSoftness ("Edge Softness", Float) = 2.0
    }

    SubShader
    {
        Tags { "Queue"="Overlay" "IgnoreProjector"="True" "RenderType"="Transparent" }
        Blend SrcAlpha OneMinusSrcAlpha
        ZWrite Off
        Cull Off

        Pass
        {
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag
            #include "UnityCG.cginc"

            struct appdata { float4 vertex : POSITION; float2 uv : TEXCOORD0; };
            struct v2f    { float4 pos : SV_POSITION;  float2 uv : TEXCOORD0; };

            fixed4 _Color;
            float4 _HoleRect;
            float  _Radius;
            float  _EdgeSoftness;

            v2f vert(appdata v)
            {
                v2f o;
                o.pos = UnityObjectToClipPos(v.vertex);
                o.uv  = v.uv;
                return o;
            }

            // 圆角矩形SDF
            float RoundedRectSDF(float2 p, float2 center, float2 halfSize, float r)
            {
                float2 d = abs(p - center) - halfSize + r;
                return length(max(d, 0.0)) + min(max(d.x, d.y), 0.0) - r;
            }

            fixed4 frag(v2f i) : SV_Target
            {
                float2 holeCenter  = (_HoleRect.xy + _HoleRect.zw) * 0.5;
                float2 holeHalfSize = (_HoleRect.zw - _HoleRect.xy) * 0.5;
                
                float sdf = RoundedRectSDF(i.uv, holeCenter, holeHalfSize, _Radius);
                // 负值=孔洞内, 正值=遮罩区
                float alpha = smoothstep(-_EdgeSoftness * 0.001, _EdgeSoftness * 0.001, sdf);
                
                return fixed4(_Color.rgb, _Color.a * alpha);
            }
            ENDCG
        }
    }
}
```

### 4.2 遮罩管理器

```csharp
// TutorialMaskOverlay.cs
using UnityEngine;
using UnityEngine.UI;
using DG.Tweening;

public class TutorialMaskOverlay : MonoBehaviour
{
    [SerializeField] private RawImage maskImage;
    [SerializeField] private Material maskMaterial;
    [SerializeField] private Canvas rootCanvas;

    private static readonly int HoleRectID    = Shader.PropertyToID("_HoleRect");
    private static readonly int MaskColorID   = Shader.PropertyToID("_Color");
    private static readonly int RadiusID      = Shader.PropertyToID("_Radius");

    private Material _matInstance;

    private void Awake()
    {
        _matInstance = Instantiate(maskMaterial);
        maskImage.material = _matInstance;
    }

    /// <summary>高亮某个UI RectTransform区域</summary>
    public void HighlightRect(RectTransform target, Vector4 padding,
                               float cornerRadius = 8f, float fadeTime = 0.3f)
    {
        var holeRect = GetNormalizedRect(target, padding);
        
        // 淡入遮罩
        _matInstance.SetColor(MaskColorID, new Color(0, 0, 0, 0));
        DOTween.To(
            () => _matInstance.GetColor(MaskColorID).a,
            a => _matInstance.SetColor(MaskColorID, new Color(0, 0, 0, a)),
            0.7f, fadeTime
        );
        
        _matInstance.SetVector(HoleRectID, holeRect);
        _matInstance.SetFloat(RadiusID, cornerRadius / GetCanvasScale());
        
        gameObject.SetActive(true);
    }

    /// <summary>平滑移动高亮到新目标</summary>
    public void MoveHighlight(RectTransform newTarget, Vector4 padding,
                               float duration = 0.4f)
    {
        var currentRect = _matInstance.GetVector(HoleRectID);
        var targetRect  = GetNormalizedRect(newTarget, padding);

        DOTween.To(
            () => (Vector4)_matInstance.GetVector(HoleRectID),
            v => _matInstance.SetVector(HoleRectID, v),
            targetRect, duration
        ).SetEase(Ease.InOutQuart);
    }

    public void Hide(float fadeTime = 0.25f)
    {
        DOTween.To(
            () => _matInstance.GetColor(MaskColorID).a,
            a => _matInstance.SetColor(MaskColorID, new Color(0, 0, 0, a)),
            0f, fadeTime
        ).OnComplete(() => gameObject.SetActive(false));
    }

    private Vector4 GetNormalizedRect(RectTransform target, Vector4 pad)
    {
        var corners = new Vector3[4];
        target.GetWorldCorners(corners);
        
        float canvasW = rootCanvas.pixelRect.width;
        float canvasH = rootCanvas.pixelRect.height;
        
        float xMin = (corners[0].x - pad.x) / canvasW;
        float yMin = (corners[0].y - pad.z) / canvasH;
        float xMax = (corners[2].x + pad.y) / canvasW;
        float yMax = (corners[2].y + pad.w) / canvasH;
        
        return new Vector4(xMin, yMin, xMax, yMax);
    }

    private float GetCanvasScale()
    {
        if (rootCanvas.renderMode == RenderMode.ScreenSpaceOverlay)
            return 1f;
        return rootCanvas.scaleFactor;
    }
}
```

## 五、对话气泡系统

### 5.1 气泡控制器

```csharp
// TutorialDialogBubble.cs
using UnityEngine;
using UnityEngine.UI;
using TMPro;
using DG.Tweening;
using System.Collections;

public class TutorialDialogBubble : MonoBehaviour
{
    [Header("UI引用")]
    [SerializeField] private RectTransform bubbleRoot;
    [SerializeField] private TextMeshProUGUI dialogText;
    [SerializeField] private TextMeshProUGUI speakerNameText;
    [SerializeField] private Image speakerAvatar;
    [SerializeField] private Button confirmButton;
    [SerializeField] private TextMeshProUGUI confirmButtonText;

    [Header("打字机效果")]
    [SerializeField] private float typewriterSpeed = 30f; // 字/秒
    [SerializeField] private bool enableTypewriter = true;

    private Coroutine _typewriterCoroutine;
    private System.Action _onConfirm;

    private void Awake()
    {
        confirmButton.onClick.AddListener(OnConfirmClicked);
    }

    public void Show(TutorialStepConfig config, RectTransform anchor,
                     System.Action onConfirm = null)
    {
        _onConfirm = onConfirm;
        
        // 设置说话者信息
        speakerNameText.text  = config.speakerName;
        speakerAvatar.sprite  = config.speakerAvatar;
        speakerAvatar.gameObject.SetActive(config.speakerAvatar != null);
        
        // 定位气泡
        PositionBubble(anchor, config.dialogPosition);
        
        // 显示动画
        bubbleRoot.localScale = Vector3.zero;
        gameObject.SetActive(true);
        bubbleRoot.DOScale(Vector3.one, 0.25f).SetEase(Ease.OutBack);
        
        // 打字机效果
        if (enableTypewriter)
        {
            if (_typewriterCoroutine != null) StopCoroutine(_typewriterCoroutine);
            _typewriterCoroutine = StartCoroutine(TypewriterEffect(config.dialogText));
        }
        else
        {
            dialogText.text = config.dialogText;
        }
    }

    private IEnumerator TypewriterEffect(string fullText)
    {
        dialogText.text = "";
        float interval = 1f / typewriterSpeed;
        
        foreach (char c in fullText)
        {
            dialogText.text += c;
            
            // 标点符号额外停顿
            if (c == '。' || c == '！' || c == '？' || c == ',' || c == '，')
                yield return new WaitForSeconds(interval * 4f);
            else
                yield return new WaitForSeconds(interval);
        }
    }

    public void Hide()
    {
        if (_typewriterCoroutine != null)
        {
            StopCoroutine(_typewriterCoroutine);
            _typewriterCoroutine = null;
        }
        bubbleRoot.DOScale(Vector3.zero, 0.2f).SetEase(Ease.InBack)
            .OnComplete(() => gameObject.SetActive(false));
    }

    private void OnConfirmClicked()
    {
        // 如果打字机未完成，先完成文本
        if (_typewriterCoroutine != null)
        {
            StopCoroutine(_typewriterCoroutine);
            _typewriterCoroutine = null;
            // dialogText已经在协程中逐步填充，直接设置完整文本
            _onConfirm = null; // 第一次点击只是快进文字
            return;
        }
        _onConfirm?.Invoke();
    }

    private void PositionBubble(RectTransform anchor, DialogPosition position)
    {
        if (anchor == null) return;
        
        var corners = new Vector3[4];
        anchor.GetWorldCorners(corners);
        
        Vector2 worldCenter = (corners[0] + corners[2]) / 2f;
        
        RectTransformUtility.ScreenPointToLocalPointInRectangle(
            bubbleRoot.parent as RectTransform,
            worldCenter,
            null,
            out var localPoint
        );
        
        var rect = bubbleRoot;
        var offset = position switch
        {
            DialogPosition.Top    => new Vector2(0,  150f),
            DialogPosition.Bottom => new Vector2(0, -150f),
            DialogPosition.Left   => new Vector2(-200f, 0),
            DialogPosition.Right  => new Vector2( 200f, 0),
            _                     => Vector2.zero
        };
        
        rect.anchoredPosition = localPoint + offset;
    }
}
```

```csharp
// TutorialGestureArrow.cs
using UnityEngine;
using UnityEngine.UI;
using DG.Tweening;

public class TutorialGestureArrow : MonoBehaviour
{
    [SerializeField] private RectTransform handIcon;
    [SerializeField] private RectTransform arrowIcon;
    [SerializeField] private TrailRenderer tapRipple;

    private Sequence _gestureSequence;

    public void ShowTap(RectTransform target)
    {
        StopGesture();
        
        PositionAt(target);
        gameObject.SetActive(true);
        
        // 循环点按动画
        _gestureSequence = DOTween.Sequence();
        _gestureSequence
            .Append(handIcon.DOScale(0.85f, 0.12f).SetEase(Ease.InQuad))
            .Append(handIcon.DOScale(1f,    0.12f).SetEase(Ease.OutQuad))
            .AppendInterval(0.5f)
            .SetLoops(-1, LoopType.Restart);
    }

    public void ShowSwipe(RectTransform from, RectTransform to, float duration = 0.8f)
    {
        StopGesture();
        
        var fromPos = GetLocalPos(from);
        var toPos   = GetLocalPos(to);
        
        handIcon.anchoredPosition = fromPos;
        gameObject.SetActive(true);
        
        _gestureSequence = DOTween.Sequence();
        _gestureSequence
            .Append(handIcon.DOAnchorPos(toPos, duration).SetEase(Ease.InOutSine))
            .AppendInterval(0.3f)
            .Append(handIcon.DOAnchorPos(fromPos, 0f))
            .AppendInterval(0.2f)
            .SetLoops(-1, LoopType.Restart);
    }

    public void ShowDragTo(RectTransform from, RectTransform to)
    {
        StopGesture();
        
        var fromPos = GetLocalPos(from);
        var toPos   = GetLocalPos(to);
        
        handIcon.anchoredPosition = fromPos;
        gameObject.SetActive(true);
        
        _gestureSequence = DOTween.Sequence();
        _gestureSequence
            .Append(handIcon.DOScale(0.9f, 0.1f))
            .Append(handIcon.DOAnchorPos(toPos, 0.7f).SetEase(Ease.InOutCubic))
            .Append(handIcon.DOScale(1f, 0.1f))
            .AppendInterval(0.4f)
            .Append(handIcon.DOAnchorPos(fromPos, 0f))
            .AppendInterval(0.2f)
            .SetLoops(-1, LoopType.Restart);
    }

    public void StopGesture()
    {
        _gestureSequence?.Kill();
        gameObject.SetActive(false);
    }

    private void PositionAt(RectTransform target)
    {
        RectTransformUtility.ScreenPointToLocalPointInRectangle(
            handIcon.parent as RectTransform,
            GetWorldCenter(target), null, out var lp);
        handIcon.anchoredPosition = lp;
    }

    private Vector2 GetLocalPos(RectTransform target)
    {
        RectTransformUtility.ScreenPointToLocalPointInRectangle(
            handIcon.parent as RectTransform,
            GetWorldCenter(target), null, out var lp);
        return lp;
    }

    private Vector2 GetWorldCenter(RectTransform rt)
    {
        var corners = new Vector3[4];
        rt.GetWorldCorners(corners);
        return (corners[0] + corners[2]) / 2f;
    }
}
```

## 七、核心状态机 TutorialManager

```csharp
// TutorialManager.cs
using System;
using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using Cysharp.Threading.Tasks;

public class TutorialManager : MonoBehaviour
{
    public static TutorialManager Instance { get; private set; }

    [Header("子系统引用")]
    [SerializeField] private TutorialMaskOverlay maskOverlay;
    [SerializeField] private TutorialDialogBubble dialogBubble;
    [SerializeField] private TutorialGestureArrow gestureArrow;
    [SerializeField] private GameObject skipButton;

    [Header("配置")]
    [SerializeField] private TutorialFlowConfig defaultFlow;

    // 状态
    private TutorialFlowConfig _currentFlow;
    private int _currentStepIndex = -1;
    private bool _isRunning = false;
    private bool _waitingForComplete = false;

    // 存档Key
    private const string SaveKeyPrefix = "Tutorial_";

    private void Awake()
    {
        if (Instance != null) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
    }

    private void Start()
    {
        // 自动检测是否需要播放默认引导
        if (defaultFlow != null && !IsFlowCompleted(defaultFlow.flowId))
            StartFlow(defaultFlow).Forget();
    }

    public async UniTaskVoid StartFlow(TutorialFlowConfig flow)
    {
        if (_isRunning)
        {
            Debug.LogWarning("[Tutorial] 已有引导进行中");
            return;
        }

        _currentFlow = flow;
        _isRunning   = true;

        // 从存档恢复断点
        int startStep = LoadProgress(flow.flowId);
        Debug.Log($"[Tutorial] 开始引导流程: {flow.flowId}, 从步骤 {startStep} 开始");

        for (int i = startStep; i < flow.steps.Count; i++)
        {
            _currentStepIndex = i;
            SaveProgress(flow.flowId, i);

            await PlayStep(flow.steps[i]);
            
            // 每步完成后等一帧，防止事件穿透
            await UniTask.Yield(PlayerLoopTiming.LastPostLateUpdate);
        }

        CompleteFlow(flow);
    }

    private async UniTask PlayStep(TutorialStepConfig step)
    {
        Debug.Log($"[Tutorial] 播放步骤: {step.stepId}");

        // 寻找目标UI
        var highlightTarget = FindUITarget(step.highlightTargetPath);
        var gestureTarget   = FindUITarget(step.gestureTargetPath);

        // 显示遮罩
        if (step.useMask && highlightTarget != null)
            maskOverlay.HighlightRect(highlightTarget, step.highlightPadding);

        // 显示气泡（等待确认）
        if (step.useDialog)
        {
            bool confirmed = false;
            dialogBubble.Show(step, highlightTarget, () => confirmed = true);
            await UniTask.WaitUntil(() => confirmed);
            dialogBubble.Hide();
        }

        // 显示手势
        if (step.useGesture && gestureTarget != null)
        {
            switch (step.gestureType)
            {
                case GestureType.Tap:     gestureArrow.ShowTap(gestureTarget);    break;
                case GestureType.DragTo:  gestureArrow.ShowDragTo(highlightTarget, gestureTarget); break;
            }
        }

        // 等待完成条件
        _waitingForComplete = true;
        await WaitForComplete(step);
        _waitingForComplete = false;

        // 清理手势
        gestureArrow.StopGesture();
    }

    private async UniTask WaitForComplete(TutorialStepConfig step)
    {
        switch (step.completeType)
        {
            case CompleteType.AutoDelay:
                if (step.autoCompleteDelay > 0)
                    await UniTask.Delay(TimeSpan.FromSeconds(step.autoCompleteDelay));
                break;

            case CompleteType.OnButtonClick:
            {
                var target = FindUITarget(step.completeParam);
                if (target == null) break;
                bool clicked = false;
                var btn = target.GetComponent<UnityEngine.UI.Button>();
                btn?.onClick.AddListener(() => clicked = true);
                await UniTask.WaitUntil(() => clicked);
                break;
            }

            case CompleteType.OnEvent:
            {
                bool fired = false;
                GameEventBus.Subscribe(step.completeParam, _ => fired = true);
                await UniTask.WaitUntil(() => fired);
                GameEventBus.Unsubscribe(step.completeParam, _ => fired = true);
                break;
            }
        }

        // 隐藏遮罩
        maskOverlay.Hide();
    }

    private void CompleteFlow(TutorialFlowConfig flow)
    {
        _isRunning = false;
        _currentStepIndex = -1;
        PlayerPrefs.SetInt(SaveKeyPrefix + flow.flowId + "_done", 1);
        PlayerPrefs.Save();
        Debug.Log($"[Tutorial] 引导流程完成: {flow.flowId}");
        GameEventBus.Publish("Tutorial.FlowComplete", flow.flowId);
    }

    public bool IsFlowCompleted(string flowId)
        => PlayerPrefs.GetInt(SaveKeyPrefix + flowId + "_done", 0) == 1;

    public void SkipCurrentFlow()
    {
        if (!_isRunning || _currentFlow == null) return;
        if (!_currentFlow.canSkipAll) return;
        
        StopAllCoroutines();
        maskOverlay.Hide();
        dialogBubble.Hide();
        gestureArrow.StopGesture();
        CompleteFlow(_currentFlow);
    }

    private void SaveProgress(string flowId, int stepIndex)
        => PlayerPrefs.SetInt(SaveKeyPrefix + flowId + "_progress", stepIndex);

    private int LoadProgress(string flowId)
        => PlayerPrefs.GetInt(SaveKeyPrefix + flowId + "_progress", 0);

    private RectTransform FindUITarget(string path)
    {
        if (string.IsNullOrEmpty(path)) return null;
        var go = GameObject.Find(path);
        return go?.GetComponent<RectTransform>();
    }
}
```

## 八、游戏事件总线（解耦适配层）

```csharp
// GameEventBus.cs - 简化事件总线，供引导系统使用
using System;
using System.Collections.Generic;

public static class GameEventBus
{
    private static readonly Dictionary<string, List<Action<object>>> _listeners = new();

    public static void Subscribe(string eventName, Action<object> handler)
    {
        if (!_listeners.TryGetValue(eventName, out var list))
            _listeners[eventName] = list = new List<Action<object>>();
        list.Add(handler);
    }

    public static void Unsubscribe(string eventName, Action<object> handler)
    {
        if (_listeners.TryGetValue(eventName, out var list))
            list.Remove(handler);
    }

    public static void Publish(string eventName, object data = null)
    {
        if (!_listeners.TryGetValue(eventName, out var list)) return;
        // 拷贝避免迭代中修改
        var snapshot = new List<Action<object>>(list);
        foreach (var h in snapshot) h?.Invoke(data);
    }
}
```

## 九、引导与剧情融合：CutsceneGuide

当引导步骤需要与Timeline/剧情动画融合时，可通过自定义 `PlayableAsset` 在 Timeline 中嵌入引导触发信号：

```csharp
// TutorialSignalReceiver.cs
using UnityEngine;
using UnityEngine.Timeline;
using UnityEngine.Playables;

[RequireComponent(typeof(SignalReceiver))]
public class TutorialSignalReceiver : MonoBehaviour
{
    [SerializeField] private TutorialFlowConfig tutorialFlow;
    [SerializeField] private SignalAsset triggerSignal;

    private void OnEnable()
    {
        var receiver = GetComponent<SignalReceiver>();
        receiver.AddReaction(triggerSignal, new UnityEngine.Events.UnityEvent());
    }

    /// <summary>由 SignalReceiver 的 UnityEvent 调用</summary>
    public void OnTimelineTrigger()
    {
        if (tutorialFlow != null)
            TutorialManager.Instance.StartFlow(tutorialFlow).Forget();
    }
}
```

在 Timeline 中操作：
1. 在 Signal Track 上右键 → Add Signal Emitter
2. 指定 `TutorialSignalReceiver` 接收
3. 引导会在 Timeline 播放到该帧时自动触发并暂停时间线

## 十、多分辨率适配

```csharp
// TutorialResolutionAdapter.cs
using UnityEngine;

/// <summary>
/// 引导中所有坐标计算需通过此适配器，
/// 保证在不同分辨率/刘海屏/平板下正常显示
/// </summary>
public class TutorialResolutionAdapter : MonoBehaviour
{
    [SerializeField] private CanvasScaler scaler;

    /// <summary>将屏幕坐标转换为安全区坐标</summary>
    public static Rect GetSafeAreaInCanvas(Canvas canvas)
    {
        var safeArea = Screen.safeArea;
        var pixelRect = canvas.pixelRect;
        
        float sx = safeArea.x      / pixelRect.width;
        float sy = safeArea.y      / pixelRect.height;
        float sw = safeArea.width  / pixelRect.width;
        float sh = safeArea.height / pixelRect.height;
        
        return new Rect(sx, sy, sw, sh);
    }

    /// <summary>检测高亮目标是否在安全区内，否则偏移</summary>
    public static Vector2 ClampToSafeArea(Vector2 pos, Vector2 size, Canvas canvas)
    {
        var safe = GetSafeAreaInCanvas(canvas);
        float halfW = size.x * 0.5f;
        float halfH = size.y * 0.5f;
        
        float x = Mathf.Clamp(pos.x, safe.xMin + halfW, safe.xMax - halfW);
        float y = Mathf.Clamp(pos.y, safe.yMin + halfH, safe.yMax - halfH);
        return new Vector2(x, y);
    }
}
```

## 十一、性能注意事项

| 问题 | 解决方案 |
|------|---------|
| 遮罩每帧重建 | 使用 Material PropertyBlock，避免每帧 Instantiate |
| 打字机协程GC | 改用 StringBuilder 复用，避免字符串 `+=` |
| UI路径查找慢 | 启动时预注册 UI 组件到字典，避免 `GameObject.Find` |
| 步骤切换跳帧 | 每步切换后 `await UniTask.Yield(LastPostLateUpdate)` |
| 引导与物理穿透 | 步骤进行中屏蔽非目标区域的 `IPointerClickHandler` |

## 十二、最佳实践总结

1. **永远数据驱动**：步骤配置在 ScriptableObject 或 JSON 中，策划可热更修改，不需要重新打包
2. **断点续导必须实现**：游戏崩溃、切后台、强退是高频场景，引导需持久化当前步骤
3. **事件总线解耦**：引导系统不直接依赖游戏业务代码，通过事件驱动完成条件判断
4. **高亮使用 SDF Shader**：圆角矩形 SDF 比 Sprite 高亮更灵活，任意尺寸零失真
5. **跳过机制必须设计**：重复游玩玩家、测试人员都需要跳过，且应记录完成状态
6. **剧情融合使用 Timeline Signal**：避免引导代码与动画系统强耦合
7. **手势动画循环**：循环频率控制在 1~2 次/秒，过快会让玩家焦虑
8. **安全区适配**：刘海屏/异形屏下气泡和手势不能被系统UI遮挡
9. **A/B 测试埋点**：记录每一步的完成率、耗时，持续优化引导转化
10. **本地化友好**：所有文本通过本地化 Key 引用，禁止硬编码中文字符串
