# 游戏分包与按需下载系统：大型游戏资源管理方案

随着游戏内容不断膨胀，包体大小对下载转化率的影响越来越显著。**首包越小，玩家越愿意下载；内容越多，玩家留存越高**——这两个目标看似矛盾，而分包与按需下载就是解决这一矛盾的核心技术。

## 一、包体优化策略

### 1.1 包体分析

优化之前，先了解包体组成：

```bash
# Unity 构建报告分析
```

典型包体构成：
```
总包体（100%）
├── 代码 (IL2CPP)         ~15-25%
├── 纹理资源              ~40-60%
├── 音频资源              ~10-20%
├── 模型/动画             ~10-20%
└── 其他（字体/着色器等） ~5-10%
```

### 1.2 纹理压缩优化

```csharp
// 通过 AssetPostprocessor 自动设置纹理压缩格式
public class TextureImportOptimizer : AssetPostprocessor
{
    void OnPreprocessTexture()
    {
        var importer = assetImporter as TextureImporter;
        if (importer == null) return;
        
        // UI 纹理
        if (assetPath.Contains("/UI/"))
        {
            SetPlatformSettings(importer, "Android", TextureImporterFormat.ETC2_RGBA8);
            SetPlatformSettings(importer, "iPhone", TextureImporterFormat.ASTC_6x6);
            SetPlatformSettings(importer, "Standalone", TextureImporterFormat.BC7);
        }
        // 场景纹理
        else if (assetPath.Contains("/Environment/"))
        {
            SetPlatformSettings(importer, "Android", TextureImporterFormat.ETC2_RGB4);
            SetPlatformSettings(importer, "iPhone", TextureImporterFormat.ASTC_8x8);
        }
    }
    
    void SetPlatformSettings(TextureImporter importer, string platform, TextureImporterFormat format)
    {
        var settings = importer.GetPlatformTextureSettings(platform);
        settings.overridden = true;
        settings.format = format;
        settings.compressionQuality = 100;
        importer.SetPlatformTextureSettings(settings);
    }
}
```

### 1.3 音频压缩策略

```csharp
public class AudioImportOptimizer : AssetPostprocessor
{
    void OnPreprocessAudio()
    {
        var importer = assetImporter as AudioImporter;
        if (importer == null) return;
        
        var defaultSampleSettings = importer.defaultSampleSettings;
        
        // 背景音乐：流式加载，不占内存
        if (assetPath.Contains("/BGM/"))
        {
            defaultSampleSettings.loadType = AudioClipLoadType.Streaming;
            defaultSampleSettings.compressionFormat = AudioCompressionFormat.Vorbis;
            defaultSampleSettings.quality = 0.5f; // 50% 质量
        }
        // 短音效：解压后缓存，低延迟
        else if (assetPath.Contains("/SFX/Short/"))
        {
            defaultSampleSettings.loadType = AudioClipLoadType.DecompressOnLoad;
            defaultSampleSettings.compressionFormat = AudioCompressionFormat.PCM;
        }
        // 长音效：压缩内存，按需解压
        else
        {
            defaultSampleSettings.loadType = AudioClipLoadType.CompressedInMemory;
            defaultSampleSettings.compressionFormat = AudioCompressionFormat.Vorbis;
            defaultSampleSettings.quality = 0.7f;
        }
        
        importer.defaultSampleSettings = defaultSampleSettings;
    }
}
```

## 二、Addressables 分包架构

### 2.1 Group 分组策略

```
Addressables Groups
├── BuiltIn（随包内置，不可下载）
│   ├── Core-Gameplay    # 核心玩法必需资源
```

### 2.3 远程 Catalog 更新检查

```csharp
public class AddressablesUpdater : MonoSingleton<AddressablesUpdater>
{
    public async Task<bool> CheckForUpdates()
    {
        // 检查是否有新的 catalog
        var checkHandle = Addressables.CheckForCatalogUpdates(false);
        await checkHandle.Task;
        
        if (checkHandle.Status != AsyncOperationStatus.Succeeded)
        {
            Debug.LogError("[Updater] 检查更新失败");
            Addressables.Release(checkHandle);
            return false;
        }
        
        var catalogs = checkHandle.Result;
        Addressables.Release(checkHandle);
        
        if (catalogs == null || catalogs.Count == 0)
        {
            Debug.Log("[Updater] 无需更新");
            return false;
        }
        
        // 更新 catalog
        var updateHandle = Addressables.UpdateCatalogs(catalogs, false);
        await updateHandle.Task;
        
        bool success = updateHandle.Status == AsyncOperationStatus.Succeeded;
        Addressables.Release(updateHandle);
        
        return success;
    }
    
    // 计算需要下载的大小
    public async Task<long> GetDownloadSize(IEnumerable<string> keys)
    {
        long totalSize = 0;
        foreach (var key in keys)
        {
            var sizeHandle = Addressables.GetDownloadSizeAsync(key);
            await sizeHandle.Task;
            if (sizeHandle.Status == AsyncOperationStatus.Succeeded)
                totalSize += sizeHandle.Result;
            Addressables.Release(sizeHandle);
        }
        return totalSize;
    }
    
    // 下载指定 Label 的所有资源
    public async Task DownloadDependencies(string label, 
        Action<float> onProgress = null,
        Action<string> onError = null)
    {
        var downloadHandle = Addressables.DownloadDependenciesAsync(label, false);
        
        while (!downloadHandle.IsDone)
        {
            onProgress?.Invoke(downloadHandle.PercentComplete);
            await Task.Yield();
        }
        
        if (downloadHandle.Status != AsyncOperationStatus.Succeeded)
            onError?.Invoke(downloadHandle.OperationException?.Message ?? "下载失败");
        
        Addressables.Release(downloadHandle);
    }
}
```

## 三、自定义下载管理器

Addressables 的下载功能有时无法满足复杂需求，需要自定义下载器：

### 3.1 下载任务管理

```csharp
public class DownloadTask
{
    public string Url { get; set; }
    public string SavePath { get; set; }
    public long TotalBytes { get; set; }
    public long DownloadedBytes { get; set; }
    public DownloadStatus Status { get; set; }
    public int RetryCount { get; set; }
    public float Progress => TotalBytes > 0 ? (float)DownloadedBytes / TotalBytes : 0;
    
    // 校验信息
    public string Md5 { get; set; }
    public int Version { get; set; }
}

public enum DownloadStatus { Pending, Downloading, Paused, Completed, Failed }

public class DownloadManager : MonoSingleton<DownloadManager>
{
    private const int MAX_CONCURRENT = 3;     // 最大并发数
    private const int MAX_RETRY = 3;          // 最大重试次数
    private const int TIMEOUT_SECONDS = 30;   // 超时时间
    
    private Queue<DownloadTask> _pendingQueue = new();
    private List<DownloadTask> _activeDownloads = new();
    private Dictionary<string, DownloadTask> _allTasks = new();
    
    // 全局进度
    public float TotalProgress { get; private set; }
    public long TotalBytes { get; private set; }
    public long DownloadedBytes { get; private set; }
    
    public event Action<DownloadTask> OnTaskCompleted;
    public event Action<DownloadTask> OnTaskFailed;
    public event Action OnAllCompleted;
    
    public void AddTask(string url, string savePath, string md5 = null)
    {
        if (_allTasks.ContainsKey(url)) return;
        
        var task = new DownloadTask { Url = url, SavePath = savePath, Md5 = md5 };
        _allTasks[url] = task;
        _pendingQueue.Enqueue(task);
        
        ProcessQueue();
    }
    
    void ProcessQueue()
    {
        while (_activeDownloads.Count < MAX_CONCURRENT && _pendingQueue.Count > 0)
        {
            var task = _pendingQueue.Dequeue();
            _activeDownloads.Add(task);
            StartCoroutine(DownloadCoroutine(task));
        }
    }
    
    IEnumerator DownloadCoroutine(DownloadTask task)
    {
        task.Status = DownloadStatus.Downloading;
        
        // 检查断点续传
        long existingSize = 0;
        if (System.IO.File.Exists(task.SavePath + ".tmp"))
            existingSize = new System.IO.FileInfo(task.SavePath + ".tmp").Length;
        
        using var request = UnityWebRequest.Get(task.Url);
        
        // 设置 Range 头（断点续传）
        if (existingSize > 0)
            request.SetRequestHeader("Range", $"bytes={existingSize}-");
        
        request.timeout = TIMEOUT_SECONDS;
        request.downloadHandler = new DownloadHandlerFile(task.SavePath + ".tmp")
        {
            removeFileOnAbort = false // 保留文件用于断点续传
        };
        
        request.SendWebRequest();
        
        while (!request.isDone)
        {
            task.DownloadedBytes = existingSize + (long)(request.downloadedBytes);
            UpdateTotalProgress();
            yield return null;
        }
        
        if (request.result != UnityWebRequest.Result.Success)
        {
            yield return HandleDownloadError(task);
            yield break;
        }
        
        // 校验 MD5
        if (!string.IsNullOrEmpty(task.Md5))
        {
            string actualMd5 = CalculateMd5(task.SavePath + ".tmp");
            if (actualMd5 != task.Md5)
            {
                Debug.LogError($"[Download] MD5校验失败: {task.Url}");
                System.IO.File.Delete(task.SavePath + ".tmp");
                yield return HandleDownloadError(task);
                yield break;
            }
        }
        
        // 重命名临时文件
        if (System.IO.File.Exists(task.SavePath))
            System.IO.File.Delete(task.SavePath);
        System.IO.File.Move(task.SavePath + ".tmp", task.SavePath);
        
        task.Status = DownloadStatus.Completed;
        _activeDownloads.Remove(task);
        OnTaskCompleted?.Invoke(task);
        
        Debug.Log($"[Download] 完成: {System.IO.Path.GetFileName(task.SavePath)}");
        
        ProcessQueue();
        
        if (_pendingQueue.Count == 0 && _activeDownloads.Count == 0)
            OnAllCompleted?.Invoke();
    }
    
    IEnumerator HandleDownloadError(DownloadTask task)
    {
        task.RetryCount++;
        if (task.RetryCount <= MAX_RETRY)
        {
            Debug.LogWarning($"[Download] 重试 {task.RetryCount}/{MAX_RETRY}: {task.Url}");
            yield return new WaitForSeconds(2f * task.RetryCount); // 指数退避
            _pendingQueue.Enqueue(task);
        }
        else
        {
            task.Status = DownloadStatus.Failed;
            _activeDownloads.Remove(task);
            OnTaskFailed?.Invoke(task);
            Debug.LogError($"[Download] 最终失败: {task.Url}");
        }
        
        ProcessQueue();
    }
    
    void UpdateTotalProgress()
    {
        DownloadedBytes = _allTasks.Values.Sum(t => t.DownloadedBytes);
        TotalBytes = _allTasks.Values.Sum(t => t.TotalBytes);
        TotalProgress = TotalBytes > 0 ? (float)DownloadedBytes / TotalBytes : 0;
    }
    
    string CalculateMd5(string filePath)
    {
        using var md5 = System.Security.Cryptography.MD5.Create();
        using var stream = System.IO.File.OpenRead(filePath);
        byte[] hash = md5.ComputeHash(stream);
        return System.BitConverter.ToString(hash).Replace("-", "").ToLower();
    }
    
    // 暂停/恢复
    public void PauseAll()
    {
        foreach (var task in _activeDownloads)
            task.Status = DownloadStatus.Paused;
        StopAllCoroutines();
    }
    
    public void ResumeAll()
    {
        var paused = _activeDownloads.ToList();
        _activeDownloads.Clear();
        foreach (var task in paused)
        {
            task.Status = DownloadStatus.Pending;
            _pendingQueue.Enqueue(task);
        }
        ProcessQueue();
    }
}
```

## 四、差量更新系统

### 4.1 资源版本清单

```csharp
[Serializable]
public class AssetManifest
{
    public int version;
    public string buildTime;
    public List<AssetEntry> entries = new();
    
    [Serializable]
    public class AssetEntry
    {
        public string path;     // 相对路径
        public string md5;      // 文件 MD5
        public long size;       // 文件大小
        public int version;     // 资源版本号
    }
}

public class DiffPatchSystem
{
    // 比较本地与远程清单，找出需要更新的文件
    public static List<AssetManifest.AssetEntry> GetDiffEntries(
        AssetManifest local, AssetManifest remote)
    {
        var localDict = local.entries.ToDictionary(e => e.path, e => e);
        var needUpdate = new List<AssetManifest.AssetEntry>();
        
        foreach (var remoteEntry in remote.entries)
        {
            // 新增的文件
            if (!localDict.TryGetValue(remoteEntry.path, out var localEntry))
            {
                needUpdate.Add(remoteEntry);
                continue;
            }
            
            // 有变化的文件（MD5 不同）
            if (localEntry.md5 != remoteEntry.md5)
                needUpdate.Add(remoteEntry);
        }
        
        return needUpdate;
    }
    
    // 找出本地多余的文件（远程已删除）
    public static List<string> GetObsoleteFiles(AssetManifest local, AssetManifest remote)
    {
        var remotePaths = new HashSet<string>(remote.entries.Select(e => e.path));
        return local.entries
            .Where(e => !remotePaths.Contains(e.path))
            .Select(e => e.path)
            .ToList();
    }
}
```

## 五、下载 UI 体验设计

### 5.1 首次进入下载流程

```csharp
public class FirstLaunchDownloader : MonoBehaviour
{
    [SerializeField] private Slider progressBar;
    [SerializeField] private TMP_Text statusText;
    [SerializeField] private TMP_Text sizeText;
    [SerializeField] private Button wifiOnlyToggle;
    [SerializeField] private Button skipButton;
    
    private bool _wifiOnly = true;
    
    async void Start()
    {
        // 检查网络状态
        if (!IsNetworkAvailable())
        {
            ShowError("无网络连接，请检查网络设置");
            return;
        }
        
        // 检查 WiFi
        if (_wifiOnly && Application.internetReachability == NetworkReachability.ReachableViaCarrierDataNetwork)
        {
            ShowWifiWarning();
            return;
        }
        
        await StartInitialDownload();
    }
    
    async Task StartInitialDownload()
    {
        statusText.text = "正在检查更新...";
        
        // 1. 检查必要资源大小
        long size = await AddressablesUpdater.Instance.GetDownloadSize(new[] { "essential" });
        
        if (size == 0)
        {
            // 无需下载，直接进入
            EnterGame();
            return;
        }
        
        sizeText.text = $"需要下载 {FormatBytes(size)}";
        statusText.text = "下载核心资源...";
        
        // 2. 下载
        await AddressablesUpdater.Instance.DownloadDependencies(
            "essential",
            progress =>
            {
                progressBar.value = progress;
                statusText.text = $"下载中... {progress:P0}";
            },
            error => ShowError($"下载失败: {error}")
        );
        
        // 3. 进入游戏
        EnterGame();
    }
    
    void ShowWifiWarning()
    {
        // 显示流量警告弹窗
        UIManager.Instance.ShowDialog(
            "需要使用移动数据",
            $"首次运行需要下载约 {sizeText.text}，将使用移动数据。",
            "继续下载", () => { _wifiOnly = false; Start(); },
            "等待WiFi", () => Application.Quit()
        );
    }
    
    string FormatBytes(long bytes)
    {
        if (bytes < 1024) return $"{bytes} B";
        if (bytes < 1024 * 1024) return $"{bytes / 1024.0:F1} KB";
        if (bytes < 1024 * 1024 * 1024) return $"{bytes / 1024.0 / 1024.0:F1} MB";
        return $"{bytes / 1024.0 / 1024.0 / 1024.0:F1} GB";
    }
    
    bool IsNetworkAvailable() => 
        Application.internetReachability != NetworkReachability.NotReachable;
    
    void EnterGame() => SceneManager.LoadScene("MainMenu");
    
    void ShowError(string msg) => statusText.text = $"错误: {msg}";
}
```

## 六、工程最佳实践

### 6.1 包体瘦身 Checklist

```
□ 纹理：按平台设置正确压缩格式（ASTC/ETC2/BC7）
□ 纹理：开启 Generate Mip Maps 仅对3D场景纹理
□ 音频：BGM 使用 Streaming，短音效 DecompressOnLoad
□ 音频：移除未使用的音频通道（单声道 vs 立体声）
□ 模型：移除冗余骨骼，LOD 组合理配置
□ 代码：IL2CPP + 代码裁剪（Medium/High Strip Level）
□ 着色器：打包前变体收集，移除未使用变体
□ 场景：不同场景的资源不要重复引用
□ 图集：UI 图集合理分类，避免超大图集
□ 字体：只包含用到的字符子集（Font Subsetting）
```

### 6.2 CDN 配置建议

```
CDN 目录结构：
```

## 总结

| 阶段 | 核心策略 |
|------|---------|
| 首包瘦身 | 纹理压缩 + 音频优化 + 代码裁剪 |
| 分包策略 | 必要包内置 + DLC/内容包远程化 |
| 下载体验 | 进度显示 + WiFi 提醒 + 断点续传 |
| 更新效率 | 差量更新 + MD5 校验 + CDN 分发 |
| 异常处理 | 超时重试 + 指数退避 + 错误上报 |

分包与按需下载是大型手游的标配能力，优秀的实现能将首包控制在 100MB 以内，同时支持 GB 级的游戏内容按需加载，给玩家带来流畅的安装与更新体验。
